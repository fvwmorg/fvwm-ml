Hello FVWM users,

I'm not sure how many guys are interested in this,
but I'd like to release the fvwm localization patch for supporting
multibyte characters, mainly Japanese kanji characters.
This patch enables to use/draw multibyte characters in menu, window
title and icon title, etc.

This patch must be applied in fvwm-pre-2.0-patchlevel-35 source tree.
I've only tested on SunOS 4.1.3/X11R6 with Japanese locale and fonts.
I think some modifications may be needed for other OS/platforms or
languages. (or some extra fonts ?)
You should set appropriate locale before executing fvwm2.
Also you should use font set for font assignment in your .fvwm2rc file.
(Sample: fvwm/sample.fvwmrc/kanji.fvwmrc)
Please refer to XCreateFontSet(3X11) man page for locale and font set.

I attached gzipped and uuencoded patch at the end of this mail.
(verrrry sorry for long mail)
Please 'uudecode' and 'gunzip' this then 'patch'.

Any comments are appreciated.
