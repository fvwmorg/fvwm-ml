Roeland Moors <roelandmoors_at_pandora.be> writes:
> I want a small border around my windows (Width 2). But then it is
> very difficult to resize a window with the mouse.
> Is it possible to create a small border but set a larger area for
> the mouse?

No, that is not possible (as far as I know).

It is possible to create a key or mouse binding that will
initiate a resize from anwhere in the window, the title, or on the root,
or use a menu to initiate the resize.

Resizes that are started outside the context of the window
want you to click first to indicate which window you want to resize.
