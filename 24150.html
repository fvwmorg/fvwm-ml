Do you want to make windows smaller so that they fit nicely on a single page?
Could you explain in detail where the problem lies so that we can better judge
if some built in mechanism suffices or if new code is needed?

Possible solutions:

 - make the window fit on the page exactly

     Resize 100 100

 - make the window fill the available space without overlapping other windows
   (This is a recent change in the development sources, not available in the
   2.2 release)

     Resize grow grow

 - clip the window to page boundaries when it is created

     not possible at the moment
