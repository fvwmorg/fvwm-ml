Brian Schroeder <BJS_at_sagres.com.au> writes:
> I want to swallow an application into a button, but still have that button
> launch another application if it is clicked on. I was under the impression
> that that can be done, however it doesn't work for me.

I am uncertain, whether FvwmButton currently supports this feature. If
you cannot get it to work, you could always code it yourself. The
modifications to "perfmeter" should be very simple.

For example code, have a look at my program at
ftp://ftp.franken.de/pub/isnd4linux/contributions/isdnbutton-2.5.tar.gz
(actually, the archive might still reside in ftp://ftp.franken.de/incoming).
This program displays some status information and allows to launch
other programs when you click in the window. It even has a pop-up menu
to select between different programs. The code is not very large and
should serve as a good basis to get started. My code is highly
configurable thru the use of application resources.

Good luck,


Markus
