%% "Randall K. Sharpe" <rsharpe_at_depaul.edu> writes:

  rks> Is the newest version (2.3.x) going to be gnome compliant ?

Note that FVWM follows Linux-like release structure: 2.3.x are the
alpha/beta releases for the next stable release, 2.4.

There is currently support for GNOME in the beta (2.3.x) versions of
FVWM. You need to use --enable-gnome when you configure it, IIRC, to
get that support.
