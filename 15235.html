I also gathered a bunch of .xpm files from the AIcons repository.
(ftp://ftp.cs.indiana.edu/pub/AIcons)

I made an interesting enhancement to some of the icons.
Using the pixmap(1x) program
(ftp://sunsite.unc.edu/pub/Linux/X11/xapps/graphics/draw/pixmap-2.6.tar.gz),
I removed the background color, so now the "background" is transparent.
This gives icons a cool standout look against the X root screen.

Now my xterm icon is simply Homer's head, not a 32x32 picture.

These icons look great in an FvwmButton bar too, since the bar provides
a common background color to all the icons.


You're welcome to check out the few icons I've converted this way:
including 32x32 pixmaps of Homer, Mr. Burns, Milhouse, Bart,
Maggie, Calvin, Hobbes, Dilbert, Betty Boop.
ftp://ftp.best.com/pub/newton/icons/trans-icons.tar.gz
