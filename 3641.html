> I notice a section
>
> AddToFunc "StartFunction" I Module FvwmButtons
> I Module FvwmPager 0 0
>
> What do you think about trying to add it there?
>
> I also tried putting this in startxwin.sh
>
> fvwm2 Restart --dont-preserve-state &

Restart isn't a command line option but a built in command. Look
for "Restart" in your configuration file and replace it with
"Restart --dont-preserve-state".

> It had no affect as it made the fvwm2 command
> unintelligible, and I added it as the last line in the .fvwm2rc
> file also without any change in behavior.
>
> *Restart --dont-preserve-state

Luckily, you got the syntax wrong. You have to remove the
asterisk for the command to have any effect. But just putting
that command at the end of the config file puts fvwm into an
endless loop restarting again and again.

> consistently.
