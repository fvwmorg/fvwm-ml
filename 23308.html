> SetMenuStyle is a very recent command. I don't think it is in any of the

Ahem, that's completely wrong ;-) SetMenuStyle was part of the
releases 2.1.3 to 2.1.6. In 2.1.6 the syntax was rewritten and
from 2.1.7 onwards there is no SetMenuStyle anymore. Specifically
there's no SetMenuStyle in the current CVS source. Its functionality
and syntax have been merged into MenuStyle. Paul seems to use
2.1.5 (at least his module path suggests this), so this should
work.
