Um, I think he meant he downloaded fvwm2.

To install fvwm2:

  1) Unpack the fvwm-2.x.tar.gz whereever you like.
  2) cd into the fvwm directory.
  3) ./configure
  4) make
  5) as root: make install
  6) Find some config file you like (there are some examples in
     the distribution).
  7) Set fvwm2 as your default window manager (e.g. if you have a
     $HOME/.xinitrc file you should start it there).

If you have any problems you should look into the INSTALL.fvwm file.
If you still can't get fvwm to work (hopefully unlikely), ask the
list.
