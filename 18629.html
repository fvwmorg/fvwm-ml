Have You tried The Dotfile Generator? (And _yes_ I'm the author of the
Fvwm modules for this program -- it's GPL'd, though;).

It's a program for visual configuration of applications, among others Fvwm
1/2. You still have to restart fvwm after editing, though, since the
program generates the .rc file (don't worry - it doesn't overwrite
.fvwm2rc by default).

If You're interested, find out more from my web-page:
http://www.imada.ou.dk/~buk/fvwm/

Please note that the latest release is still in beta -- but then again, so
is Fvwm2 ;-) The next release (due out next week, probably) will include
a parser to read existing .fvwm2rc files (or parts thereof).

Please tell me what You think of my modules.
