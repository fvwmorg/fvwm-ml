On Sun, Feb 22, 2004 at 10:27:45PM +0100, Roland Hautz wrote:
> Hello,
>
> Defined key bindings like
>
> Key Left A SM Scroll -100 +0
>
> only work for me, when pressing alt before shift. That misbehavior was
> introduced such a while ago that I'm already used to it. Still I'd prefer
> return to normality.
>
> Tried with fvwm 2.5.7 and 2.5.9 (cvs head) on XFree86 4.3
>
> I'm not 100% sure that my Xkb is not the culprit. But my Xkb configs are
> unmodified from suse 9.0, and I can spot nothing unusual in these.
>
> Is it a fvwm bug?

I don't think so. Run

  $ xmodmap -pm

and

  $ xmodmap -pke

. This displays which modifiers are defined and which keys
generate them. The Shift_... or Alt_... key is probably
configured in an odd way.
