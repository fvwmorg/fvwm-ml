David Hansen <david.hansen_at_physik.fu-berlin.de> writes:
> On Wed, Jun 04 at 1:38+0200 Marco Righele wrote:
> > Greetings to everybody,
> > is there a Style setting that makes a window not appear in FvwmPager ?
> > I looked in the manual and I didn't find it, but
> > there are so many Style options that I could have missed it
>
> I think "WindowListSkip" is working.

Yep. Set the Style WindowListSkip for the windows you want to
skip, and set UseSkipList for the pager:

       *FvwmPager: UseSkipList
              Tells FvwmPager to not show the windows that are
              using the WindowListSkip style.
