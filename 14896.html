Hi all,

I can see that when specifying a percentage of vertical screen space, the
maximize function begins from the top of the screen by default, and expands
the specified percentage of the vertical screen downward.

To me it seems most ergonomic to have my toolbar at the top of the screen
since all of FVWM's window buttons are there as well as all applications
menu's. With the toolbar at the top of the screen I'm no longer constantly
running my mouse from the bottom (when starting an ap) to the top (when using
an ap).

I'm using tkgoodstuff at the top of my screen, and I'd like my maximized
windows to cover 90% of the screen's vertical length, but starting from the
bottom; leaving room at the to so that TkGoodstuff is never covered up.

Is there anyway to achieve this?
