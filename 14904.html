olvwm has something like this where you can specify a number of columns
in the menu definition.

Another nice feature of olvwm is that ability to specify a directory as
a menu item. It then creates a whole sub-menu consisting of every
executable in the directory.

These might look like:

AddToMenu MainMenu
+ "" Columns 2
+ "Main Menu" Title
+ "Programs%mini.xlogo.xpm%" Popup Programs
+ "MyProgs" Dirmenu $HOME/bin
