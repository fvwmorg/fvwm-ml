Hi, all!

I've been using .41 for a while now and everything's great. I
downloaded .42 today and compiled it; no problems there. The problem
is the 'new and improved FvwmButtons' module. I can't seem to get it
to do _anything_, let alone what it was doing before. FvwmButtons is
in the Init and Restart functions, and I rewrote the *FvwmButtons
stuff to conform to (I think) the new format. The relevant lines are
included below. All they do now is start up the Pager in the lower rh
corner of the screen. I get no errors in .xsession-errors, no
disk-thrashing, nothing. Can someone take a look at this, or perhaps
confirm that something might be wrong (doubtful, it's probably me
doing something stupid)?

Thanks a lot for your time.

->justin zaglio

----------------------------------------------------------------------

#------------------ FvwmButtons
*FvwmButtonsFore Black
*FvwmButtonsBack #c3c3c3
*FvwmButtonsFont -adobe-helvetica-bold-r-*-*-10-*-*-*-*-*-*-*
# Geometry - really likes to pick its own size, but giving a position is OK
*FvwmButtonsGeometry -0-0
# Layout: specify rows or columns, not both
*FvwmButtonsRows 1

# Define the buttons to use.....
*FvwmButtons(2x1, Swallow "xclock" `Exec xclock -bg \#c3c3c3 -geometry -1500-1500 -padding 0 &`)
*FvwmButtons(2x1, Swallow "xload" `Exec nice -16 xload -nolabel -bg grey60 -update 5 -geometry -1500-1500 &`)
*FvwmButtons(Title 80x24, Icon rterm.xpm, Action `Exec "XTerm" xt rxvt bash &`)
*FvwmButtons(Title 80x50, Icon rterm.xpm, Action `Exec "XTerm" lxt rxvt bash &`)
*FvwmButtons(Title brox, Icon rterm.xpm, Action `Exec "XTerm" lxt brox rlogin -x brox &`)
*FvwmButtons(Title cnet, Icon rterm.xpm, Action `Exec "XTerm" xt columbianet telnet cnet &`)
*FvwmButtons(Title cunix, Icon rterm.xpm, Action `Exec "XTerm" lxt cunix telnet cunix.cc.columbia.edu &`)
*FvwmButtons(Title manila, Icon rterm.xpm, Action `Exec "XTerm" lxt manila rlogin -x manila &`)
*FvwmButtons(Title simile, Icon rterm.xpm, Action `Exec "XTerm" lxt simile rlogin -x simile &`)
*FvwmButtons(Title watsun, Icon term.xpm, Action `Exec "XTerm" lxt watsun rlogin -x watsun &`)
*FvwmButtons(Title kerberos, Icon snoopy.xpm, Action `Popup Kerberos`)
*FvwmButtons(Title kill, Icon rbomb.xpm, Action Destroy)
*FvwmButtons(Title lock, Icon atom.xpm, Action `Exec "XLock" xlock -mode hyper -nolock`)
*FvwmButtons(Title logout, Icon disk_run.xpm, Action `Popup Quit-Verify`)
*FvwmButtons(3x1, Swallow "FvwmPager" `FvwmPager 0 0`)
