KONTRA Gergely <kgergely_at_mlabdial.hit.bme.hu> writes:
> Hi!
>
> I want to achieve an afterstep-like effect:
> When I click on a button (or the mouse is over the button) some other
> buttons appear (like a Start button in WIN, but buttons). Is it
> possible?
>
> And what would you suggest, if I want a little area on my desktop for
> 1-key commands (eg. when I press x an xterm window appears)?
> Root window is almost good, but what if I want to maximize a window,
> there is no root window :(

FvwmButtons has slide out panels. Look for "panel" in the man page.

As for the root window being covered, you could always change the
maximize function being used to cover less of the window. For example:

Maximize 100 -60p

Or, you could reserve one of your unused keyboard keys to activate
a menu regardless of context.
