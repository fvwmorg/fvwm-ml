> > >
> > > Is it possible for an FvwmForm button to clear the contents of an
> > > FvwmForm entry?
> >
> > By setting the action on a button to "restart" all the form's input
> > fields are reset to their initial values. Is that what you want?
>
> Not really. I have some forms which have default values but I want
> a button that will clear the form entries rather than resetting
> them to their initial values.

You can use a button to recreate and restart the form with new input
values.

I assume you know the keystroke ^k will clear an individual input field.

You can request an enhancement for another type of refresh. You might
want to explain why this is useful if you make the enhancement request.
