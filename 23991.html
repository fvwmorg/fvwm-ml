Hmm...they know how to resize and move windows with the mouse, but
don't know how to use keys. How about a big fat "MOVE" label on the
F7 key?

Or...you could bind the mouse in a window to do a move, if you have
a spare mouse button. Netscape is primarily concerned with buttons
1 and 3, 2 doesn't appear to be used:

Mouse 2 W A Popup Move-or-Resize

but if this is a Netscape only application.

and position the window so that the frame is off-screen.
