In case it matters, I forgot to mention my way of restarting. I use
the following:

AddToMenu Quit-Verify "Really Quit Fvwm?" Title
+ "Yes, Really Quit" Quit
+ "Restart Fvwm" Restart fvwm
+ "Start twm" Restart ctwm
+ "Start mwm" Restart mwm
+ "Start olwm" Restart /usr/openwin/bin/olwm
+ "Start tvtwm" Restart vtwm
+ "Start dummy" Restart xterm
+ "" Nop
+ "No, Don't Quit" Nop

Note I have `fvwm' instead of `fvwm2' (my version of fvwm-2.0.41 is
installed as `fvwm', as I'm the only one here who uses it so
far). Should I be passing an arg to fvwm above? (Is that even
possible?) Doesn't it re-read .fvwmrc upon restarting by default?
