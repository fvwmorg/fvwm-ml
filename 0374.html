Xander van Wiggen <xvw_at_warande0147.warande.uu.nl> writes:
> Hi,
>
> Is there a way to disable the mouse pointer turning to 'O' when clicking
> the border? I.e. another way than hacking the sources?
>
> I would like fvwm2 to behave *exactly* like mwm, that's why ;)

Always tell us what version you are using.

We have to know which button you are clicking with and what command
you have bound to it. For example, heres mouse button 1 on the frame
with any modifier set to run function Resize-or-Raise:

Mouse 1 F A Function "Resize-or-Raise"

and Resize-or-Raise does a "Raise" on a single click:

AddToFunc "Resize-or-Raise" "M" Resize
+ "M" Raise
+ "C" Raise
+ "D" RaiseLower

Some fvwm commands can result in the pointer changing which you can
find in the man page under "Cursor".
