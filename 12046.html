What's the best way to effectively accomplish this in my .fvwm2rc file?

if xdpyinfo | /bin/grep -q XINERAMA ; then
        Xinerama on
        XineramaPrimaryScreen 0
        Style * StartsOnScreen c
        DeskTopSize 4x1
else
        DeskTopSize 6x1
fi
