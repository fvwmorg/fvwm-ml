> fully ICCCM2 compliant ? This problem can not be reproduced with this wm.

Well, you will probably not find a single window manager and not a
single application that is completely ICCCM2 compliant. The
problem that window manager authors have is that they have to work
with a multitude of buggy applications. So we can not simply say
"fix it, your application is broken" because if we did this, only
one of two applications would run properly. So we keep making
options to work around many bugs in applications but keep telling
the authors to fix them.
