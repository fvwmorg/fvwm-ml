> I am trying to create a couple of 'goodies' on the task bar. I, for
> starters, would like to put a pager button on the task bar and maybe a
> couple of other 'little things' a-bit later. Also I want to remove the
> clock that resides on the far right side of the taskbar - How can I do
> this?

Easiest way is to use FvwmButtons + FvwmIconMan to create your own
personal task bar. This is what I have done. Instead of a start
menu, I have an "xload" showing my workstation load. In the "system
tray" area I have:

- An icon that runs a screen saver
- An icon that allows me to run some window selection scripts
- An icon that warps to Netscape (or runs it if it isn't already
  running)
- An icon that displays the status of the printers attached to my
  systerm.
- An icon that runs xclipboard (VERY useful!)
- An icon that allows me to either modify my .fvwm2rc files or restart
  fvwm (via menu pick)
- An icon that terminates fvwm (effectively logging me off)
- A "swallowed" tcl/tk application that displays the current time, and
  (if pressed) warps to my mailer (or runs it if it isn't currently
  running).

I have found FvwmButtons very versatile for providing this sort of
functionality, and FvwmIconMan provides the "task list" in a similar
manner that FvwmTaskBar does. And with FvwmButtons, you have COMPLETE
control over how your task bar looks.

Anyway, I've MIME-attached a tiny GIF file that shows you what my task
bar looks like. And, if anybody is interested, send me private email
(NOT A REQUEST ON THIS LIST!) and I will send the pertinent sections
of my .fvwm2rc files. My example assumes a 1280x1024 screen, but the
"magic numbers" can be fudged... for example, I have a slightly
modified version of this configuration at home that works equally well
on my 1280x1024 and 1152x864 screens, thanks to the magic of FvwmM4.
