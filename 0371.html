Another option is probably to change invocations of System and Net Sites
with Module FvwmButtons System and Module FvwmButtons Net\ Sites.
I really don't like the extra symbolic links way of doing things in fvwm
precisely because it makes the config files harder to use by users
who are not familiar with fvwm.

E-mail me your config file and I'll have a look at it and see if
I can change it to a config file which does not make use of extra
symlinks.

Best Regards,

Neil
