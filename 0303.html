"Olivier F. Beauchemin" <obeauche_at_bbnplanet.com> writes:
>
> Hello.
>
> I'd like to use these two modules, however, it won't seem to work under:
>
> fvwm2 -version
> [FVWM][main]: Fvwm Version 2.1.7 compiled on Feb 10 2000 at 17:01:21
>
> Is this even possible? If so, what should I add to my .fvwm2rc file? If
> not, is there any chance that a developer would add this?

FvwmAnimate and FvwmWinList only work together in the latest betas.
You would need to install all of fvwm. The modules certainly won't
work with fvwm 2.1.7.

The man pages describe what you need to do.
