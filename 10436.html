Hi all,

A new version of FvwmTabs is available.

You can get it at:

http://users.tpg.com.au/users/scottie7/fvwmtabs.html

ChangeLog:
- Allow re-ordering of tabs via mouse (drag-&-drop) & key bindings.
- Fixed really annoying bug where some tab-managers would not accept
  key-bindings.
- Updated fvwmtabrc file.
- bugfix: titlebar was expanding when windows were resized.
- Remember selected tab b/w restarts.

SCoTT. :)
