> I have one question about images used in titlebars and other panels.
> There doesnt seem to be a guideline as far as how large an image should
> be in titlebars. Is there a size that is too big that would possibly
> use too much in the way of resources?

That depends on the resources available on your X server. On one
hand, rendering images in all title bars might be too slow for
your liking, on the other hands the images consume RAM on the
server (about Width x Height x (Depth / 8) bytes). So, you should
not put wallpapers in window titles. Usually, images in titles
are pretty small, but remember that the X server stores a copy of
the original image for each window where it is used as a
background.
