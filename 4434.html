> Hi there,

> How buttons supposed to show up in the title bar? If I define ButtonStyle or
> AddToButtonStyle nothing happens, even with Style * Button [N] only the title

Defining the button is not enough, you need to bind an action to
the button.

For example, this is an action on button 1:

Mouse 0 1 A Iconify
