AHA! Thanks to an unrelated message posted by Jerry Alexandratos
asking how to turn off FvwmAuto, I now have what I was looking for. By
using the following combination: 

--
AddToFunc InitFunction
+             "I" Module FvwmAuto 0 Raise Nop
AddToFunc RestartFunction
+             "I" Module FvwmAuto 0 Raise Nop
Key Tab     A       M       Next (CurrentPage) Focus
Key Tab     A       C       Prev (CurrentPage) Focus
--
I now have my quick focus&raise. I'm guessing that the default raise
delay is 200ms or so? :$}
