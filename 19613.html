My friend Paul (an as-of-yet non-FVWM user) asks:

  I was wondering if there was something like "fvwmclient" which would
  allow you to call those functions from the command line. So you could
  write a little wrapper script like:

  #!/bin/sh
  emacsclient $1
  fvwmclient 'Focus Emacs ; WarpToWindow Emacs'

Is there a way that, whenever he opened a file with emacsclient (a small
app which communicates with an already-running emacs and instructs that
instance of emacs to open the passed file) fvwm would raise/de-iconify the
emacs window and warp to it?

We're running FVWM 2.0.45 on SunOS and X11R5.
