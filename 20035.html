I've tried a couple of approaches which would come close to giving the
look and feel of what I'm looking for but so far I haven't even had
any luck getting those to work.

The first thing I tried was to set up a style for the application
(such as netscape) with NoIcon. This works great but I haven't been
able to find a function for de-iconifying except through WindowList.
WindowId looks like it might be the function I need but I'm not sure
how to figure out the window id.

The other approach I've tried is to start the application during
startup with the -iconic switch and swallowing the result into a
button. This resulted in a blank button and no action occurred when
I clicked on the button.

Any ideas on how to get one of these approaches to work? Or is there
a different approach that works?
