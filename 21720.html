estes_at_cipic.ucdavis.edu said:
> I'm using the same ~/.fvwmrc file as before my upgrade from RedHat 4.2
> to 5.0. I had it set up so that only the titles of the icons were
> used, and so that they stacked up in the lower right part of the
> screen. After the upgrade, when I iconify windows, they disappear - I
> don't have any icons at all, so there is no way to recall a window.

Try checking colour availability. I've found that in most cases where icons
were disappearing, it was because I'd run out of colours. Run xdpyinfo, and
make sure you're getting the right kind of screen depth (this'll probably be
16, 24 or 32 bits truecolor; if it's only 8 bits pseudocolor, then that's
probably your problem). It may be something else, of course, but it's easy to
check for colours.
