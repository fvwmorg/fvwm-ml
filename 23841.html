Graham Gough wrote:
>
> Is there any way in the latest fvwm to use styles to have autoraise
> turned on/off selectively for different classes of windows?

Well, it is not style but running this script with FvwmCommand should
do the job.

Make sure you kill FvwmAuto, run FvwmCommandS, and place FvwmCommand
in your command path.

Invoke, from a shell, something like,

perl selectable-raise.pl '(XTerm|Eterm|Netscape)' 0.4
