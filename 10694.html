On 05 Dec 2003 05:55:40 -0800, Anupam Kapoor wrote:
>
> is it possible to group windows in a WindowList ? i.e. all emacs
> instances together, all xterms together etc. and what would be really
> nice is to have a WindowList something akin to normal menu stuff (with
> window mini-icons !) etc. so we have an emacs icon which expands into
> all emacs instances etc.
>
> can you please let me know on how to achieve this ?

You can't have one icon occupying several menu items. But one emacs
mini icon in every consequent menu item should do what you want.
Use SortByResource option. This is what I have, for example:

  Mouse 2 A SM WindowList SelectOnRelease "" \
    SortByResource, NoDeskSort, NoGeometry, NoHotkeys

You may also group applications into sections by any criteria (name/class
regular expressions) using FvwmWindowMenu module, then application
sections are separated from each other using horisontal line. But this
module has some issues yet. Maybe in the future FvwmWindowMenu will
support section submenus and showing only the first application icon per
section (if its author or someone else adds these features).
