Sometime in the late 1900s Timothy Lottes (tlottes_at_students.uiuc.edu) wrote:

TL> One really nice feature of twm that was cut in the creation of fvwm was
TL> the use of non X11 standard cursors by the window manager. Twm allows
TL> for the user to set a cursor and mask file instead of just a reference to
TL> a standard X cursor.
TL>
TL> I'm not sure how many others would also like to be able to create and use
TL> there own cursors for use with fvwm, but it would be a nice feature (and
TL> hopefully an easy one to add)...

   I'd agree that it would be a nice add.on feature. How about support
   for animated cursors? So the user could specify a sequence of cursors

TL> Then again it probably isn't very high on the list of things to get done

   And I'd also agree here too...
