On 31 Jul 2002 06:42:49 -0700, Jeff Gibson wrote:
>
> I'll give 2.5.2 a shot. Is there a tool that comes with fvwm2 that I
> can use to take s screenshot?

In fvwm-themes you may create a screenshot of a specific window or the
whole desktop automatically using the "Window Utility" menu.

You may do this manually of cause too. Use xv for this, pretty convenient.
Or import utility from ImageMagick:

  % import -frame /tmp/shot-window.png # one window
  % import -window root /tmp/shot-window.png # desktop
