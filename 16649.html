I am having problems getting fvwm-2.0.42 to work. I untar the distribution,
check the Fvwm.tmpl without doing modifications, type MakeMakefiles and make
and it compiles fine (only thing it complains about is not finding
readline/readline.h while doing makedep in FvwmConsole directory). Then I do
'make install'. Then, when I try to start X and the new fvwm2 with my own
.fvwm2rc, it loads a while and then shuts down X and returns to console with
no error message. With the supplied system.fvwm2rc it doesn't do this. I
have found the reason: Fvwm works and doesn't crash if the config file
doesn't
use any of the functions that handle with XPM's. FvwmButtons doesn't work
either (because of the same reason obviously). My system is Linux 1.3.66
and XFree v3.1.2 and I _do_ have /usr/X11/lib/libXpm.so.4.7

Is my libXpm too old or why doesn't it work?
fvwm 2.0.41 worked 100% fine!
