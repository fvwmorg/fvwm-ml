bs> I've had trouble mapping any of the "special" function keys on the Sun
bs> keyboard. I'm using a type 4 keyboard.

I'm using a type 5, but this ought to work.

# Props
Key L3 A C Module FvwmTalk
Key L3 R N Module FvwmTalk
# Front
Key L5 A N RaiseLower
# Close
Key L7 A N Iconify
# Find
Key L9 A C Module FindWin
Key L9 R N Module FindWin

# Pause/Break
Key F21 A S Exec "" xlockmore -mode random
Key F21 A C Restart /usr/local/share/bin/fvwm
Key F21 A CM Quit

Enjoy,
 Andrew.
