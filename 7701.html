Hi all,

A new version (1.5) of FvwmTabs is available.

http://users.tpg.com.au/users/scottie7/fvwmtabs.html

This release implements proper focusing, so now apps like
Mozilla work properly. (You can enter URL text, etc.)

Also, a bug which caused windows to go off into the great, big
window-manager void when FvwmTabs is destroyed has been fixed.

& I cleaned up some of the Perl syntax.

SCoTT. :)
