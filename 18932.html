I've just installed 2.0.44 (from 2.0.42) and I'd like to see what all
the new features do, but I'm too lazy to figure them all out from the
man page. I'd especially like examples using mini icons, BorderStyle,
TitleStyle, ButtonStyle, etc. I could try the examples in the man
page, but I don't have the pixmaps (marble, granite, up, down,
inactive, etc.).

It'd be nice if fvwm came with a style sampler that would let one
browse various combinations. I'm thinking of something like a .fvwm2rc
file with menus that would pop up windows with different styles
showing the .fvwm2rc code used to achieve them. E.g., a menu for
various BorderStyles, another for TitleStyles, another for
ButtonStyles, etc. I realize that'd be a significant chunk of work,
though.

I'd be happy if a few people would just bundle up their favorite
configs--including .fvwmrc and pixmaps--and posted them to the
list. Perhaps they could be bundled with 2.1.x.

-Dave
