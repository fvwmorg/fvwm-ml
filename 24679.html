> I have search all the docs and archive but have found no answer. hope you

>
> how do I make the transient (the little windows for search, open file
> etc.) to automatically get focus? [when using sloppy focus]
>
> I remember that I've read about the module that would do this somewhere,
> but can't find any information about it now. wonder if that was just my
> imagination...

The 'GrabFocus' and 'GrabFocusTrasient' styles are implemented since
2.3.2. Note that this does not work well if the transients are opened
with the mouse from a menu of the application because the program
would immediately get the focus back when its menu pops down.
