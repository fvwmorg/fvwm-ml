I am having a problem getting modules to load properly using fvwm2 v2.2.4. This
may be related to how fvwm2 was installed.

fvwm2 is in /usr/local/bin and the modules are in /usr/local/fvwm-2.2.4/modules.
However, each module is in its own directory, e.g. FvwmPager is in
/usr/local/fvwm-2.2.4/modules/FvwmPager. The ModulePath in .fvwm2rc is
/usr/local/fvwm-2.2.4/modules. When I start fvwm2 it gives a "cannot execute"
error for each module. If I change ModulePath to
/usr/local/fvwm-2.2.4/modules/FvwmPager:/usr/local/fvwm-2.2.4/modules/FvwmAuto
then it works. If this is correct, it is a change from 2.0.46, which we were
using.

I suspect that something went awry in installation, but don't know what. I
would have expected the module executables to have been moved from the
module-name directories to ../modules, but that apparently didn't happen.
