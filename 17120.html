Lots of people have said xman works fine, or that there is no problem with it.
Odd, I can start xman, get a manual page up, but thats it.

I can't use either menu, so I can't actually get to a manual page or
browse through commands. To me, this is broken! Maybe its not a
widespread problem. Lets see, I have Solaris 2.5 and openwin 3.?
I'll just go and check that using different mouse buttons etc doesn't fix
it. Nope. Xman might as well be a static picture for all the use I can
get out of it. I have the last fvwm beta, compiled as normal, no patches.
I can get the menus up in xman, I just cannot select anything in either
options or sections. Hence, no use! :-)

I thought that xman would be a better example than say, the Sun tools,
many of which exhibit problems, especially with menus. The menus
problem is really annonying! Maybe I was a bit rash saying it renders many
programs useless, but it certainly inhibits their full use. :-)

In filemgr for example (Sun's fine File Mangler) some option menus
just vanish behind the main window. Hard to set options then!
