I have a new FvwmCommand module, which is under testing right now,
which can help you. Basically, it can monitor fvwm transaction and
send commands accordingly.

I use it for the exact same purpose, giving a focus to 'find' dialog
window every time I hit Alt-F. I also move 'download' window to move to
the side so that it won't be hidden under main Netscape window. I'd like
to know if the downloading has ended by just looking at Pager from any
desk, any page. ( Somehow, Netscape's X resource file doesn't work to
set this window to a particular location.)

I have sent this module to fvwm-worker list recently.
If you want to try it out, just let me know. I will send you the
most updated version and the example script above during this weekend.
(I don't have them with me here.)
