Is there a way to turn off key & mouse bindings when entering a
particular window? I've looked through some of the documentation but
haven't found anything but I may have looked hard enough yet. ;-)

For a simple example, suppose I have

        Key F1 FSTWI N RaiseLower

and I'm running fvwm2 locally. I'm also using a vncviewer to attach
to a remote Xvnc which is also running fvwm2. What I would like to
do is when I'm in the vncviewer window, I want the "F1" key event to
pass into the vncviewer to the remote fvmw2 instead of being acted upon
by the locally running fvwm2.

Any suggestions or comments would be appreciated.
