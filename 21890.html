Near 2000, Steve Sanbeg mumbled something about:

> I have noticed a bug in the version of fvwm I am currently experimenting
> with. When trying to restart fvwm from a menu, it often doesn't work
> properly.

AddToFunc GoodRestart
+ "I" Exec (/home/foobar/bin.share/erica)

AddToMenu MainMenu "Main Menu" Title
+ "&Restart" Function GoodRestart

Where erica is a shell script that has x-permissions set for you and does the following:

1) kill fvwm
2) start fvwm

Hope this helps.
