Use the latest daily snapshot and you should have fribidi out of the box
for Hebrew, Arabic and unicode fonts. It is not really fully tested and
not documented yet in the man page, only in INSTALL.fvwm file.

Just remember that if you use MultiPixmap titlebar, snapshots after
2002-03-01 do not support it back yet. And we speak about unstable code.
