Funny title isn't it? The discussion on vector buttons gave me
an idea on how to facilitate designing fvwm vector buttons,
without having to write an entire drawing program.

I just made a perl script, that converts encapsulated postscript
files to fvwm vector button definitions. This way, you can draw
your buttons using xfig, then export them to eps, and run the script,
which then produces your button definition. The script probably won't work
on very complicated eps files like your girlfriend's picture, it only
looks for polygons. But I guess this will suffice.

Go see and try:

http://www.xs4all.nl/~josvanr/eps2vector.pl

Have fun,
