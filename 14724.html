This is a bit off topic, but some people might find it interesting.

There is a win95look patch to bowman, which is a window manager originally
based on fvwm 1.24r. Bowman is a NeXTSTEP lookalike with fvwm-1.24esque
rc file. win95look is a patch pp_at_friendly.netppl.fi (Pekka Pietik{inen)
made as a joke for bowman. You can find bowman at

  http://www-acs.ucsd.edu/~byang/bowman/

and the win95look at

  ftp://ftp.netppl.fi/pub/linux/win95look/

Since I don't use win95, I don't know how close this is to "the real thing".

I don't think PP is planning to port this to fvwm 2.0 (yet), but he did
mention that he's interested to know if people like it. So email him
praises if you like it. :-)
