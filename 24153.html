ctwm has a MaxWindowSize option and makes larger windows smaller. I
think this applies to newly created windows as well as already
existing windows. Here's the relevant man page excerpt:

,----- from ctwm man page
| MaxWindowSize string
| This variable specifies a geometry in which the
| width and height give the maximum size for a given
| window. This is typically used to restrict windows
| to the size of the screen. The default is
| "30000x30000".
`-----

Dunno if this is what Josh wants, but maybe it is.
