Sorry to junk if not interested:

                                AnotherLevelUp

a `new' fvwm2 configuration package based heavily on Red Hat's AnotherLevel,
but with much more flexibility in decor and persistence of user settings.

See: http://www.cs.man.ac.uk/~jtl/ALU

for details and download (GPL, no warranty, beta release, comments welcome and
regular updates promised).

It has many features which may be of interest to this list, including, e.g.:

        Loading/saving named theme/preferences sets
                with Shared themes (~50 ready made)

        FvwmScript for editing XTerm X resources,
                with FvwmScript Colour browser

        Table driven `toggle functions' -- persistent on/off dynamic user
                settings, e.g. Console window, xbiff, GMC

Thanks, John Latham
