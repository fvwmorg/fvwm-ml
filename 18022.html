I just got a replacement keyboard (old one died under warranty), and
the new one has those annoying Windows keys. I thought it would be
useful to have the WindowList on one, so I could easily switch windows
with the kbd.

Unfortunately, while the 1.x Windowlist warped the pointer to the
window and focussed it, the 2.x one does not. I can't use
FvwmWinList, because it apparently doesn't have a kbd interface.

Is there any way to use WindowList and give the selected window focus?
Or an alternative window list with a kbd interface?
