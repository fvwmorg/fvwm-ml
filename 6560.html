On Wed, Oct 16, 2002 at 09:10:14PM -0500, Andrew Morelock wrote:
> While I know this is a feature, does anyone know how remove the pointer from
> displaying for use on a touch screen environment?

Add "CursorStyle ROOT None" to your config file for removing the
root cursor. You can remove the cursor for others context in the
same way, see the fvwm man page.
