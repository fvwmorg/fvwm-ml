On 05 Oct 2000 22:55:46 -0700, Perry Hutchison wrote:
>
> I have run into a situation where I would like to change a window's
> name (as shown in its title bar), and/or its icon name, without
> shutting down and restarting the application. Several hours of
> digging into the manpage, FAQ, and source code have not turned
> anything up. Is there some command which could be issued via
> FvwmConsole to perform this sort of operation?

In a window title and an icon title you see a real window name and a real
icon name, which your application supplied. Unless this application wants
to change its names dynamically, you can't change this in fvwm. As of now.
