>>>>> zinc writes:

  zinc> anyway, i use xbiff to indicate new mail (any better ones out
  zinc> there?). i have FvwmButtons swallow xbiff. i want to be able
  zinc> to click on this as well and have it start vm in XEmacs (using
  zinc> gnudoit -q '(vm)'. i can run the gnudoit part as a script,
  zinc> that's not the problem. the problem is convincing FvwmButtons
  zinc> that this is a button.

Clicks on buttons that have swallowed an app are always passed to the
app. But you can tell the button to have a title then specifying an
action means that you can click on the title to execute the action.
I did this and it worked.
