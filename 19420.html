Hello. I'm a newbie and not quite sure if this is the right place to ask
this question, so please no flames.

I'm running fvwm95-2.0.42a-5 and would like to be able to set the
background (border) colour on my button bar to a separate colour than is
specified by the DefalutColors winbackcolor variable:

DefaultColors Black grey50 #c0c0c0 grey60

I've tried specifying it using the Style tool, thus:

Style "FvwmButtons" NoTitle, NoHandles, Sticky,\
                     WindowListSkip,BorderWidth 5, BackColor Grey10

But I still get the same background colour as the titlebar and borders of
my unselected windows (actually the borders of all the windows are the
default color).

I can't believe that it is impossible to set the colour of the button bar
separately. Any help would be appreciated, and should probably be
mailed directly to myself.
