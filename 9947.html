> If I'm not mistaken, this has nothing to to with FvwmTheme.

You're correct, it does not. The module "FvwmTheme" is there to provide
one with a way of using coloursets and the like (man FvwmTheme).

> Will fvwm themes be included in fvwm.

You are confusing the issue here.... fvwm-themes is a separate "project"
from fvwm, in that all fvwm-themes is, is a set of highly
intelligent/configurable scripts to be used with the window manager: FVWM.

> So, if I want themes, there is nothing "official", but I can happyly
> install fvwm-themes?

That is correct :) All the hard-work has been done for you, with
fvwm-themes.
