I'm currently using exmh 2.0alpha. It's got a few bugs (alpha release 'n
all) but it's a very good mailer. It has excellent MIME support,
integrated PGP, mail fetching via POP3 (and can do multiple methods of
incorporating mail), filtering via either the separate procmail package or
the MH slocal package, coloured folder labels to indicate new mail, an
icon which acts as a biff replacement, facesaver support, glimpse fulltext
searching (making it easy to find that email you know you have somewhere)
HTML documentation, etc etc. It's definitely at the high-power end of the
spectrum, although I've not used some of the mailers mentioned in this
thread such as Zmail.

Of the Windows mailers I'd say it surpasses Pegasus by miles and Eudora
Pro 3.0 by a fairly good margin.
