hi list!

i have several different xterms (aterms in fact) which are used to log
on to different machines. to recognize better where i am in, i colorize
the titles in a different way for each server. when i start them up,
the title changes to the machine name, so i can easily create a decor
for each of them. that is working fine.

the other point is, that i'd like to have some default styles on all of
my windows. that is buttons, title position and so on. right now, i have
to define one "Default" Decor for all windows, and several other decors
for all the machines. all those decors contain *all* the styles right
now: TitleStyle, ButtonStyle, BorderStyle... . when i want to change for
example the appearance of a button, i have to change this in all the
styles which is really annoying (for i am still searching for *the*
buttonstyle *gg*).

is there a way to have something like this:

# these are global settings, used for ALL windows!
AddToStyle Global
+ TitleStyle RightJustified
+ Button 1 Pixmap bla.xpm
+ ButtonStyle All -- UseTitleStyle
+ BorderStyle -- HiddenHandles NoInset

# this is for xterms only (e.g)
AddToStyle xtermStyle
+ TitleStyle HGradient 128 MediumSeaGreen black
# all the rest will be taken from Global

is this possible? i already tried with "AddToStyle Default", but this
works only for windows without *any* decoration. Also i tried to use
UseStyle within AddToStyle, this didn't work either.

can you help me, will you?

thanks in advance

andré
