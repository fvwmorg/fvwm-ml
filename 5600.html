On 21 Jul 2002 01:05:26 +0500, Denis Danilov wrote:
>
> How to change the default look of icons in fvwm, when their size depend
>
> on icon pixmap, to mwm-like style. In Mwm all icons have equal size on
> quare base.

There are 2 different solutions.

1) Disable program-specific icons (there are several style options,
ask if you need a help) and define icons of equal size for all
applications, the wm-icons project includes several such icon sets.

2) In pre-2.5.3 use "Style * IconSize 48 48", this does not resize icons,
but pads or strips to 48x48 if needed.
