Well, I had been using a hacked fvwm-2.0.43 / fvwm95-2.0.41a (basically
just adding the stuff that gives it the Win95 look and feel). After
hearing about the BorderStyle and related patches (my sources were too
hacked up to patch correctly, so I waited PATIENTLY for 2.0.44), I've
decided to put together a Win95 appearance for fvwm.

So far... It's PRETTY CLOSE... but still no joy.

MenuStyle Black grey75 grey40 adobe-helvetica-medium-r-*-120-*
AddToDecor win95
+ TitleStyle LeftJustified ActiveUp (Solid Navy -- flat) \
        ActiveDown (Solid Navy -- flat) Inactive (Solid grey51 -- flat)
+ ButtonStyle 1 MiniIcon -- UseTitleStyle flat
+ ButtonStyle 2 Pixmap win95-close-full.xpm -- UseTitleStyle flat
+ ButtonStyle 4 Pixmap win95-maximize-full.xpm -- UseTitleStyle flat
+ ButtonStyle 6 Pixmap win95-minimize-full.xpm -- UseTitleStyle flat
+ BorderStyle Active -- HiddenHandles NoInset
+ BorderStyle Inactive -- HiddenHandles NoInset
+ WindowFont -adobe-helvetica-bold-r-*-*-12-*-*-*-*-*-*-*
+ HilightColor white grey60

Style "*" UseDecor win95, ForeColor Black

The win95-*.xpm files are pix of the Win95 buttons from the fvwm95
distribution, and grey75.xpm is a 1-pixel XPM file that contains a
single grey75 pixel (BorderStyle could possibly use a COLOR command!).
All these .xpm files are attached to this message via MIME.


Also, the fvwm2 man page could do some explaining about 'ActiveUp' and
'ActiveDown' and how they apply to the TitleStyle and other functions.

Anyway, I'm posting this as an attempt to drive a stake in the ground to
get something that looks like Win95 - NT4.0. If anybody wishes to tweak
with the values that I got empirically, feel free! (And let me know!)

Does anybody have any mwm, olwm, twm, etc. decor styles? I'm hoping we
can get a few of these styles so that we can create fvwm2 functions
like:

AddToFunction StyleWin95
"I" Style "*" UseDecor win95, ForeColor Black
"I" MenuStyle Black grey75 grey40 adobe-helvetica-medium-r-*-120-*

AddToFunction StyleMwm
...

We could then make a pop-up menu, or FvwmButtons display that will allow
the user to select the desired window display style... this would make
a powerful enhancement to the system.fvwm2rc file.

Anybody game for such a project?
