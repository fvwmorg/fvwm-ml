Michael Han <mikehan_at_best.com> writes:
> Previously from Geoff Ludwiczak
> >I'd like to know if it's possible to set a default cursor with a
> >provided xbm file for everything, not just the root window, i know how
> >to do that with xsetroot. But I want to set one for everything, whether
> >I'm in Netscape, or whatever..
>
> I don't think there's any way to change the cursor used *within* an
> application context, except by configuring/hacking the application.
> For example, the little "hand" cursor in Netscape when you rollover a
> link is private to Netscape. It's an area of weakness in X, IMHO.

I disagree. I find this stuff in my Netscape.ad file:

! Mouse cursor when over an anchor.
*linkCursor: hand2
! Mouse cursor when awaiting a reply.
*busyCursor: watch
! Mouse cursor when "Save Next Link As..." mode is on, and over a link.
*saveNextLinkCursor: hand2
! Mouse cursor when "Save Next Link As..." mode is on, and not over a link.
*saveNextNonlinkCursor: crosshair
