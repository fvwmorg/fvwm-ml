>>>>> stefano lacaprara writes:

  stefano> i have a simple question: is there a way to change desk
  stefano> WITHOUT using the mouse?

I use the following in my .fvwmrc file:

Key Left A 4 Desk -1 1
Key Left R A Desk -1 1
Key Right A 4 Desk 1 1
Key Right R A Desk 1 1
Key F1 A 4 Desk 0 0
Key F2 A 4 Desk 0 1
Key F3 A 4 Desk 0 2
Key F4 A 4 Desk 0 3
Key F5 A 4 Desk 0 4
Key F6 A 4 Desk 0 5
Key F7 A 4 Desk 0 6
Key F8 A 4 Desk 0 7

hth,
