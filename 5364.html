Neither them nor their combinations with control or alt or shift keys no longer cause any thing in X sessions. >
> Can anyone help me how can I give back the existence of Enter key to my X session ? Any comment will be appreciated.

We can't really help much with fvwm95 because it was split off
from fvwm itself years ago. Fvwn itself has changed a lot since
then and provides almost all functionality that fvwm95 has too.
Since fvwm95 is no longer maintained, I suggest you use the latest
fvwm release instead (2.4.8, http://fvwm.org). It has an exmaple
configuration in the sample.fvwmrc directory of the source
distribution. I once created it from the fvwm95 default config.
And I think the fvwm-themes package also has a similar
configuration that is easy to install (?) Should you have any
trouble with fvwm, please do report it here.
