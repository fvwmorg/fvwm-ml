On Wed, Dec 16, 1998 at 06:46:08PM +0100, tg_at_skynet.be wrote:
> Chris Rankin wrote:
> >
> > This patch (166 lines, but can't be helped) fixes the signal handling in
> > FvwmPager. It will now exit cleanly when it receives either a TERM, INT,
> > QUIT, PIPE or HUP signal.
>
> This will sound quite stupid to some of you, but how do apply the patch
> to the source? I know I should use the nice "patch" command, which read
> from stdin, but I don't know which parameters it needs.

cd into the directory from which the patch was created (usually
the directory with the patched files but maybe something above
it if you have files in multiple directories) and then:

patch < foo.patch

where foo.patch is the name of the patch (plus path).
