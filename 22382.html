Hi,

    I'm running fvwm 2.0.46 on an SGI Indy (IRIX5.3). I can't run some
programs from either the fvwm menus or FvwmButtons.

>From my ~/.fvwm2rc:

*FvwmButtons(Title "-", Icon xfm_draw.xpm,\
    Action `Exec /usr1/misc/bin/ivtools/idraw &`)

*FvwmButtons(Title "-", Icon info.xpm,\
    Action `Exec /usr1/misc/bin/tk/tkinfo &`)

*FvwmButtons(Title "-", Icon xpaint.xpm,\
    Action `Exec /usr1/misc/bin/xpaint &`)

*FvwmButtons(Title "-", Icon xv2.xpm,\
    Action `Exec /usr/local/bin/xv &`)

Xpaint and XV work fine - idraw and tkinfo don't even start up (if I use
the hangon extension to Action the button stays depressed). They all work
from the command line. I've tried different permissions and program
locations - nothing helps.
