Good morning,

> > I just want to be able to tranfer a window on my machine from the display
> > falco:0.0 to falco:0.1.
> You cannot. There is no general way to move windows from one display to
> another. This is almost equivalent to moving a window from host1:0 to
> host2:0, it can't be done. (I wish that it could, my Ultra has 2 heads.)

The comp.windows.x FAQ tells about "xmove", which is "a pseudoserver (aka
proxy server) which allows you to dynamically move an X application
between servers, and screens within a server." The v2 beta seems to date
to Augost. ftp site: ftp.cs.columbia.edu:/pub/xmove. I have of course no
idea about performance.

http://www.cis.ohio-state.edu/hypertext/faq/usenet/x-faq/part6/faq-doc-4.html
is the faq answer with several other programs.
