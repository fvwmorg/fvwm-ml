I was wondering if it is possible to have the taskbar pop up a menu
when it is clicked such that the actions in the menu affect the window that
the taskbar refers to, not the actual task bar window. I tried this:

  AddToMenu "Window-Ops2" "&Kill" Destroy
  *FvwmTaskBarAction Click3 PopUp Window-Ops2

But, the Destory even is apparently being sent to the actual TaskBar, not
the window that the TaskBar refers to. Is it possible to change that?

I am using fvwm95-2 if that matters...
