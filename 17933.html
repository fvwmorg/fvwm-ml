On Mon, 23 Sep 1996, Jerome GOUT wrote:

> I'm tired to see my old poor xmailbox (an xbiff-like tool) and I
> wonder if somebody knows (and uses) some pretty one.
>
> I use the TaskBar module, Is it possible to insert some mini-icon or
> mini-animation to show when a new mail arrives ? There is already the
> time displayed, it can be fun to diplay a little mailbox flicking ...

I'm using xmailbox-2.4 (use archie to find it). It supports arbitrary
xpm's for full and empty mailboxes. Additionally has animation for
a full mailbox and can play a sound file on various platforms when
a mail arrives. I've got it running under Digital Unix 4.0 and
it says "You've got mail" when a mail arrives and the mailbox flashes.
