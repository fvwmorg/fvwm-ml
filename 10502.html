Hi there,

We start the first stage of logo competition now. Everyone
subscribed to one of the fvwm mailing list or who has
submitted a logo for the competition should receive a mail
which contains a personal voting id which allows one to vote
for fvwm logos. This id is used to access the voting page.

The first voting stage ends at November 30.

Please vote for the logos in the fvwm logo competition and
help us to find the best logo for fvwm.

Cheers, Uwe
