On Tue, 8 Oct 1996, Michael Hirsch wrote:

> Wile I'm on the subject of autoraise, twm (and variants) have per
> window autoraise. You can dynamically toggle autoraise on a window as
> well as have a default behavior.
>
I kind of have it set up this way... read on.

> I would really love this. Sometimes there is a window that I really
> don't want to autoraise, but usually I want to. Is there any way to
> do this with fvwm?

I don't really know if this will help but I have a few windows that I
rarely want focus to. I have them styled with ClickToFocus. On my
machine, the combination of ClickToFocus and Autoraise have the desired
effect -- all other windows are autoraised but the ones with ClickToFocus
are ignored by autoraise.

Relavent portions of my .fvwm2rc.

AddToFunc InitFunction "I" Module FvwmButtons
+ "I" Module FvwmAuto 150

Style "xconsole" ClickToFocus, ForeColor black, BackColor gray
Style "xmessage" ClickToFocus, ForeColor black, BackColor gray
