Nazgul <poizner_at_ecf.toronto.edu> writes:
> HI, I am currently trying to install FVWM 2.42 under SunOS
> when I am doing make install I get an error on:
> make: Fatal error: Don't know how to make target `../libs/libfvwm2.a'
> Current working directory /u/0T1/poizner/fvwm-2.0.42/fvwm
> *** Error code 1
> make: Fatal error: Command failed for target `install'
>
> I tried to find this file but unsuccesfully
> What can it be, or where can i get this file

This file should have been created when you did make. Instead of
doing make install, do make from the top of the tree. At some point
you should see it go into the libs directory and do a make. This make
should create the libfvwm2.a file.

Then do make install.
