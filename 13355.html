On Mon, Oct 11, 2004 at 07:02:53PM +0800, David Sun wrote:
> and two more newbie questions:
> 1. (emerge when reading the fvwm2 manual) what does the default layer=4
> means?

> all the application windows sharing a common default layer?

Yes, unless the fvwm config or the application requests something
else.

> 2. how can we tell if two application windows are in the same layer or
> not? say we start a application window and then another.

If you can raise one above the other and vice versa, they are in
the same layer. Otherwise, the upper window is on a higher layer.
For more information, you can run the FvwmIdent module. Among
other bits of information, it displays the window's layer.
