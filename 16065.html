>>>>> "Kevin" == Kevin Werner <kwerner_at_holli.com> writes:

    Kevin> I am using Fvwm 1.24. Is it possible to use variables in
    Kevin> the .fvwmrc file? I'd like to create some variables for
    Kevin> colors, so I don't have to change the colors throughout the
    Kevin> file.

If I remember correctly, that version of fvwm processed its rc file
through the m4 macro preprocessor. You could use m4 to define those
colors and use thier symbols where needed. Just change the
definitions to change the color of the display. See m4(1) for more
details.
