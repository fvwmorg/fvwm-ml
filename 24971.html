From: Brian Servis <servis_at_purdue.edu>

I am using fvwm 2.2.2 and was wondering if there is a way to control the
alignment of the icon label in the icon box. Currently it appears that
IconBox only left aligns the labels when not in focus and then centers
when in focus. I have an annoying commercial application that has
really long icon labels and the distinguishing component of the label is
on the right end of the label. Is it possible to right align the icon
label? Currently I have the following setup to get around the problem.

*FvwmIconBoxMaxIconSize 156x0
*FvwmIconBoxFont 5x8
*FvwmIconBoxSetWMIconSize

So I have the IconBox really wide with a small font. This is a bit hard
on the eyes. I am able to set the ICON_NAME resource globally for the
application but then I loose the distinguishing label, i.e. all icons
would have the same label.
