On Sun, 31 Aug 1997, Jamie wrote:

> Dear ALl,
>
> Simple question; n the FvwmPager module, each desk has the title above it.
> IE, Desk 0, Misc, whatever. How do I turn this off? I only use one desktop
> of 3x3, and its annoying. :)

set the following property for the font:

*FvwmPagerFont none
*FvwmPagerSmallFont 0x0

that should do it :)
