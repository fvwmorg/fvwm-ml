On Wed, 17 Sep 1997, Randy J. Ray wrote:

> Now that I am opting for multiple one-page desktops instead of my old
> multi-page single desktop, I'd like a pager more like the one in TkGoodStuff,
> that is, where each desktop is a single button without window outlines,
> preferably labelled. I'd like it to allow me to change desks, with colors
> changes to note the current desk, but I want it to be much much smaller than
> FvwmPager is for my 3x2 arrangement. Is there a feature I have overlooked
> already out there?

I think so. Just say "DesktopSize 1 1" and bind the commands "Desk 0",
"Desk 1", ... to buttons in FvwmButtons. For the color changes you can
use FvwmBacker. This all assumes you're using fvwm2.

Greg
