Well, here it is, the patch is made against fvwm-2.0.46, and adds
some files, if it doesn't work post a note and I would repost it
again, I was in a hurry, and have little time to test it.


The structure of the MenuStyle command has changed, now it admits
next (in the same position that mwm or fvwm) and then it needs more
parameters

        Here are some examples

        
        MenuStyle .... next NWSE Black White 20 WEW MidnightBlue White 20
                             | | | |
                gradient style | | |
                        start color | |
                                end color |
                                     # of steps

                                                The same for title style
        MenuStyle .... next NSN Red Maroon 20 None
        

        The possible GradientStyle are:
          -> None (No gradient. It doesn't need any more parameters)
          -> NWSE (NorthWest to SouthEast)
          -> NS (North to South)
          -> NSN (North - South - Norht )
          -> WE (West to East)
          -> WEW (West - East - Weast )

   
    Questions are welcome.
