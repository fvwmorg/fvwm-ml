This is probably not related, but you know that there is a kernel update
for RH 7.1, right?

http://www.redhat.com/support/errata/rh71-errata.html


I use RH 7.1 and fvwm2.2.4 with all of the updates and don't have any
problems. One major difference, however, is that I simply start fvwm
using a .xsession file in my home directory.


There is also a more recent version of gdm on rufus

http://rpmfind.net//linux/RPM/rawhide/1.0/i386/RedHat/RPMS/gdm-2.2.3.1-17.i386.html

which I'm finding fixes a number of other small problems with gdm... You
might want to give it a try. (There are a couple of other packages which
you will also have to install... )
