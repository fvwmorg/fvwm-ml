On Sat, 2003-04-19 at 17:52, Felix E. Klee wrote:
> Hi,
>
> when I start an application on a certain desktop, say 1, and switch to
> another desktop, say 2, I find it annoying that the application pops up on
> desktop 2. Is there a setting to force all applications and their child
> windows to appear on the desktop where they were started?
>
> Felix

Well, that's a little tricky... FVWM doesn't know which desktop an
application was started on, but you can say that you want to start a
certain application on a certain desktop all of the time with the Style
command, or you can have an application say it wants to start on a
certain desktop with an X resource.

Check out 'THE VIRTUAL DESKTOP' section in the manpage.
