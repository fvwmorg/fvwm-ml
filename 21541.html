amasso <amasso_at_conexis.es> writes:
> I configured and ran Xwindow with FVWM. When using some programs, there
> are errors telling me that I don't have my fonts configured???...When
> using, for example arena, it opens, but imediately disapears. What is
> the cause of this problem?

I don't think your problem is caused by fvwm. Fvwm doesn't do
anything to interfere with your ability to use fonts.

You can look at the directories that your X server is using to get
fonts from by issuing the command "xset -q".

Part of the output will look like this:

  Font Path:
    /opt/public/pkg/x32/fonts,/u/dane/fonts/fonts,
  /usr/openwin/lib/X11/fonts/Xt+/

If you need to add fonts to your fontpath, use "xset +fp".
