Near 2000, Look, Joseph mumbled something about:
>
> I would like to know is it possible to do some cleanup (running my own
> script) before executing the built-in Quit when the user select to quit
> fvwm.
> For example, in the .fvwmrc file the Quit selection look like this
> AddToMenu "Shutdown" \
> "Quit fvwm%mini-exclam.xpm%" Quit
>
> Can I do something like:
> AddToMenu "Shutdown" \
> "Quit fvwm%mini-exclam.xpm%" Exec my-program , Quit


AddToFunc Byebye "I" Exec (command 1)
+ "I" Command 2
+ "I" Etc...
+ "I" Quit

AddToMenu "Shutdown" Quit fvwm%min-exclam.xpm%" Function Byebye

Hope you get the idea,
