Previously from Rohan Oberoi
>1. Does anyone know if it's possible to order the TaskBar to only show
> applications in the current virtual desktop (ie. FvwmPager mode)?

I don't think it's possible.

>2. Since I suspect from reading the man page that it is not possible
> to do that, is it possible to create a simulated TaskBar using
> FvwmButtons and FvwmIconMan?

Sure. Depending upon how exact you want things, it's easy or hard to
do. Take a look at http://www.best.com/~mikehan/fvwm2rc.html and
fvwm2rc.tgz for an example of the same principle (though what I've
done is created two small FvwmIconMan instances, one of which is
showing current page and the other showing everything not on the
current page.

Anyway, the simplest example I can give you would probably look like
this:

*FvwmButtons( Swallow FvwmIconMan "Module FvwmIconMan" )
*FvwmIconMan*resolution desk

After that you'd need to start tuning things to look nice, put in the
Start button and clock, etc.
