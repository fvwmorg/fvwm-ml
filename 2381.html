On 23 Apr 2001 11:40:55 -0400, Louis LaFontant wrote:
>
> Is there a way to control the client decoration symbols ; the "minimize", "Maximize", and "Close" buttons located on
> the upper right hand corner of a window/dialog box. Alternatively, is there a way to re-display a dialog box until the user displays the correct entry.
>
> What I would like to do within a FVWM Script is to display a dialog box until the user has entered the correct input.
> When the user tries to exit the dialog box, using the "x' on upper right hand corner, an invalid entry mesage is displayed and the main menu appears gain. This repeats (loops) until the user select one of two entries.

You may simply wrap your dialog into a while loop, something like:

  while test "`xmessage -buttons 'Yes:2,No:3' 'Correct?' -print`" == ""; do
    echo "Nothing chosen, repeating..."
  done

There is also an option to disable close and iconify buttons:

  Style xmessage NoButton 2, NoButton 6
