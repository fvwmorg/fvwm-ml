I have implemented CleverPlacement (thanks to Trent Piepho for the name).
CleverPlacement is like SmartPlacement, but instead of falling back to
RandomPlacement or ActivePlacement when it can't find a place to
put the window, CleverPlacement finds the area where there is the least
amount of interference with other windows and places it there. It
generally puts the window more or less where you would have put it
yourself.

To try it, just replace the standard placement.c with this one:
http://www.engr.csulb.edu/~amartin/placement.c
and recompile.

Note that currently, this just replaces SmartPlacement, so you will need
to have the SmartPlacement style in your .fvwmrc file.

I tested this on fvwm-2.0.43, but it should work fine with fvwm95, as I
think the placement code hasn't changed.

I would like to hear about successes or failures.

Tony.
