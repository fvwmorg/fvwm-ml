> Another question, which might not be FVWM related has to do with
> netscape. I start netscape with the -component-bar option however, it
> never comes back in the place that I specify... Any ideas?

I'm not exactly sure how you specify where you want the component
bar. If I issue

  netscape -component-bar -geometry +x+y

It always opens at +50+54, regardless of the actual values of
x and y. The only explanation that comes to my mind is this:
netscape ignores the geometry hint you specify but uses its
own idea of the correct values instead. Furthermore it sets
the 'user specified position' hint which forces fvwm to place
the window exactly where netscape wants it. If this hint wasn't
set, fvwm would have placed the window like any other window.
This is very likely a netscape bug. It's a known fact that
netscape does some unusual things with the 'user specified
position' hint.
