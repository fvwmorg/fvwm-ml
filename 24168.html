On Fri, May 14, 1999 at 02:36:31PM -0400, Kim Edwards wrote:
> Hi,
>
> When I use my middle mouse button to drag a window within the Pager, the window is not being dragged and I simply change viewports. I believe that there must be some configurable option that I am missing. The middle button works in an Xterm when I use it for the paste operation.

No, you aren't missing anything. There is no such option.

> I am using the same .fvwmrc that I used for a Solaris box and it works fine. The problem only occurs on my Linux box. Any comments or solutions?

I guess you have picked up one of the versions that had this bug on
your Linux box. With fvwm2.2 and up this is no problem. Only some
of the 2.1.x betas had this. Please make sure you are really using
the correct FvwmPager. Perhaps your path points to an old one.
