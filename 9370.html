Probably because you use only one desk. Start fvwm with this initial rc:

  Key Tab A M Next (Visible, AcceptsFocus) Focus
  Style FvwmConsole Sticky
  Style FvwmPager Sticky
  FvwmPager 0 3
  Wait FvwmPager
  FvwmConsole
  Wait FvwmConsole
  Exec xterm -name xterm1
  Wait xterm1
  GotoDeskAndPage 1 1 1
  Exec xterm -name xterm2
  Wait xterm2

Press Alt-Tab. Visible is ignored.
