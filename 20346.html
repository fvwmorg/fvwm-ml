Christopher Wolf <cwolf_at_micro.ti.com> writes:
> Is it possible to move a window relative to it's current position?
>
> Say I might want to have a key to bump a window a little to the right
> or left of it's current position, maybe to help with overlapping
> windows.
>
> How is this/could this be done? I can only find commands to move to
> absolute positions ("relative" in the man page's sense means relative
> to the page size -- a questionable usage as written).

If you use "move" without arguments, as in:

Key F7 A M Move

You then get to use the cursor keys to move the window where you want.

As far as dealing with overlapping windows, don't forget FvwmTile
and FvwmCascade.
