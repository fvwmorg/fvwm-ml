>>>>> Balasubramanian Raju writes:

  Raju> Is there any place where there is step by step instruction for
  Raju> how to configure ??

I think the INSTALL file is rather clear.
untar fvwm, then edit Fvwm.tmpl and config.h (there are comments in
it, so you'll know what to put in there), then type xmkmf, then make
Makefiles, then make, then make install, then make install.man.

(I didn't need to edit config.h, so you should start editing Fvwm.tmpl
first, then only if something doesn't work look at config.h.)
