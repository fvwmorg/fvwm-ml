> hi there...

> right direction in order to implement the SqueezeTitleBars funtionality
> that is in twm...
>
> basically SqueezeTitleBars makes the title bar about 1/3rd the width of
> the window...
>
> i've tried poking around with this a bit but have had no success

Well, this enhancement request is waiting to be done for a while
now. I thought a bit about a possible implementation, and the
best design I could come up with is this:

 1) Use the Shape X extension to hide the right half of the
    title bar.
 2) Modify the border drawing and button placement code in
    borders.c.
 3) Adapt animated window shade and opaque resizing to these
    changes.
 4) Introduce a new style (style.c, update.c, style.h, fvwm.h,
    window_flags.h).

(this all refers to the latest beta versions).

Overall this feature will become very difficult to implement.
