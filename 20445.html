I have been using Fvwm 2.0.45 for a while now. I use a pixmap title-bar
and pixmap buttons (which are set to flat)... My question is this:

Since I use a pixmap title-bar.. I would like my buttons to appear over
the same background pixmap that is on the title-bar. So far, I believe
the only way to accomplish this is to set the buttonstyle to my
title-bar's pixmap and then AddButtonStyle my (semi-transparent)
button-pixmaps. The only problem with this is that the
background pattern re-starts on each button.

Is there any way to avoid drawing the button's background pixmap... so
that the pixmaps for the buttons will just be drawn over the title-bar's

again)

If the only way to get this to work is by modifying the code... could
someone give me a tip or two on where to go? (I gather that it's in
borders.c) [specifically, how to get the buttons not to draw a
background...]
