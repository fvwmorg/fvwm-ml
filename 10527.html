You don't need to disable the Nautilus background, I don't think.

This is a quote from the fvwm man page, section "EXTENDED WINDOW MANAGER
HINTS":

> There is a new Context 'D' for the Key, PointerKey, Mouse and Stroke
> commands. This context is for desktop applications (such as kdesktop and
> Nautilus desktop).

So, add "D" to your context for the commands you want enabled on the
desktop window and you should have no problems.
