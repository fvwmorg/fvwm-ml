> > I'm wanting to give some users access to one program when they log

> > fvwm,
>
> It sounds like what you really want is to just set that one program as
> the window manager for that environment. Does the program really need
> to have window handling support?

You could start the program up without any window manager, using xinit.
For example :

xinit ./quake2 +set game action -- /usr/bin/X11/X :1

(You don't even need to start X to do this)
