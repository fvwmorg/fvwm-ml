On Sat, Mar 06, 2004 at 07:39:12AM +0000, Ukpong Ukpong wrote:
> hi all
> I am using a shell scripts to create a string in a given format
> then want the contents of that string (array) to be displayed in the menu
>
>
> +DesktopNumber DesktopName+
> + 0 Mail +
> ++++++++++++++++++++++++++++
> I have a few ideas but can't figure out how to make PipeRead use the
> contents of the array.....

The easiest way is to make a shell script and call that, e.g.

  ------------------ my_script.sh -----------------
  echo "AddToMenu foo"
  echo "+ 1 $array[1]"
  echo "+ 2 $array[2]"
  ...
  -------------------------------------------------

  PipeRead my_script.sh
