I've noticed a problem with FrameMaker document windows which I believe is
related to fvwm.

My Frame windows are very big because they show a document page at 100%.
Generally, Frame window geometry gets saved with the document, so that when
I open a document, it opens wherever I put it last. However, I find that with
fvwm, the windows are coming up slightly too far down on the screen, making
the bottom go off the bottom of the page.

I've been experimenting. I put a window at +12 (from the top), save it, and
close it. When I reopen it, it is at +25.

Any ideas why this is happening?
