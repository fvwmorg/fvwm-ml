> On Tue, Jan 13, 2004 at 01:28:35AM -0800, Joshua Eckroth wrote:
> > I want to load a FvwmPager module on desk 1, say. I've tried
> > GotoDesk 0 1 in StartFunction, but it never seems to be executed.
> >
> > How do I start a window on another desktop?
>
> Set a style:
>
> Style FvwmPagerDesk1 StartsOnDesk 1
>
> and start the pager with the name given in the style:
>
> FvwmPager FvwmPagerDesk1 ...
