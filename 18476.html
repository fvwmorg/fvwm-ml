Although I didn't mention it, eventually the plan is that these will
be options/modifiers on the ClickToFocus style option, but since there
aren't enough style flags until after the GSFR, and since people keep
asking about an option like that, I figured I'd add them as global for
now. BTW, I have a few other things planned to add like this as well
for 2.0.44:

  ClickToFocusDoesntRaise/Raises - so have only clicking on border
                                   raise, like compiling with
                                   CLICKY_MODE_1 right now,
  SmartPlacementIsReallySmart - to make it use the new CleverPlacement
                                stuff

Eventually (post GSFR) these will also become styles/style modifiers,
but I figure that global is better than nothing right now.
