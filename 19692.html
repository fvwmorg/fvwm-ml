'clearmake' is part of the ClearCase configuration management system.
The X consortium recently switched to using it so the imake system that
comes with X11R6 has support for it.

However, if you don't use clearcase, you shouldn't have a problem.

Looks like there's something wrong with your X configuration/setup.
