On Thu, Nov 13, 2003 at 10:54:10PM +1100, Scott Smedley wrote:
> Note for Xavier & Tavis making FvwmTabs distributions:
> My patch to X11::Protocol v0.51 (which was bundled in my own distro
> v0.51_2) has been merged into v0.52 so this should be considered the
> definitive version of X11::Protocol to use from now on.
>

Thanks Scott, if anyone using Gentoo would like to try the update, you
can do this:

// fetch the latest updates to the portage tree
# emerge sync

// you need to accept packages marked ~arch (testing), use this command
// to check everything looks okay and you have all the USE flags set you
// want (you need perl and tcltk set to get FvwmTabs).
# ACCEPT_KEYWORDS=~x86 emerge -pv fvwm
(...)
[ebuild U ] dev-perl/X11-Protocol-0.52 [0.51-r2]
[ebuild U ] x11-wm/fvwm-2.5.8-r1 [2.5.8] +readline +truetype +ncurses +gtk +stroke -gnome -rplay -xinerama +perl -nls +png -bidi +imlib +tcltk +debug +gtk2

// if that looks okay...
# ACCEPT_KEYWORDS=~x86 emerge fvwm
