On 28 Jul 2004 16:37:11 -0700, Manish wrote:
>
> I'm using it on cygwin/win2k. I put the following in
> my .fvwm2rc following the FAQ but it shows me all
> windows (in all desktops; I have 2x2 virtual desktops)
>
> Key F8 A N WindowList Root c c CurrentDesk NoGeometry

This works. Do you mean the current page rather than the current desk?

  WindowList (CurrentPage) Root c c NoGeometry, NoHotkeys

Still, it is better to place a comma between options, i.e. after
CurrentDesk in your example, and possibly use "A"ny modifier instead of
"N"one, so that the binding works with NumLock pressed too.
