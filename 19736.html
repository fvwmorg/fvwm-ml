<gripe>
Every time I go to download the latest FVWM-2 beta, I look at the
monster size and say "nah... I'll do it later." As a result, I'm
still all the way back at 2.0.43 (prehistoric, when it comes to
development...).
</gripe>

<proposal>
Patches (preferably unified diffs) should be available for downloading.
</proposal>

<apology>
Does someone already maintain an archive of these patches? If so, it
would be really nice if they were merged with the main site...
</apology>

<obligatory offer :) >
If no one at all is willing to do this, then I'll do it myself for the
benefit of everyone else... Though it'll certainly waste bandwidth,
since I'll have to download each new release and upload each patch (and
28.8 is molasses); and it certainly won't help my problem :)
</obligatory offer>
