"S. Anderson" <sndavis_at_crosswinds.net> writes:
> On Sun, Jan 07, 2001 at 08:17:03PM -0700, pzhao wrote:
> > In my .fvwm2rc file, I add following statement
> >
> > Style Netscape* StartsOnPage 1 0, RandomPlacement, SmartPlacement
> > *FvwmButtons(Title Netscape, Icon netscape.xpm, Action 'Exec "Netscape"
> > netscape')
> >
> > but when I click the "Netscape" button, I get the netscape run with
> > maxmized window size, covering all other windows. This makes me feel
> > very inconvenient. How to tell the system a defined window size of
> > a specified application(e.g. netscape)
> >
> > thanks
> >
>
> Its a little off topic but you could try putting:
>
> Netscape.Navigator.geometry: =856x600
>
> into your ~/.Xdefaults file. Or mabey look into MaxWindowSize, if you
> are running the development version.

For me, Netscape windows "remember" their most recent size.
If you don't like full screen windows, you should resize them
smaller. The next Netscape window you create should be that smaller size.
