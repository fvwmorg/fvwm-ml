Here's a suggestion for a future release of fvwm2. Not being a
particularly great programmer, especially when it comes to X windows, I
wouldn't know where to start, but here goes anyway -

        When I used to be an avid Acorn RISCOS junkie, when I used to
write programs for it, dialogue boxes could be sprouted from popup
windows, to allow for text entries etc. This was very useful, since
things like formatting disks and stuff could be done without starting new
applications.

        It would be nice to be able to do something similar, say using
FvwmButtons or FvwmForm as the popped-up dialogue box. Moving on from
that, it would be nice to be able to popup say FvwmButtons, which would
allow you to use it as an alternative to current menu styles.
