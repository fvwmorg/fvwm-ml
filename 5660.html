On 25 Jul 2002 18:08:50 +0200, Wolfgang Pfeiffer wrote:
>
> I don't believe it:
> To change my icons for iconified windows I edited
> .fvwm/themes-rc-3 to this value:
> Read "/home/user/.fvwm/themes/personal/settings/iconstyles/native"
>
> I changed .fvwm/themes-rc-2 resp.
> .fvwm/themes/current-test/themes-rc-2 to that value:
> Read "/home/user/.fvwm/themes/personal/settings/iconstyles/native"
>
> I click on
> Theme Management > Refresh and Reload > Refresh with no cache
> and the changes from above have been destroyed, reset to the values
> the files from above had before.

The rule in fvwm-themes is you don't edit any existing files, especially
autogenerated ones. You may create and edit your own personal parts that
will be detected by issuing "reset" from the menus. Or in some cases
optionally use GUIs if exist to create such personal parts.

> So how can I change these icons?
> Is there a menu point or file that I didn't see?

Please create ~/.fvwm/themes/personal/iconstyles-extra file and put any
Icon and MiniIcon lines that you want to define. Then do the usual
"Refresh with no cache" and choose this newly appeared iconstyles-extra
component from the Themes menu or Themes Center GUI.
