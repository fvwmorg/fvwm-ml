Unfortunately (partly due to my haste) if you downloaded the tbsp4
patch there is only a chance you got a good patch. To remedy this, a
brand new patch, tbsp4a is now available which works (compared to
tbsp4 at least :). If you tried tbsp4, please try tbsp4a before you
email me of problems (of course, email me if you like / dislike
features too so I can pursue certain avenues).

However, as a bonus, you now get a working HiddenHandles style for the
borderstyle. This time it works nicely.

You can get the latest patch at:
http://www.rpi.edu/~veliaa/projects/fvwm_patches/fvwm_border_picture.html
