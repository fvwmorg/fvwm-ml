On Thu, 11 Sep 1997, Jeppe Buk wrote:

> Hi
>
> Does a module exist that allows a shell script to send commands to fvwm
> (e.g. via a named pipe)?
>

There is a module called FvwmCommand available at
  
 www.concentric.net/~isogai/fvwm2/fvwm2.html

(FvwmCommand come with 2.0.46 probably will not compile as it is.)
(I also have to update the one on my site for SunOS.)


Put it under extra directory of fvwm2 source. Untar it, then run
xmkmf
make
make install
make install_man.

A man page and samples scripts are in the package.
