On 02 Dec 2003 11:05:15 -0600, Shay Hill wrote:
>
> Does fvwm have any mechanism for making the grabbing of window edges and
> handles any easier?

Do you mean Resize defined using Key or Mouse? They have different "feel".

If you bind Resize to window borders using Mouse and make window borders
wide enough (5-7), then resizing is pretty comfortable, I think.

> MS Windows, for instance, has a barbell shaped
> cursor which allows a little bit of play when grabbing an edge.

You may define 8 different cursors that appear in different corners of
the window border before Resize is started, and one cursor for Resize
itself. See CursorStyle. Any custom cursors are possible, and the latest
XFree86 supports even animated translucent cursors. Is this what you want?

> I believe that Afterstep will "snap" the cursor to a window border.
>
> Anything similar in fvwm?

Nope. The pointer is not currently automatically wrapped to the window
border on Resize. It is not clear to which from the 4 borders it should
be wrapped and to which point inside that border (say, border width 6).
