Hi,
        I don't know how many of you checked out the rc's I mentioned
URL's for, but on one of them, I packaged everything as a self installing
shell script. I did it because I wanted to try to do something functional
with shell scripting (I'm sorta new at this :) Anyway, it seemed someone
was very impressed with the idea of downloading a script that was self
contained and installed all the necessary stuff. He suggested I submit it
as a sample config for the distribution, but somehow I doubted this would
actually happen. It did however give me an idea for a homepage...

        The one thing that this person liked was that you see the config,
download the script, and it installs.... No figitting with settings, etc,
etc... It would take the hassle out of trying out someone's available
.fvwm2rc ... The idea that a user can look through a bunch of "looks" and
then pick and install one that they liked without too much trouble is
actually kinda cool. I was wondering if maybe someone would like to try
and start up an FVWM Easy Config homepage.
        Basically, the idea would be that a user could go to the homepage,
see enough of a screen capture to get an idea (no ghastly 1280x1024 jpegs
that take 10 years to download), download a script and voila! An easy
install. I would do this myself but I don't have the time. I have
converted both of my configurations into this installation script format
and they're both available at:

        http://www.fiu.edu/~kvarga01/computing/linux

If anyone wants to actually see what I'm talking about by an easy install.
I'f anyone would like to work on this and coordinate this or talk about
it... let me know...
                        -peace
