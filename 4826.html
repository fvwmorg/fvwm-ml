I use desktops. If you use pages, I can't help you. I have 9
desktops visually displayed as a 3x3 grid. I have a key binding for
GotoDesk 1 0 8 and GotoDesk -1 0 8. These commands wrap from desk 8
to desk 0 and vice versa. I'm pretty sure this has worked, modulo
command renaming, for a very long time. At least since 2.2.* and
probably since 2.0.*. Dare I suggest even 1.24* :-)
