On Mon, Feb 25, 2002 at 01:53:34PM +0000, Thomas Adam wrote:
> Ok,
>
> My sliding panel is working a treat now, except for
> one thing.
>
> Since I have swallowed an xterm, what I am finding is
> that when you open/close the sliding panel, the prompt
> on the xterm:
>
> [thomas_at_grangedairy]$
>
> gets mangelled, i.e. gets pushed out of sequence. The
> only way to correct this is to type in:
>
> clear
>
> into the xterm.
>
> Is this a common problem?? Is there a way to
> automatically refresh the xterm window, if not???

Terminal window usually don't like to be shrunk too much. The
problem should go away if you disable the animation. Add the
panel option "steps 0" to the appropriate line, e.g.

*FvwmButtons(Panel(steps 0, indicator) foobar "exec xterm -T foobar")
                  ^^^^^^^^^
