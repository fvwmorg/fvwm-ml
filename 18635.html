Quick Intro:

The BorderStyle patch is a patch which adds value to your existing
fvwm-2.0.43. There are many things that it does. The best way to
know what you can do with it is to read the associated man page
entries it changes for ButtonStyle / TitleStyle / AddButtonStyle /
AddTitleStyle and BorderStyle.

I've put a new patch on my WWW page. There are a number of notable
changes in it (i.e., the rewrite is starting to come of age :). Note
that this is still in development. Style tagging is not there.

Changes include:

         fix for MWMBorder flat button

         more man page updates

         consistency improvements changes
         (borderstyle/buttonstyle/titlestyle now very similar in operation)

         ButtonStyle now has an Inactive state

         ButtonStyle flags (which include borderstyle / titlestyle )
         can now be negated by prefixing an !

         when changing flags, only changed flags are actually changed (i.e., for setting of multiple
         states at once)

         titlestyle sunk, raised, flat flags removed (see next)

         all buttonstyles now have a sunk and raised flag. This removes
         the need for the titlestyle flat, sunk and raised global options (do
         'titlestyle -- sunk' for a global effect).


Read the man page updates, since they are important. If you find a
bug, make sure you have the latest version before you do anything.

As usual, it is at:
http://www.rpi.edu/~veliaa/projects/fvwm_patches/fvwm_border_picture.html

Enjoy,
