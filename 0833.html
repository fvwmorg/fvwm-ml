Peter,

First let me suggest that you do not edit the files directly. This way
you always have a working set-up to go back to in case of trouble. When
things work as you like then make changes if you prefer. Instead, copy
the file you want to edit to your home directory and rename it with a "."
in front and droping the ".m4" suffix.

fvwm2rc.modules.m4 -> ~/.fvwm2rc.modules

As for tutorials and docs on editing the files, all I can suggest is the
man pages for fvwm2 and m4 man pages. The AnotherLevel package is just m4
preprocessing and fvwm2. Besides that the people on this list are very
helpful.

As for the Scale and Geometry not working, look at the fvwm2rc.modules.m4
file and note the section on FvwmButtons above the section on
FvwmPager. See the Swallow command. What you see as the pager is a
swallowed app, so the configuration of FvwmButtons also modifies
(currently constrains) the size of the pager.

Tom
