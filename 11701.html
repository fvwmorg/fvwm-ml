Use the ImagePath command in your .fvwm2rc (sen man page).
 
> 2) Are there any suggested programs that I should have
> on my system for working with fvwm. such as perl? etc
>
> 3) If a user sets up a theme for the fvwm window
> viewer what else is required it terms of installation

Running the configure scripts in fvwm (and fvwm-theme?) prints a
summary at the end. Check whether all features you would like to
use are enabled. If not, the reason is printed too. Install all
missing reqirements until you get yes for all the features you
want.
