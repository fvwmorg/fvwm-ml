This is untested, but perhaps a function like this:

AddToFunc position_at_start "I" Exec $1
+ "I" Wait $2
+ "I" Next [$2] Focus
+ "I" Move $3 $4

would do the trick?

The usage would be such that $1 is the command to run, $2 the window
name of that application and $3 and $4 the x and y positions (probably
want to use the "p" suffix for pixels rather than percentage).
