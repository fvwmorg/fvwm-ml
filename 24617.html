On Tue, Jul 20, 1999 at 12:48:54PM -0700, Krishna Behara wrote:
> Hi,
>
> I have been trying to get icons to work with my
> fvwm , i installed xpm ,cxpm etc but these
> are in a bin directory in my home directory,
> I also downloaded a whole set of icons in xpm
> format so that i could set them for the different applications
> but nothing seems to work,i keep getting the default
> icons.

1) You must have compiled fvwm with xpm support.

   If you have compiled fvwm yourself you have to make sure
   you have compiled it with xpm support. When you run 'configure'
   it will tell you this at the end.

2) You must set the ImagePath to point to your icon directory.
   (or the PixmapPath command in earlier versions).

     ImagePath path_to_icon_directory:another_directory_path:...

3) You must tell the applications which icons to use like this

     Style * my_default_icon.xpm
     Style netscape my_netscape_icon.xpm
     Style xterm Terminal.xpm
     ...
