Hello,

to start off with some praise: xine is brilliant software and it works great
for me - so is fvwm2 ;)

However - these two don't work together. Whenever I start xine, it exits
as soon as it begins playing a Movie. With any other wm (I've tried a lot
of them, including old versions of fvwm2) it works just fine. And that only
happens when playing back via XFree4's XV-extension.

My machine is a P-III 450, 256M RAM, Matrox G200
XFree4.0.1, fvwm-2.3.3{0,2}, xine-0.4.{01,2} (I've tested both).
I'm running GNOME on a Mandrake-7.2 Distribution.

I've had a look into the source, however I don't know about X-Programming, so
I couldn't even track the problem down to either fvwm or xine, so I posted
this on both lists.
I have added the xine output, and if that helps I have the output produced
by strace online at
http://gargamel.weh.rwth-aachen.de/xine/DestroyNotify.strace
