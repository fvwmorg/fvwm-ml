 MF> I would like to pop up fvwm pager using a key binding to show the
 MF> desktop layout.

 MF> I don't have the desktop space to keep it open all the time (14"
 MF> laptop screen ;-( and having it dock away is kind of a waste of
 MF> memory as I don't use that much (most of the time I'm on one
 MF> desktop).

 MF> Is it possible to make it just appear in the middle of the screen
 MF> or something one a key is pressed and then make it go away again
 MF> later?

  man FvwmPager:
---8<---------------------------------------------------------
  If the pager is started with the -transient option, the next time a
  button is released the pager is closed. Note that this option does
  only work if the window style of the pager window is 'sticky' (see
  the fvwm man page). You should use the 'StaysOnTop' style too.
  Example:

       Style FvwmPager Sticky, StaysOnTop
       *FvwmPager: Rows 1
       *FvwmPager: Columns 1
       Mouse 3 R C Module FvwmPager -transient

  With this in your .fvwm2rc, if you press control and button 3 in the
  root window the pager pops up under the mouse and while the viewport
  moves with the mouse.
--------------------------------------------------------->8----
