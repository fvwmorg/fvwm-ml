     TogglePage
          Temporarily disables edge scrolling. Edge scrolling can
          be re-enabled by calling this again.

was very convenient to be put under a button in my buttonbar to temporarily
disable paging, when doing some CAD stuff.

Using:
EdgeScroll 0 0
EdgeScroll my_normal_params
would use up two buttons (or both the click & the doubleclick function of
one button - if the doubleclick option worked, that is :) )
