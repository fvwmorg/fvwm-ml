Using Fvwm2, I have an FvwmButtons module located in the top left corner
of my screen. It extends down about 40 pixels and across to the right
about 800 pixels. No border, Always on top, to describe it briefly.

When I start Netscape, Netscape positions itself slightly underneath the
buttons, so each time I start Netscape or ask Netscape to open a new
browser window, I immediately have to pull the window down below my
FvwmButtons area.

Is there an Fvwm2 or Netscape configuration change I can make so that
Netscape will be placed according to the open space on my screen, not
under FvwmButtons?

FYI, I have these already:

Style "*" ActivePlacement, SmartPlacement, RandomPlacement
Style "*" MWMFunctions, MWMDecor, HintOverride

Style "Netscape" Icon netpositive.xpm

And no other general purpose or Netscape specific settings that I can
find. I'm very interested in hearing if others are just accepting this
annoying behaviour. :)
