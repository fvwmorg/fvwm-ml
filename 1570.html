Well, the FvwmButtons man page on my system lists several options that
should be of interest to you:

  * FvwmButtonsButtonGeometry, which lets you set a geometry for a
    single button. So, you can make each button be a specific size if
    you like.

  * Or, FvwmButtonsPadding can increase the padding between the edge of
    the button and its contents (normally this is 2 pixels).

There may be others; check the man page.

Or, you can find larger pixmaps and use those instead of the ones you
have, then FvwmButtons will automatically increase the size of your
button bar accordingly.
