By itself, fvwm offers just a basic default configuration, requiring an
extra add-on package to allow switchable desktop themes.

You do not mention it so one can only assume that you have not yet
installed fvwm-themes available at http://fvwm-themes.sourceforge.net/
which would not only save your last loaded configuration but also offers
session management support.

As far as I know this is the only official fvwm theme engine, and
although there are several other theme formats for older versions of
fvwm, none have the features, support and the continuous development
concurrent with the latest fvwm development version, like the
fvwm-themes project enjoys.

Please supply more details of your actual configuration if you require
further assistance.
