On Mon, 14 Dec 1998 15:32:50 +0100, danny.pauselius_at_kvaerner.com wrote:

> I'm currently working with fvwm 1.xx and i would like to know if
> it's possible to create a menu, that doesn't popup with a
> mouse-click or keyboard stroke, but stays on my desktop all the
> time? Can somebody help me out? Thanks

I was about to reply that there is an easy way to do this in fvwm2,
when I decided to check and see if it still works in fvwm 2.1.4 and
discovered that it DOES NOT work in 2.1.4!

What I was using in earlier fvwm2 versions was something that started
like this:

        AddToMenu Window-Ops
        + "Window Ops" Title
        + "Stay" Popup Window-Ops

When clicking on "Stay", the menu would stick around. Sure, it would
disappear as soon as I did anything else, but it would stay around
longer than otherwise. Yes, I know there are other ways to do this,
but my point is that this no longer works. It's broken in my
installation of fvwm 2.1.4.

Has anyone else seen this?
