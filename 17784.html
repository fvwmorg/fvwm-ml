Hello,

I'm trying to put the pager in a button. I've tried this:

*FvwmButtonsFore Black
*FvwmButtonsBack #c0c0c0
*FvwmButtonsFont -adobe-helvetica-bold-r-*-*-10-*-*-*-*-*-*-*
*FvwmButtonsGeometry 1124x60-1-1
*FvwmButtonsRows 1

*FvwmButtons xclock clock.xpm Swallow "xclock" Exec xclock -bg \#c3c3c3
        -geometry -1500-1500 -padding 0 &

# etc.

*FvwmButtons(2x1) - whatever SwallowModule "FvwmPager" FvwmPager 0 0

*FvwmPagerBack grey60
*FvwmPagerFore black
*FvwmPagerFont -adobe-helvetica-bold-r-*-*-10-*-*-*-*-*-*-*
*FvwmPagerHilight #c0c0c0
*FvwmPagerDeskTopScale 50
*FvwmPagerGeometry -1-1
*FvwmPagerLabel 0 Desktop
*FvwmPagerSmallFont 5x8


But the Pager is then giant. So, I made this change:

*FvwmPagerGeometry 120x58-1-1

Now the pager is almost in the right place, but it is not in a button.
It simply overlaps the location where the button would be.

What can I do?
