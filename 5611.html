I think it is explained well in the man page, but I agree that it may look
strange for the users new to FVWM. The syntax of the Mouse command is
explained in the Mouse entry of the "COMMANDS FOR MOUSE, KEY AND STROKE
BINDINGS" section. And the syntax of user functions is explained in the
AddToFunc entry of the "COMMANDS FOR USER FUNCTIONS AND SHELL COMMANDS"
section.
