Hmph. You can try to forbid gkrellm to do anything with its own
windows. Try the following styles in this order:

  Style gkrellm NoPPosition

If that does not help, add

  Style gkrellm NoUSPosition

...

  Style gkrellm GNOMEIgnoreHints

...

  Style gkrellm FixedPPosition

...

  Style gkrellm FixedUSPosition

If that still does not help, use xosview ;-)
