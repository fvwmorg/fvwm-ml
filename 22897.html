Humble apologies if this is a repeat, I think my previous message didn't get
through.

We're running Solaris x86 HW 5/98, with DU3 (I've stuck the list of
relevant-looking patches at the bottom) on RM PC 6333 (PII/333MHz, 64MB, ATI
Rage Pro 8MB), and it all works fine, except ... fvwm-2.0.46 reliably hangs
the X server when trying to move a window off the screen (and certain other
resize operations when the window is on the edge of the screen). I've
compiled it with gcc-2.8.1 and egcs-2.90.29, no change. Has anyone got any
insights / suggestions on chasing this problem ? Are we the only ones to have
seen it ? Any comments would be well received, since people are in tears
about losing fvwm2.

Cheers,
