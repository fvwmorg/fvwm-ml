Hi all,

<1> How to place Icons side by side without GAP in between?

I use 'Style * IconBox 130 12 800 250, IconFill left top' and
Thumbnail function from the FAQ to iconify windows. Usually,
the iconified windows are placed this way:

+----------+ +----------+
| | | |
| 1 | | 2 |
| | | |
+----------+ +----------+ ... ...
+----------+ +----------+
| <title> | | <title> |
+----------+ +----------+
            ^
            |-------------- Gap

I want them to be placed this way:

+----------+----------+
| | |
| 1 | 2 |
| | |
+----------+----------+ ... ...
+----------+----------+
| <title> | <title> |
+----------+----------+
           ^
           |--------------- No gap

Any ideas?

<2> How to prevent IconTitle from auto-expanding?

Every time I focus a iconified window, it's title
expands to the length of the title text. This is
sort of annoying for me because the title text is
too long most of the time.

Any ideas how to cut the title text to fit Icon width,
or disable IconTitle auto-expansion?

<3> StippledTitle of uniconified windows, but not iconified windows.

I use:
  Style * Icon, StickyAcrossPagesIcon

And the iconified windows get stippled titles, which are not looking good.
How can I disable ionified windows' stippled title, but keep stippled title
for sticky uniconified windows?

Thank you VERY MUCH!
