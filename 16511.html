On Sun, 28 Apr 1996, Kevin Hilman wrote:

> Is there a way to enable screen savers (using xlock) after a certain
> amount of time in fvwm (still using version 1).

I use xautolock, which allows any arbitrary program to be run after some time
(usually xlock) and also allows the startup of the program if the mouse is
placed in a corner.

There are other programs as well, but I have found xautolock works best for
me. I can't remember where I picked it up (it was before I started logging
all my software FTP's) but I think you might be able to find it at ftp.x.org.
