> h '/users/samk/fvwm'
>
> I don't know of any modules of this sort ?

When you installed fvwm, there should have been a file copied to
<install-prefix>/etc/fvwm named "ConfigFvwmDefaults". The function
is defined in that file and fvwm attempts to read it during startup.

That function is used by the WindowList command.
