And then some.

The configuration of an X workstation is a problem that needs addressing.
I don't think things like CDE have really solved them. Rather, some way
of managing it without depending on a limited-configure setup like CDE is
needed.

It is relatively easy to write a program to manage any one individual
file. But across multiple files, most of which are totally different from
each other, it is much harder.

Still, I wish something like that existed... maybe something of a
theme-editor for X that understand the differenent files and how they
related. I have seen anything written about this in any kind of serious
detail.
