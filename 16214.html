1) Even a simple 1-button definition doesn't work. Nothing appears.

This reminded me of earlier complaints on the mailing list about
incompatibilities between FvwmButtons and certain versions of libXpm. I'm
using version 3.4h. It so happens I have a copy of 3.4a lying around, so
I used that instead, and voila! It works.

Apparently there's still some incompatibility in there. Unfortunately, I
don't have the time to narrow it down any further. Hopefully Jarl or someone
else can use this information to find the problem.
