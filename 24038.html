Is there anyway to delete a title bar button that has been previously
defined? I looked through the docs and could find no way of doing this.
I would think something like,

DestroyButton 1
or
DestroyButton all

would be great. The latter would delete button 1 and all mouse bindings
associated with it and the former would remove all button setups and
bindings. If a mouse binding is set to button 1 and no ButtonStyle is
defined for button 1 then a default button is used, so the destruction
of the button must also unbind the mouse. This could be useful in a
function that is used to redefine the button setup for title bars or
other change a theme on the fly that has different amounts of buttons
defined, etc.

I am using fvwm 2.2.
