On 16 Jun 2004 18:53:18 -0000, Eduardo Gargiulo wrote:
>
> I want to configure fvwm 2.5 to use another mouse cursor; IMHO the
> default "X" cursor contrast with a pretty cool desktop decoration. Is
> there any way to change it? How?

You may do this either using external utility xsetroot or CursorStyle
command, see the man page.

Run "FvwmForm FvwmForm-RootCursor" to experiment with different X cursors
and colors.

> Could I use a pixmap as mouse cursor?

You may use special 3 color pixmap as explained in the man page
(search for CursorStyle).
