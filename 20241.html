> > My question is: it is possible to start direcly netscape (or other applications)
> > preserving the settings done in .login and co.?
>
> Yes. In your .xsession file (or whatever file you use to start up
> fvwm2), source the .login file BEFORE calling fvwm2. That way, the
> environment variables are inherited by fvwm2, which will be passed
> along to whatever applications are started by fvwm2.
