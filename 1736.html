On 29 Nov 2000 03:41:00 -0500, David Meleedy wrote:
>
> Why doesn't fvwm2 have a themes page at this site anymore?

Because the first and then the second maintainers either lost an interest
some time ago or had no free time on this work anymore. Add also a luck of
new theme contributions. You probably know, themes.org is not a static web
site. Sections which are not updated for months have no much sense.

> Or better yet, since fvwm2 is embarking on gnome compliancy, is
> it possible to get it's themes page reinstated at this web site?

It is possible once there are some enthusiasts in the FVWM Themes area to
spend their time (probably on a daily basis) to maintain the site. If there
are volunteers, they are invited to write about their offer to the list
fvwm-themes-devel_at_users.sourceforge.net, where all details are discussed.

> I've been an fvwm2 user for a long time, and I really don't want
> to use any other window manager.

Very good. Still this does not quite help with fvwm.themes.org. :-)
