The address for asmail is http://www.rby.hk-r.se/~pt95pli/asmail.html
I also recomment the module Wharf of Afterstep. Wharf works as Goodstuff
but it has more features such as you can creat a button which will draw
out some other buttons if you press it and the buttons will draw back
after you click it again. Wharf can be used in fvwm like a module as
Goodstuff.
