        There are 2 possibilities I can think of.

        1) In xman, did you look for "fvwm" (all lower case) or "fvwm2" - all
the module stuff shows up early in the index, because the names all
start with "Fvwm", but fvwm is lower case, and therefore shows up
farther down.

        2) The xman config can be set up to show things differently from the
standard search order you would expect judging from the MANPATH. I doubt
that this is the case, but if it is you would see a file called mandesc
in one of more of the root directories of the various man hierarchies -
e.g., /usr/man, /usr/local/man, /var/man, etc.
