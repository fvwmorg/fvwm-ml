> the problem is, i'm trying to get the ATL-Graph key
> mapped to be a CONTROL-R key so that 'alt-graph plus
> arrow' can move across the virtual desktops but am
> having no fun using xkeycaps. does anyone out there
> have a successful .xmodmap that allows the alt-graph
> key to work as a control (and also the left 'diamond'

I don't see the need to change the alt-graph mapping.
On my Sun Type 4 keyboard, AltGraph is known as "Mode_switch"
with "mod2" assigned as the modifier.

To have fvwm react to AltGraph/Left use:

Key Left A 2 <action>

The left Alt key seems to want to use the Modifier "M".
