On Sat, 13 Dec 1997, Craig Hutchison wrote:
> Does FVWM or X11R6 have the ability to display a backgroup
> wallpaper. If so, how is it done? If not I would be interested in
> doing it. How difficult do you think it will be? Where in the source
> code would I start? I do ALOT of coding in UNIX using C, but I don't
> have much exposure to the X Windows coding environment.

Fvwm itsself doesn't do wall paper. The background of the root window can be
set by any program. One might use xsetroot to get solid color or stipple
patterns. For a more textured background xpmroot (which comes with fvwm) will
let you load in an xpm file as a background. If you want to load in a jpg or
gif or something, try xv. xv -root -quit foo.jpg will load the picture into
the root window, then have xv exit. xpmroot is faster than xv, but can only
load xpm files.
