> CleverPlacement is now cleverer. Now CleverPlacement tries more possible

I have installed CleverPlacement and it works great for me.
However, and I admit I am not sure if I want to bring this up
because I dont want to start an other discussion on the following
subject, but,

When there is plenty of room, lets say a clean workspace,
the first window is placed at the Top Left corner of the screen.
For me, I like to start with the windows at the Bottom Left corner
and work my way up as I place new windows.

But really, its not important. I see a great improvement as it is.
Especially when I use seyon, first one window always obscured the other
and now everything is accessible.
