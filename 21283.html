hi,

i have a problem that i have posted here some days ago (no replys :( ) :
i use 2 fvwmbuttons config-files and i change them with a fvwmbutton.
but i get allways a : "Major opcode of failed request: 4
(X_DestroyWindow)"
error msg :(
what can i do against this ??

i change using 2 functions:

AddToFunc fvwm2b_inet
+ "I" KillModule FvwmButtons
+ "I" Module FvwmButtons File /root/.fvwm2b_inet

AddToFunc fvwm2b_nrm
+ "I" KillModule FvwmButtons
+ "I" Module FvwmButtons File /root/.fvwm2b_nrm
