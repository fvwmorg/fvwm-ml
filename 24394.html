Hi, All.

I've been using fvwm2.2 for about a month now. Very nice. (Currently
running 2.2.2 on sun-solaris, w/o c++ modules.) I have one problem I just
don't understand, though.

In my .fvwm2rc, I have

Style "*" ClickToFocus

and

Key F9 A M Next [ CurrentScreen CurrentDesk ] Focus
Key F10 A M Prev [ CurrentScreen CurrentDesk ] Focus

So that I can use the keyboard to switch the window focus around the
screen. The problem is that every now and then it just stops working
right! Some windows at random become "unaccessable" -- I can't move the
focus to them with the keyboard, only by moving the mouse onto them and
clicking.

Once I've clicked on the unaccessable window, usually the other
group becomes unaccessable. It is like there's some barrier in the
window list preventing the Next & Prev commands from getting the whole
way through.

(Another detail: I give some of my sticky windows, like xbiff, the
attributes WindowListSkip and CirculateSkip, so that they never get focus
from my keyboard shortcuts. Sometimes it seems like this may be the source
of the problem, but I can't be sure.)

Any insights?

Thanks,
                        --- Marc.
