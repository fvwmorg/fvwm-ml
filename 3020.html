After installing X developement libs (or any other lib that you want to be
automatically found) you should run "ldconfig" as root.
There is a solution without running ldconfig by giving parameters to
configure, but you'd better just run "ldconfig".
I hope, you have /usr/X11R6/lib line in your /etc/ld.so.conf.
Then rerun "./configure; make".

> PS should I upgrade XFree86?

You don't need.
