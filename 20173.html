I order to get functions like the following to work, it seems I need
to include the CirculateHit in the Next command.

Since I can find no information in the man page on what application
CirculateHit would have in this position, I wondered if anyone could
tell me if it's legal here, what effect it might have on the Function,
and it's relation to the CirculateSkip and WindowListSkip style
options.


AddToFunc warp-or-run-cmd
+ "I" None [$0 CirculateHit] Exec exec $1 &
+ "I" Next [$0 CirculateHit] Iconify -1
+ "I" Next [$0 CirculateHit] Focus
