>> When I press Alt-Tab a fvwm window list appears
>> even though I have the following line in my .fvwm2rc file:
>>
>> Key Tab A M -
>>
>> How can I disable this key setting (FVWM version 2.4.3)?
    Dominik>
    Dominik> In principle, that's the correct command. If it doesn't work, a
    Dominik> different binding must be somewhere in the config files. Look for
    Dominik> bindings with the tab key in your config file and the
    Dominik> ConfigFvwmDefaults that comes with the distribution. You have to
    Dominik> negate all bindings by issuing the same command to "-" as the
    Dominik> action.
    Dominik>

it works. Looks as if that line above doesn't work properly.
