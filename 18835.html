Hi,

> I am using FVWM 2.0.43 on an ULTRA Spac running SUNOS 5.5, can anyone tell me how
> to make the Front key on my sun keyboard work within FVWM, that is, either raise
> or lower windows? I was previously using FVWM 1.24r and it worked fine. I am sure
> it is just a setting in my .fvwm2rc file that I have missed.

Within your .fvwm2rc file there is a key definition section (the lines
starting with Key...).
The correct key names for the addtional SUN functions keys on the left are:

F11 - Stop
F12 - Again
F13 - Props
...
F20 - Cut

(if theses don't work for you, try to figure out the correct X events by using
a program like 'xev').
So, in your case, you just have to add a line

Key F15 A N RaiseLower

to your .fvwm2rc
