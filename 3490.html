You could install AnotherLevel from the latest Red Hat distro that contained
it (RH6.2), but you might instead take a peek at

        http://www.cs.man.ac.uk/~jtl/ALU/

It's a much improved version called AnotherLevelUp, including built-in themes
and user defined preferences sets. It is being actively developed, whereas
AnotherLevel development has ceased.

In case it's not obvious, AnotherLevelUp is actually developed by me, so I
might be a bit biased when I say I think it's excellent! :-)
