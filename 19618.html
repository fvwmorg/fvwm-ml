Hello:

 I've built FvwmCDE on IRIX 5.3 with XForms-0.85 and it bombs on startup.
So a note for those people who want to use FvwmCDE on SGI boxen: use your
stock IDO CC, _not_ GCC. I don't know what's causing this, but the
executable built with GCC always SIGSEGV's mysteriously while parsing,
while that compiled with the IDO works fine.

 Thanks (see, I already found a bug :)
