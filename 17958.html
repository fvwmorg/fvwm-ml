Hi,

I had a quick think re: my gripe about pixmap borders and
highlight colours. Basically my problem is that I use the same
pixmap for active/inactive windows (I just rely on titlebar
colour to indicate focus). The highlight and ordinary
style "*" colour must be set in such a way that it doesn't look
terrible against the chosen pixmap. So I use the same colour for
highlight and inactive borders (the titlebar indicates focus
anyway).

The problem is that when scooting over my icons its not very
obvious which one has the focus as the colours don't change.
The functionality of a highcolour is still needed in this case.

I'd like a very different colour for highlighted icons while
retaining the same highlighted/ordinary colour for pixmaped
border windows. Might I suggest extending (and slightly
complicating, but only slightly) the pixmap borders patch to
include a config file line like "Icon highlight colour"
or something. Most fvwm users will be fine with this, but
for "pixmap border patch" users this would be useful.
I reckon this would be very simple to implement.

/end{rant}
