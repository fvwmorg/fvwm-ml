On Thu, Feb 15, 2001 at 12:05:14PM +0100, Andre Bonhote wrote:
> hi there!
>
> i am using windowmaker dock apps and other 'transparent' apps in
> my FvwmButtons. Now unfortunately those apps don't appear to be
> transparent, instead they're using the background color I
> defined for FvwmButtons.
>
> how can i define them to be REALLY transparent? is this possible?

It seems that the original author of the transparency patch in
FvwmButtons forgot to handle shaped application windows. I have
just committed a patch to resolve the problem. Transparency will
work properly with the next beta release (2.3.29) , current CVS
or the next snapshot.

> also, i would actually like FvwmPager to be transparent instead
> of grey. With my actual background image, it doesn't change much,
> but i am using other images with which it's really not very
> beautiful.

With the current beta version you can give the Pager a transparent
background by using a transparent colour set (look for colorset in
the man pages of fvwm2, FvwmButtons and the FvwmTheme module. This
does not work well, though. It's probably better to make a
snapshot of the background at the place where you put the pager
and use it as its background image.
