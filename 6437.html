I'm running fvwm 2.5.2 with nautilus-1.0.6-ximian.14. My .fvwm2rc
contains the following lines (relevant to focus):

ColormapFocus FollowsFocus
Style "*" SloppyFocus
Style "xload" NoTitle, Sticky, WindowListSkip, ClickToFocus

If I switch to a virtual desk or otherwise end up with no windows having
focus, my Nautilus windows do not get focused when the mouse enters.

If I click on an xload window to bring it into focus, the nautilus
window still doesn't get focus.

However, if I have any other window open, that window will get focused
with no problems, and after that the nautilus windows get focused as
well.

As far as I can tell, this behaviour only happens with nautilus windows.

Any ideas on how I can make nautilus get focused?

All this is on a RedHat-7.2 with Ximian desktop.
