Using 'Next [CurrentScreen] Focus' will select windows on another desk,
but on the same virtual page number. It seems to me that CurrentScreen
should only select windows in the current view port (ie screen). What
CurrentScreen does should probably be called CurrentPage.

Then you would have:

        CurrentScreen (only windows you can see)
        CurrentPage (only windows on the same virtual page number)
        CurrentDesk (only windows on the current virtual desktop)

What does everyone else think? Change CurrentScreen and add CurrentPage?
