On Fri, 5 Jun 1998, Pete Forman wrote:

> Neil Zanella writes:
> > The problem with the color names given as output to the showrgb
> > command is that unfortunately not every color has a name so the
> > list is incomplete in that sense.
>
> That is deliberate. The number of colors in use is limited, typically
> to 256. A shorter list of color names encourages sharing of colors
> between applications.

What is it that limits the number of colors to 256? I guess you are
referring to the video card. If I am correct you can configure some video
cards to allow for 56K colors (true color). This is somehow less much less
than 256^3 which is the total number of possibilities for RGB colors.

>
> BTW the #RGB style of color specification is deprecated. Numeric
> specification of colors should be done with something like rgb:.
> See the man page for XParseColor or a book on X11 such as the Xlib
> Programming Manual.
