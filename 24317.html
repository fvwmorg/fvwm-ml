On Thu, Jun 03, 1999 at 08:57:27AM -0400, NedungA_at_utrc.utc.com wrote:
> Hello,
> I have used FVWM in the past on a Solaris box, but would
> like to set it up on my SGI machine. I have an R10000
> Octane running IRIX 6.5. I would be very grateful if you
> could tell me exactly how to get FWM running. That is,
> do I have to have a .xsession file and what do I need to
> put in there etc. Also, any tips you might have on compiling
> on an SGI would be appreciated as well. Are there pre-compiled
> binaries out there somewhere? I thank you in
> advance for your help and look forward to hearing from you.

Simply put a command like this in your .xinitrc or .xsession
file (whichever your system uses):

  fvwm2 <options>

or

  exec fvwm2 <options>

or

  fvwm2 &
  ... (more commands)

Note that X will exit when the last application started from
this file exits.
