Hello Everyone,

I was wondering if anyone has or knows where to get more of the
background pixmaps that can be used with xpm. One of these is
mentioned in the man page for FvwmBacker,

*FvwmBackerDesk 3 xpmroot /usr/include/X11/pixmaps/background2.xpm

I found one called background8.xpm, but I cannot find any others.

Also, is it possible to include a command such as the one above
in a menu so that the user can select a background pattern for the
current desk, thus sending a command directly to the FvwmBacker
module, such as

*FvwmBackerDesk CurrentDesk xpmroot /usr/include/X11/pixmaps/background2.xpm


Has anyone implemented a background menu with perhaps submenus for
backgrounds of solid color, gif images, and background patterns?
If so, could it be mailed to the list?
