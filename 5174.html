> On 21 May 2002 00:02:02 +0200, Marcus Lundblad wrote:
> >
> > With sloppy focus some windows doesn't take focus (like xclock).
> > By turning on Style * Lenience this is overriden, but not for fvwm
> > tear-off menus.
> > I know most people prefer not to have windows that you don't type in
> > "steal" focus, but I prefer to have them change to focused so I can
> > easier see where my pointer is.
> > Is there a way to get all windows accept focus?

> The way to make a decoration look focussed is like you said:
>
> Style fvwm_menu SloppyFocus, Lenience
>

There is no way to make tear off menus behave like other
applications regarding focus. All I can do is hack something in
so that menus get a 'focused' decoration when the pointer enters
them.
