Well, I created my first module! I took the Xse program and adapted
it for use as an Fvwm module. Attached is the tar and gziped file
containing the module. I compiled it as though it was in the modules
subdirectory. Since I have done very little unix programming, I will
only garentee that it will compile on my system (A Sparc Startion 2
running Solaris 2.5.1 using the GNU gcc compiler). Try it on other
systems and see if it runs.

I do not have an ftp or WEB site that I can place this program
(company policies does not allow me to but this on our site).
Therefore, could someone make this available on the WEB?

I believe that this is a very useful function that would be nice to
include in the final distribution.
