i noticed the same problem with FvwmBacker. ie, using it as the
manual suggests does not work.

i had to call explicitly call xsetroot like this:

*FvwmBackerDesk 0 xsetroot -solid gray13
*FvwmBackerDesk 1 xsetroot -solid midnightblue
*FvwmBackerDesk 2 xsetroot -solid darkslategray
