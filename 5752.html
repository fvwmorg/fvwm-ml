Bart Raatgerink <b.raatgerink_at_emaxx.nl> writes:
> Hello,
>
> I'm looking for a step by step fvwm2 tutorial to start with an empty
> fvwm2rc file. Do you have any pointers ?

Start up Fvwm with no .fvwm2rc file.
Click on the root window.
Choose either "Setup Form" or "Setup 95 Script".
Either one will create a starting .fvwm2rc file complete
with comments.
