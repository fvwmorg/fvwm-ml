I did it and use it. (FVWM 2.5.8 and Mandrake 9.2)

(this is for having menu automaticaly)
- I have install fvwm fvwm2 fvwm2-icons, etc... from the contrib
- I have made a copy of the file in directory /etc/X11/fvwm2/ in order to
have a correct menu and add at the end of .fvwm2rc
Read /etc/X11/fvwm2/system.fvwm2rc-menu
So I have a correct menu.
- I have uninstall all fvwm* rpm except fvwm fvwm2-icons


- Some *-devel package are need for compiling.
- Now I compile fvwm
  ./configure --prefix=/usr --bindir=/usr/X11R6/bin
  make
  make install

The problem was to make the fvwm2 the default WM. It's why I have left
fvwm rpm and select as default fvwm. Its, not a pb because fvwm is a link
to fvwm2.

Another pbs is *-devel rpm need to compile
WM binary file are in /usr/X11R6/bin directory

>Or any suggestion or pointers.
