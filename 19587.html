  Anyone have a recipe for starting fvwm instead of VUE? Our HP's seem
  to ignore my .xsession file.

Unofficially speaking, I managed to change the /usr/vue/config/Xconfig file
on HPUX 10.10. I changed the line about Vuelogin*xdmMode. These lines now
look like:

############################################################################
# To set a display to run the user's XDM $HOME/.xsession script as the
# default session. Ensure that the $HOME/.xsession script is executable.
#
# Vuelogin*xdmMode: True
Vuelogin*xdmMode: True

############################################################################
