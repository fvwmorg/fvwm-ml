ActiveColorset and GreyedColorset only affect menu items, not the whole
menu. MenuColorset does affect the whole menu, but if I change the
MenuColorset for a menu's style while the menu is open, the menu does not
visibly change. Similarly nothing happens if I change which menu style is
applied to the open menu.
