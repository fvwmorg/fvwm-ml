I noted the "Key" command became case sensitive when I moved from 2.4.0 to
2.4.2

I had: Key R A M Popup RootMenu

This stopped working
Instead: Key r A M Popup RootMenu
made it work again as expected.
This could possibly be a problem for some users (I'm thinking about point
0 in the FAQ: "My keyboard/mouse stopped working")
Just a thought.
