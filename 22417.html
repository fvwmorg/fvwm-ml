Previously...
>I am using the Maximize/Maximize vertical functions in fvwm but when
>I type them it seems like my windiws get almost maximized vertically but
>not quite. Usually I use text based Xterms.
>
>Is this some sort of bug in fvwm?

Probably not. I see the same thing and never regarded it as a bug.
Anyway, since xterm uses fixed-width fonts, its geometry
characteristics aren't done by the pixel, but rather by character
columns and rows. Doing a maximize 0 100 (for example) on an xterm
will make it as tall as it can be without growing beyond the desktop
size. This usually leaves a spare pixel or ten.
