Hi Mikhael,

: > The Ctrl-Alt-number key-bindings are the fastest way to switch
: > b/w tabs. ie. Ctrl-Alt-0 will select/display the first tab.
: I do not currently use FvwmTabs, but I would dislike such bindings.
: Our keyboard has "1"/"F1" as the first number.

All the key-bindings for FvwmTabs are configurable with the standard
FVWM "Key" command.

: I would use 1 2 3 4 5... to identify tabs. And 0 means switch to the
: last tab used (then 0 may be used to switch between any two tabs).

An option to display the last tab is a good idea. I will add it soon.

: FYI, it may be a bit problematic for someone with fvwm-themes to use
: exactly these bindings, since Ctrl-Alt-1 to Ctrl-Alt-4 are used there to
: switch between desks, and Ctrl-Alt-0 to switch to the previously used
: desk. There is a similar idea for switching between pages using keypad.

Thanks, I wasn't aware of this. The key-bindings are configurable &
can be easily changed. The fvwmtabrc file on the website is actually
the one I use (ie. it's my settings). Do you think I should provide
a modified version of the fvwmtabrc file that has bindings less
likely to clash with other FVWM users?

Approximately what percentage of FVWM users use fvwm-themes anyway?
(It's infinitely larger than the percentage of FvwmTabs users I'm sure! :)
