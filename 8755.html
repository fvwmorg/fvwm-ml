> * Bowman (well, this is the AfterStep predecessor)
> * AmiWM (unlikely, the sources have no any indication)
> * Lesstif-MWM (it is true, but this WM was abandoned on Feb 1997)

of enlightenment releases, rasterman decided to start from scratch. So
yeah, the first versions were fvwm based, but none after that.
