Yes, the feature is not so difficult to implement, but I wanted to
clean up the unholy mess in the icon hints code. It took me two
days before I gave up, unable to single out the icon hints from
the ewmh/gnome code :-/ Certainly, you remember my frustrated
mail back then ("hint code clean up" or something like this).
It's still one of my top priorities - not because the code were
badly written (it isn't), but because it has become so complex and
relies on so many implicit dependencies that it has well become
unmaintainable.
