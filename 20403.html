>
> FvwmAuto seems to make all the windows autoraise.
> Is there anyway to make only a selected list of windows autoraise?
> (Something like: Style FvwmPager AutoRaise)

Not with Fvwm-2.0.45 or lower. FvwmAuto can invoke any function on a window
getting the focus and if you combine this with the "Current" patch you can get
what you want. The "Current" patch is in the mailing list archives. Search for
"current.patch" on the Web site below. If you don't have Web access email me
and I'll post it to you. If you can't compile Fvwm I am pretty sure that the
patch will be in 2.0.46.

AddToFunc InitFunction FvwmAuto 500 my_autoraise

AddToFunc my_autoraise "I" Current [FvwmPager] Raise
+ "I" Current [BlahBlah] Raise
etc. etc.

Cheers,
Tim.
