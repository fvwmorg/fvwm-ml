misha21_at_ihgp.ih.att.com:
> Also while looking thru the archives, I noticed a reference to an olvwm
> feature that will create a menu from a directory of executables (e.g.
> $HOME/bin). Is there any plan for this feature.

I have a setup that does this, although not automatically. It can't
be done completely automatically, since some programs must be run inside
an xterm, other need special arguments, and so on. In practice, most
programs need special handling. I use a configuration file to create the
menus, with one line per menu entry. This allows me to add any suitable
options for each program, and also to put in a mini-icon before the menu
entry. A bit like Win95, actually. :-)

My setup should still be in
http://www.iki.fi/liw/misc/fvwm2config-0.1.tar.gz and
http://www.iki.fi/liw/misc/fvwm2debian-0.1.tar.gz . It's been working
splendidly for me.

As soon as I have the time, I will start working on getting this into the
Debian Linux distribution (and possibly the master fvwm distribution, but
I'm not sure Chuck wants it there).
