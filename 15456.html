> Is there an easy way to set up keys to take me directly to a mapped
> desktop without having to be in the root window (or the virtual
> desktop). I used to have my 9-key numeric keypad mapped to my 3x3
> virtual desktop using tvtwm (running Solaris 2.5), and it was nice
> not having to move my pointer off my active windows before using
> navigating with the keypad (like with olvwm). Any suggestions
> are greatly appreciated!


I map my numeric keypad to a 3x3 virtual desktop like this:

Key KP_1 A A GotoPage 0 2
Key KP_2 A A GotoPage 1 2
Key KP_3 A A GotoPage 2 2
Key KP_4 A A GotoPage 0 1
Key KP_5 A A GotoPage 1 1
Key KP_6 A A GotoPage 2 1
Key KP_7 A A GotoPage 0 0
Key KP_8 A A GotoPage 1 0
Key KP_9 A A GotoPage 2 0

Curt
