On 12 Apr 2003 14:14:47 -0400, Ayan Mahalanobis wrote:
>
> Great it works.
> The switchdesk only knows fvwm, I have fvwm2 and want to start from
> fvwm-themes-start.
> How do I teach switch desk to do that?

The only thing switchdesk does is creates a file ~/.Xclients that
contains one line, it does nothing else (maybe backups the old one).

Just create this file yourself:

  % echo exec fvwm >~/.Xclients
  % chmod a+x ~/.Xclients

Or:

  % echo exec fvwm-themes-start >~/.Xclients

> Another thing, how can I play with the clock?
> In good old days it was in the .fvwm2rc :) where it asks some thing to
> swallow the clock. Where is it said these days??

Decide whether you use .fvwm2rc (start fvwm), or learn how fvwm-themes
works. There is a documentation and mailing lists for fvwm-themes.
