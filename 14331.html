Umm... We fvwm-users-in-japan mailing list have already developed
these feature (fvwm internationalization) and released (with documents
including japanese-translated-manual and supplementals) as
fvwm-users-jp-kits.

(Our approach is just like your approach, i.e., using X11's I18N feature.)

If you have interested in, you can ftp to get. Be aware that all
documentation is written in JAPANESE. The latest release, named
fvwm-users-jp-kits.95Sep09.tar.gz, is available from followings:

<URL:ftp://ftp.sra.co.jp/pub/x11/fvwm/jp/fvwm-users-jp-kits.95Sep09.tar.gz>
