This is really odd, I wonder if anyone can shed any light on the subject.
Since I use vim on an aterm a lot, I decided that it would be a good idea
to create a keyboard shortcut for it. I did it like so:

Key v A SCM Exec exec aterm -name vim -e vim

added to my fvwm2rc file (well, actually, to a separate file that holds all
my keybindings, and that is read by fvwm by the Read command). All looks
good, except when I decide to save and quit (via :wq). Then fvwm crashes.
