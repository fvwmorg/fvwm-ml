> How do I get rid of a window's border. I have rxvt set with a transparent
> background and no title bar, but I can't get rid of it's border. I'm using:
>
> # Get rid of the default border pixmap
> DestroyDecor recreate RXVT
> AddToDecor RXVT
> + BorderStyle Active TiledPixmap none
> + BorderStyle Inactive TiledPixmap none
>
> Style "rxvt" UseDecor RXVT
> Style "rxvt" NoTitle, sticky, BorderWidth 0, ParentalRelativity
>
> "BorderWidth" doesn't seem to do anything. I've tried it at many settings but
> the border remains unchanged.
>
> Using 2.4.2
>

When I don't want an window border I set both this variable:

BorderWidth 0 and HandleWidth 0
