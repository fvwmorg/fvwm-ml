I've been trying to build fvwm on my Solaris 2.4 system.

I believe that the README.Install file ought to at least mention the
fact that people who are trying to build this on Solaris will need to
also fetch, build, and install the xpm library (and associated header
files) from ftp.x.org. (Neither the header file(s) nor the library
for xpm are present by default on Solaris systems.)
