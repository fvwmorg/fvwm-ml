Mark Hewitt commented that tkgoodstuff does all that I asked for, and he's
quite right. Don't know why I haven't been using it all along, but the 8.0
version seems to work fine. It's a bit offputting that it seems to be
unsupported at present (the programmer having given up development), but
presumably someone will take over to keep it working with subsequent Tcl/Tk
releases.

then the libraries will be shared, so it's not so bad.
