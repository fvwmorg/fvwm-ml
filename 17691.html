> I am not sure I can be of much help. All I know is that fvwm picks up my
> resources from ~/.fvwm (or ~/.fvwm2rc if using version 2). If this file
> does not exist it will look for system defaults. These could be anywhere.
> Good places to look are /usr/lib/X11/fvwm/.fvwm2rc or
> /usr/lib/X11/fvwm/system.fvwmrc (ours happens to be in
> /opt/X11/lib/X11/fvwm/system.fvwmrc).
