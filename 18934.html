the reason I am asking is, that I have found the following setup to be very
conveniant ...

[...]
*FvwmButtons(2x13, Frame 0, Padding 0 0,\
        Swallow(NoHints) FvwmIconBox FvwmIconBox)

[...]
# FvwmIconBox
*FvwmIconBoxPlacement left top
*FvwmIconBoxHilightFocusWin
*FvwmIconBoxBack gray74
*FvwmIconBoxIconBack LightSteelBlue3
*FvwmIconBoxIconFore Black
*FvwmIconBoxIconHiBack PaleVioletRed3
*FvwmIconBoxIconHiFore Black
*FvwmIconBoxHideSC horizontal
*FvwmIconBoxMaxIconSize 69x0
*FvwmIconBoxSetWMIconSize 48x48
*FvwmIconBoxLines 1
*FvwmIconBoxPadding 1
*FvwmIconBoxSBWidth 8
*FvwmIconBox "Fvwm*" -
*FvwmIconBoxFont 10x9
*FvwmIconBoxMouse 1 Click RaiseLower
*FvwmIconBoxMouse 1 DoubleClick Iconify
*FvwmIconBoxFrameWidth 0 0
*FvwmIconBoxSortIcons ResName
