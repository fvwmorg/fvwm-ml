Thomas Adam <thomas_adam16_at_yahoo.com> writes:
> Dear List,
>
> My question is this: While I am using fvwm2 I often
> get information appearing in that window. Is it
> possible to get fvwm2 to de-iconify that window (thus
> restoring it) when information is piped into this
> window???

The best you can do with xconsole is start it with "-notify"
on the command line. The icon name changes with this option.

There is a program called "contool" that does what you want
and a lot more.
