On 26 Sep 2004 17:33:36 +0200, Roeland Moors wrote:
>
> I started using gnome with fvwm. It works nice, but there is one
> small problem.
>
> The function InitFunction in my .fvwm isn't started anymore.
> It looks like a conflict with the gnome-session.
> The splash screen from the gnome-session doesn't dissapear
> automaticly.
> The function StartFunction works just fine.

Please search for "InitFunction" in fvwm man page. This section explains
all special functions and their relationship to a session manager.

If you run gnome-session, then you should start the programs using it. I
don't use gnome-session, but I think if you "Save Session" then the next
time you start gnome-session it runs the same programs, including fvwm.
