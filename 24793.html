2.3.7 is a particularly unfortunate version to start using panels.
I have rewritten the panel code from scratch between 2.3.8, so it
would be best if you don't try to get it running with 2.3.7. Fetch
a copy of 2.3.8 as soon as it appears on our ftp server (may be
already there). I wrote some instructions on how to set up
panels with the new code in the new FvwmButtons man page
('CREATING SUBPANELS' section). Don't hesitate to ask is that
leaves open questions.
