> How do I find out how to config FVWM?

You have 2 main choices, read the fvwm man page and
edit your config file with an editor, or use fvwm-themes.

You can find fvwm-themes here:

http://fvwm-themes.sourceforge.net/

> Is the
> rc file the only config file?

By default, fvwm looks for the config file:

$HOME/.fvwm/.fvwm2rc

You can override that from the command line.
Any config file can read other config files, so your config
file can be organized into as many parts as you like.
