I've placed my fvwm2 WindowShade function patch on my
BorderStyle/UseDecor patch page at:

http://www.rpi.edu/~veliaa/projects/fvwm_patches/fvwm_border_picture.html

There are two versions, one for plain vanilla fvwm2 and one for use
with the BorderStyle/UseDecor patch (they are exactly the same, just
patched to different fvwm2 trees).

There are some minor but important changes from the one's I've posted
before:

        - the child window is no longer resized

        Before, many of my client windows didn't seem to know how
        to handle being resized so small (Netscape, rxvt, etc.), so
        now this problem should be completely alleviated since the
        child window is no longer resized.

        - It should be a bit faster

        - It fully compiles out

Enjoy,
Andrew
