I added a new option MinimalLayer that fully controls the layer FvwmIdent
is placed on.

The default is 4, i.e. FvwmIdent jumps to the layer of the target window
or to the layer 4 whichever is greater.

If the value is 0 it works as it worked in 2.4.x, always jump.

If the value is "none" it works as it worked in 2.2.x, never jump.
