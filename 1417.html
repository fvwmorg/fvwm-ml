Hi all. While testing procmail I lost the messages I was keeping from
previous posts, so thanks to those who helped me start off down the
happy road to mouse-less operation!

Here's what I've been using so far with good success. Any comments &
suggestions are more than welcome.

thks again. 

--
Key Tab     A       M       Next (CurrentPage) NextFocusAndRaise
    AddToFunc NextFocusAndRaise
    + I Next (CurrentPage) Focus
    + I Raise
Key Tab     A       C       Prev (CurrentPage) PrevFocusAndRaise
    AddToFunc PrevFocusAndRaise
    + I Prev (CurrentPage) Focus
    + I Raise
Key Left    A       SCM     Desk -1 0 2         # next desktop to the left
Key Right   A       SCM     Desk 1 0 2          # next desktop to the right
# The Maximize functions take in to account the space _at_ the bottom of
# the screen taken up by things like the pager, etc.
Key F1      A       M       Maximize 100 88     # maximize
Key F2      A       M       Maximize 0 88       # Maximize vertically
Key F3      A       M       Maximize 100 0      # Maximize horizontally
Key F1      A       C       Menu RootMenu
Key F2      A       C       Menu Window-Ops
Key R       A       CM      Restart fvwm2
Key X       A       C       Exec xterm
