The window manager used with gnome in Redhat 6.0 is changeable in the Gnome
Control Centre under the descriptive name "window managers".

simply add fvwm, test it works with try and then click ok. make sure you save
settings before you log out.

on this subject can the developers give some idea when fvwm will be compatible
with both KDE and Gnome. Having looked at Enlightenment I'm quite impressed.
sure it's all just eye candy and a big resource hog but its configuration
program is amazing.
I think it should be relatively easy to change this program's parsing to
configure Fvwm. When I've got time I'll have a look at it, but since I'm very
busy don't rely on me.
