The clock is updated each minute : I force the the 'select' in
'EndlessLopp' to give up :

      tv.tv_sec=60;
      tv.tv_usec=0;
      if ((ret = select(fd_width,&readset,NULL,NULL,&tv)) <= 0)
        { DrawGoodies(); goto hack; }
