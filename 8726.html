> In my fvwm I run a sticky gkrellm to monitor what's going on.
> I would like new windows to not overlap this one application.
> I guess it's done with a Style command in .fvwm2rc but I couldn't
> find it. Any tips?

If your version supports Layer style (at least 2.5.[56] do), you
could supply the maximum layer number (see fvwm(1) man page) as
a style for your application. Otherwise, search for "StayOnTop" in
the man page.
