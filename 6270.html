I am new and really happy user of fvwm. I found some tips for
improvements:

1,FvwmPager would benefit from having separate colorsets for desk labels.

2,FvwmIconBox has troubles when combinig

*PanelIconBox: Resolution Desk
*PanelIconBox: Placement Left Bottom

3, I already asked about if it is possible to bind action to transparent
parts of icons, but now I submit it as a recomended feature. Sometimes
transparent parts of icons are really nasty thing. Mwm solved this by
creating those miniature "windows" around each icon. Fvwm is said to be
mwm-mimickable, so this useful feature would be really nice.
