On 1 Jun 1998, Paul D. Smith wrote:

> 2) New core feature for SmartPlacement: have SmartPlacement try extra
> hard to avoid overlapping windows tagged StaysOnTop.

CleverPlacement can already do that, although currently the parameters
that determine how hard it avoids certain types of windows is hard coded.
Once the style flags are fixed, this can be made configurable. To enable
CleverPlacement, use the following line in your rc file:

GlobalOpts SmartPlacementIsReallySmart
