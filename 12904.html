Hi,

Would it be possible to add a style option to disable stippling
entirely? There's already an option to enable it globally for all
windows (StippledTitle), but for Sticky windows, the stippling seems to
be hard-coded. Every time I upgrade FVWM I have to go and hand-remove
the lines from the source which draw the stippling. (In icons.c and
borders.c, specifically.) Stippling is really hideous, at least with
the titlebar styles I tend to use. Just a suggestion. (If I've somehow
managed to completely overlook an already-existing option to disable
stippling for Sticky windows for the last few months, someone feel free
to shoot me.)
