fvwm-2.2 versions require also c++ (or g++, cxx, cc++) compiler.
It is possible that you have it, but it is not installed propertly.

If this is the case, specify ./configure --without-cxx .
You will not have one minor module, written in C++.

fvwm-2.3 versions do not use C++.
