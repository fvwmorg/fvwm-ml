I like my FvwmButtons. There is a screenshot at

http://www.physik.fu-berlin.de/~dhansen/fvwm.png

It has several swallowed apps (xosview, xbuffy, dclock
and two window manager applets) and swallowed Fvwm
modules.

The "erotic, violence and cute" wallpaper is from

http://momonekodou.cool.ne.jp/mad-lovers/

The fvwm.rc as well as some mini icons and the window
decorations is in

http://www.physik.fu-berlin.de/~dhansen/fvwm.tgz

You can get a lot more very nice window decoration
From http://fvwm-themes.sf.net

But be warned, the configuration file isn't well
commented and may need some minor fixes. You will
need fvwm > 2.5 for it (get it from cvs, don't worry
about stability, it's working very well).

Regards,

David
