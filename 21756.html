Jin Ko wrote:
> How can I set certain windows "not resizable"?
> i.e. I don't wanna allow some sticky windows -- like xclock, xload
> ... - to be resized.

You can remove the titlebar, and/or the handlebars which provide the obvious
ways of resizing them (and reduce screen estate wastage). You'd still be able
to resize by going to the menus, though. (Assuming a standardish
configuration; you could remove resizing from all menus except those
accessible from windows which you want to be able to resize, I suppose.)

What are you trying to achieve? i.e., how much work is it worth?
