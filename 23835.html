My setup is:
  Linux RedHat 5.0 for intel -- 2.0.31 kernel
  XFree86 3.3
  fvwm 2.0.46
  vim 5.3 running on xterms
------------------------------------------------------------------------



The plan is that i would like to use keys F20-F29 in fvwm to do mappings
like the following:
        Key F20 A A Popup "Utilities"
        Key F21 A A WindowList

To achieve that, I want to use the windoz keys in my keyboard as modifiers.
(i have already placed them as osf keys in my xmodmap keycode table)

Thus, I would like to produce keys F20-F29 by typing (for example)
    osfSelect + a
    osfSelect + b
        ...

The modifier map follows:

        xmodmap: up to 2 keys per modifier, (keycodes in parentheses):

        shift Shift_L (0x32), Shift_R (0x3e)
        lock Caps_Lock (0x42)
        control Control_L (0x25), Control_R (0x6d)
        mod1 Alt_L (0x40), Alt_R (0x71)
        mod2 Num_Lock (0x4d)
        mod3
        mod4
        mod5 Scroll_Lock (0x4e)

To be honest, in spite of reading the man page and a couple of HOW TO's,
i still don't have a very good grasp of how the modifiers work.

The only thing that i know is that Alt-<char> currently gives the
corresponding char with the high bit set. And since i use those Meta's
in my editor (vim), i would not want to change how that works.

[i say that because I see that Alt is my modifier 1 up above]

any help and enlightment will be appreciated...
