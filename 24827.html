The next official release of FVWM, 2.4, will contain native support for
GNOME (i.e., be a "GNOME-compliant window manager").

We are well on our way into development on this; a number of beta tests
have been released. If you're interested in playing with beta code, you
can find it on the ftp.gnu.org site. Note this is _really_ beta code,
not "kinda beta but really released" code like the 2.0.x stuff.

There are also daily source snapshots available. And, of course, for
those who just gotta be current, you can use anon-CVS to get
up-to-the-second code.

There's a lot of new, cool stuff here, but it's still beta. If you use
it please read the info on reporting bugs carefully. You may also
consider tracking the fvwm-workers mailing list more closely.
