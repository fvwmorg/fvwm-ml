You need a pixmap editor that can process xpm files.

XPaint is OK for doing icons & stuff like that. It can usually be
found on most Linux Distributions. My local slackware mirror has it
at

ftp://sunsite.doc.ic.ac.uk/packages/Linux/
  slackware-mirror/slakware/xap2/xpaint.tgz

It doesn't do transparent colour: (and arbitrarily assigns a colour to
the colour `None' in an xpm file). It is not too difficult to work
around this; just annoying if you want to make up umpteen icons with
transparent backgrounds in a hurry.

Of course, there is always Xemacs... talk about using a 50 kilonewton
steam hammer to crack a peanut...
