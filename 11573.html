>
> I only ask because I think I lack the understanding to do it (atm).

If you update WindowStyle MiniIcon, then this is theoretically possible
(the parent subject modified Icon). However this also affects the
MiniIcon titlebar button if you have one (you definitelly want to use
dimentions larger than 16x16, such large icon is useless for titlebar).
This also requires a lot of CPU, because unlike iconified windows, you
want to capture the current image constantly, not just on iconification.
There is no good solution to solve these 2 problems. Other than this, it
may work.
