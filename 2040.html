Re-read Dominik's comment, pay special attention to the phrase "one of
the releases that had the responsible code screwed up".

He's trying to tell you that this code was recently fixed.

> also i checked ./configure file it has './configure
> --disable-shape' option in there, does this has
> something to do with it ?

Yes, if you disable the shape extension, there is a chance that
the icon will have a solid background. You would have to reconfigure
2.3.25 and rebuild it. That may or may not work. I don't remember
anyone doing this on purpose.

I still think the best solution would be to use a more current
beta.
