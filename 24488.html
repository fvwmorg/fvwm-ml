Type "find / -name SysUtil.h" to find out where this file is located
because the build process can't locate it. If you have a lot of mounted
directories you might try finding in the root subdirectories such as
/usr or /opt.

Let us know where it is then we can perhaps tell you what went wrong in
your configure stage. I hope you have already mentioned to the list the
system your building on (uname -a) and the version of fvwm you are
building.
