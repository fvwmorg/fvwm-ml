I can't say anything on the subject (PNG on Solaris), just some notes.

On 22 Sep 2003 18:23:56 -0700, Ben Sferrazza wrote:
>
> I built the snapshot from June 7, but not only do PNGs not display
> still, but I'm having other problems.
>
> What's a good, stable snapshot on Solaris that would include any updates
> related to PNG?

The latest cvs is always thought to be prefereable, at least in the
feature freeze period.

> *FvwmConsole version 2.5.7
> [FVWM][GetIconFromFile]: <<ERROR>> Failed to load /path/to/my/pngfile.png

It should say 2.5.8 (from cvs) if it is really the snapshot from June 7.

Can you please build the latest cvs, so we don't speak about past
versions, and install it using "make install". Verify that fvwm-config
--info lists png support and you run the correct executable.

Actually, you don't even need to install anything to verify the png
support in FVWM. After "make", just run bin/fvwm-root with your png file
to see whether it works, it should set the root background.
