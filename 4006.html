I did a similar thing to allow title bar buttons to be drawn 'sunken' based on
the layer the window is in (I have title bar buttons set up to put the window
in layer 2, 4, or 6, as a kind of "stay on top", "stay on bottom" or "float
normally" facility).

I posted my patches (for 2.4.3 I think) to fvwm-workers last December : see

    http://www.hpc.uh.edu/fvwm/archive/0112/msg00093.html

but never got any feedback on them :-(

My patch basically just extended the set of 'MWM hints' to include a layer
number option.
