}

I ended up using xmodmap to remap the keypad so it always works as a
keypad, disabling numlock forever.

! .Xmod-pad
! xmodmap this file to make keypad always work
!
! xev shows keycodes
! key symbols are in /usr/X11R6/lib/X11/XKeysymDB
!
! fix the keypad so the NumLock doesn't do anything and the keypad always works
clear Mod2
! Mod2 is NumLock
! Now make keypad keys always do numbers
keycode 87 = KP_1
keycode 88 = KP_2
keycode 89 = KP_3
keycode 83 = KP_4
keycode 84 = KP_5
keycode 85 = KP_6
keycode 79 = KP_7
keycode 80 = KP_8
keycode 81 = KP_9
keycode 90 = KP_0
keycode 91 = KP_Decimal

remove Lock = Num_Lock
keysym Num_Lock = Escape
! make the NumLock do something useful
