Fvwm will and does correctly use the PPosition hint if you set
it in the WM_HINTS structure. If you specify NoPPosition, then it won't
use that hint. It will always use USPosition hints. If it works for some
window managers, but not others, then maybe you're not setting the hint
before you map the window, or maybe you're getting it a little mixed
up, and fvwm just doesn't quite parse the hints correctly
