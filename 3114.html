Marcus, I don't doubt that fvwm2 is reading your .fvwm2rc file, but I'd be
willing to guess that the problem is not there. If you put a simple
script like

#!/bin/bash
exec fvwm2 > ~/.Fvwm2-errors 2>&1

in a file called .xsession in your home directory (with execute
permission) and log in through gdm without selecting any particular window
manager, then the .xsession script will be run, etc.

Do you still have the same problem when loging out?

Tom
