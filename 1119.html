On Wed, Jul 12, 2000 at 09:53:07AM +0200, Dude wrote:
> hi,
> how can i remove all buttons except delete ?

Edit your configuration file and find all lines that look
like this:

  Mouse <digit> ...
  Key <digit> ...

The <digit> is the number of the button (1, 3, 5, 7 and 9 for
the buttons on the left side and 2, 4, 6, 8 and 0 for buttons
on the right side, counting from outside to inside). Delete
or comment all the lines for the buttons you do not want to get.

In fvwm, a button appears as soon as a function is bound to it
with the commands 'Key' or 'Mouse'. To disable it you have to
remove all bindings on this button. An alternative way is to
specify the 'NoButton' style for each unwanted button. However,
this is less clean and efficient.

  Style * Nobutton 3, NoButton 5, ... (and so on)

(See man page for details)
