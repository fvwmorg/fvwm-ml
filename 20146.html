On 5 Jun, Mark Krischer wrote:
> i've been checking out netscape communicator and it looks like they're
> combining the mail and news reader. what this means for me is that it
> will no longer work for me to use the mailer icon as my indicator that i
> have new mail.
>
> so what i'm looking for is a version of an xbiff type program that can
> do POP, will ideally let me choose the icons, and works well with
> FvwmButtons (everybody's always talking about it, i figure i might as
> well give it a try).

I use tkRat, which requires tcl/tk. I don't use it with FvwmButtons,
but if you iconize the application, it can alert you when new mail
arrives. It does POP3 and IMAP4 as well as other stuff.

It's home page is: http://www.dtek.chalmers.se/~maf/ratatosk/
