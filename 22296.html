status report:
-------------
Good news, I passed my prelims yesterday.
Mixed news, I'm flying to Des Moines to visit my dying grandma (sad part) and
eat lots of corn-fed prime rib (happy part).

So, I haven't gotten much done on 2.0.47 except look it over, and won't until
at least next week.

volunteer request:
-----------------

I'd like two things:

1. A list of people who want to alpha test. Along with a name I want to know:
  a. architecture/OS used
  b. modules used (me: FvwmM4, FvwmButtons, FvwmIconMan, FvwmIdent)
  c. "extensions" used (things with defines in Fvwm.tmpl) (me: shape,
     miniicons, pixmap buttons, titlestyle)

People on this list will get to alpha test the new release (probably future
releases too) and will be expected to give a complete testing/report (reminder
email sent after a week). Sounds like fun, eh?

2. A repository of fvwmrc's on the website (along with all icons, scripts,
   etc. used and a list of any environment variables and options in Fvwm.tmpl
   which need to be set so that people can just run them without going mad)
   organized by features. So if someone wants to use some fvwmrc's using
   gradients, they can find them. It would probably also be cool if people
   submitted "truncated" fvwmrc's which only had the configuration for some
   module, for example, all of their *FvwmButtons lines long with any Style
   commands which modify how FvwmButtons looks. I would like to have a
   volunteer to get this running too. This volunteer (unlike those poor souls
   for #1) would probably get lots of happy feedback.
