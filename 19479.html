On Fri, 7 Mar 1997, Ives Aerts wrote:

> What do you people use to edit xpm's (icons, textures, ...)? Currently
> I use gimp but doesn't know how to save them, so I have to save them
> as tiff's, then use XV to convert them, then use a text editor to
> change one of the colours to transparent.
>
> Surely there must be an easier option, not?

I use pixmap which is available as source code. It's neat; it makes it
easy for you to center pixmaps, etc, and even to make them transparent --
just select the transparent color and floodfill.

Pixmap works just like bitmap (which you should have) and can use either
Athena widgets, or, since you're using an SGI, the Motif widgets, which
makes it look way cool.. :)

The only thing missing from pixmap (although there might be a command, I'm
no expert) is some cropping function and how to resize pixmaps.
