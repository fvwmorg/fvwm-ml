Well, first of all we need to know the actual release you are using
(2.2? 2.2.2? 2.2.4?).

> I whould like to change the behaviour for dragging a window around.
> At the momentI click, move the window and another click releses the
> window.

This sounds as if you were using 2.0.46 or even an older release.
Since 2.2, pressing a mouse button while you move a window cancels
the motion. So I can't believe you have 2.2.x. If so, please
download the latest stable release (2.2.4) from our web site
(http://fvwm.org).

> But I would like to have the old behaviour. Pressing the mouse button,
> moving the window and releasing the mouse button also releases the
> window.

If upgrading doesn't help or if you already have 2.2.2 or higher,
please send me your config file so I can see what's going on.
