On Fri, Sep 17, 1999 at 09:19:32AM -0500, Matthew W. Roberts wrote:
> I just saw tkdesk for the first time and was pretty impressed. One
> thing I was wondering: is there a way to "hide" FvwmButtons like
> you can in tkdesk. If you haven't seen it, there is a small button at
> the top of the tkdesk "button bar" that you press and the whole
> thing (except the small button) disappears. Press the button
> again and it comes back.
>
> Maybe the real question is, do a lot of people run tkdesk with
> fvwm, or is there a way to use fvwm modules to do the same thing?

How aboout this (with FvwmButtons):

 1) Draw a pixmap you want to use for this button. Lets call
    it iconify_buttons.xpm.

 2) Tell fvwm to use the pixmap as FvwmButtons's icon:

      style FvwmButtons Icon iconify_buttons.xpm

 3) Tell fvwm where to place the icon (modify the coordinates
    to suit your setup):

      style FvwmButtons IconBox +0 +0 +31 +31

 4) Set up FvwmButtons:

      *FvwmButtons(1x1+0+0, Icon iconify_buttons.xpm, \
                   Action Next (FvwmButtons) Iconify)

This won't work very well if you move your button bar and
requires some work to find the right place for the icon, but
it should work.
