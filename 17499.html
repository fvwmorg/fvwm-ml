Your problem is probably the video card (i.e. the chipset). Since there is no
accelerated X-server for the trident chipset, programs are going to run
inherently slow, as far as drawing windows is concerned. Netscape is slow
anyway, though. I believe there is an accelerated server available, but it is
commercial.
