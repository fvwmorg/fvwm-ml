Here's a sorta trivial question, but one I can't seem to get to work.

I have two desks, each 2x2. I want to be able to move to a specific page
on a specific desktop using the function keys and shift. So if I hit F1
it'd put on on page one of desk one, F2 page two of desk one, F3 page
one of desk two, etc. Here's what I've tried:

Key F1 A S Desk 0 0,GotoPage 0 0
Key F2 A S Desk 0 0,GotoPage 1 0
Key F3 A S Desk 0 1,GotoPage 0 0
Key F4 A S Desk 0 1,GotoPage 1 0
Key F5 A S Desk 0 0,GotoPage 0 1
Key F6 A S Desk 0 0,GotoPage 1 1
Key F7 A S Desk 0 1,GotoPage 0 1
Key F8 A S Desk 0 1,GotoPage 1 1

While the Desk command works, I move to the proper desk, the
GotoPage part doesn't get acted on. What's the proper way to bind
multiple commands to a function key?
