I posted a CursorStyle patch a while back. But it doesnt seem to have made it
into patch level 19. So at the risk of repeating myself, here's another copy.
Although this one fixes the stupid mistakes I made in the first one...

BTW: Whats the best way to submit patches for fvwm? And to whom?

The idea is to allow you to change the cursors used by fvwm. For example I
prefer the more Mwm like:

CursorStyle SYS XC_left_ptr
CursorStyle Destroy XC_left_ptr
CursorStyle Title_Cursor XC_left_ptr
CursorStyle Menu XC_arrow

Possible things to change the cursor for are:

POSITION DEFAULT SYS TITLE_CURSOR MOVE MENU WAIT SELECT DESTROY LEFT RIGHT TOP
BOTTOM TOP_LEFT TOP_RIGHT BOTTOM_LEFT BOTTOM_RIGHT

To install the patch use (something like):

uudecode patchmail
uncompress cursorstyle.patch.gz
cd <fvwm path>/fvwm
patch <cursorstyle.patch

Otherwise use whatever method you can for installing the changes from a context
style diff.
