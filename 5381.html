On 27 Jun 2002 09:57:44 -0400, Mike Bourassa wrote:
>
> Thanks very much for the quick reply! I'm not sure though where the
> info you provided should go.

It should go to the file ~/.fvwm/.fvwm2rc. You said you installed fvwm
yourself on Cygwin, so you don't have a system wide configuration.
But then you said you were able to configure your xterm and have
several titlebar buttons. Did you use "Setup Script 95" or "Setup Form"
on the first start? If yes, you have some files in ~/.fvwm that are
included from .fvwm2rc file, change either .fvwm2rc or these files.

> Does it go in my fvwm-config file and if so is there a particular place
> or format?

fvwm-config only gives you information of what and where you installed.

> Is there a man page for the config file?

Yes, "man fvwm2".
