%% "Olivier F. Beauchemin" <obeauche_at_bbnplanet.com> writes:

  ofb> Why is this happening:

  ofb> checking whether setvbuf arguments are reversed... configure: error: can
  ofb> not run test program while cross compiling

Your compiler is not installed correctly, or something. Configure
thinks you're cross-compiling, which you're not: that means that it
couldn't get your compiler to work.

If you look back up through configure you'll see it decide you're
cross-compiling.

  ofb> Trying to complie version:
  ofb> fvwm-2.2.4

OK, but you didn't specify your OS.

It's probably your C++ compiler that it's having problems with; try
re-running configure with --without-cxx (remove the config.cache file
first).

Please read the INSTALL.fvwm file; it talks about these kinds of
problems.
