The best way I know of the read man pages is inside Emacs or XEmacs.

1. start (x)emacs.
2. type Esc X man Return
3. Type the name of the man page "fvwm2" Return.

You now have an emacs buffer with the man page in it. All the emacs
search, and positioning commands work.

If thats too much for you, try "xman".

I believe most versions of Linux use "less" to display the man page.
Do "man less" for more information.
