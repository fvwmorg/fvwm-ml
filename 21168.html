  I caught this line in an .fvwm2rc.m4 and was wondering if anyone had a
sample script that showed off how to create the dynamic menus. I was
specifically wondering how to get xterms popping up and cd'ing to different
directories automatically.

AddToFunc Dynamic-Menu-Func "I" DestroyMenu "Dynamic-Menu"
+ "I" FvwmM4 lib/fvwm/dynamic-menu
+ "I" Menu Dynamic-Menu
 
Thanks, Nils.
