Sorry to junk if not interested.

I have just made a new release of AnotherLevelUp (a much improved version of
Red Hat's AnotherLevel Fvwm2 configuration, with persistent user options and
themes).

See: http://www.cs.man.ac.uk/~jtl/ALU

for tar file or Red Hat rpms.

Recent main new features:
        * option for random theme selection at start up
        * improved componentisation
        * monitoring of error output from programs started via the menu

Thanks, John Latham
