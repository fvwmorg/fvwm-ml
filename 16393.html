> I've had trouble mapping any of the "special" function keys on the
> Sun keyboard. I'm using a type 4 keyboard. I looked in the man
> page, which told me to look at the/usr/include/X11/xkeysymdef.h
> file. [...]

Get the program xkeycaps from ftp.x.org. Not only will it tell you
what the keysym names are, it will generate appropriate xmodmap code
for you to modify them if necessary. In fact, you can use it to make
your keyboard match your .fvwm2rc :-)
