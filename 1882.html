On 15 Jan 2001 18:51:16 +0000, John.Lonergan_at_ubsw.com wrote:
>
> Is there anyway to get fvwm to always give focus to a new window when
> one pops up due to a command line operation, eg when I run vim.
>
> With my current setting's (focus follows mouse) the new window only
> gets focus if it happens to materialise under the cursor.
>
> What I'm looking for is an orthogonal focus mode like
> 'NewWindowAlwaysGetsFocus'.

If you use 2.3 versions, this is done by:

  Style * GrabFocus, GrabFocusTransient

In fact, GrabFocusTransient and in some cases GrabFocus is default,
see the man page.

For a more general solution (not restricted to focus only) of the "I want
this and this when a new window pops up" see FVWM FAQ, question 3.18.
It also provides solutions for 2.2 versions, but 2.3 is more flexible.
