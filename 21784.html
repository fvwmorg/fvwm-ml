On Mon, 16 Feb 1998, kmbehara wrote:

kmbehara>could someone tell me how i could edit my .fvwm2rc
kmbehara>file so that the VT fonts are not set to default but
kmbehara>can be set to something that i want instead of having
kmbehara>to press control and then right clicking over all the win
kmbehara>dows. any help will be appreciated.:w

Add a line to your ~/Xdefaults file, similar to this one:

XTerm*font: 10x20

using whatever font you like, using the full x-style name, or nickname
like the above.

For more on XTerm resources, "man xterm"
