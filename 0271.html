> When you iconify or deiconify a window, FvwmAnimate draws lines on the
> root window between the window and the icon.
>
> If you are using a 2.2 version of fvwm and you are using an icon manager
> instead of the root window icons, you won't see anything.
>
> In the 2.3 betas, additional configuration would be needed to get animation
> with the icon managers.
>
> If you have a very fast machine, you may have to increase the line width
> or iterations. The effect may be going by too fast for you to see anything.
> Try a line width of 16 and work down.
>

I am using the IconManager, yet I don't see any on the FvwmAnimate features.
