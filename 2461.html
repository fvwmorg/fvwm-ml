I've used Exceed quite a bit and I don't see any full screen
refreshes.

There are some settings that might affect this.

I run windows in truecolor mode. If you don't run in truecolor mode
and a new color is allocated, windows does some colormap balancing
the might look like a full screen refresh.

Under Exceed, Xconfig, Screen Definition, Server Visual I selected
"Auto Select". You can pick TrueColor or PsuedoColor. Combined with
the windows setting, a mismatch might cause problems.

Under Exceed, Xconfig, Video, there are some Palette options. I
picked Defer Changes and Optimize To Reduce Flashing On Focus Change.
I've also got Preserve System Colors selected, Preallocate system
colors in default colormap selected and Enable System Pixels not
selected. Since I'm running with TrueColor I don't think these
setting affect me much.

I use FvwmAnimate with the builtin IconBox with no problem.
