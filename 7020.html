Hi,

I set up key combinations (CA-Left, CA-Right, CA-UP, CA-DOWN) to move around
pages on a desktop by 100% window size. But when pressing eg. CA-Left I
expect the currently visible window below the mouse cursor to have input
focus. To do this I guess it is easiest to focus the window below the cursor
immediately after changing the page. But how do I set the focus to the window
below the mouse cursor?

You might wonder why I don't use desktops instead. Well, the reason is that I
find it more intuitive to move between pages as described above than to
switch between desktops that have no geometric relation.
