Dominik Vogt wrote:
>
> Again I don't know how it is in 1.24, but with 2.2 you need to
> add this style:
>
> Style "*" DecorateTransient

        This is the correct one, if my fvwm2rc is any indication.

        However, in version 1, DecorateTransients (note the different spelling)
was a standalone command; it hadn't yet been made into a style option,
so it's an all or nothing proposition.

        Bob also asked:
>
> I would like to know if I can source another file into the ~/.fvwmrc file.

        As far as I can tell, there wasn't any such facility in version 1. In
version 2, you definitely can. My .fvwm2rc is little more than a
skeleton containing a bunch of read commands, and the real work is done
by about a dozen other files sourced via those read commands.

        I believe Dominik mentioned the conversion script. I can't comment on
its effectiveness, because I've never used it: I just gritted my teeth
and did the whole thing by hand years ago. But as he said, feel free to
ask us if you run into any snags.
