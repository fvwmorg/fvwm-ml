Version 2.4.0 reads the config file in
$HOME/.fvwm/.fvwm2rc

There are some incompatible changes between 2.2.x and 2.4.x.
If you have an existing config file, you may want to run it
thru fvwm24_convert which comes with 2.4.x. There is a bug
in the 2.4.0 version of that script, change
"$$HOME" to "$ENV{'HOME'}" in 2 places.
