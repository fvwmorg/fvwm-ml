> netscape to get going. What would be really neat is to have fvwm
> remember that I started netscape (or any other ap for that matter) on
> the screen I was in when I selected the menu item (effectively running
> a function), and then automagically made sure that the ap actually
> appears in that screen, regardless of which screen I am currently in.
> The ap would still think it's running on the normal screen, but it
> would actually just appear in the background, as it were. (I can still
> see it in the Pager for example).

I think you should be able to specify which desktop program loads on,
maybe going left to right then down, 0, 1, 2, 3, 4, etc. So perhaps:

Style "Netscape" Desk 0, Screen 4, Focus 0 (so it doesn't steal the
focus.

Or something to that effect would put it on the 5th screen, letting you
do other tasks while letting Netscape load. This might be easier than
having FVWM remember all the windows. That could get annoying anyhow.
