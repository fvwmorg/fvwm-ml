Hi,

I'm a former vtwm user, and I'm now using fvwm 2.0.45, with
FvwmPager and
FvwmIconMan. One of the nice features of vtwm (or at least the
version I was using) was that if I deiconified something from the
Icon Manager, it would show up on whatever my current screen was,
instead of in the screen that it was on originally. This behavior
was configurable with the "DeIconifyToScreen" parameter. I often used
this to move a window from another screen to my current one.

Is there a similar parameter in FVWM or FvwmIconMan so that when I
click on the icon in the Icon Manager the window shows up in the
current page?
