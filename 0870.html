Peter Bismuti wrote:
>
> I'm having problems with my colors, I was
> told to run 'xdpyinfo', I'm not sure now
> to interpret the results, what should
> I be looking at?

> default screen number: 0
> number of screens: 1

You have one screen

> screen #0:
> depths (1): 24

of depth 24

> number of visuals: 1

You have one visual

> default visual id: 0x22
> visual:
> visual id: 0x22
> class: TrueColor
> depth: 24 planes

which is a TrueColor.

What this all means is that your graphics card can display bazillions of
colors and if something isn't displaying the correct colors it isn't
because the card has run out. It's more likely to be a configuration
problem with the client.
