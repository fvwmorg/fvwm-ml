X programs expect all headers to be in an X11 directory under the
include directory. That is, they all do things like:

  #include <X11/xpm.h>

So, you need to copy that header to /usr/local/include/X11/xpm.h, not
/usr/local/include/xpm.h.

Don't know if that helps; you definitely should have seen some kind of
message about XPM not being found correctly in the configure output if
it didn't work.
