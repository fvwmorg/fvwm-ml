On 19 Mar 2004 at 13:11:27 +0100, Dominik Vogt wrote:

> You can use the WarpToWindow command and specify the other
> screen's root window.

Of course, it is even in the man page :-|
I thought this is impossible since there are two separate
fvwms.

This one does exactly I was looking for:

Key Left A S4 WindowId root 1 WarpToWindow 50 50
Key Right A S4 WindowId root 0 WarpToWindow 50 50

Thank you very much for the hint - I should have asked
ealier ;-)
