On 14 Mar 2002 09:47:24 -0800, Li Tan wrote:
>
> Hi, I just started to use FVWM. It looks pretty configurable. But I almost have no idea how to do it. I downloaded FVWM2.4.6 and FVWM themes. I can choose the themes, but how can I reconfigure them?

Well, if you use fvwm-themes, it helps to read it's documentation.
Usually all questions similar to yours are answered in the FAQ.

> 1,When I tried to quit and save session, it did nothing, the only one worked is Quit FVWM only.

You don't run FVWM under a session manager, so you should not choose this
option in the themes settings menu or GUI. Restore the "None" choice.

> 2,I remember there was a FVWmrc file I can setup color and fonts, but I cannot find it and setup color after i used FVWM themes, how could I setup my color?

FVWM Themes uses its own file structures (system wide and user) for themes.

  % mkdir -p ~/.fvwm/themes/litan
  % cp `fvwm-config -d`/themes/default/colors ~/.fvwm/themes/litan/

Then edit this colors file as you want, with help of "man FvwmTheme".
Then "Refresh the current theme" from the menu and choose colors_at_litan.

> 3,how can I setup and configure screen saver?

This is not configurable using menus/GUI yet, but this may be done like:

  % mkdir -p ~/.fvwm/themes/personal
  % echo 'AddToFunc InitFunction + I Exec "xscreensaver -no-splash"' \
> ~/.fvwm/themes/personal/startup-extra

Then again "Refresh the current theme" and load startup-extra_at_personal.
The screensaver will be available on the next startup.

Write to fvwm-themes-users_at_lists.sf.net if you have problems with
personalizing fvwm-themes.

Or don't run fvwm-themes-start, run fvwm2 instead, and create your own
~/.fvwm/.fvwm2rc after reading other configs and "man fvwm2".
