bhicks_at_turing.une.edu.au said:
> I have downlaoded a version of fvwm and get the file
> fvwm-1.24r.tar.gz what do i do now.

Delete it, and download fvwm-2.0.45.tar.gz?

bhicks_at_turing.une.edu.au said:
> I am able to remove the .gz by using gunzip but have no idea and have
> been unable to find out how the remove the .tar or what to do once I
> have done that.

Use tar. A suitable command is "tar xf fvwm-1.24r.tar".

bhicks_at_turing.une.edu.au said:
> Also is there anywhere where i can get basic instructions on how fvwm
> works and how i implement it only my account or computer.

I don't think so. You're just supposed to know. To be fair to writers of
books and FAQs, almost every X installation is different in signficant enough
ways to make it difficult to give useful advice, other than "ask your local X
guru". What is your machine, anyway? If you gave more information, people
could help more. For example, if you're accessing a VMS machine using VT200
terminals, then you can't use fvwm; if you're using a Linux machine, then
you've probably already got fvwm, and if we know whether you're using xdm or
not, we can offer help on changing your window manager.
