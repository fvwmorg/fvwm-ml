"Araujo, Scott" <SAraujo_at_boundless.com> writes:
> i would like to start netscape as a full screen window that cannot be moved
> or resized but i would also like any child windows that netscape spawns to
> *be* movable *and* resizable. i can start it full screen with -geometry and
> specify 'Style "Netscape" NoHandles, NoTitle, BoundaryWidth 0' but this
> makes all the child windows unmoveable too. is there a way to do what i
> want using the fvwm config file? if i can't do it through the config file,
> could i modify the fvwm source or would i have to modify netscape? i am
> using version 1.23b but i could upgrade if i need to.

You don't need handles, frames and the like to move or resize windows.
Do something like:

Key F7 A M Move
Key F8 A M Resize

(Meta F7/F8 anywhere move and resize windows)
