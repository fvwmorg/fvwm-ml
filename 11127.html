 --- Joel Smith <joel_at_earthlink.net> wrote: >
> i'm not quite sure how to explain this. the symptom is that the text
> at the top of every window is set too low, so the top of the
> application cuts it off. this happens for most buttonstyle,
> titlestyle i've copied from other examples. i believe the default
> style, i.e. .fvwm2rc is blank as far as buttonstyle goes, is fine. is
> there something i'm missing here? thanks.

Without a screenshot, this is difficult to diagnose, but it sounds to me
like your locale settings are out and so the font that it has defaulted
back to is ugly. What have you got $LANG and $LC_ALL set to? If you're
running debian:

apt-get install locales

and/or:

dpkg-reconfigure locales

I use en_GB.UTF8
