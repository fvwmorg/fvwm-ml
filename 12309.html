If I am not mistaken, what you are looking for is the translucency
patch (translucent windows allow showing windows hidden under them and
I believe this is not yet in FVWM, transparent windows show the
background, but not overlapped windows, that is supported in mainstream
fvwm).

If you get the patch you will need to recompile the package (unless
someone supplies a package with the patch included).

I don't know how your chip will deal with that though (didn't know
there were still 1MB graphic cards out there).

I don't know how things are with freebsd (don't think it has
frame buffer support) but if it does there is DirectFB which allows
translucency (don't know how hard it is to setup).
