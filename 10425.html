jlm <jsado_sc3_at_earthlink.net> writes:
> I have this line in my .fvwm2rc file:
>
> Style *GVIM* NoTitle, BorderWidth 0, NoHandles
>
> When gvim starts up, it has a border. If I type this into the
> FVWMConsole, the gvim border dissappears. I want all of my gvim windows
> to have no border. What am I doing wrong?

Gvim, doesn't use the name "*GVIM*" as it starts.

Try:

 Style gvim NoTitle, BorderWidth 0, NoHandles, HandleWidth 0

works for me.
