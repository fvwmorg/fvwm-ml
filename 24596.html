> Osvaldo Rentas wrote:
>
> I installed redhat 5.2 and the default window managers were fvwm95 and
> afterstep ... I looked around and liked fvwm. So I uninstalled the
> other fvwms with the -e oldfvwm.rpm type of command. I installed the
> new version. Now it seems in .Xclients I edited it to start the new
> one, but it never works. WHere is the executable installed to when I
> start X I can get fvwm, so all I get are errors. Another thing that
> seems to happen is that all the characters in console will get all
> messy and be in disorder .. so basically, can anyone help me install
> fvwm ?

try this:
1) comment out everything in your ~/.Xclients so all it reads is:

exec fvwm2

2) make sure fvwm2 is installed and in your path. to find out where it's
installed do this command: find / -name fvwm2

3) you could try uninstalling AnotherLevel, if you dont like it.( I
believe this is redhats fvwm95 program your talking about)

4) make sure you have a .fvwm2rc in your hom dir, you can get get sample
ones in the fvwm2 distribution.
it should then work, if I doesnt try putting the full path to fvwm2 in
your .Xclients

Good luck,
