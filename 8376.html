No, sorry, not currently. Nothing except for MultiPixmap works with the
actual text length.

I am going to rename MultiPixmap to Multi or MultiFace anyway, so maybe
in the future someone will add a support for Vector to it (what you call
multivector). Currently you should use transparent images for this.

See http://www.netspace.net.au/~rossh/fvwm-bug/description.html
to get an idea how to replace Vector with alpha-blended png.
