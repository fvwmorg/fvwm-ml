I want to remap Alt+Tab to pop up a list of the windows on the current desk
(normal+iconised). I want the mapping to be such that when I release both
keys, the menu is left up and I can pick one of the windows by pressing the
correct 'short-key' number.

If I do

    Key Tab A M WindowList Root c c CurrentDesk, NoGeometry

the menu disappears when I release the Alt key, and the 0,1,2 etc keys don't
work whilst Alt is held down. However, if I map

    Key Tab A C WindowList Root c c CurrentDesk, NoGeometry

I can use Ctrl+Tab to do exactly what I want. Why can't I remap Alt in the
same way? Does that modifier have special handling that I can't remove?
