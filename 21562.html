I finally decided to compile and install version 2.0.46, after many
months with .45...
All's well, except for a little glitch: Focus doesn't seem to also raise
a window, as it should.
I've always got the Alt-tab key combination binded to a Prev Focus
command.
It has always worked as expected in fvwm 2.0.45, but as I installed
version .46 typing the combination now only switches focus, without
rising the focused window.
The manual page says that Focus "Raises the window if needed to make it
visible." Indeed it worked that way with the previous version.
Now I had to write a small function to explicitly Focus and Raise a
window. Is it a known problem or did I miss something?
