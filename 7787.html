Hi guys and gals,

I have made a very nice FvwmButtons with a lot of buttons
with pretty pixmaps.

Is there a way to have "balloon" help for these buttons?
That way when the mouse pointer hovers over a button, a
tiny window will pop up right there with a brief description
of the button's function.

I don't want to use "Title" to add labels to each button,
because that detracts from my very artistic design :-)
My setup is being used by a large number ( >100 ) of people.
They would rather have a balloon help rather than find out
by trial and error what each button does.
