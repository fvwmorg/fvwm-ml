Kai Grossjohann wrote: 
>>>>> Scott  writes: 

  Scott> i would like for my desktop to all fit on my screen without 
  Scott> any scrolling. 

DeskTopSize 1x1 

Units are in physical screens. 

If that doesn't do it for you, you're probably using a PC-type machine 
with an SVGA-type graphics card which does hardware scrolling (they 
call this "panning" I think).  I'm not sure how to turn it off, but 
I've seen a "Virtual" statement in XF86config once. 
  If you are using XF86 you will need to go to your XF86Config file and change the virtual resolution in whatever section your paticular graphics server is in. SVGA accelerated whatever. Then simply set the virtual resolution to match true resolution. If you have a resolution of 800x600 set virtual to 800x600. 
I had the same problem, just changing fvwm wont work if XF86 is set to virtual. Hope this helps.
