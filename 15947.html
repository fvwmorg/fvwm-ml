I installed fvwm 2.0.41 and while playing with the configuration I found
that while it works fine on the user's .fvwm2rc, it would not load
the system-wide configuration if there was no user's .fvwm2rc. Fvwm2 loads,
with nothing except background and cursor (movable). Mouse buttons do
nothing, killing the X server unjams it though.

I am running Slack 3.0, 1.2.13 ELF. I installed fvwm2 by hand (to keep it
from screwing my stuff up with that install mystery-script[1]),
also installed over 2.0.40, so I might not have gotten everything right, but
it seems kinda fishy that it would work fine with ~/.fvwmrc but can't use
/etc/X11/fvwm/system.fvwmrc, fvwm/system.fvwm2rc, fvwm2/system.fvwmrc, or
fvwm2/system.fvwm2rc.

Sorry, I'm not on the list. Flames - don't bother. I'm going to try a
reinstall, but just thought I'd report my difficulty.
