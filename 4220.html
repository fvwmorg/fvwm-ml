On Thu, Feb 07, 2002 at 11:29:27AM +0100, Klaus Zeitler wrote:
>
> Hello,
>
> I've checked in my .fvwm2rc file and there's no definition for the tab key,
> but when I press Alt-Tab a fvwm window list appears. I guess that it's
> defined somewhere in fvwm as a default. How can I disable this key setting, so
> that I can use this key in my application?

It's defined in the ConfigFvwmDefaults file. To turn it off,
issue the same binding command again but with "-" as the command:

  Key Tab A M -
