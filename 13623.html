In the 4th edition Linux In A Nutshell from O'Reilly, there is a
section on fvwm2 which includes the following sample configuration:

    *FvwmWinListAction Clock3 Module "FvwmIdent" FvwmIdent

which does not work. The manpage says this should do the trick:

    *FvwmWinList: Action Click3 Module FvwmIdent

which seems to not work either. No error message from either attempt,
and I tried several variations.

I am running gentoo with fvwm version 2.5.10-r3 on an opteron system.
