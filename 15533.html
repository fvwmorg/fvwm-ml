Ron> I was wondering if it would be possible to add an addition to the
Ron> maximize function built in to fvwm2. What I want fvwm2 to do is
Ron> allow maximization from the _bottom_ of the screen rather than
Ron> the top of the screen. I thought it might be easiest to allow
Ron> negative numbers as arguments to the Maximize function.

addition to the code. I'll see about getting it in there.

Ron> prefered placement area. As I tend to have all my "needed"
Ron> windows at the top of the screen, I don't want any new windows to
