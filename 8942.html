Hello,


Currently I use two monitors, a 17" and a 15".
I'm using fvwm 2.5.6

the following problem occurs,

I have the following options in my .fvwm2rc:
Xinerama True
XineramaSls False
Mouse 0 3 A Maximize 100 100


Wenn I maximize my windows, everything works fine.
So when I click maximize, the windows is maximized on
the monitor it is on. (So not spread over both monitors)

But wenn I click on the desktop on the second monitor,
still the menu turns up on the first. So it is not
possible to get the menu to turn up onder the pointer
on the second screen.

When I set "XineramaSls True" the menu problem is
solved, so the menu always turns up under the mouse pointer
also on the second screen, however, now the
maximizing of the screens now goes wrong, when clicked the
window spreads over both monitors.
Setting Maxmize 50 100 is not an option, because then
the maximized windows will always move to the first monitor.

Regards,
Marc Dirix
