  Subbarao> How do I change sessions between an xterm and a console
  Subbarao> terminal and vice versa? I am talking about the sessions
  Subbarao> like switching from MS Windows and DOS.

  Subbarao> I am using fvwm 1.24. Is this feature a part of the window
  Subbarao> manager or part of xdm?

Neither nor. You can switch between the virtual consoles with the
Alt-F keys. Alt-F1 is the first console, Alt-F2 the second, and so
on.

Within X, this has been rebound to Ctrl-Alt-F-key, so while running X
you can switch back to the first console with Ctrl-Alt-F1 and so on.

To switch *to* X, you must choose the first virtual console that
doesn't exist. Often, people have six virtual consoles (Alt-F1 thru
Alt-F6), so Alt-F7 is where they find X.

This is all part of the Linux console driver, but some X support is
required, too.

Btw, Ctrl-Alt-Backspace unconditionally terminates the X server,
useful in case there is some emergency.
