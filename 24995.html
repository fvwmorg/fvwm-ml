I'm going to have to go with Dominik, try doing a ps and see what the
path to the pager fvwm is invoking then do a which FvwmPager and find
out which one you are picking up. If your installation differs from the
way you compiled, then you need to set ModulePath in your rc file to
fvwm where to look, otherwise it looks in $prefix/libexec/fvwm/$version
which got defined during configure
