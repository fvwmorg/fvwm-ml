Hi,
You can use another tool to acomplish that.
I use [1] fbsetbg, which is just a wrapper used by fluxbox to set the
wallpaper, with [2] feh.
I'd recomend that you take a closer look at it :>
+ I Test (init) Exec fbsetbg -l
sets the last used wallpaper.

* you may need to edit the script(fbstebg) to change the path where the
last wallpaper used is stored.


[1] http://www.xs4all.nl/~hanb/software/fluxbox/fbsetbg.html
[2] http://www.linuxbrit.co.uk/feh
