On Wed, 2004-05-19 at 19:22, Mikhael Goikhman wrote:

> > I still can't seem to get mouse stroke to work. Does someone have a
> > sample fvwm2rc file that works with mouse strokes? I keep getting
> > errors with my fvwm2rc file.
>
> Did you try the file I suggested?
>
> Start FvwmConsole and type these commands in it:
>
> Mouse 3 R A StrokeFunc FeedBack DrawMotion StrokeWidth 2
> Stroke 0 0 R A Exec xmessage "Just click mouse 3 on the root"
> Stroke N852 0 R A Exec xterm -n "Draw line from up to down"
> Stroke N456 0 R A Exec xterm -n "Draw line from left to right"
