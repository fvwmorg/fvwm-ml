Markus Grunwald <markus.grunwald_at_gmx.de> writes:
> Hi!
>
> I saw in the list, that there is a version 2.5 of fvwm. Will my old
> config files work with that one without lots of editing?

If you are coming from 2.2.5, you should go to
the stable official release. Ie. 2.4.6.
There is a conversion script included
that will handle the differences.
I think its worth it.

The 2.5 branch is beta software, there are lots of great improvements,
but its occasionally unstable. I use 2.5.x every day with only minor
problems, but I wouldn't install it for others to use.
No config file changes are needed to go from 2.4.x to 2.5.x.
