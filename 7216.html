Command/prefix Pick is used to select a window and execute command in the
selected window context or not execute if window selection is canceled.

If there is master FvwmTabs and several aliases, you may do the following.
To create a new tab from the master, send this or similar command:

  Pick Module FvwmTabs FvwmTabs-3 # starts a new module with a new window

or (depending on implementation):

  Pick SendToModule FvwmTabs createnew

The last may send "FvwmTabs FvwmTabs-3" in the received window context.
To add a to-be-selected window to an existing tab, this tab may send:

  Pick SendToModule FvwmTabs-3 addme
