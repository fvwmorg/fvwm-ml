> On 16 Jan 2002 02:41:26 +0000, Ian Truelsen wrote:
> >
> > I just installed fvwm from the rpms on the fvwm site, but what I am having
> > difficulty doing is figuring out how to get it up and running as my window
> > manager. How exactly do I go about it?
>
> There are several solutions, which may work depending on how you start the
> X System. Try this one:
>
> % echo "fvwm2" >~/.Xclients

You'll probably also have to make .Xclients executable:

  % chmod 755 ~/.Xclients

If you're at runlevel 5 (i.e. xdm) and using the gnome or kde login
screen you'll also need to select the "default" session type to have
xdm read your .Xclients.
