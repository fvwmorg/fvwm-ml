Is there a way to set a short delay on SloppyFocus before it brings a different
window into focus? I line up all my windows to the upper left of the screen,
immediately under the task bar. And I leave about 10p of space on the right
side clear so that the root window shows. This allows me to move the cursor
over the root window so I can easily access the customized menus I have with the
mouse. However, when I do this, if one of the under-lying windows extends
beyond the focused window, even if it's just a pixel or two, I lose focus of
the window I'm working on. Is there a way to set up a short delay so I can pass
over another window on route to the root window without losing focus of the
window I'm working on?
