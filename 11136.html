> I'd like to make my Gaim conversation window sticky, but not the buddy
> list. Gaim sets
>
> WM_CLASS(STRING) = "gaim", "Gaim"
>
> for all its windows, and claims it's doing The Right Thing as you can
> read here:
>
> http://gaim.sourceforge.net/faq.php#q20
>
> So I need to be able to set style based on WM_WINDOW_ROLE. How do I go

It is currently only possible to define different window styles based on
window name, class, resource and id.

Does Gaim also set the same window name (WM_NAME) for all its windows?
If this is the case, then ask them to fix this.
