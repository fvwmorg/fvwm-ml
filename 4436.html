On Thu, Mar 07, 2002 at 02:34:05PM +0000, Fernando Herrero Carrón wrote:
> Hi there,
>
> How buttons supposed to show up in the title bar? If I define
> ButtonStyle or AddToButtonStyle nothing happens, even with
> Style * Button [N] only the title is shown.

Buttons show up if some action is bound to them with the Key or
Mouse commands. For example, if you want to use button 7, define
an action:

  Mouse 1 7 A Beep
