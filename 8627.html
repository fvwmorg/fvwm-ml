Catherine Luedecker <luedchic_at_attbi.com> writes:
> I am stuck. I am not able to get pixmap buttons on my titlebars.
>
> I thought I just had to do this
>
> ButtonStyle 1 Pixmap mac-window-shade.xpm --Top Left
>
> However the button doesnt show up. I have tried this also with
> AddButtonStyle.

Buttons only display if you make them do something.

For each button you want to display add a line like this:

Mouse 0 1 A "Iconify"
Mouse 0 3 A Windowshade
Mouse 0 5 A Module FvwmIdent
