Previously from greg shebert
>does anyone know where i can get a different pager module (besides
>fvwmpager)...
>
>i can't seem to configure the pager's desktop to be an arbitrary number
>of screens in a MxN configuration... it seems that the pager's desktops
>are only 4 screens in a 2x2 config...

Generally you want to tell FVWM itself that you want more pages. For
example:

DeskTopSize 1x12

Then Pager picks this up.

>also... there doesnt seem to be any way to change the key bindings to
>allow me to switch between the screens of the desktop

man fvwm2
/GotoPage

Should do it for you. Were you looking in the docs for FvwmPager for
this? Again, it's in the fvwm2 core.

FvwmPager isn't responsible for virtualizing the desktop -- it just
shows you that you have them. fvwm itself creates the virtual
desktops.
