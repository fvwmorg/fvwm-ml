I'm using RedHat 6 and fvwm2.2.4.

I've defined my focus policy like this:

GlobalOpts ClickToFocusRaises
Style "*" ClickToFocus

This works fine for a while after fvwm2 starts (a few minutes up to an
hour...). But suddenly it's gone and i've to click the titlebar (or my
FvwmWinList) to regain focus. Restart fvwm2 won't help. Exit to shell
and relaunch fvwm2 does.

Appart from GlobalOpts, i've noticed that Style "*" ClickToFocus only
disables the default MouseFocus behaviour. To get actually focus when
using Style "*" ClickToFocus, you also need an explicit Focus function
in your mouse bindings.

Am i wrong?...
