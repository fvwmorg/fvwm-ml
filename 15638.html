Here's some progress in trying to bind the Microsoft Keyboard left window key
to Fvwm's menu.
As I'm not in Japan, I can use the Japanese keysms without consequence.

The following binds the ms$ left window key to the Kanji keysym

        xmodmap -ev "keycode 115 = 0xFF21"

In .fvwmrc this binds the Kanji keysym + my Left ALT key to the menu

        Key Kanji A M Popup "Utilities"
 
I'd like to call up the menu with just one key, but if I leave out the "M" in
the above line, ONLY the FVWM Modules menu pops up for me. -- is this a
feature?

Can I call up the main menu (Utilities on my system) with just one key??
