Adam said:
> How can I configure FVWM so that the mouse buttons are reversed for a
> left hander? I've switched all the 3's to 1's and vice versa in the
> .fvwm2rc file, but it doesn't work.

Mmm. Should work basically.
Say for example I have the line:

Mouse 1 R A Menu Utilities Nop

This means that mouse button one on the root window pops
up a menu.

Now, if you wanted to reverse this so that mouse three
on your left did the same just do this:

Mouse 3 R A Menu Utilities Nop


Er..don't forget to restart fvwm or log out/in to see any changes.
(Just in case you're a newbie!)

Have another go, hope this helps.
