On Thu, 9 May 1996, Rasmus Peter Tamstorf wrote:

>
> Secondly, if I use *FvwmButtonsRows 1, I would like to be able to
> specify the width of swallowed modules (xload and FvwmPager), but no
> matter what geometry I specify FvwmButtons apparently gets the last word
> and decides for itself what the size should be. How can I get the last
> word ?
>

this i found in a .fvwm[2]rc file somewhere:

        *FvwmButtons(4x2) - crap Swallow "console" Exec xterm -C -T -blah...
                    ^^^^^

i guess this lets you decide how much space FvwmButtons allows for the
swallowed stuff. so, if FvwmButtons is screwing with the resizing of your
swallowed apps, try that.

>
> Finally, for Fvwm 2.0.41e in general I miss the L-shaped corners. Has
> that something to do with using the Win95-hack-version, and/or can I get
> them back.
>

i'm using fvwm-2.0.41 and the l-shaped resize corners are present. i
guess they were removed in Fvwm95, because they don't look like win95's.
