The build/install process is one of the few things that's really

root> Of course I didn't have imake, had never heard of it, so I had
root> to go find that first.

Just so everyone knows his reason for saying this, for some unknown
reason Imake is not installed by default on IBM machines, which is
most unfortunate.

If you can manage to install the Xamples tree (under
/usr/lpp/X11/Xamples/) from your install medium (it should be in there
somewhere, as I found it), the imake related files will be there and
can relatively easily be compiled I believe. If I remember correctly
(it was a while ago), that's what I did.
