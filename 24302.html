> Is there a command I can use for shutdown from a Popup menu?
> For instance, change the Quit command to execute shutdown -h now?

  AddToMenu "Shut down" Exec shutdown -h

or if you need to do more

  AddToMenu "Shut down" Exec my_shutdown_script.sh
