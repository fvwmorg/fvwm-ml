: Question:
: How do I select which page (desk grid #) to start
: a program in?

You can always specify -geometry in the command line of the app being called;
for example, my Sun desktop is 1152x900, so the command line:

xterm -geometry +1152+0

gives me an xterm in the upper-left-hand corner of the page to the right of my
start page.

hope this helps...
