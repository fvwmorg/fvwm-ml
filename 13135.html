Please double check whether fvwm runs fine without a configuration
file. Start fvwm with the command

  fvwm -f /dev/null

e.g. from $HOME/.xinitrc. Start FvwmConsole from the root menu,
then type

  *FvwmPager: Geometry 71x53+0-0
  FvwmPager

This should work. If it doesn't, check the file
ConfigFvwmDefaults (may be installed in /usr/local/share/fvwm).
This file should not be modified, but if it is it may contain
conflicting settings.

If the pager comes up fine, check all your config files for
settings that override the pager geometry.

You may also add

  DestroyMofuleConfig FvwmPager*

before the pager configuration to make sure you're not inheriting
other settings.
