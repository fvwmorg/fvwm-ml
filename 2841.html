Hi, I have this enhancement request:

I use different desktops for different tasks, e.g. desk 0 for
mail and www, desk 1 for development.

When I'm switching among the desktops, I usually want to go to a
different page as well (I have 3x4 pages), like I'm on the page 1
0 on desk 1 and I'd like to go directly to 0 0 on desk 0 because
that's where I was last time.

Imagine I have a terminal with a long compilation on desk 1, page
1 0 (quite a common situation ) and I want to be there
automagically when the compilation ends and it beeps and I press
the button that gets me to desk 1. And all that regardless of
where I was on the other desktop, reading fvwm news while the
compilation was in progress. 8-)

Maybe the attached diff of my quick-and-dirty solution will help
if my previous text is not clear enough.
