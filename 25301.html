Elliot Sowadsky <elliot_at_c-cube.com> writes:
> SloppyFocus is a mouse-to-focus. click-to-focus shouldnt drop the focus w/o
> a click elsewhere. I use click-to-focus because it allows hold-the-key-down
> copies and moves (an xview thing). What is unclutter?

unclutter is a program you run in the background that makes the mouse
pointer disappear when you haven't moved the pointer for some amount
of time. You should be able to find it at ftp.x.org or thru a web
search. You seem to want to use click-to-focus for a different reason,
so I guess it wouldn't help you.

> Key L5 FSTWI0123456789 N RaiseLower didnt touch the window the mouse was
