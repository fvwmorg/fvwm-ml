>>>>> Barry D Benowitz writes:

B> I would like to set up FvwmPager to only display the active desktop,
B> not all the possible desktops. I would use another method of switching
B> between desktops (probably tied to a button).

B> Is This possible?

my pager does this all the time:

*FvwmButtons(4x2, Swallow FvwmPager "FvwmPager 0 0")
########################### Pager #########################################
*FvwmPagerGeometry -1500-1500
# to be swallowed

*FvwmPagerBack #908090
*FvwmPagerFore #484048
*FvwmPagerHilight #cab3ca

*FvwmPagerFont none
# -adobe-helvetica-bold-r-*-*-10-*-*-*-*-*-*-*
*FvwmPagerSmallFont 5x8

#*FvwmPagerLabel 0 Emacs
#*FvwmPagerLabel 1 Misc
#*FvwmPagerLabel 2 Make
#*FvwmPagerLabel 3 Spare

i.e: no labels, no font.

I get a 4x2 set of pages representing the current desk.
If i accidently start an application on desk2 (I usually just use desk 1),
it's nearly unreachable (except from winlist)

hope this helps,
