Just for fun I played with fvwm2 recently, and was sufficiently
impressed with new Style and Decor stuff (and the toolbar enabled by
FvwmButtons) that I've junked my old fvwm.

I'm using 2.0.45. I have a few questions. My .fvwm2rc is available
at http://www.ccil/~esr/fvwm2 in case it turns out to be relevant.

1. I think I'm using the stock button styles and bindings. My
   window frames have a thick dash and a thin dash at the left,
   and a small and large square at the right.

a) What does the small dash do? The only effect I can see is that
   it changes the title-panel texture, but it must be doing something
   subtler and more significant than that, right?....

b) Where are these default buttons (and other defaults) set up? Are
   they wired into the code? If so, is there an fvwm2rc equivalent
   to the defaults I can peruse somewhere?

2. Like everybody else, I'd like to be able to launch my MUA by
   double-clicking a swallowed xbiff. I've read the last month
   of fvwm-list archives, but am not clear on whether there is a
   known recipe that works.

Thanks in advance.
