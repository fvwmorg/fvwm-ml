I just found in the mail archive how to make text shadowed.
It looks great! As for calculating the color, in my setup
I use the 'selection' color from the motif color calculations.
This one is a bit lighter than the 'shadow' color.

A problem with drawing a shadow behind text, is that it only
looks good when the shadow can be made darker than the color
of the actual text. Eg if the text color is black, the fgsh
maybe should be set to the background color so that effectively
no shadow appears. Otherwise, one gets an ugly doubely drawn
text, in almost the same color.

Furthermore, I also tried to use the feature in the pager, for
the desklabels. (Like is done in CDE) This hoever, doesn't seem
to work (yet?).
