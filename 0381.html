  it can be done, that's how my setup works:

EdgeScroll 0
0
EdgeResistance 400 0

  relevant parts of manpage:

EdgeResistance scrolling moving
...Note that, with "EdgeScroll 0 0", it is still possible to move or
resize windows across the edge of the current screen.
...

EdgeScroll horizontal vertical
...If you don't want any paging or scrolling when you hit the edge of a
page include "EdgeScroll 0 0" in your .fvwm2rc file, or possibly
better, set the EdgeThickness to zero.
...

  I'm using 2.2.4 on solaris and something close to 2.2.4 mon linux, it
works on both (mouse does not change viewport unless you move/resize
windows).

        erik
