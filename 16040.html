my suggestion for a to-do in fvwm 2.0 is a dynamic icon bar which changes
depending on which icon you push. This way, you can establish a nested
tree-like structure of icons. If you click on System Setup icon, the
vertical bar changes into icons for setup with new icons such as: RAM,
disk, cache, etc. Click on Networking and the icons get replaced with
icons pertainting to network setup
