NOte, I am willing to believe that the following might show my
cluelessness. I am currently using fvwm 1.24r. If there is someway to
accomplish the following, even with just an upgrade to the beta 2.0,
please let me know.

***Control over the icon pixmap vs. the 'icon text' the floats under
the icon pixmap (or is the sole piece of the icon when there is no
pixmap):
        I'm really picky on space, and for programs like cm the icon
pixmap is sufficient and I don't want the icon text to even
appear. For most other things, I don't want there to be any icon
pixmap, just the icon text (trying to give a small 'invisible' pixmap,
just results in the icon text being as small in width as the pixmap).




***Ability to make a key equivalent to a certain mouse click:
        I've got in the habit of using the keyboard to move my mouse
pointer around, but in some cases, this is useless unless I can click.




***Ability to use a double-click when the second click is held down:
        I currently use the following setup, which allows me to click
and release to get a menu, as well as click and hold to get a menu,
but I can't double-click and hold to get the other menu.

Popup "Mouse1Popups"
        Popup "Click" FVWM_control
        Popup "Motion" FVWM_control
        Popup "DoubleClick" screen_locks
EndFunction



***Better control of moving and resizing from commands:
        What I mean here are things I can link to the click of a
button, like Maximize. But I hate the way maximize set the position to
0 0. Is there any way give a window a size or position?




***Ability to change the button pixmaps
        I have a feeling I just don't know what to do on this.




***Ability to set the pointer inner and outer color.
        Am I missing something?




***Have a StartsOnPage like StartsOnDesk



***Question, why do commands like iconify and delete not wait for a
window if you set them to a mouse-click in the root window? I
currently use the following, which works, but I was just curious:

Function "Iconify"
        Iconify "Click"
EndFunction

Mouse 0 AT142 MS Function "Iconify"





That's all I can remember---
