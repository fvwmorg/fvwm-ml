Actually, none that I know of. I had a similar problem on my network. I
like to use 1024x768 resolution, but the monitors on my network don't go
past 800x600. Most Xwindows clients do have a configurable option to set
the screen size, but this changes the size of the client window. You can
only change the display size of the X server on the machine through the
Xconfig file.
