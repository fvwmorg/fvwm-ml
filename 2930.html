Dirk Wright <wright_at_kreative.net> writes:
> After looking at the man files that came with my slackware 7.1, all I
> see are files about modules. Obviously, I'm missing something here. I
> got fvwm2 going by copying the supplied system.fvwm2rc file to my home
> directory and renaming it ".fvwm2rc". Then I hacked it up a
> little.....but I don't have much of a clue about what I'm doing. I
> appear to have fvwm version 2.2.4.....OK, so I went to the fvwm web page
> and found the huge man page for fvwm2. How do I get this in "man page
> format" whatever that is? I saved it as html, but I don't think xman
> reads that format. I've got some reading to do....

You should be able to bring up the man page by typing:

man fvwm2

The man page in html format was created from the man page in
man page format. There is now way to convert it from html to man
format.

If the fvwm2 man page got deleted from your setup, you should be able
to download fvwm 2.2.4 and extract the man page from it, or you
might want to just install the current release 2.4.0.
