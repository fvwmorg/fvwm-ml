On August 14, 2002 at 15:14 +0200, Rasmus Stenfalk wrote:
> I noticed that it is not possible to set a window to stay on top or bottom (ie
> floating/sinking) in FVWM.
> I'm using 2.2.4. Should I upgrade or has it not been added?

You should. In version 2.4.x and higher

Style * StaysOnTop
Style * StaysOnBottom
Style * Layer n
