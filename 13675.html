As an experiment, I've created the config

Key Home W M Move +0 +0
Key Left W M Move +0 w
Key End W M Move +0 -0
Key Up W M Move w +0
Key Down W M Move w -0
Key Prior W M Move -0 +0
Key Right W M Move -0 w
Key Next W M Move -0 -0
Key Insert W M Iconify

..with the idea that, by holding down the Meta key and pressing the
appropriate cardinal direction key, the focused window will jump to
that edge/corner of the desktop.

Problem: Mozilla uses Meta-Left for "Back".

Is there any way to define an exception case for a particular window,
or do I need to suck it up and configure (and learn) a different Modifier?
