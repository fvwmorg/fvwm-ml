On Tue, 2003-04-08 at 08:04, Michelle Konzack wrote:
> Hello,
>
> I have some Multiuser-Systems and running fvwm only without
> GNOME or KDE. System default is 'C'
>
> Now I have User-Settings like
>
> export LC_MESSAGES=de_DE_at_euro
> export LANGUAGE=de_DE_at_euro
> export LANG=de_DE_at_euro
>
> But I can not use the 'LANG' because fvwm exits every time
> without any error messages.
>
> WHY ???
>
> This Error occur with ALL possibel settings like tr_TR,
> ar_SY, fs_IR ar_MA and fr_FR_at_euro...
>
> Thank you for your Help
> Michelle

How are you starting fvwm? If you are starting fvwm from .xsession,
check ~/.xsession-errors to see if there is any output from fvwm. If
you're using ~/.xinitrc, check the terminal where you started X. You
might also check to see if fvwm crashed and left a core file.

My LANG is en_US.UTF-8, and I haven't had any trouble starting fvwm 2.4
or 2.5, although the Debian package of fvwm 2.4 gave me some font issues
(probably because of my UTF-8 locale.)
