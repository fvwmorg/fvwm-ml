On Wed, 23 Oct 2002, Denis Perelyubskiy wrote:

> Hello,
>
> I'd like to run fvwm2 --version from my menu, and show
> output.
>
something like this would perhaps do good:
Exec exec xmessage `fvwm --version`

or (for a more brief string):
Exec exec xmessage `fvwm --version | awk '{print $1 " " $3;}' | head -1`

There is some option for xmessage that might be interesting, see the
man-page
