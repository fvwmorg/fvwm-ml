> The following is waht I have in my .fvwm2rc file:
>
> # Button 1: Close
>
> # Button 4: Minimize (image of large window being shrunk down)
>
> # Button 2: Maximize (image of window being enlarged)

Buttons are numbered 1,3,5.. on the left side, 2,4,6.. on the right side.
First, you should define a button's look, just like it is done for your
buttons 1, 2 and 4, using vector graphics, pixmap, mini-icon or other
method, read about ButtonStyle in the man page. If you don't define a
button's look, the default look for that button will be used (dot,
rectangle or empty square), which may be not what you want.

Second, you should bind some Key or Mouse command to a newly defined
button, unbound buttons are not shown.

A working example:

ButtonStyle 3 7 70x15_at_0 30x55_at_1 50x55_at_0 40x85_at_1 80x45_at_0 60x45_at_1 70x15_at_0
Mouse 0 3 A WindowShade
