Le Dimanche 10 Mars 2002 20:43, vous avez écrit :
> On 10 Mar 2002 08:23:37 +0100, Denis Prost wrote:
> > I'm using fvwm 2.4.3. Is there in further releases an option to tell
> > fvwm-menu-directory to generate keyboard shortcuts on the menu items
> > first letters, or would it be possible to add it in a future release.
>
> To do this for any menu, not just autogenerated, add:
>
> MenuStyle * AutomaticHotkeys
>
> If you like to do this only for fvwm-menu-directory (or other fvwm-menu-*
> script) menus, run this script with, say, "--menu-style dirmenu" and add:
>
> MenuStyle dirmenu AutomaticHotkeys
>
> Regards,
> Mikhael.
