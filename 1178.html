>
> I can reproduce the problem by issueing the command
> "xterm -geometry 200x10000". If I use this command to create a display

> physical screen. If I display the same application on a system running
> FVWM then the created display is much larger than the physical screen.
> If I Maximize the X display then the window reduces in size so that it
