I have been following the discussion started by Hucka about window
resizing features, and it makes me recall a module written by Elad Eyal,
<elad_at_cs.bgu.ac.il> a year or so ago. My last update on it was Aug 1997.
It is a module with which you bodily take the named window and put it
where you want it and resize it as you go. Click of the mouse gives one
corner, release gives the other corner. I gave the author marginal help
ni testing his module, and flagged some issues with other color depths
than he was using. It seemed very usable, but my normal window habits
are not very conducive to using FvwmGeom. If the files are not available
from the author, I could tar up the source again and send it to interested
parties.

I don't think FvwmGeom is part of the regular distribution. If it is,
please ignore me gracefully.
