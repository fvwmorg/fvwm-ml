I've figured out most of what's wrong with my .fvwm2rc and various other
manager config files. Most of it was assuming that my older .fvwm2rc
wouldn't cause things to break, but it did. 2.5.9 fvwm chokes on some of the
2.4.x options.

Anyway, I'm running version 2.5.9 compiled for AMD64 under Mandrake 9.2 for
AMD64 and am having a problem with the fvwm Icon Manager. Specifically, the
little square to the left of the button label that indicates whether or not a
window is iconfied or not. This square is usually absent. The only time
this square is present is if I resetart fvwm while a window is iconified.

is no longer present.

I haven't upgraded to 2.5.10 yet. Will this fix the problem or is there
something else funny going on? I didn't see any option in the FvwmIconMan
man page to indicate the little square is even an option that can be
disabled.
