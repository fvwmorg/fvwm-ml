: FvwmTabs (v3.1) started with: Perl 5.8.4, X11::Protocol 0.52, Tk 800.025.
: image "" doesn't exist at /usr/lib/perl5/Tk/Widget.pm line 196.
: at /usr/lib/fvwm/2.5.10/FvwmTabs line 464
: (in cleanup) Can't call method "name" on an undefined value at /usr/lib/fvwm/2.5.10/FvwmTabs line 850 during global destruction.

Oh ok. You're using an oldish version of Tk & it doesn't appear to handle
empty strings as an image name the way later versions of Tk do. I'll
modify FvwmTabs to get around this in the next release. For now, if
you make sure a valid image is specified for the 4 toolbar icons it
should work ok:

*FvwmTabs: killIcon red.png
*FvwmTabs: addIcon yellow.png
*FvwmTabs: releaseIcon green.png
*FvwmTabs: menuIcon green.png

& make sure the .png images are in your ImagePath.

Let me know if this works.
