  de> You should be able to download the source and compile it without

Well, not without _any_ problems.

FVWM now requires an ANSI C compiler. The default cc that comes with
SunOS is a moldy old K&R compiler (but hey, it's a compiler!)

You'll need to either get a binary copy of GCC or use your moldy old K&R
cc to build GCC, whereupon you can use GCC to build FVWM.

Unless you have an ANSI compiler for SunOS 4.1.x lying around somewhere
else, of course.
