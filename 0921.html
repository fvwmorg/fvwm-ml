"Nikolaj G." wrote:

> 1. Im using a filemenu item called "Files" with a list of files I use
> often, each called with "emacs <filename>") - how can I send a file to
> an existing emacs window, whitout opening a new emacs window?

Use gnuserv.

 1- Get and install gnuserv package
 2- Add this to ~/.emacs :
     
  (setq gnuserv-frame (window-frame (selected-window)))
  (require 'gnuserv)
  (gnuserv-start)

 3- edit a file from the shell:

  $ gnuclient -q somefile.txt
