On 08 Nov 2002 14:39:22 +0100, Borg.Enders_at_pdv-FS.de wrote:
>
> I want to implement a Session Manager for a program with a lot of windows,
> so that all windows are on the same pages, when i have shutdown fvwm2 and
> restarted it.
> So is there any possibility to get the Number of the page from within a
> C-program for some WindowWidget.
> By the way I'am using Xt and Motif.

I think you want in your program to force some placement of your
application windows. I don't think this is what users (at least some of
them) want.

You may make your program session management aware by using libSM.
Then hopefully the last used place will be used for your windows by a
compliant window manager when run under a session manager.

Or you may just let the user to decide where windows of your application
will be placed by not doing anything special. Then a user (if he wants)
may just define:

  Style YourApp StartsOnPage 1 2
  Style YourApp CascadePlacement
