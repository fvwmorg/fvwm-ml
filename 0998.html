%% Anthony Grossi <grossia_at_asterisk.sproj.psi.net> writes:

  ag> Does anyone know how to supress console messages from popping up
  ag> on the bottom of my window?? Everytime they appear I must
  ag> "Redraw" to clean up my desktop.

The simplest thing to do is run a program that grabs the console output.
Then the console messages will go to that program instead of the screen.
One of these is xconsole. You can also start an xterm with the -C
option to grab the console output.

See the man pages (this isn't a function of your window manager).
