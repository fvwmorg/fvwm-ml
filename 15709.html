Charles Hines wrote:
|>
|> RTFM. You should be able to this now via a function and the 'None'
|> directive. The example that shows this in fvwm2.man:
|>
|> AddToFunc MailFunction "I" Next [$0] Iconify -1
|> + "I" Next [$0] focus
|> + "I" None [$0] Exec $0 $1
|>
|> The 'None' line says only do this if no window matching $0 exists.
|> You should be able to use something like this with FrameMaker.

Yes, but if I use such a function as a button-box command, it won't
do the button depressing stuff because it only looks for "Exec"
commands. I don't think there's a general solution to this problem
short of adding support into the button-box module.
