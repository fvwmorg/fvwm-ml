> Works for me as expected. What is your full FvwmButtons configuration?

I don't see any difference if I add that either. There is a screenshot
at http://johan.svedberg.pp.se/files/tmp/FvwmButtons.png. Strange, I
must be doing something wrong. Here is the full relavent configuration:

Style "FvwmButtons" BorderWidth 0, ClickToFocus, NoHandles, NoTitle, StaysOnTop, Sticky, WindowListSkip

SetEnv FvwmButtonsBorderWidth 0
SetEnv FvwmButtonsHeight 32

PipeRead `printf "SetEnv FvwmButtonsWidth "; expr $[vp.width] - $[FvwmButtonsBorderWidth] - $[FvwmButtonsBorderWidth]`

*FvwmButtons: BoxSize smart
*FvwmButtons: Colorset 2
*FvwmButtons: Columns 4
*FvwmButtons: Font None
*FvwmButtons: Frame 0
*FvwmButtons: Geometry $[FvwmButtonsWidth]x$[FvwmButtonsHeight]+0-0
*FvwmButtons: Padding 0 0
*FvwmButtons: Rows 1
*FvwmButtons: (1x1+0+0, Padding 5 8, Swallow "XClock" 'Exec xclock -bg white -fg black -face "Bitstream Vera Sans:bold:size=8" -digital -strftime "%b %d %R" -padding 0')
*FvwmButtons: (1x1-0+0, Padding 0 3, Swallow "FvwmPager" 'Module FvwmPager')

Colorset 2 Foreground black, Background white, Shadow black
