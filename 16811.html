Hi Jamie

Aha, you're swallowing the pager. I'm going to assume you're using 2.0.42
because this happened to a lot of my users upgrading from 2.0.39.

> *FvwmButtons(3x1) - NULL SwallowModule Exec "FvwmPager" FvwmPager 0 2

You have to change 'SwallowModule Exec' to 'Swallow' here:

*FvwmButtons(3x1) - whatever Swallow "Fvwm Pager" FvwmPager 0 2
