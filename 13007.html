back to the GDM) with OpenOffice 1.1.

1) start ooimpress
2) create an empty presentation
3) create an empty text document
(using the menu New/Text Document)
4) type anything, select it, press CTRL+C
5) go back to the presentation using
the menu Window/<presentation> window
