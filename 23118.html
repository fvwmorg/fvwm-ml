>> Where should I actually put my bug reports? To the fvwm-workers

  dv> Bugs should be reported in Jitterbug.

Note that the fvwm-workers list automatically receives a copy of all
"incoming" bug reports, so there's no need to send anything to
fvwm-workers when you report a bug. Info on any particular testing you
do, as Dominik mentions, can be reported to fvwm-workers.

Thanks.

PS. Also note that your bug report will probably not appear immediately
    on the web! Jitterbug works via email, so your bug report will
    generate an email which needs to be processed before it shows up on
    the web.

    Don't despair! And don't send it twice! :)
