Actually, that's not a pager. It's a button bar, called `Wharf' in
AfterStep. A pager allows navigation of a virtual desktop, not
starting of programs.

The functionality you're asking about has already been done. FvwmWharf
is a module that comes with fvwm2, but doesn't compile by default. The
recent Red Hat Linux releases include FvwmWharf as part of the fvwm2
package.
