Viren,

Let me describe what I'm doing.

1. Select a command line login from the CDE login window.
2. Hit return once and get good old tty login ("hostname login:").
3. Login
4. Run openwin. You need to make sure that you have a ~/.xinitrc file and run
   fvwm from this file (/path/to/fvwm2). Openwin will start the X server for you
   and will start with your specific application reading this file. The window
   manager is just another program (like xterm or xclock) from that point of
   view).
5. I also put a soft link from my ~/.xinitrc to ~/.xsession since some (old)
    XDM login tools are looking for this file instead.


Good luck,
Yehuda.
