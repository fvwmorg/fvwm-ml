Hello,

> I have been using fvwm very happilly for a number of years. Recently
> I got a brand new Sun with 24-bit color frame buffer. My default
> installation still doesn't seem to give me access to more than 356
> colors.
>
> What should I do to access the 24-bit color realm and still enjoy
> fvwm?? Should I use a special Xserver?

I use the following command to start X on my Sparc system with a ffb and cgsix
frame buffers:

#!/bin/sh
#
# Start X using the Sun supplied X server and set the color depth for
# the ffb to be 24bits
/usr/openwin/bin/xinit -- /usr/openwin/bin/X :0 -dev /dev/fb0 defdepth 24 \
        -dev /dev/cgsix0 -auth /homes/boyda/.xsun.molecule:0

Tom.
