Please note that you're trying to compile on a Solaris 7 system with a
GCC build on a Solaris 2.5.1 system. This is very unlikely to work,
_especially_ with a version of GCC as old as 2.7.2.2.

I suggest you get a version of GCC compile on Solaris 7, and try that.
You can get prebuilt versions at sunfreeware.com, if you don't want to
build it yourself.
