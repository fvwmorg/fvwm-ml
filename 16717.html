On Thu, 9 May 1996, Alessandro Vozza wrote:

> I have just compiled and installed fvwm95.I then started it from xinitrc,
> and all seems to go well. But, fvwm can't find the varius .xpm icons, and
> the start menu show , instead of the icons, the name of the icon that are
> in .fvwm2rc95 file.I think I have Xpm libraries installed correctly, but
> I'm not sure. Anyone can tell me the correct place for the Xpm lib and
> other files related?

The installation process does not install the mini-icons by default, you
have to copy them manually to a /usr/local/icons or /usr/local/mini-icons
directory. If you copy them to some other place you have to update the
PixmapPath option in the .fvwm2rc95 config file.

Hope that helps,
Hector Peraza.
