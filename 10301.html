On Sat, 18 Oct 2003, Brian W.Kernighan wrote:

> How can I add a Windows like "Star Here" button to my Fvwm2?
>
> Now I have a root menu, and I want to bind it to a button("Start Button"),
> so When I click on it , the Root Menu will display on the screen ,
> just as all
> Windows "Start Here" button do.(It may use fvwmbutton, I guess.)
>
> Is there anybody can help? Thanks in advance!

I started off with fvwm95 and kept some of the features over the years
when I upgraded. One of these was the little-used Start button...except I
renamed mine "Slut" um...because I could.
There's a screenshot and a copy of my rc file on the fvwm.org web site, if
you're interested.
I think the part you're missing is this:

*FvwmTaskBarStartName Slut
*FvwmTaskBarStartMenu Slut

Or, well...you get the idea. I hope.
