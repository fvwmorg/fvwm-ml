This might sound like a stupid question, but while you're using X, did
you turn on numlock? I had the same problem, I'd be working and at
seemingly random points, all of my Meta key commands would stop working.
I finally realized it happened when I turned on numlock. There's an
explanation for it floating around if you're interested, but I'm not
completely sure I remember where I saw it. I think it's on fvwm.org in
the FAQ.

If this is the case, put the following in your .fvwm2rc file:

        IgnoreModifiers L25

That will keep the Meta strokes working when numlock is on.

- J
