I had exactly the same problem, (with an other application though)
untill I discovered that I had the BorderWidth set to 0...

this did the trick for me,
< Style "Fvwm*" NoHandles,BorderWidth 0,NoTitle,WindowListSkip,Sticky 
---
> Style "Fvwm*" NoHandles,BorderWidth 11,NoTitle,WindowListSkip,Sticky
Especially, if you are using dotfile, dotfile will set various
BorderWidths to 0 by default, I think the NoHandles ones.
You have to enable them in the menus explicitly.
Style * is not enough, you also need Style Fvwm*
