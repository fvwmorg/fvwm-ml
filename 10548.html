        What release of Java are you using? I don't know how the widget
hierarchy is set up, but I did notice that AWTApp and VendorShell no
longer got used for the app's Class and Resource as of 1.4. Both Class
and Resource seem to take their name from the Java class name of the top
level window; e.g., say I have a simple app with a class called MyClass
that extends JFrame (it also happens to be the main() class, but I
don't think that matters), both the Class and Resource will be named
MyClass.
