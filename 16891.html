>>>>> "Dan" == Dan Espen <dane_at_bae.bellcore.com> writes:

Dan> Tim Phipps <tim_at_hpcpbla.bri.hp.com> writes:
>> Is it possible to navigate around the menus using keyboard shortcuts?
>>
>> I'm moving from ctwm and this is the only thing I can't replicate.
>> With ctwm once a menu is popped from the keyboard you can move to
>> the menu items and select them just by typing the first letter of
>> the item name. I have gotten into this and it is very fast for
>> going down three or four levels of menu nesting.

Dan> Yes. An ampersand in the menu item name indicates the hot letter:

Dan> AddToMenu Window-Ops "--Window Operations--" Title
Dan> + "&Destroy" Destroy
Dan> + "D&elete" Delete

Dan> I think you need fvwm2, I'm not sure, but I don't think this was in
Dan> fvwm (1).

I believe the ampersand works in 1.xx as well.
