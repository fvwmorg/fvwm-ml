On Sun, Sep 14, 2003 at 07:21:51AM -0300, romildo_at_uber.com.br wrote:
> Hello
>
> I am starting a pager with the following commands in my
> configuration file:
>
> AddToFunc StartFunction I Module FvwmPager 0 3
>
> Style FvwmPager NoTitle
> Style FvwmPager BorderWidth 0, HandleWidth 0
> Style FvwmPager Sticky, StaysOnTop
> Style FvwmPager WindowListSkip, CirculateSkip
>
> *FvwmPager: Geometry x$[MyTaskBarHeight]+0-0
>
> Now I want to determine its width when calculating the
> position and size of an IconMan window, which I want
> to put side by side (to the right) of the pager.
>
> How can I accomplish this?

Use FvwmButtons to swallow both, the FvwmIconMan and the
FvwmPager. This way, they both get the same height without any
manual intervention.
