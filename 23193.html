Hello,

With a former release (2.0.45), I had :

EdgeResistance 250 50
EdgeScroll 0 0

This allows me :
- to switch from one page to another by moving a window across border of
screen
- to NOT have automatic scrolling when mouse reaches border of screen.

With fvwn 2.1.45, this behaviour seems to be impossible despite the man
page:

          Note that, with "EdgeScroll 0 0", it is still possible
          to move or resize windows across the edge of the
          current screen. By making the first parameter to
          EdgeResistance 10000 this type of motion is impossible.
          With EdgeResistance less than 10000 but greater than 0
          moving over pages becomes difficult but not impossible.

This behaviour was very useful for new users. I use Solaris 2.5.1
