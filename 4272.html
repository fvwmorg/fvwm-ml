I can't speak for focus-link. Run an FvwmCommandS to accept external
commands and then FvwmCommand to stream events to a perl script which
watches for stuff. WHat stuff? Bloody Mozilla, like you do!

Anyway, this lets you watch the window titles, which suffices. When you
see the right ones you spit commands back to the FvwmCommandS.
