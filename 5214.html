Are you sure you installed the cvs version as you said (make install)?
There is a FONT section in the man that describes Free Type font loading.

Try this line (such font names obviously do not work in 2.4.7):

  Style * Font "xft:Luxi Serif:Bold:pixelsize=14:minspace=true"

> If you are using FVWM from CVS, the file name for the man page
