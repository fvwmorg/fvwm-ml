On Tue, 2 Dec 1997, Bruce Stephens wrote:

> Nidhogg_at_mnd-net.no said:
> > What I'm wondering is, should I use 1.24 or the latest 2.x ? As 1.24
> > is the latest "official" version, that's what I have. But does 2.x use
> > the same configurations ? Like, can I use the old config options + new
> > ones, or are the two versions not 100% compatible ? Also, are there a
> > lot of bugs in 2.x ?
>
> Use the latest 2.x (2.0.46, I think). The configurations are different, and
> doubtless there are bugs in 2.x, but you aren't likely to notice them. RedHat
> comes with TheNextLevel, a set of configurations for fvwm95 (a modified fvwm2)
> which you may find convenient to start off with.

There is a utility out there for converting your files from version 1 to
version 2. I forget what it is called.

I disagree about TheNextLevel; I don't like it at all. And it is fvwm-95
I believe. Also, don't quote me on it for a few more days, but I'm pretty
sure Red Hat has abandoned TheNextLevel in Red Hat 5.0.
