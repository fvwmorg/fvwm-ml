Paul.Olberz_at_mail.state.ky.us writes:
> I have downloaded the latest version of FVWM to my pc which runs Windows XP
> home edition. I've unpacked it into its various folders but I can't figure
> out how to 'compile' it or run it. Can anyone explain what I need to do?

Fvwm is an X Windows window manager.

X Windows is not a flavor of Windows XP.

If you really want to run X Windows applications,
install the Cygwin tools first. Then you will be able to install Fvwm.
