David Barth says:
>
> Hello,
>
> I've seen a popup-patch on Jarl's page, but I don't see how to use it...
> Could you (Jarl or anyone else) explain the syntax to popup a menu from
> a module.
The syntax is the normal 'Popup Somemenuname', but for the menu to popup
automatically the pointer has to be free, i.e. if you want to get the popup
on a ButtonPress, you have to do an XUngrabPointer and then do a SendInfo()
to fvwm. If you wait for a ButtonRelease first it will not work.

> I hacked fvwm to make it parse "Popup "Menu" x y , popping up a menu at
> absolute coordinate x,y. Does this patch allow the same ?
No, my patch just puts the menu where the pointer is.
