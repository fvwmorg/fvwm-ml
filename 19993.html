Hi,

I have just compiled fvwm 2.0.45 for AIX 4.1.4.0 after using it for a
long time under SunOS.

fvwm seem to work until it tries to Exec something. Then it locks.
Swallowing anything except fwvm modules in FvwmButtons cause the same
lock. Any ideas?

I had a problem compiling it, xmkmf complains:

[zeise] 4:fvwm-2.0.45>xmkmf
mv Makefile Makefile.bak
imake -DUseInstalled -I/usr/lib/X11/config
"./Imakefile", line 51.1: 1506-215 (E) Too many arguments specified for macro NamedTargetSubdirs.

Line 51 reads:
MakeSubdirs($(SUBDIRS))

Instead I used ./MakeMakefiles, it caused the same error in all
subdirectories, but Imake had done _something_, at least the libXpm
directory had successfully been put into the Makefiles, and fvwm
compiled without problems.

Any help will be appreciated. I can live with mwm for a while,
but... :-)
