On 18 Feb 2003 05:05:36 +0300, Alexander Kotelnikov wrote:
>
> Is there any way to avoid default actions for Mouse 4 and 5 in a
> manner similar to 'Key Tab A M -'. It seems that 'Mouse 4 R A Nop'
> works, but there is always subtle difference between no action and
> null-action...

The following should do it for you:

  Mouse 0 R N -
  Mouse 1 R N Menu MenuFvwmRoot
  Mouse 2 R N Menu MenuFvwmRoot
  Mouse 3 R N Menu MenuFvwmRoot

I agree here, this should be the built-in default.
Actually, I would only use "Mouse 1 R A" instead of "Mouse 0 R N".
