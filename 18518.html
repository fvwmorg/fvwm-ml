Jamie Sparks wrote:
>
> How does one cause a window to be automatically iconified upon login?

AddToFunction "IconifyOrElse"
+ "I" Next ["$0" !iconic] Iconify 1

AddToFunction "InitFunction"
+ "I" IconifyOrElse WindowName

Where 'WindowName' is the name of the window you want to iconify. You
can get the windowname using the FvwmIdentify Module.
