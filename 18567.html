On Sun, 1 Dec 1996, Ted Freitas wrote:

> this is great when I press the left shift key the computer will back space
> for me! great! well I read a message about keymaps and stuff and it showed
> that keymap 50 = BackSpace but it looks like it maped my left shift key,
> how do I set it up so that I can use the backspace key in Netscape and in
> tkgoodstuff?

AFAIK it's keymap 22. Someone correct me if I'm wrong... :)

> and for my second question how do I get the autoraise and auto focus
> functions to work?

Add this to .fvwm2rc. It'll raise windows 500 milliseconds after they got
focus.

AddToFunc "InitFunction"
+ "I" Module FvwmAuto 500

Use the following to give focus to windows the mouse is in, it won't
un-focus when you get above the desktop.

Style "*" SloppyFocus

> what the heck and for my third and final question why does the cut and
> past function not work? I can not use the shift-insert to past some text.

I always cut/paste by selecting text with the first mouse-button and
pasting it with the second/middle button. BTW: That is COPY/paste. I
wouldn't know how to do CUT/paste.
