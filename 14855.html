> II. My pager in fvwm v1.24 was about half the size it is now (for the same
> of 3 virtual 2x2 desktops. What controls the geometry of its size?
The builtin pager of v1.24 is gone. You have to use FvwmPager instead.
If you only want to see the contents of the current desktop, iconify
the FvwmPager.v1.24 or put this in your .fvwm2rc:
*FvwmPagerStartIconic

Then you can control the size of the icon by doing this:
*FvwmPagerDeskTopScale 35

Unfortunately it doesn't work to specify the position of the icon
like this:
*FvwmPagerGeometry -1+1
If you happen to find a way to do that be sure to let me know, please.
