Mark Borges (mdb_at_cdc.noaa.gov) wrote:
:
: So, does anyone now if the download protocol/mechanism to X terminals
: is generic enough to accept third party window managers? Or is this
: something proprietary such that HP would have to effectively rubber
: stamp any window managers that are downloaded?
:
Sorry but you can't download 3rd party window managers to run locally
inside the X Terminal. The primary reason is that the code must be
compiled for the intel 960 and even then, you'd need access to HP's
internal libX11,libC libraries.

The Aptrex terminals will load Enware software and you can have the
following local window managers: dtwm,mwm,vuerx,vuewm,olwm, and twm.
(Twm is provided but not supported).
