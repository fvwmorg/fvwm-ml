Jon A. Christopher <jon_at_amanda.tamu.edu> wrote:
>
> I'm wondering why SaveUnders was removed from FVWM.

While we're on the subject, I'll add that I also miss the other option
that was removed, "BackingStore". As part of my work, I often find
myself exporting windows to my screen, over a 64K ISDN connection.
Believe me, backing store makes quite a difference for these apps, even
though they are simple xterms! They don't request the backing store
themselves, though, so having FVWM do it on their behalf is quite nice!

I find myself switching back to fvwm 1.24 whenever I need to run these
apps over my ISDN connection, then switching back to fvwm2 when
finished.

Is there some way to add BackingStore back again, especially as a Style
option? That would be super! There are some windows I don't want to
enable BackingStore on, and so a Style option for that would be great.
