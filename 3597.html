The GNOME panels explicitly forbids fvwm to move it by settin one
of these (braindead) GNOME hints. Normally, you can only move it
by pushing a button somewhere inside the panel and dragging.

However, using

  Style panel GnomeIgnoreHints

should have allowed you to move the panel, but didn't. There was
a well hidden bug in the handling of some gnome hints, but that
should be fixed as soon as I can connect the fvwm cvs server to
commit my patch.

But -- the patch won't help you at all. Although you can move the
panel araound as you like, the panel will snap back to the
original position as soon as you release the mouse button :( I
guess you'll have to flame the gnome developers about this ;-)
