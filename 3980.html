Srivathsa Rao U <srivathsa_at_india.ti.com> writes:
> hello,
>
> I have downloaded the fvwm 2.4.4 . Now im using 2.0.43 , how to
> upgrade this to 2.4.4 . I need to install it or any other way to upgrade?

After you install 2.4.4, you will find it contains 2 conversion
scripts, fvwmrc_convert and fvwm24_convert. These 2 scripts
will change your .fvwm2rc file so that it works with 2.4.4.

Read the man page for both scripts.
