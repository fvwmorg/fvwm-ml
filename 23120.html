There appear to be TWO official FVWM pages, which I have arbitrarily
classified into old:

http://www.hpc.uh.edu/fvwm/ (which has NOT been updated)

and new:

http://fvwm.math.uh.edu/ (which HAS been updated)

I realize that it is unreasonable to maintain two different pages, but
there should at least be a notice (prominently) displayed on the old page
to point people to the new page.

I was unable to find the e-mail addresses of the maintainers of either
page (and so I apologize for sending this to a list I am not a member of).
I also noticed that the FAQ on the new page refers to the old page as the
'official site'! What is going on? Finally, there is not even a version
number or date on the FAQ!

The documentation for FVWM has always been a bit sketchy, but this is
ridiculous.

If anyone wants my help with this documentation, feel free to ask. I
could integrate these two web pages and add the missing information to the
FAQ. Right now all you are doing is alienating the user community!
