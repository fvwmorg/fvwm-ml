> The text in my FVWM button looks funny. It has a square following every
> letter.

Not sure how helpful this is, but ...

On my Redhat 9 machine I edited /etc/sysconfig/i18n to look like this:

LANG="en_AU"
SUPPORTED="en_AU.UTF-8:en_AU:en"
SYSFONT="latarcyrheb-sun16"

Presumably there is a similar config file for your own system.
