This is still about the problem I described last time about not able to
get panels to work in FvwmButtons. I finally got it working. The problem
seems to be that I had to specify the direction of the Panel :
*FvwmButtons(Panel (up) test,Title test)

Although in the man page it says that up is the default, it doesn't seem
so in my case. But anyway, it's working. I plan to use this instead of
FvwmWharf as I did before. It would look much better actually if it has
animation.
