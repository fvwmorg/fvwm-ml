Werner Fleck <wf_at_zaphod.prompt.de> writes:
> Hi,
>
> Can anybody tell me how expensive it would be to extend the
> FvwmPager module so that it displays miniatures of the real
> windows. This way if you are using your workstation primarily for
> diplaying pictures you would have a nice way to navigate on your
> virtual desktop.

There are a number of programs that already draw their own icons.

For pictures, the "display" program in the ImageMagick package does
what you want.

The one I am most familiar with is XEmacs which includes the live-icon
package which does a sketch of the XEmacs window. Since XEmacs is
appoximating characters to pixels, and ignoring title bars and
menubars the logic is fairly simple. However, if the window manager
attempted to draw the icon, I think it would fail miserably. When it
went to reduce a spot in the window containing a character to a pixel,
the color in the background would overwhelm the color of the
character. So when it tried to draw a picture of the XEmacs window,
with all its glorious color, all you would see is the background,
menubar, toolbar and status line.

> If anybody has done or is working on this please let me know. Or

As I said, ImageMagick would be a place to start.

> maybe someone is willing to this if he gets some financial
> sponsorship?

My guess is that money could buy anything.
