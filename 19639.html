I would like to report the following "bug" with FvwmCpp and FvwmM4:
The man pages are out of date w.r.t. fvwm2.

In particular, the man pages indicate that the respective modules can
be invoked as follows:

        fvwm -f /usr/lib/X11/fvwm/FvwmCpp

        fvwm -f "FvwmM4 -m4-squote { -m4-equote } .fvwmrc"

Well, this is hooey in fvwm2. The correct form is only (and
fortunately) stated in the main fvwm2 man page as:

        fvwm -cmd "FvwmM4 .fvwm2rc"

(and likewise for FvwmCpp).
