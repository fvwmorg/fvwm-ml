I'm running FVWM 2.4.4-2 under RedHat 7.2 now. It took some work to
convert the dot file to work with the newer version. I think the default
version that RedHat 7.2 installs with is 2.2.5-4, but you can get the
newer version from www.fvwm.org. I suggest upgrading to the latest
version. After you update, run fvwm24_convert to convert the 2.2.x dot
file to 2.4 syntax. It won't pick up everything, though. You'll still
have to edit things by hand.
