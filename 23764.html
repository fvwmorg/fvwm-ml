Fvwm and fvwm2 are basically the same thing. The older version 1 (fvwm)
is not actively developed, distributed or supported anymore. A few days
ago we had the first release of version 2 (i.e. stable, non-beta release).
Please check our web site:

  http://www.fvwm.org/

I assume that you were using fvwm95 (a $%^*#_at_ 95 like spawn of fvwm2)
since you got this task bar by default. In fvwm2 itself this is
called FvwmTaskBar and is started in your .fvwm2rc with something like this:

  Module FvwmTaskBar

Removing this line will remove the task bar.

For your desktop dimension you put this command in the .fvwm2rc:

  DesktopSize 1x1

And swallow a FvwmPager module in your FvwmButtons bar. Basically
you have to start the pager with

  Module FvwmPager 0 9

which tells the pager to display the desks 0-9. Please check the
manpage for FvwmPager for details. If you need further assistance
I can send you an example.
