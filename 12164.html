I am trying to have 2 FvwmPager: one for Desk and one for Page

I want the one for desk show only desk and the one for Page show Page of
current desk.
I have the following problems:
- The Desk pager show page
- The Desk pager must have 2 Columns and the Page pager must have 1
columns. How make differents configuration? How use a module 2 times with
differents configuration?

I use fvwm 2.5.10 and the following config for the pager:

Style "FvwmPager" StaysOnTop

...

########################### Pager #########################################
*FvwmPagerBack #700070
*FvwmPagerFore bisque3
#*FvwmPagerFont -adobe-helvetica-bold-r-*-*-10-*-*-*-*-*-*-*
# turn off desktop names for swallowing in above button bar example:
*FvwmPagerFont none
*FvwmPager: Columns 2
*FvwmPagerHilight #cab3ca
*FvwmPagerGeometry -1-1
*FvwmPagerLabel 0 Misc
*FvwmPagerLabel 1 Maker
*FvwmPagerLabel 2 Mail
*FvwmPagerLabel 3 Matlab
*FvwmPagerSmallFont 5x8
*FvwmPagerBalloons All
*FvwmPagerBalloonBack Yellow
*FvwmPagerBalloonFore Black
*FvwmPagerBalloonFont lucidasanstypewriter-12
*FvwmPagerBalloonYOffset +2
*FvwmPagerBalloonBorderWidth 1
*FvwmPagerBalloonBorderColor Black
*FvwmPagerMiniIcons

...

#Pager
*FvwmButtons: (1x2 Frame 2 Swallow(UseOld) "FvwmPager" "Module FvwmPager * *")
*FvwmButtons: (1x2 Frame 2 Swallow(UseOld) "FvwmPager" "Module FvwmPager 0 3")
