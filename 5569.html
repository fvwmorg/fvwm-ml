On 16 Jul 2002 11:50:24 -0400, Bob Krovetz wrote:
>
> I used to use fvwm and when I clicked on a window in the pager, it would not
> only switch to the desktop in which that window appears, that window would
> also get the focus. I'm now using fvwm2, and when I do the same thing I
> still switch to the desktop, but the focus does not change until I click
> on that window. Can anyone tell me what I need to do in order to have the
> focus change when I click on a window in the pager?

I think that currently the only way to achieve what you want is to click
on a mini window in the pager twice, once using the mouse button 1 and
then (without moving a pointer) using the mouse button 2.

Maybe sometimes clicks in the pager will be configurable.
