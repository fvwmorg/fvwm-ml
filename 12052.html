When I use windows list to change windows using alt-tab:
Key Tab A M WindowList (CurrentDesk) Root c c NoDeskSort, CurrentAtEnd, IconifiedAtEnd, ShowPageX
The pointer is moved to the top-left corner of the new window. How can
I make it stay at the same location? Its quite annoying having to move
it back to its place every time. This also causes focus problems with
border-less windows and sloppy focus. If I have to move the pointer, a
better behavior would be to move the pointer to the center of the new
window, this would also solve the focus problems I had with sloppy
focus and focusing a window that is not under the mouse when the mouse
is over a different window.
