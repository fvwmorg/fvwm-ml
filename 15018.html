Hello,

I would have to agree with Rob. I have several hundred xpm icons in my
directories and I am running on an 8-bit display, so if all of these are
read in then I would absolutely no colors left.
Remove your cons one by one until you find the ones that are causing the
problem.

By the way, Rob, your solution for the Sun IPC's worked fine. Starting
the X server with the -zaphod worked great. The sideways page flipping
problem was cured. However, we noticed that there were two processes of
everything still running, such as two fvwm2 processes and two consoles,
etc. Once we started fvwm2 with the -s option that was also cured. It
seems like this is a problem for all Sun IPCs, so it might be helpful
put a note about this in the FAQ's.
