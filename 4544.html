Dan Espen wrote:

> I'm not sure what "acc" is though. We have Sun's compiler
> here, but I don't have anything named "acc" in my path.

We bought Sun's Forte package (formerly Workshop). On our server, it's
/opt/SUNWspro/WS6U1/bin/acc. We have that directory PATH'd. When it's
installed, "man cc" references /usr/ucb/cc, which is a script that wraps
/usr/ccs/bin/ucbcc, which is a symlink to /opt/SUNWspro/WS6U1/bin/acc.
