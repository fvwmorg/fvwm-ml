hey, all

since i was keeping up with alot of the latest patches, especially from
Andrew and Brady, i put up my source tree to make some of the latest
add-ins available to all. this is meant just a single access point for
some of the nicer additions, and should help those who have are having
trouble keeping up with all these pieces. even after the next beta, i'm
sure there will be add-ins that pop up, so i will try to stay on top of all
of it! :-)

the site should generally have the latest patches available, except on
weekends when i'm not usually available (the machine is at work). right
now, the tree is patched up with the UseDecor v9b patch, the windowshade
patch v1 (??), and the MiniIcons patch ( v??), hence the patch file is
called 'fvwm-2.0.43.tbsp9b.wsp1.miniicons.diff', and also includes Andrew's
Cascade and Tile modules, and Brady's IconMan v9 module.

and as i said, right now the site is just a link to the source tree, so
there's no pretty index to guide you around. i'll try to set something up
before the end of the week, though.

it's laid out somewhat sensibly. there's a 'patches' dir to hold patches,
'modules' for the modules, etc. 'fvwm-2.0.43' will be the current tree,
with some others probably lying around. the machine is a Linux/Pentium
box, so that's what any compiled code will be for.

the URL is <URL:http://phantom.iweb.net/fvwm/>

let me know of any suggestions, problems, etc.
greg
