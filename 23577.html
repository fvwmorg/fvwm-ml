FvwmScrollFore, default grey seems te be ignored and FvwmScrollBack,
default black, defines the color for the window background, scrollbars,
buttons. So when I don't specify the background color, I don't see
anything but a black region. If you specify another color, all items got
that color (you can still see the bars and buttons thanks to shadows
etc.) Setting the foreground color manually doesn't help and I can't
discover any place where that color is used.
