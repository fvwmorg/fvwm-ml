  You have to edit the file 'modules/FvwmPager/x_pager.c', changing the
line containing "LineOnOffDash" to "LineSolid". Now recompile fvwm and
enjoy. I do this every time I upgrade because I think it looks better,
too. I think this should be a config option... if I weren't too busy, I'd
do it myself.

Good luck,
-jason
