I assume you're talking about "sunfreeware.com". They tell you on the
first page in a very ugly color to read the Downloading/Installation
instructions before you download any software.

The instructions tell you to use Sun's pkgadd program, which must be
run as root, and won't tell what it is going to do. I don't like to
mess with my system without knowing what's going on, so I usually get
the source code and compile it myself. If it's on sunfreeware, it
must be possible to compile it on solaris after all.
