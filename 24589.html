> I installed redhat 5.2 and the default window managers were fvwm95 and afterstep ... I looked around and liked fvwm. So I uninstalled the other fvwms with the -e oldfvwm.rpm type of command. I'm not quite sure if you are aware that fvwm95 and fvwm (aka fvwm2)
are different window managers. No development has been going on on
fvwm95 for about two years now, so plain fvwm is probably what you
want. If you feel uncomfortable with the red hat setup you can
fetch a copy of the latest stable release from our home page:

  http://fvwm.org

If you compile and install this package you will get a standard
installation in the standard location (/usr/loca/...). Be warned
that the deafult configuration isn't very nice yet (in fact it's
quite ugly). There is a section about RH setup in the docs/FAQ file
in the distribution.
