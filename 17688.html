Kees de Bruin wrote:
>
> Hello Fvwm(95) users,
>
> When I use fvwm 1.x or 2.x I can configure fvwm so that I only see the
> sticky icons and the icons for the windows on a specific desktop. However,
> the FvwmTaskBar module wants to show the active ans iconized windows of all
> desktops. Is there a configuration setting to let the TaskBar just show the
> windows related to the current desktop (and all sticky windows).
>

I dont think the TaskBar does this. But the latest FvwmWinList(blame
me) in Fvwm95 does what you are asking. Set the
FvwmWinListShowCurrentDesk option. FvwmWinList also has the MiniTitle
Icons and Highlights according to Focused window, like TaskBar.

I would Summit it to Fvwm2 also, but there is no #define FVWM95 for
ifdefs to wrap around the MiniTitle Icon stuff.
Or is there ? Hector?
