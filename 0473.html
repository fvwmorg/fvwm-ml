Hop Le <hople_at_cisco.com> writes:
> Hi,
>
> I'm kind of new to FVWM w-manager. I've just setup FVWM2 for my Sun
> Solaris machine and I'm wondering if there is a way to display an image
> (gif, jpg, etc.) as a desktop background. Is it possible to do that?

Yes, its possible. Unless you are using FvwmBacker, this is not really
an fvwm issue.

You can put an image on your background with "xv" "xloadimage", "xli",
and probably lots of other programs.

If you have 8 bit color, be sure to limit the number of colors used.

You probably want to put the command to set the background in
$HOME/.xinitrc.
