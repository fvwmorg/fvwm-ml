hello,

I'm running fvwm 2.4

and, after an uptime of 2 days, FvwmPager begin to crash when i change from
desk...

I had to add this line to make the pager re-appear :
Key F10 A S FvwmPager * *
(and i have to press shift-F10 when the pager crashes)


relevant parts of .fvwm2rc :
--------------------------------


Style "Fvwm" Sticky
Style "FvwmPager" Sticky


Style FvwmPager NoTitle
OpaqueMoveSize 10
Style "*" SloppyFocus
*FvwmPagerBack CadetBlue
*FvwmPagerFore Black
*FvwmPagerFont none
*FvwmPagerHilight Lightgrey
*FvwmPagerGeometry 200-5-2
*FvwmPagerSmallFont -adobe-times-*-r-*-*-12-*-*-*-*-*-*-*
*FvwmPagerDeskTopScale 32

DeskTopSize 10x7

FvwmPager * *
#AddToFunc "InitFunction" "I" Module FvwmPager * *


*FvwmPagerGeometry 230x120+1037+890

Key F10 A S FvwmPager * *

Key 1 W C MoveToDesk 0 0 0 3
Key 2 W C MoveToDesk 0 1 0 3
Key 3 W C MoveToDesk 0 2 0 3
Key 4 W C MoveToDesk 0 3 0 3


Key 1 A M Desk 0 0 0 3
Key 2 A M Desk 0 1 0 3
Key 3 A M Desk 0 2 0 3
Key 4 A M Desk 0 3 0 3

EOF

Is there a way to print debug information ? when the pager crash ?

thanks

xavier
