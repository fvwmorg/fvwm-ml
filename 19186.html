>>>>> Henry McNair writes:

H> I have managed to build fvwm2 on HPUX 10.10. It seems to be acting
H> very well for me. However, I have a couple of questions. I
H> am unable to build FvwmAudio because I am missing the header
H> file rplay.h. I was wondering where I can get this header file
H> and what additional functionality does it provide? Do I need rplay.h?

no you don't need it.
If you
        #define HasRPlay NO /* YES */
in Fvwm.tmpl, FvwmAudio compiles without rplay support.

for the rplay sources, the man page says:
          rplay can be obtained via anonymous ftp at
               <URL:ftp://ftp.sdsu.edu/pub/rplay>
          or <URL:ftp://ftp.x.org/contrib/audio/rplay>
I checked it last week.
It worked then.
