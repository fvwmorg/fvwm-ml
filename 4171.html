I have a suggestion for the pager.
I would like if it were possible to get the current desk marked by
surrounding that area in the pager by a frame in some colour.
As it is now it's only possible to either designate a certain bitmap or
colour for the background or by using titles.

My suggestion would resemble the way it's done in IRIX (4dwm).

I looked at the source for FvwmPager, but I'm not exactly sure where the
drawing of the contents is done.
