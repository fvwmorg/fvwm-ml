Since you don't have a root window to click on, you'll have to make a
keybinding to bring up the menu (I don't think there is one by default.)
 If you used the setup form when you first installed fvwm, it probably
created a .fvwm2rc for you (look for ~/.fvwm2rc or ~/.fvwm/.fvwm2rc),
and in that case you can just add something like:

Key F12 A M Menu MenuFvwmRoot

to the .fvwm2rc, which binds meta (M) + f12 in any (A) context to Menu
MenuFvwmRoot (which is defined in the configuration that the setup form
generates.)

Hope that helps,
