On Thu, Nov 08, 2001 at 12:56:42PM +0100, Ripalda-Marin, Miguel-Angel wrote:
> Hi,
>
> I have just migrated to 2.4.3 version and have some trouble with my .fvwm2rc
> file, because this new version doesn't seem to accept the following option:
>
> Mouse 1 R N -
> Mouse 2 R N -
> Mouse 3 R N -
> Maybe this command has become deprecated in the new version. Do I have to
> change it? Best regards and thanks in advance,

The "-" command in a binding tells fvwm to remove the binding
that was created with exactly the same parameters. In 2.4, the
default binding command is

  Mouse 0 R N Menu MenuFvwmRoot

You can't remove this with

  Mouse 1 ...
  Mouse 2 ...
  Mouse 3 ...

but have to use

  Mouse 0 R N -
