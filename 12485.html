Hi.
this is how i do it:

AddToFunc StartFunction
+ I Test (Init) Exec exec fbsetbg -l

fbsetbg automatically sets the last used wallpaper after you startx :)

you can get it here:
http://www.xs4all.nl/~hanb/software/fluxbox/fbsetbg.html


This is Better because the path to the image isn't static, so you dont
need to edit the config everytime you change wallpaper.
