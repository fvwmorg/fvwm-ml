  pk> "icons.c", line 622: syntax error before or at: xpm_attributes

  pk> do i need to get a copy of this header file?

If you want to compile FVWM with pixmap support, you need to obtain the
Xpm library and install it (which will give you the xpm.h header, too).

You can get Xpm from ftp://ftp.x.org/pub/ somewhere (I forget exactly).

Or, modify the configuration of your fvwm and disable pixmap support.
