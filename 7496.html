On Thu, Jan 09, 2003 at 01:03:27PM -0600, Barry A. Warsaw wrote:
>
> Hi all, please CC me as I'm not on this list.
>
> I hate application icons. All I want when I iconify a window is to
> see a nice little rectangle with the window name in it. I must be a
> bonehead, because I can't figure out how to make this work. I've
> tried to use a style with "IconOverride,Icon unknown.xpm" but that
> doesn't seem to work. E.g.:
>
> Style ** IconOverride,Icon unknown.xpm
>
> but it still seems like apps like xchat and gaim still show me
> pictures when they're iconfied. I've tried "Style * ..." and
> "Style "Gaim" ..." but those don't help much either.
>
> FWIW, I'm using Fvwm 2.4.14.
>
> Any assistance would be greatly appreciated.

The icon selection code needs some clean up. Especially, it's not
easy to disable icon pictures completely. There are two
possibilities:

 1) Explicitly set the Icon style of every application that annoys
    you:

      Style mozilla Icon non_existing_icon.xpm
      Style emacs Icon non_existing_icon.xpm

 2) Make a completely transparent xpm icon that is 1 pixel high
    and as wide as you like the icon titles to be displayed, then
    set

      Style ** my_icon.xpm
      Style * IconOverride

    To be safe, do this near the top of your .fvwm2rc and never
    issue another "Style * Icon" or "Style ** Icon" style again.
    They seem to interfere with each other, depending on the
    styles that were set in between.
