Version 1.2 of the "Iconboxes patch" is available at:

ftp://ftp.hpc.uh.edu/pub/fvwm/version-2/unofficial_patches

This patch implements multiple icon boxes per style, icon grids and
full icon fill direction control.

Changes in version 1.2:

1) When filling right to left, icons used to go outside left boundary.

2) Now icons can now be filled in any direction, creating either rows
or columns.
