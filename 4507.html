On 11 Mar 2002 09:19:59 +0100, Uwe Pross wrote:
>
> Is there any way that certain windows are not shown in FvwmPager?
>
> I would like to keep track of some files using aterm with tail -f. Using
> the Styles Sticky, ClickToFocus, StaysOnBottom, HandleWidth 0,
> BorderWith 0, WindowSkipList do exactly what I want, since I can use
> copy and paste within this aterm which is not possible by using
> root-tail.
>
> I would prefer if those windows would not be displayed in FvwmPager.
> Can anybody help me to get this work?
>
> (Currently I use fvwm2 version 2.4.3, maybe I have to upgrade)

You may upgrade to 2.5.0 or some february's snapshot (the current week
snapshot is less recommended). Then use:

  *FvwmPager: UseSkipList

BTW, you may try NeverFocus for this window, so you don't lose focus
when pasting from it. You also want CirculateSkip if you use Next/Prev.
