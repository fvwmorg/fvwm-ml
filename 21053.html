Hyoung-Kee Choi <hkchoi_at_cc.gatech.edu> writes:
> Hi
> How do you use AddButtonStyle
>
> I got an error like
> [FVWM][executeModule]: <<ERROR>> No such module /usr/local/public/bin/
> AddButtonStyle
>
> How would you handle it?

AddButtonStyle is not a module, its an fvwm2 command.

If you have shown us the literal output, I would have to guess that
you are not using fvwm 2.0.46, since in that release the message would
read like this:

[FVWM][executeModule]: <<ERROR>> No such module 'xxxxxx'
 in ModulePath '/usr/dje/fvwm-2.0.46/fvwm/'

Make sure the fvwm2 you are running matches your documentation. If
that doesn't help, show us the lines in your .fvwm2rc that apply.
