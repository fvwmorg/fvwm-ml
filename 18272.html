sylvain_at_ctron.com (^8= Eric T. Sylvain =8^) writes:
> I would like to have my shell resize whenever I do a
> resize of a window with FVWM2. I know I can use tcsh
> and the shell would do this for me, but is there a way
> to get this functionality by using fvwm2 to direct
> a command to the shell connected to my current xterm?

When resizing an xterm window, your application should receive
SIGWINCH signals telling it about the change. Sometimes, this does not
appear to work; in most cases that is because you run a "telnet" or
"rlogin" session in the window and your network software does not
properly transmit the signal. Try switching between "telnet" and
"rlogin", and check if that makes any difference.
