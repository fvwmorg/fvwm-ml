I guess by virtual desktop, you mean FvwmPager?
It doesn't matter, I don't know how to do that.

> Is there a way to execute an application maximized?

Usually, an application will accept the "-iconic" flag,
as in:

Key N A SC Exec /local/netscape-4.79/.arch.os/bin/netscape -iconic

There is also the "StartIconic" style.

> Is there an way that fvwm does not load ConfigFvwmDefaults?
> or do I have to delete that file?

It will always attempt to load the default config file.

> Where is fvwm 2.4.4 - it is announced at the web page, but the links to
> 2.4.4 are broken and it is not at the ftp server?!

It was released a couple of hours ago, wait a little while.
I seemed to be able to download it, perhaps you should reload
the page.
