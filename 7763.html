On Mon, Feb 03, 2003 at 08:04:27AM +0100, Dory Thomas,RM wrote:
> _at_ FVWM-Team
>
> After I updated my RedHat 7.1 to 7.3 there was a Problem when starting the FVWM-Desktop (version 2.4.6).
> The X-Console reported the problem: Could not load font adobe-helvetia-12-*-*-...
> This Font is defined in the fvwmrc for the buttons _at_ the fvwm-desktop.

could it be that you use de_DE_at_euro ISO-5589-15 as locale? in this case
you need to install the transcoded fonts, or fvwm won't find them. i
don't know if redhat packs the transcoded fonts into their default font
packages or not. for debian the transcoded fonts are in the packages
xfonts-*-transcoded.

i had this problem too, when i switched my locale to de_DE_at_euro and fvwm
didn't want to start at all because it could not load fonts.
