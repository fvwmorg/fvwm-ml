Howdy Robert,

We have used it as follows:

In /etc/dt/config/C/Xresources:

Dtlogin*altDtName2: FVWM2
Dtlogin*altDtKey2: /usr/local/X11/bin/fvwm2
Dtlogin*altDtStart2: /usr/openwin/lib/xdm/Xsession
Dtlogin*altDtLogo2: oilio

We used the same fvwm2 configuration and XDM Xsession
file that we used with XDM. The logo can be any logo
that you desire!
