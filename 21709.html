There is no facility for copying decors directly in fvwm2, and it
would really need code modification. Maybe something along the lines
of (I think a copy would be more appropriate than something like
InheritDecor since changing the common decor later would not modify
any intended descendents, that would require major restructuring):

AddToDecor "MyNewDecor"
+ CopyDecor "CommonDecor"
