On 16 Nov 2000 10:13:15 +1100, Shao Zhang wrote:
>
> Is it possible to make the FvwmPager transparent with the background? Or
> use an xpm for the BackColor instead...

Yes, it is possible using both these methods, but only in the 2.3 series.

The first method uses colorsets. This is a tricky method, you should learn
FvwmTheme man page (including "Transparent" option) to do this.

For the second method, 'man FvwmPager', search for *Pixmap options.
