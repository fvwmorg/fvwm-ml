Hi,

ctwm has MovePack and ResizePack which I quite like. MovePack lets
you move a window `against' other windows. Just like EdgeResistance
allows you to easily move a window against the screen edge, MovePack
allows you to easily move a window against other windows, because
these other windows offer a little `resistance' against moving.

ResizePack is the same function for resizing which allows you to
resize a window till it fills the available space.
