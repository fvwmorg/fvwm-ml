I'm using fvwm-2.0.46 on a linux 2.0 system. Everything works fine except the
startup of fvwm. I'll set some default styles in my ~/.fvwm2rc which are used
by all the windows.

However, when I first start fvwm then some windows do not get the default
style (no, I don't have a special style entry for them). When I do a manual
"Recpature Screen" (using the menu), then these "broken" windows get the
correct style. It seems, that this happens to all windows, which open *before*
fvwm initializes. The windows which open *after* fvwm has started get all
their correct styles. But I'm not sure about this.

Has anyone a explanation?
