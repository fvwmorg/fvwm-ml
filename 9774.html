> I have not installed Gnome nor KDE; There is a way to
> change the pics,colours... used in the scrollbar,
> menubar... ? I don't want to use any external window
> manager.
>
> I have tried with Xdefaults, but the changes are related
> to separated applications. I want something more radical.

Changing pics is quite difficult for some applications for
others nearly impossible.

Colors for more than one x-application can be set using
x-resource with wildcards e.g.

*background: #f0f0f0
*foreground: #000000
*borderwidth: 3

For kde-applications there is a file ~/.kderc. Gtk (gnome)
applications use the settings in file ~/.gtkrc.mine. All
three settings could be made due shell-scripts or fvwm.
