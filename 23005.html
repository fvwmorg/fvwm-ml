The "How do I enable click to raise" issue has been mentioned a few times
in recent days. I use the following mouse binding:

Mouse 1 W M RaiseLower

to enable <Alt>-Button1 as a RaiseLower mechanism. Such things can fail
if the application also wants to use this as an input mechanism from the
user. I ran into this one when one application failed to run with "Cannot
access private resource" errors (I don't remember the exact syntax). It
was educational, at least. I hope this isn't entirely a waste of
bandwidth.
