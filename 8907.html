Michelle Konzack <linux4michelle_at_freenet.de> writes:
> Hello all,
>
> I like to make screenschoots from fullscreen but with some opened menus...
>
> How does it work ???
>
> On each keyboard there is a button 'Print' so I like to use it for this.
> Has anyone done this before ??? - and if yes, - HOW ???

If you want to use xwd, do something like:

key Print Exec /bin/sh -c 'sleep 10;xwd -root -out /tmp/xwd.out'

bind it to the printscreen key, hit the key, then activate the
menu.
