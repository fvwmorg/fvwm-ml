> Some programs start w/ their title bar off the top of the screen. I
> assume fvwm's getting confused about where 0,0 is. for the window.

> Is there a way i can tell fvwm that if i window goes above the
> screen, to push it back down....preferably generically?

I get the same behaviour with an older version of fvwm. I don't think
it's confused about where 0,0 is because you can grab the window, not
move it at all and let go and the window will snap to 0,0 on it's own.
Also, if you move it around and then move back so the little window
position indicator shows 0,0, then it does end up where it should.

It only happens when a new window opens.
