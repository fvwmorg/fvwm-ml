"Ville Pätsi" <drc_at_zoo-gate.fi> writes:

> Is it possible to make all windows to show their contents when you move
> them?(not showing the "bars") Ezppp uses this, do i have to modify all
> the progs source to make this happen? Or is the modification of fvwm
> config/source all that is required?

Put this near the start of your ``~/.fvwm2rc'' file and change the
number to 100 if you have a fast graphics card and CPU.

## This style of window movement draws the entire window as it
## moves, instead of the "rubber-band" outlines.
## It has a number (N) attached to it (default 5). If the window occupies
## less than N% of the screen, OpaqueMove is used. 0 <= N <= 100
OpaqueMoveSize 25
