If u use xinerama you can specify it like this:
+0+0_at_p the @p specifies that it starts on the primary screen

Other possibilities are _at_c (current screen containing the pointer), @g (global
screen, in your +0+0 screen it will start at the same position as regular
+0+0) and _at_X where X is the screen number you wish to display your window on.

Before this works you need to enable xinerama:
XineramaPrimaryScreen 0 (or 1 figure that one out yourself)
Xinerama On

man fvwm gives you some more info on this.
