Hi there :-)
Today I got FVWM 2.5.1 from CVS (checked out 2002/03/24 14:12:20), and
I found two slight problem with it:

1) When windows pop up, the MiniIcon in the titlebar only appears
   after the window gets focus. If it then loses focus, the MiniIcon
   remains... But if it starts without focus, it won't have the
   MiniIcon until it gets the focus...

2) While moving/resizing windows, there is on the top left a box with
   the geometry of the window being moved/resized... If there is an
   icon there also, when the box disappears, the part of the icon that
   was hidden by it becomes grey.

  Hope those little bugs will be fixed...
