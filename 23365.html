I'm having a problem whenever I restart fvwm2. I get some default
decoration on the windows instead of what is in my .fvwm2rc file. Yet,
everything works fine when I start fvwm2 the first time.

[FVWM][executeModule]: <<ERROR>> No such module
/home/fxb/fvwm2/libexec/fvwm/2.1.4/ BorderStyle
(the list goes on and on with all the commands related to style).

I don't quite understand why "BorderSytle" is interpreted as a module.

Does anyone have the same problem?

I'm using v2.1.4, but the same thing was happening with 2.0.46.
