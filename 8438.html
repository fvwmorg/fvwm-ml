You may use rpms linked on fvwm.org. Should work on RedHat 7, 8, 9.

> I did the rpm and all that. Which command says thar FVWM2
> is installed. But I can't swith from Gnome to FVWM2. I deleted .xinitrc
> and .xsession and ammended it to contain FVWM2. Nothing seems to help.

Do you boot your machine immediatelly into X?
If no (i.e. you use startx or xinit to run X), you may create ~/.xinitrc
script and chmod +x it, and it will be called.

There are other ways. Try to run switchdesk, it has "FVWM" as an option
if you have the old RH rpm installed, this should create ~/.Xclients with
a call to fvwm2.

The XFree86 setup is not principally different on different distributions.
You may always start FVWM.
