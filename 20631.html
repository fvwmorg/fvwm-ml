Hi people,

  This is my first post to this list. I have been reading the list
  and looking in the archives hoping someone would've mentioned my
  question but I have had no luck with finding any discussion on the
  subject.

  I used to use AfterStep till I noticed its unstability (compared
  to fvwm 1.xx). But during my use of AfterStep I got very used to
  how each window is represented to the user, meaning each window
  has a titlebar, and a bottom border with no side borders. I like
  this simple look.

  I wonder if fvwm 1.xx or 2.xx are able to emulate this look? If
  so, can anyone let me know how I can achieve this look and feel?
  If not, how hard would it be to implement this feature to allow
  users to define in .fvwm2rc how they would like their borders to
  be in the next fvwm 2.xx releases?

  Thanks!
