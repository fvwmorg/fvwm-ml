Nick Fortune <nick_at_nymar.demon.co.uk> writes:
> On Thu, Apr 28, 2005 at 03:04:50PM -0400, Dan Espen wrote:
> > If you are setting ModulePath in your .fvwm2rc, don't.
>
> Does it break things if we do?

Only if you install a new release and the modules are in
a new location.

> What should we do instead
> if we have non-distribution modules and can't (or don't want)
> to install them in FVWM_MODULEDIR?

Append or prepend to the ModulePath:

ModulePath ${HOME}/lib/fvwm/modules:+

Note the colon, plus sign.
