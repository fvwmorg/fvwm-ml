On 19 Oct 2002 21:55:30 -0700, Denis Perelyubskiy wrote:
>
> 2.5.4 (cvs) Fvwm:
>
> is there any way to make windowlist switch between windows
> back and forth on alt-tab. For example, here is what I am
> using now:
>
> Key Tab A M WindowList (CurrentPage) NoGeometry, CurrentAtEnd, IconifiedAtEnd
>
> This makes it convenient to cycle through windows.
> However, if i have windows a, b, c is it possible to go from
> window a to window b by hitting alt-tab, and then back to
> window a by hitting another alt-tab?
>
> Holding alt and hitting tab repeatedly would then cycle
> through the rest of the windows (it does so anyway)

You should be using a not up to date cvs version, because the current
one does what you describe. This was broken two weeks ago (and in 2.5.3).

As something totally unrelated, you probably know that you may center
the window list menu:

  Key Tab A M WindowList (CurrentPage) Root c c NoGeometry, CurrentAtEnd, IconifiedAtEnd

or maybe even:

  Key Tab A M WindowList (CurrentPage) Window c c NoGeometry, CurrentAtEnd, IconifiedAtEnd
