| to echo asterisks during entry? Nothing important, one user

The xdm xlogin widget won't echo anything during password entry
because it exposes the password's length. I briefly checked the
xdm man page to see if there was an resource that could be set to
enable feedback on password entry but there doesn't appear to be.
You'd probably have to hack it into the source if you really
wanted it.
