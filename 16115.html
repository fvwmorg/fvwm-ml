Well, conisdering the flurry of activity on the list today, I just
thought I'd put in my 0.02$ worth.

As some of you may know, a version of XFree86 has been released for
OS/2. The code is considered ALPHA, but has in fact proven to be
quite stable. It can be found at the usual XFree ftp site and
at its mirrors. You can also check out
http://borneo.gmd.de/~veit/os2/xf86os2.html
for the latest info and list of bugs, ports of X apps to OS/2 etc.

I did some porting work on fvwm2, the main problem was getting the
pipe connection to modules to work. The more interesting modules
work, and fvwm2 seems to work quite well. Hopefully I can clean up
the patches and send them to Chuck for inclusion in the standard
distribution.

You can find fvwm2 at:
http://aix1.uottawa.ca/~s521936/xf86ports.html

The feedback I have received has been very positive. Again, congrads on a
great window manager!



Below is the original announcement I made on the XFreeOS2 mailing list:

________________________________________________________________


Version 2.0.41 of the fvwm window manager is now available. This new
version includes support for modules, and includes a few of the more
interesting modules, such as:

FvwmPager: virtual workspaces and virtual screen support.
FvwmButtons: Button bars, can also "swallow" other windows.
FvwmTalk: provides interactive interface to fvwm

and a few other modules. Note that with the above modules, the
customization options are nearly endless.....

It is available at the following address:

http://aix1.uottawa.ca/~s521936/xf86ports.html


A few notes on this release:

- The XPM library is REQUIRED. Check out Holger Veit's page to get it.

 
- The fvwm version 2 code is still somewhat in flux (fvwm2 is still
beta). Expect updates to fvwm, especially for the modules.

- Following the convention set by the fvwm developpers, the new
version is called "fvwm2.exe" to avoid any name clash with the old
version. Note also that the configuration file is called ".fvwm2rc"
and is located in /xfree86/lib/X11/fvwm2
You should look through this and customize to your taste.

- The archive includes a collection of pixmaps.
