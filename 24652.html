Hello,

Having made the switch from tvtwm, there's one thing I miss, namely
having a dynamic number of icon managers. With tvtwm I had an icon
manager for each host I was logged into. The config file 'IconManagers'
entry defined about 60 icon managers, one per host, but if I was logged
into 5 hosts then I'd only see 5 icon managers. With fvwm2 I'd be seeing
60 icon managers all the time.

Is there a way of avoiding this?
