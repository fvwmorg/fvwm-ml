I'm running fvwm on Digital Unix 4.0 and XDec. Sometimes, when I log
out or restart the window manager my FvwmButtons module doesn't get
killed. Then the modules runs away with the processor, using 50-90% of
the processor. This is VERY bad, because the computers on our network
are used assigned jobs from a batch queue to run. If the load is always
high because of buttons, no jobs get run.
