Rohan Oberoi wrote:
>
> Hi,
>
> 1. Does anyone know if it's possible to order the TaskBar to only show
> applications in the current virtual desktop (ie. FvwmPager mode)?

It is not possible with the 2.2.x (or inferior) version. But, it is
possible with the devel version since 2.3.? (for fvwm-devel: NEWS does
not mention this! and I've not found any entries about this in the
change log). So if you use your machine for fun try the latest devel
version 2.3.15 (current status: feature freeze beta, need help for
debugging!).
