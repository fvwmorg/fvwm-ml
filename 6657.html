Tenebrae <tenebrae_at_niceboots.com> writes:

> That reminds me, though...
> I guess I should go ahead and ask the mailing list the stupid question of
> How can I make this particular instance of Eterm appear with no titles?
> I want the other Eterms to have the normal decorations, but I'd like THIS
> instance to be NoTitle, NoHandles, etc.

I may have misunderstood the question, but isn't one way to use the
"name" option, assuming term has it or something similar.

eg for rxvt

rxvt -bg midnightblue -fg cornflowerblue -fn 10x20 +sb -name gterm

Put this wherever you want it in your key or menu sections.

Then you can set style options for gterm.


Style "gterm" notitle, nohandles, BorderWidth 3, HandleWidth 3

or whatever!!
