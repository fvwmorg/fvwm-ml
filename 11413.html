I just discovered two more problems when using the FullPush and
NoBorder options. When a button is pressed a 2-pixel-wide
"pressed-in" relief is still being drawn around it and not only
the icon is moved to the left and down but the transparent
background moves with it. Just wanted to let you know, I'm going
back to using FvwmButtons because it's more flexible and allows
different button sizes.
Would it be possible to add a feature like "FullPush" to
FvwmButtons or would that be against the feature freeze policy?
This way one could get some visual feedback on a button press
when the buttons are transparent or flat, I also remember someone
asking about that feature on this list about a month ago.
