  Charlton> I used to have my icons go to the bottom-left of my
  Charlton> desktop when windows were minimized uner fvwm1. Now that
  Charlton> I've installed fvwm95 (fvwm2) I can only get a small
  Charlton> text-box of the window-title instead of an icon and the
  Charlton> position of that text-box is where the mouse happened to

You can assign icons to programs using the Style directive. Example:

Style "XTerm" Icon foo.xpm

Also see the PixmapPath and the IconPath directives.

  Charlton> Does anyone know how to configure the .fvwm2rc file to get
  Charlton> my icons to behave the way they used to (like icons behave
  Charlton> in the Windows 3.1 environment)?

See the IconBox directive.
