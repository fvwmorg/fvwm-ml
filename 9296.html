Set WindowListSkip style for FvwmButtons, FvwmPager, xclock. (Style
FvwmButtons WindowListSkip).
Then tell FvwmPager not to display windows with this style (*FvwmPager:
UseSkipList).
Problem is that FvwmButtons, Pager and xclock will not appear in you windows
list anymore. Maybe there is another, cleaner way to go...
