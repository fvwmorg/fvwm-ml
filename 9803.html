Uwe, please spend some minutes and install fvwm and then fvwm-themes
properly (i.e. using "./configure; make; make install") without any
copying of directories or correcting paths, otherwise you will surely
experience problems in one or another place.

Then if there is any problem, I will be glad to help. The CDE and Luthien
themes depend on FvwmM4 and awk, in case the problem still persists.
