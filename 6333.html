> If I turn off everything (emulating a "splash screen"), I can still

Why would you want to prevent the user from moving the window, even
a splash screen?

Even though the user can't yet do anything with that application
(since it's starting up), the user might want to do something else
(with some other application). Making that difficult is _not_
user-friendly.
