1) Check to see if there is another .FvwmTaskBarGeometry somewhere else in the
file.

2) If you start X from the console, try

startx 2> zzz

This will place any X messages in a file called "zzz". You can check it for any
error messages that the FvwmTaskBar module might be kicking out.

3) Make sure you're editing the right .fvwm2rc file (I'm out of ideas <grin>).
I've had it happen more than once where changes I was making weren't doing
anything, only to find out that I was editing the wrong file. If you've
recently switched to one of the newer beta versions from 2.2.4 ~/.fvwm2rc no
longer works. You need to edit ~/.fvwm/.fvwm2rc.

Glen
