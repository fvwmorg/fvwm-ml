>>>>> On Mon, 30 Oct 1995, Gruwez Guido said:

  Gruwez> I' like to use linux with fvwm as an X-terminal on a
  Gruwez> Digital Ultrix sparc .
  Gruwez> Can i really use it or have i to buy an X-motif variant ????

Yes, Linux with fvwm is sufficient, no need to buy commercial
software. Some commercial Unixen (Unices?) use the Display Postscript
extension (notably SunOS and Solaris) for some programs; on these, you
might not be able to run all programs remotely. But Motif isn't
needed at all. All Motif clients run fine when started on your Ultrix
with redirected display to your Linux machine. You can even run mwm
this way, if you really want...
