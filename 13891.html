"Thomas A. Horsley" <tom.horsley_at_att.net> writes:
> I have a feeling I'm going to want to use fvwm, but at the moment
> I'm being overwhelmed by bells and whistles :-).
>
> All the documentation I've been reading so far has emphasized how
> there are all sorts of fallback configs that provide useful root
> menus, etc.

If you start Fvwm with no config file,
you get a very small menu with just a few choices,
one of them is quit. Very few things are configured,
the man page lists the things that are configured and
how to unconfigure them.

The menu will offer to create a starting config for you,
but you don't have to select that option.
