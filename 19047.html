I guess the title sums it up well. This is a patch to fvwm that makes the
icons have the shaded look that is typical to NextStep. I acteded without
shame and took most of the code from AfterStep. As with Afterstep all the
icons need to have atleast one pixel that is of color "None". I have been
using this patch agenst fvwm 2.0.43 for 5 months with no problems. My
goal is to make this yet another style; I.E. shaded icons, but for now it is
an all or nothing deal.

If there are any questions or problems please address me and not the
mantaner(s) of FVWM. They have enough work.

This patch is posted on my webpage along with a screen shot.

http://hepserver.nhn.ou.edu/~spielman/fvwm/index.html
