Ali Faour <afaour_at_newbridge.com> writes:
> Hi Guys,
>
> First thanks for all of who answered my questions. But questions
> never end. In fact,
> after I install fvwm, how to make it work? I replaced 'olvwm' by 'fvwm2'
> in '.xinitrc'
> and I got nothing. I had to reboot my machine with CDE to make it work
> again.
> I hope you can help me.

If you are using a 2.2.x version, copy
"sample.fvwmrc/system.fvwm2rc" to $HOME/.fvwm2rc
and make the changes you want. Then start fvwm again.

If you are using a 2.3.x version, click on the background or hit F1.
