> Dear List,
...
> The only thing I have changed, is the variable "ModulePath", in the per user
> file ".fvwm2rc.m4" (I am still using AnotherLevel -- Lestiff style). I
> changed the variable from the following:
>
> ModulePath /usr/X11/lib/X11/fvwm2
>
> to the following:
>
> ModulePath /usr/local/libexec/fvwm/2.3.28
>
> (I assume that is what I am supposed to do??)

The 2.3.x releases contain a 2.2.x to 2.4.x conversion script.
You can use that as a guide to what should be changed in user config
files.

In the case of ModulePath, it should be deleted from user config files
since the ModulePath gets built in to fvwm now.

A similar change should be done to ImagePath.
