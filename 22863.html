Thanks to Chris Rode, i made a taskbar using FvwmButtons and FvwmIconMan.
I have a little problem with FvwmIconMan in the *FvwmIconMan*action definition :

I've included in my .fvwm2rc :

*FvwmIconMan*action Mouse 1 N sendcommand "Iconify -1", sendcommand Focus, sendcommand Raise
*FvwmIconMan*action Mouse 2 N sendcommand "Module FvwmIdent FvwmIdent"
*FvwmIconMan*action Mouse 3 N sendcommand "PopUp Window-Ops "

where Window-Ops is:


AddToMenu Window-Ops
+ "Resize" Resize
+ "Raise" Raise
+ "Lower" Lower
+ "(De)Iconify" Iconify
+ "" Nop
+ "Destroy" Destroy
+ "Delete" Delete

If i click with the third mouse button on an iconified window button in FvwmIconMan, i have the
menu appearing but whatever the choice is ( Raise, Iconify, Delete ...) the resulting action is to quit fvwm2 !
