I don't think I fully understand your question.

Every window (son or uncle) has its own name, resource name and class.
All three names can be used for the first parameter of the Style command.
To determine the window names, execute FvwmIdent on the window.
For example, press Shift-Mouse3 on the window title after the binding:

  Mouse 3 T S FvwmIdent

In case some windows have similar names, you can use a patern, like:

  Style "myapp*" StyleOption1, StyleOption2...
