Thomas,

I was able to get around this by changing the default IconBox style setting in
the original .fvwm2rc file:

Style "*" IconBox 1000x100+0+15

This uses a standard geometry setting for X, placing the icon box across the
screen but about 15p below the top of it so that it doesn't interfere with the
FvwmTaskBar.

Glen
