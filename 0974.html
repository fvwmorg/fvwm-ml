On Wed, 7 Jun 2000, Dominik Vogt wrote:

> On Mon, Jun 05, 2000 at 08:01:10PM -0230, Neil Zanella wrote:
> >
> > Hello,
> >
> > I would like fvwm2 to resize xterm windows that get mapped onto
> > the desktop for the first time as follows: if the xterm has a
> > font such that its height or width is larger than the viewport
> > height or width then fvwm2 resizes the xterm so that it fits
> > entirely inside the viewport.
> >
> > Is this possible?
>
> Try using the MaxWindowSize style.

Perhaps I should have been a little bit clearer. The MaxWindowSize style
will indeed resize the xterm window so that it is entirely contained in
the viewport but if I later decide to resize xterm so that it spans two
or more viewports then I cannot do it. It would be nice if there were
a Style option called MaxWindowStartupSize which would act exactly like
MaxWindowSize except for once the window pops up the user can resize it
to any size. The MaxWindowStartupSize would only affect the geometry of
the window when it gets mapped onto the desktop for the first time.

Thanks,

Neil
