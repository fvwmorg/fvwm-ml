If you use "Style * Icon, NoIconOverride" you will see these
icons in fvwm (when the window is iconified).

So, yes.

> For example one could use it as mini icon. This would be

This is not possible in general ... This is done only if the
application provide an EWMH icon (or/and mini-icon). GNOME2 and KDE>=2
(and others) provide such icons but not the applications you mention
(but galeon "2" ??)
