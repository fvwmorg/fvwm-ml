Mabey you could try this (I've only tested it with cvs version YMMV),

DestroyFunc MovitFunc
AddToFunc MovitFunc
+ I None ($0) Exec exec $*
+ I Wait $0
+ I All ($0) Raise
+ I All ($0) Move +0 +0

then do a

MovitFunc plaympeg blahhh.mpeg
