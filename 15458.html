tevis_jim wrote:
-> Thanks Curt/Robert...I'll try that. What's the difference between
-> A and N in the Modifiers field for this definition?

A is "any", N is "none". These affect whether or not shift, control,
and meta change the key binding. If you use "A" then, for example,
<shift>+(whatever) will have the same effect as (whatever) by itself.

You can also specifically refer to:
 S shift
 M meta
 C control
RTFM on fvwm for more info; look for the section on "Key" and "Mouse"
bindings.
