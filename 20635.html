On Fri, 15 Aug 1997, Noel Yap wrote:

> I would like my FvwmPager to appear flush against the bottom right of my
> screen but it seems that fvwm translates the "-0" as a "+0". Does
> anyone know how to fix this?

well... +0+0 is upper left corner, and -0-0 is the lower right corner...

(+0-0 should be lower left, and -0+0 upper right... if not, switch these
last two around).

I think the first number reffers to the # of X pixels. If it starts with a
+, the number represents the number of pixels the image starts from the
left... If it starts with -, the number represents the number of pixels
from the right. This is similar for the second number (the Y pixel
coordinate).
