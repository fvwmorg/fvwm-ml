On Thu, Dec 19, 2002 at 02:53:38PM -0600, Amy Bergmann wrote:
> Hello,
> I am looking for a solution to a problem:
> I need to keeping an app on the screen without being able to move it
> around. I basically built a Kiosk using RH 7.2, displaying HTML pages
> with Netscape. However I can move the window up/down, I took care of
> left/right with (EdgeScroll is 0 0).
> I want to keep a window with it's tool bar, maximized, but not
> moveable. I covered up the menu bar with an image so users do not have
> access to any control options. Also, I turned off all control buttons
> in the title bar.
> I already tried Style "*" FixedPosition, Style "netscape"
> FixedPosition and it still allowed me to move the window.
> Any suggestions??

Go through your config file and remove all "Key", "PointerKey" and
"Mouse" commands that bind the Move, Resize, Maximize etc.
commands to the window decorations. To disable the built in
defaults, edit the file ConfigFvwmDefaults that comes with the
distribution.
