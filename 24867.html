I've now downloaded the latest CVS, and am now running it.
The problem persists. I've also done some testing to see
exactly when it happens.

1. Start X without starting unnecessary programs.
2. Start for example an rxvt and iconify it.
3. Switch to a virtual screen other than 0.
4. Restart windowmanager (I have Restart fvwm2 as a menu choice)
5. Voila. The iconified window is gone. It still shows up in
   WindowList and in FvwmPager though.

How to get the icons to reappear:

Either get them to pop up by using WindowList, or:

1. Switch to virtual screen number 0
2. Restart windowmanager.


As far as I can see, the icons don't disappear as long as I
look at screen 0 when I restart the windowmanager.

I have fvwm set up with eight virtual screens that are all
the size of the physical screen.

Hope this helps, because I have absolutely no idea about where
to start chasing this in the sourcecodes..
