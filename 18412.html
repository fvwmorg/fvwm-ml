Hi,

Excuse me if this has been brought up before or is already solved:
What I would like to see in FVWM (I'm running 2.0.43) would be
that it remembers which windows was active on a certain desktop
and automatically activates this particular window when this
desktop is switched to. This would save a lot a mouse clicks
I currently need to make.

An example to make it clear if my explanation above is too fuzzy:

1.) I got my pager configured for 3 separate workspaces containing
    4 desktops each.

2.) I'm on desktop #1 in workspace #1 and I got two DECTERM windows open.
    I got another DECTERM in a separate deskop #2 in workspace #1.

3.) When I switch from desktop #1 to desktop #2 (within the same
    workspace) I need to make the window active manually by clicking
    on it.

Note that this works when switching *between* workspaces and not between
desktops so the active window is saved per workspace and not per desktop.

Can this be changed somehow?

Thanks -- Tom
