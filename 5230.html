On Mon, May 27, 2002 at 08:37:21AM -0700, Jukka Lehti wrote:
> Hi,
>
> I've finally decided to move from fvwm-1.24r to latest
> fvwm2. I've already converted my config file
> succesfully, but two minor problems remain. Could any
> one help with these?
>
>...
>
> 2) I have a menu containing few Popups:
>
> AddToMenu "Utilities"
> + "Program Menu" Title
> + "xterm" Exec exec xterm &
> + "Mozilla" Exec exec mozilla &
> + "" Nop
> + "Applications" Popup Apps
> + "Utilities" Popup Utils
> ...
>
> There seems to be short delay before a Popup is shown
> when I move pointer there. How can I disable this
> delay?

Take a look at the MenuStyle command. You prabably want something
like

  MenuStyle * PopupImmediately

There are a whole lot of options to the MenuStyle command that you
might want to check (all in the man page).

> PS. Is there any plans for fvwm3? If so, does it
> understand my new fvwm2 config files?

Yes, there are plans for a major version 3 and no, if it is ever
released, the config file will be incompatible in large parts. We
will provide a script to convert configs as usual, but my guess is
that it won't do a very good job. All this is far into the
future, though (at least a year or two).
