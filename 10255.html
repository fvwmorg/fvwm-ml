Hello,
I have a 4x4 virtual desktop, and 5 desk on my FVWM. So in my fvwmrc, it's
something like:

DeskTopSize 4x4
...
AddToFunc "InitFunction" "I" Module FvwmPager 0 4

So in total I have 4x4x5=80 virtual desktops.

What I want is to have my email apps (KMail) to stay on virtual desktop 3
only, but in all desk. What is the correct 'Style' option?

For example, if I use this style:
Style "gkrellm" NoTitle, NoHandles, Sticky, WindowListSkip, BorderWidth 0,
CirculateSkip

this will make 'gkrellm' appears in all virtual desktops, all desks. That is
*not* what I want, since I only want Kmail to appear on certain virtual
desktops, all desks.
Can this be accomplished?

I'm using lastest stable fvwm-2.4.x. Any help is greatly appreciated.

Thanks in advance.
