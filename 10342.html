Hello,

I'd like to be able to have the background of the text for certain xterm
icons be different colours to others, to make it easier to find them
when minimised - in this case to make it easier to tell root terms from
user terms.

This is no problem - I can do it with a Style line. The problem is that
if I 'su' in a normal terminal FVWM doesn't change the style for that
xterm to the root term style.

The title bars of my xterms begin with text in the form
'(<user>_at_<hostname>)' which updates after an su, but FVWM doesn't react
to the change.

Is there any way to make FVWM change the style for a window without
restarting it?

Cheers,

Andy
