On Mon, Oct 08, 2001 at 05:52:10PM +0100, David Wales wrote:
} I hate double-clicking to deiconify, so I tried to set single click to
} do this:
}
} Mouse 0 1 A Iconify
}
} On my Debian Linux boxes this seems to work just fine. Downloading the
} latest fvwm2 release and compiling it on a DEC workstation under Digital
} UNIX V4.0E I find a different behaviour: single click deiconifies, but
} only when the cursor is moved after the click. The click does not seem
} to register until the mouse moves. I have tried playing with ClickTime

I have this:

Mouse 2 I A Iconify

Mouse button two deiconifies icons with a single click.
Works with RedHat, Mandrake, SuSE, and HPUX.
