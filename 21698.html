Near 2000, Chris Wasshuber mumbled something about:
>
> Several applications pop up windows (file browsers, ...) which
> don't have a title bar and which I can not move using the mouse.
> I guess these windows are called override or transient?
> How can I configure fvwm2 to allways show a titlebar, so that I can
> allways easily move these windows with my mouse on my screen.

1) Try something like Style "*" Title, Borderwidth ...
2) Why would you need a frame for moving windows ?
Mouse 2 W A Move
makes that you can move a window by dragging with the middle mouse
button. You could of course choose another mouse button or
specify a key modifier, but the point is that you don't need a
frame around the window.

Hope this helps,
