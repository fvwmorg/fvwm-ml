Ilia Chipitsine <ilia_at_cgilh.chel.su> writes:
> Merry Christmas, everybody.
>
> Is it possible to bind Alt-F4 in such way that if any window exists it
> will be closed, but if no window exist pressing Alt-F4 will cause exit
> from X (from startup script I have "exec fvwm2") ?
>
> Regards, (Наилучшие пожелания)
>
> Ilia Chipitsine (Илья Шипицин)

This worked for me:

AddToFunc CloseQuit
+ I Next Close
+ I None Quit
KEY F4 A M CloseQuit
