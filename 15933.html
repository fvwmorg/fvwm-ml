I just saw in another post, that you can select a character to be
underlined in a menu, by putting a & in front of it. I tried it and it
works, but not quite :-)

select the item. But if I popup the menu with a key binding, then the
first character I type is ignored.

that using the keybinding again would pop the menu away (I often popup
the window list just to see which windows I've got, without actually

Thanks for a great window manager, and any tips which you might have
for getting around these slight behavioural problems.
