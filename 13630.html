> I'm running fvwm-2.5.12-4; the problem is that sometimes (I couldn't find when
> exactly) the key-bindings stop working, and I can't use them even restarting
> fvwm. After that i can use the mouse to access menus, windowlist, etc, but not
> using keys.
>
> I'm thinking on mozilla-firefox block pop-up feature; when this problem
> happens, allways is mozilla-firefox running, but I'm not sure because I can't
> trace the error.
>
> Which could be the problem? How can I fix it? Any help would be appreciated.

This sounds like a FAQ 0.1 problem... ^_^
Make sure this doesn't have to do with CapsLock/NumLock/ScrollLock modifer.
If it does, then you can add

IgnoreModifiers L25

in your config file.
