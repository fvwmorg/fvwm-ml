Frank Hrebabetzky <frankh_at_terra.com.br> writes:
>
> Thanks to Mikhail Goikhman I know now how to produce buttons on the
> screen. So I don't need the default buttons any more (virtual desktop
> switcher, system load, postbox, clock). How can I get rid of them?

Look for commands like this:

*FvwmButtons: (2x1, Swallow "FvwmPager" "FvwmPager 0 1")
*FvwmButtons: (Swallow "asclock" 'Exec asclock -shape -led green -noblink')

and remove them.
