I found out where the problem is!
It's the 'ClickTime' command,ie.if you set a short clicktime(ClickTime 150)
you can reproduce this problem.
I have a simple .fvwm2rc like this:
#BEGIN
    IgnoreModifiers L25
    ClickTime 150
    TitleStyle InActive -- Flat
    TitleStyle Active -- Flat
    ButtonStyle All InActive -- Flat
    ButtonStyle All Active -- Flat
    Mouse 0 2 A Maximize
    Mouse 0 4 A Iconify
    Mouse 1 TS A Move-or-Maximize
    AddToFunc Move-or-Maximize
     + I Raise
     + M Move
     + D Maximize
#END

If you set 'ClickTime 750', doubleclick works right.
