Since nobody seems to have recognized the possibilities of my
latest patches:

With the new function contexts and the North/South option of the
WindowShade command one can now shade a window in different
directions, depending on the part of the frame that was clicked:

  Mouse 1 [<-^] N WindowShade North
  Mouse 1 >_v N WindowShade South


  +------------------+ click on top +------------------+
  | title |--------------->| title |
  +------------------+ border +------------------+
  | |
  | |
  | |
  | |
  +------------------+

  +------------------+
  | title |
  +------------------+
  | |
  | |
  | | click on +------------------+
  | |--------------->| title |
  +------------------+ bottom border +------------------+

Maybe I'll add left and right titles and W, E directions and
possibly even NW, SW, SE, NE too.
