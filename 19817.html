>
> FVWM is an ICCCM compliant multiple virtual desktop window manager for
> X11 created by by Robert Nation and derived originally from twm code.
> THANKS ROB!!!
>
> Rob has decided to pass the torch on though, so FVWM is now maintained
> by Charles Hines (me) with the help of various people all over the
> internet who are kind enough to use FVWM and submit patches,
> suggestions, etc.
>
> Also, the mailing list and official ftp site have moved with the
> change in maintainer (see below).
>

> Version Info
>
> * Last Official 1.xx Version: 1.24r
> * Latest Official 2.xx Version: N/A (still in beta)
> * Latest Beta Release: 2.0.45
>
> Just click on the version numbers above to download them.
>
> And please keep in mind that the Beta code can potentially become
> unstable or buggy at any time, so USE AT YOUR OWN RISK. You will of
> course find bugs in the more stable Officially released versions too
> (see the man page for a note on this). Please report new bugs to the
> FVWM Mailing List (see below).
>
> I hope to have the First Official 2.xx Version (it'll be 2.1.0) out
> "Real Soon Now". It requires a little more polishing, like Doc
> updates, Compile/Install procedure fixes, some more bugfixes, etc.
>
> ----------------------------------------------------------------------
>
> Features
>
> Features Common to 1.xx and 2.xx (quick list):
>
> * Multiple Disjoint Large Virtual Desktops
> * Smaller memory usage (more so in 1.xx)
> * Dynamically extensable via modules
> * Recognizes Motif MWM hints
>
> Features New to 2.xx (quick list):
>
> * New more powerful and sensible rc file format
> * Change more features on per-window basis
> * Change many features on the fly
>

> FTP and Mailing List Info
>
> The new Official FTP site for fvwm is
>
> ftp://ftp.hpc.uh.edu/pub/fvwm/
>
> and the new mailing list addresses are:
>
> fvwm_at_hpc.uh.edu (Discussion and questions list)
> and
> fvwm-announce_at_hpc.uh.edu (Announcement only list)
> and
> fvwm-workers_at_hpc.uh.edu (Beta testers and contributors list)
>
> They are maintained by Jason Tibbitts, and are Majordomo based mailing
> lists. To subscribe to the list, send "subscribe fvwm" in the body of
> a message to majordomo_at_hpc.uh.edu. To unsubscribe from the list, send
> "unsubscribe fvwm" in the body of a message to majordomo_at_hpc.uh.edu.
>
> SUBSCRIPTION REQUESTS SENT TO THE LIST WILL BE IGNORED!!!! PLEASE
> FOLLOW THE ABOVE INSTRUCTIONS FOR SUBSCRIBING PROPERLY!!!!
>
> To report problems with any of the mailing lists, send mail to
> fvwm-owner_at_hpc.uh.edu.
>
> Also the mailing lists are now archived!
