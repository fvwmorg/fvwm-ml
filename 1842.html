On Tue, Jan 02, 2001 at 03:13:21AM -0600, Glen Lee Edwards wrote:
> How do I configure True Type Fonts in .fvwm2rc so that I don't have to use
> a magnifying glass to read the screen?
>
> Glen
>

I suggest using a font viewing app to get the font name( e.g. gfontsel or
xfontsel). then put:

WindowFont "-*-yourtruetype font-*-*-*-*-72-120-*-*-*-*-*-*"

In your fvwm2rc file. If you want to change your menu or icon font see
the man page, its all there.

HTH,
