%% Ken Roberts <ken_at_worldnet.att.net> writes:

  kr> I've searched the archives for a reference to key modifiers.
  kr> Unless I've missed something, (a distinct possibility) there is no
  kr> way to add a key modifier.

You're looking in the wrong place.

You redefine your keyboard, including modifiers, via the X server not
the window manager, typically using xmodmap. There's a cool graphical
program called xkeycaps that makes it _much_ simpler to do.

AFAIK FVWM will support any modifier you bind to your Windows key.
