Here's a little form to configure FvwmBacker
It provides 6 command lines (as described in "man FvwmBacker")
per desk.
Any nuber of desks can be configured, with Page x y and pixmaps etc...
Try it out, and if you like it, let me know. If there's something wrong with it
PLEASE let me know! :-)

Alex
<awol_at_dove.net.au>

P.S. Instructions for use are on the top of the form. Invoke it with
      Module FvwmForm FormFvwmBacker.
