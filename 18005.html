> Does anyone know how to capture a screen-shot under X-Windows ?
>
> I am using FVWM (under Linux). Is there a way to ask the X server
> (in this case XFree86) or the window manager (fvwm) to take a snapshot
> of the screen and save it as an image ? I looked at the manual of FVWM

This is an X server issue, not a wm issue.

You can do 'xwd -root' or use the grab function of xv, and I think
there are more utilities knocking around that do much the same thing.
