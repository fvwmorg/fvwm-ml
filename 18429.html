my MiniIcons patch to fvwm2.0.43 from mahler.cs.arizona.edu.

anonymous ftp server here (I'd have to actually ask somebody to get something
put there!) so I've moved all that to my webpage. You can find the latest
FvwmIconMan and whatever at:

http://www.cs.arizona.edu/people/bradym/software/

It seems to work for me, but if it doesn't work, please send me email and
i'll fix it.
