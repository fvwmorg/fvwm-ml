On Thu, 07 Oct 1999, Tim Phipps wrote:

> There was a complaint that the separators were confusing with the
> NoGeometry option so I removed them.

Ok.. =)

In that case, I have another suggestion. As I find the geometry stuff
shows me a bit too much information, and the one thing I like about
it is that it shows me which windows that are iconified (and maybe
the desk number that I guess the complaint you talk about was all
about), may I suggest a new option for WindowList.

* When using NoGeometry, you don't know which desktop number each
  of the windows are located on.
* The window number row at the left edge of the menu is not really
  that useful (at least I still haven't found a use for it).
Taking these two things into consideration I would suggest adding
the desktop number at the left side of the first of the items for
each desktop.
Also, placing the parenthesises around the programname instead of
the gemetries, so that they are visible even when you don't want
to see the geometrics.

A drawing:

 ------------------------------- -----------
 Desk 3 Geometry
 ------------------------------- -----------
 1 Rxvt 80x24+1+342
    Netscape xxxxxxxxxxx
    Rxvt (rincewind.dc.luth.se) xxxxxxxxxxx
 ------------------------------- -----------
 2 Rxvt xxxxxxxxxxx
    Rxvt xxxxxxxxxxx
    Rxvt xxxxxxxxxxx
   (Rxvt) xxxxxxxxxxx
   (Rxvt) xxxxxxxxxxx
 ------------------------------- -----------
 5 Knews xxxxxxxxxxx
    Mutt mailer xxxxxxxxxxx
 ------------------------------- -----------

An alternative would be to have the window number listed to the
left (as it is now):

 --------------------------------- -----------
   Desk 3 Geometry
 --------------------------------- -----------
 1. 1:Rxvt 80x24+1+342
 2. Netscape xxxxxxxxxxx
 3. Rxvt (rincewind.dc.luth.se) xxxxxxxxxxx
 --------------------------------- -----------
 4. 2:Rxvt xxxxxxxxxxx
 5. Rxvt xxxxxxxxxxx
 6. (Rxvt) xxxxxxxxxxx
 7. (Rxvt) xxxxxxxxxxx
 --------------------------------- -----------
 8. 5:Knews xxxxxxxxxxx
 9. Mutt mailer xxxxxxxxxxx
 --------------------------------- -----------

Or something similar... =)

What do you think about this?
