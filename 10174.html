> > > border, so each one looks like a button. Is there a way to do
> > > this already that I am missing?

> > You're right. Currently you get the 3d look only when the item

>
> How hard would it be to implement this as an option for normal
> menus? I think it would give a nice look. I'd be happy to try
> it myself, but I might need someone to give me an idea of what
> needs to be changed.

It's quite some work because we don't have any item specific
options yet. You'd have to add some menu styles for button
decorations and paint them into the menu background pixmap (the
foreground is too slow). In addition, button specific
decorations have to force that the background pixmap is as big as
the menu itself. Tiling H- and Vgradients isn't possible anymore.
