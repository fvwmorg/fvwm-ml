Salvatore Leone <ototl_at_tiscali.it> wrote:
> I've got some problem trying to install the fvwm2(stable version)
> averytime it says to me that I need to install "libreadline.so.4"...
> but I know I'vr got it becouse when I write in the console:
> "whereis libreadline.so.4" it says to me: "libreadline.so:
> /lib/libreadline.so.4 /usr/lib/libreadline.so.4"... So Ithink that I
> have to put the library in another directory too, but which one??

On most Linux distributions libraries are put in two separate package.
For instance in SuSE Linux, the readline library comes in packages
named readline and readline-devel. The header files are in the second
package. Install both and you will be OK.
