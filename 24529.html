  pp> I have created .Xdefault file under my home dir

I assume you really created .Xdefaults (note plural "s"!)
                                     ^

  pp> *wmStartupCommand: /home/gsm/u08/pushpak/.fvwm/bin/fvwm2
  pp> *waitWmTimeout: 1

  pp> when I login in my CDE I am running fvwm2 but the PROBLEM is that
  pp> I get no buttons,

What do you mean, "no buttons"? By default there is no buttonbar in
FVWM. If you want one, you'll need to invoke the FvwmButtons module (or
get a 3rd party module like tkgoodstuff) and configure it for the
buttons you want (see the FvwmButtons man page).

  pp> my mouse menu works.

That's good.

  pp> and when I try to quit from fvwm2 it does not quit, I mean it
  pp> quits fvwm2 but something still running.

  pp> Can you please?

See the FVWM FAQ; there's a question there (#2.2) about using FVWM with
CDE. It says this (among other things):

> Finally you have to quit with
>
> /usr/dt/bin/dtaction ExitSession (or whatever your path is for dtaction)
>
> Now you can happily use cde programs and fvwm.
