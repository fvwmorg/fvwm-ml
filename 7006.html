How do you (or dtlogin) start fvwm?
What command do you use for Restart?

You should see an error message in the place where you have all X output.

There is almost no chances that Restart without parameters would fail.
Only if you removed or incorrectly replaced your currently running fvwm
executable (I don't know for sure about solaris8, but linux allows this).

Or maybe it always exits with your fvwm2rc (for example you have Quit
command in it), not just on Restart? If you can't find the problem
yourself, send your fvwm2rc. But first find the error message.
