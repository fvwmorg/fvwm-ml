Hi,

Since you started with 2.0.43, you should first run fvwmrc_convert
and then run fvwm24_convert. Since neither conversion is perfect,
you should look at each file after the conversion.

After the conversions, there should not be a ModulePath statement
in your config file $HOME/.fvwm/.fvwm2rc. If there is, remove it.

Also look for error messages being printed by fvwm during startup
on your console.

If you are still having problems, post the parts of your config
file that are not working. To start a module, you should have
something like this in your config file:

AddToFunc StartFunction "I" Module FvwmTaskBar

of this list don't like it.
