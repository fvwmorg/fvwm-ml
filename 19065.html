%% albrecht_at_auto.tuwien.ac.at (Albrecht Kadlec) writes:

  a> why not just have the actual maximize/minimize functions do nothing, if
  a> maximize/minimize is forbidden in the MwmHints and the hints are to be
  a> obeyed.

Unless there is some visible indication that the buttons are disabled, I
think this is absolutely NOT the right thing to do.

If all windows have min/max buttons, but some work and some don't, the
user is just going to be mightily confused and bug reports on this will
pour in to both FVWM development and other software developers forever
more.
