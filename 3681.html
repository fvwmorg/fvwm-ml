On Tue, Nov 20, 2001 at 06:41:49PM +0100, KONTRA Gergely wrote:
}
} - In icewm there is a processor-usage monitor. I'd like to do a similar
} effect in fvwm. I've configured up xosview, and like it, but it would
} be nice, if FvwmTaskBar can swallow it. And of course when I click on
} it, an application (eg. top) should appear ;)

I have xosview in my fvwm control panel:

*FvwmButtons(3x3 Swallow(UseOld,NoHints,Respawn) "xosview" `Exec xosview`)

You can use the "Action" argument to make clicking on it do something.
See man FvwmButtons.
