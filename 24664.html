I'm trying to get different icons for Gimp's different sorts of
windows. ie

wilbur.xpm - Main Gimp Window
gimpicon.xpm - Image windows
gimpmicon.xpm - Tear off menus

I have the following set in my .fvwm2rc. The Main gimp window gets the
correct icon as do the image windows (Via the match on "Gimp"). However
if I tear off the file menu in Gimp I get a window with the following

WM_CLASS(STRING) = "gimp", "Gimp"
WM_LOCALE_NAME(STRING) = "C"
WM_CLIENT_MACHINE(STRING) = "landlord.gbdirect.co.uk"
WM_ICON_NAME(STRING) = "File"
WM_NAME(STRING) = "File"

Which gets no icon at all! Any ideas?
