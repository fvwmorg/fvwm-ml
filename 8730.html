> Hello everyone,
>
> In my fvwm I run a sticky gkrellm to monitor what's going on. I would
> like new windows to not overlap this one application. I guess it's done
> with a Style command in .fvwm2rc but I couldn't find it. Any tips?
> Thanks!

In fvwm-2.5.x you can fine tune the placement algorithm to avoid
placing windows over specific other windows. Look for
MinOverlapPlacementPenalties and
MinOverlapPercentPlacementPenalties in the man page.
