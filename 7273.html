< I've recently installed XFree-4 on a laptop and slapped fvwm-2.5.3 on it.
<
< I copied over my trusty .fvwm2rc (carryover from fvwm-1.22 days) and made
< the standard system-specific tweaks. One disappointing item of note is that
< my "Move" and "Resize" (F2 and F4) key mappings don't work.
<
< The relevant lines in the .fvwm2rc file look like:
<
< Key F2 A N Move-or-Raise
< Key F3 A N Menu Window-Ops
< Key F4 WTFS N Resize-or-Raise
< Mouse 0 1 A Menu Window-Ops2 Close
< Mouse 1 TS A Move-or-Raise
< Mouse 1 F A Resize-or-Raise
< Mouse 2 TSF A Resize-or-Raise
< Mouse 2 R A Menu Window-Ops Nop
< AddToMenu Window-Ops "Window Ops" Title
< + "Move" Move-or-Raise2
< + "Resize" Resize-or-Raise2
< AddToMenu Window-Ops2 "Move" Move-or-Raise
< + "Resize" Resize-or-Raise
< AddToFunc Move-or-Raise "I" Raise
< + "M" Move
< + "D" Lower
< AddToFunc Move-or-Raise2 "M" Raise
< + "M" Move
< + "D" Lower
< AddToFunc Resize-or-Raise "I" Raise
< + "M" Resize
< + "D" Lower
< AddToFunc Resize-or-Raise2 "M" Raise
< + "M" Resize
< + "D" Lower


I replaced all those IMD type funciotns w/...

AddToFunc Clk-Dclk-HldMv
+ C $0
+ D $1
+ H $2
+ M $2

AddToFunc Op_Raise
+ I $0
+ I Raise


Mouse 1 FS A Clk-Dclk-HldMv Raise RaiseLower "Op_Raise Resize"
