> Before I dive into fvwm2 source tree, can someone tell if it's
> possible to have fvwm2 look like 95? Thank you.

Yes. There are some sample decors in the distribution of fvwm-2.0.45
that contain a MWM and a Win95 look and feel.

There's an application called tkgoodstuff which can run as an FVWM
module that will produce a nice looking task bar as well.
