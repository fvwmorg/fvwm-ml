The reason is that there is a subtle change in the requiremnets in the
.fvwm2rc file that were accepted previously. I think the nuance may
have occurred when I ran the conversion script to go from fvwm1 to fvwm2
but it has been so long that I forget now.

Here is what I had to change in our global fvwm2rc file. It works correctly
with fvwm2*.

< *FvwmButtons(2x1) - whatever Swallow "FvwmPager" Module FvwmPager 0 1
< # *FvwmButtons(4x1) - whatever Swallow "FvwmPager" Module FvwmPager 0 3 
---
> *FvwmButtons(2x1 Swallow "FvwmPager" "FvwmPager 0 1")
> # *FvwmButtons(4x1 Swallow "FvwmPager" "FvwmPager 0 3")
