I'm not too knowledgable about X, but I was wondering if it would be possible
to in impliment something like Mac's WindowShade. In case you don't know what
that is, it allows you to click twice on the windows title to collapse the
window to just it's title row.

I know that this can be simulated by just Iconifying the windows to their
title without an icon, but this doesn't keep the windows original width
information when it's iconified. By this I mean, I like to tell which window
I'm selecting by the windows width as well as it's title, since sometimes I'm
working on different windows with the same title but at different widths.

Plus, I'm sot sure if it's my settings, but when I Iconify a window the icon
is placed to the right of where the actual window was located, which is
annoying.
