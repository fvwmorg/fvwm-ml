On 2001-Mar-15, Jan Theofel wrote:

>> do you know an easy way to hide the mouse? I need that for a full-screen
>> presentation.

Jan,

   Have a look at the 'unclutter' utility. It's not a part of FVWM,
   but you can install it very easily from the RPM file at:

      ftp://rpmfind.net/linux/contrib/libc6/i386/unclutter-0.8-4.i386.rpm

   If you're not running on an RPM-based system, I'm sure a quick search
   of the web will find the source code and installation instructions
   for 'unclutter'.

   It's a simple little utility that makes the mouse cursor vanish after
   some user-specified period in which there has been no mouse movement.

Cheers, Joseph
