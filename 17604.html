On Mon, 12 Aug 1996, Henry McNair wrote:

> Hello:
> I am a novice at building fvwm and X in general; so please excuse any
> question that might seem trival or can be answered with "RTFM". :-)
>
> When I am building fvwm, I am asked to enter the command
> "xmkmf;;make Makefiles"
> I have access to imake, but what is xmkmf? How can I get it? Can
> I use imake instead? The command ./MakeMakefile also uses xmkmf. So
> I am stumped. (I have modified the configuration files correctly,
> I hope. :-)
>
> I would really like to try using the new version. So any help
> you can give would be appreciated. I am trying to build on an
> HPUX 9.x system.

xmkmf is an imake command.
You should check to see that your OPENWINHOME environment variable is set
to /usr/openwin.
I'm no expert, but this is what I had to do.
Also, if possible, type: make Makefiles
instead of using the MakeMakefile script.
