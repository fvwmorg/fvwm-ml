Hi Dorothy,
        I just answered this question before I saw yours, so try

*FvwmButtons Xemacs gnu.xpm Exec "Xemacs"
/usr/bin/xemacs -geometry 95x41+22+9 &

Where gnu.xpm is an Icon, this is just an example. I am using
fvwm2-2.0.43-1 on Linux and it works fine.

Make sure you have a path like this to your Icons:
IconPath /usr/X11/bitmaps/:/etc/X11/fvwm2/icons

Hope this helps,
