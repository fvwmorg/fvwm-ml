> From: Barry D Benowitz <bbenowit_at_telesciences.com>
>
> I would like to set up FvwmPager to only display the active desktop,
> not all the possible desktops. I would use another method of switching
> between desktops (probably tied to a button).
>
> Is This possible?
>
> Barry

Start the pager up iconic ... It will become an "active" icon of the
current desktop. When up de-iconify it you will be able to use all of the
desktops. Add the line "*FvwmPagerStartIconic" to your .fvwm2rc.
