> This works, except that the second (or later) time I run it, whichever
> xterm had the mousecursor over it doesn't get grown, and an xterm with
> the same name on another desk gets ungrown.

The function relies on unique window names. If you run it a second
time before all those windows were closed, it fails. Try

  + I (xterm1, CurrentDesk, CurrentPage) Maximize grow grow

Or give the xterms unique names:

  + I Exec exec xterm -n "xterm_+$[vp.x]+$[vp.y]_at_$d" -g +0+0
  ...
  + I (xterm_+$[vp.x]+$[vp.y]_at_$d) Maximize grow grow
  ...
