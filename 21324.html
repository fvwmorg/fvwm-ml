Sorry if this is in the manual, but I couldn't find it. Is there a way to
turn off xforms' use of backing store? It seems that the default is
WhenMapped, I'd like to change it to NotUsefull for my app.

Actually, what I really want to do is have xforms redraw its windows when I
use fvwm's refresh command. I've noticed that all the windows with backing
store disabled will redraw themselves, but the xforms windows won't. I'm
guessing that xforms is relying on backing store to do the re-draw, which
isn't working for my app.
