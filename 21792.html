It looks like the problem is the placement of your PixmapsPath line.
The docs claim that .fvwm2rc is no longer dependant upon order as in
1.24, but there are still some things where the order matters. Try
moving your Path definition lines to the top of .fvwm2rc. As for docs,
try hitting the FVWM site at http://www.hpc.uh.edu/fvwm/. Full
manpages are distributed, as well as links to samples. The site also
maintains a link to the tkgoodstuff8 home (last I checked) though
tkgoodstuff has been orphaned by its developer. The docs should be
available from the site, all the same.
