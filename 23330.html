CVS == Concurrent Versioning System

This is a source code control system used by many of the Open Source
projects. One of its prime advantages is that it is "internet aware"
and you can check out and in code across the net. And it's free
software, of course :).

Also, you can check out (only) code anonymously, so if you have a CVS
client you can obtain up-to-the-second source code directly from the
development tree to play with, if you like.

  http://www.cyclic.com/

I'm surprised you didn't find anything about CVS on the web pages, I'm
pretty sure there's a discussion of how to obtain sources via CVS, etc.
Yep, look under the "Docs, FAQ, & other Info" link:

  http://fvwm.math.uh.edu/docs.html
