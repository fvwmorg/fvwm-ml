Jos wrote:
> So my question: is it possible to manually set these
> hilight and shadow colors? (Ok, I'm being somewhat picky..)
>
Yes, but not with the stable release. In the current beta you can use
FvwmTheme to
create a colorset with whatever shadow and hilight colors you want. You
then have to configure fvwm2 and the modules to use the colorset (see
the FvwmTheme man page for examples).
