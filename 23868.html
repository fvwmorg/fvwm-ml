Brad Giaccio wrote:
>
> I know if I have a key bound to Next [<some window>] Focus it will give
> the window I specify Focus and leave the cursor in whatever window I was
> in.My question is is there a way to give focus back to the window with
> the cursor in it?

You must be using SloppyFocus/MouseFocus. If you have a menu bound to a
key that always works you can hit that, then Escape. The pointer should
be returned and the focus changed back. Alternatively try this:

Mouse 2 A M NOP

It uses up alt-btn2 to do nothing but it does change focus.
