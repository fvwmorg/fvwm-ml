> complete client side-application. Do I use GTK, TK, TCL? I've never

GTK, TK, Qt, Fox, fltk, ....
You cold use them all.
Glade can be used to build an interface. (GTK)
Qt also has an interface builder, but I can't remember the name.
It's also possible to use a scripting language to use these guis.
Ruby for example has bindings for all these guis.
