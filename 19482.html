fvwm-2.0.45, FvwmIconMan-0.9.7: anyone else experiencing the following?

have FvwmIconMan configured horizontally (to act similar to TaskBar in
fvwm95), with fixed geometry. far right 20 or 25 pixels appear to be
cut off (i.e., root window shows through). relevant configuration is
attached.

other than that, it appears to work quite nicely (thanks, Brady!).

--jim

############## jim knoble ###### jmknoble_at_pobox.com ################

AddToFunc "InitFunction" "I" Module FvwmPager 0 0
+ "I" Module FvwmIconMan

AddToFunc "RestartFunction" "I" Module FvwmPager 0 0
+ "I" Module FvwmIconMan

Style "FvwmIconMan" NoBorder, NoHandles, NoTitle, StaysOnTop, Sticky

*FvwmIconMan*nummanagers 1
## the below evaluates to 96x20 for a screen resolution of 1024x768
*FvwmIconMan*buttongeometry m4_eval(_WIDTH_ / 8)x20
*FvwmIconMan*managergeometry 8x1+0-0
## helvetica-12 is an alias for the appropriate helvetica font
*FvwmIconMan*font helvetica-12
*FvwmIconMan*format "%i"
*FvwmIconMan*background DarkGray
*FvwmIconMan*foreground Black
*FvwmIconMan*plainbutton up
*FvwmIconMan*focusbutton down
*FvwmIconMan*focusandselectbutton down
*FvwmIconMan*selectbutton up
*FvwmIconMan*title {TaskBar}
*FvwmIconMan*iconname {TaskBar}
*FvwmIconMan*drawicons true
*FvwmIconMan*followfocus true
*FvwmIconMan*sort true
*FvwmIconMan*usewinlist false
*FvwmIconMan*dontshow resource=FvwmIconMan resource=FvwmPager resource=FvwmButtons resource=asclock resource=xmailbox resource=xeyes
## ^^^^^^^^^^^^^^^^^^^^
## could this be the culprit? ___/
##
*FvwmIconMan*resolution desk
*FvwmIconMan*action Mouse 1 A sendcommand "Iconify -1", sendcommand "Focus", sendcommand "Raise"
*FvwmIconMan*action Mouse 2 A sendcommand "Iconify"
*FvwmIconMan*action Mouse 3 A sendcommand "Iconify -1", sendcommand "Focus", sendcommand "Lower"
*FvwmIconMan*action Mouse 1 S sendcommand "Iconify -1", sendcommand "Focus", sendcommand "Raise", sendcommand "WindowShade 2"
*FvwmIconMan*action Mouse 3 S sendcommand "Iconify -1", sendcommand "Focus", sendcommand "Raise", sendcommand "WindowShade 1"
