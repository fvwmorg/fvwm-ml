This is a very nice feature of olvwm, another feature of the OLIT set
that I'd like to see put into fvwm is a Menu feature.

Anyone that knows OL has seen this as the "DEFAULT" option under menu
building.

In explanation: DEFAULT does quite a few things.
For 1> if you go to select a menu with a default item and release the
mouse button, the default is activated just like you had actually moused
into the menu and selected it.
2> when the menu does come into view (off of another menu) it is aligned
such that the default selection lines up with the menu activator
3> if you mouse out of the menu (and it's still there) and mouse back
into it, the mouse warps to the default (I may remember this part wrong
;)

Any other nice features to think of?
