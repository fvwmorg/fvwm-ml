xwin> I was wondering if there is anyplace on the net where I could
xwin> get some FVWMmodules that don't come with FVWM.. Also I was
xwin> wondering if the modules work for both version of fvwm or if
xwin> they are version specific..

I've collected (in my archive named CFAN :-) some modules which is not
included in fvwm distribution. Here is a list.

* FvwmAdlLog
        Research oriented module to get data about the user action.
* FvwmWish
        One of Tcl/Tk extension, to handle module interface.
* fvwish
        Yet another Tcl/Tk extension.
* TkGoodStuff
        Exactly this is NOT a module but an application,
        includes TkFvwm (Tcl/Tk extension for fvwm)
* FvwmTaskBar
        Windows95-like taskbar. Be with to fvwm95.

I think most of all modules are come with 2.0.42. You don't worry :-)
