Hi,

Try

fvwm2 -cmd 'FvwmM4 -debug /etc/X11/TheNextLevel/fvwmrc.m4'

you get few errors on icon but fvwm2 comes up correctly with the similar menu
that fvwm95. The bug come from the file /etc/X11/xinit/Xclients that attempt to
start fvwm2 and fvwm95 in same manner with "fvwm2 -f 'FvwmM4..." command. Note
the -f instead of -cmd.

Hope you this help.
