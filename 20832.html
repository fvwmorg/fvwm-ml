Hi folks

I've added some more features to the Pager balloons. Now ...

 - horizontally, the balloon is centred on the pager window, but
   will move so as not to go off the edge of the display;

 - vertically, you may specify a y offset: +n places the
   balloon n pixels below the pager window, -n places the
   balloon n pixels above the pager window;

 - it is possible to set the balloon window border width and colour.


You can get the new patch or a tarball from

    http://www.giccs.georgetown.edu/~ric/software/fvwm/pager.html

Have fun!
