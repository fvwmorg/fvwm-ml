Hi there,

I wonder if it is possible to use fvwm to insert (often used) text in
applications as it would have been typed by the keyboard.

Some text one has to type very often e.g. the own email address or
account numbers. It would be nice if this text can be inserted by a
key stroke or a mouse click.

Any suggestions how this can be achieved? Thanks in advance.

If this is possible one could think about a dynamic menu which holds
for example 10 often used entries. New entries can be added by a fvwm
function the oldest will removed automatically. Maybe someone has
already implemented something like this.
