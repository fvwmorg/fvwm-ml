Rick Jones <ethyr2000_at_yahoo.ca> writes:
> Is there a way to have an xterm started in a buttonbar
> start a console app on startup? Of course, this only
> applies to the xterm in the buttonbar.

Start up the xterm using the -e arg:

xterm -e tail -f /var/log/messages
