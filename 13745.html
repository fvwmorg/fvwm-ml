For a while I've had a button bar at +0+0, but now I want to move it.
Unfortunately, I've had quite a bit of trouble doing this. When I use the
config below, with the "Module FvwmButtons DesktopButtons -g +0+116", I
get a really small (16x16) FvwmButtons at the correct location. If I use
"Module FvwmButtons DesktopButtons", I get an FvwmButtons at +0+0 and of
the correct size.

Thanks,

jpkotta



Version info:

fvwm 2.5.12 compiled on Oct 24 2004 at 03:36:42
with support for: ReadLine, XPM, PNG, Shape, XShm, SM, Xinerama, XRender,
XFT, NLS



Config file:

KillModule FvwmButtons
DestroyModuleConfig DesktopButtons: *

*DesktopButtons: Geometry +0+116
*DesktopButtons: ButtonGeometry 16x16
*DesktopButtons: Columns 8
*DesktopButtons: BoxSize smart

*DesktopButtons: (8x8, Frame 0, Swallow "xclock" "Exec exec xclock")

Module FvwmButtons DesktopButtons -g +0+116
#Module FvwmButtons DesktopButtons
