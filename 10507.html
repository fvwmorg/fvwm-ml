It is not clear from your message whether you use fvwm-themes (and which
version) or just copied some configuration from it.

If you do use fvwm-themes, then run "fvwm-themes-config -e | grep ^Style
| less" from command line to see all style lines.

I guess there is something like:

  Style "FvwmButtons" NoTitle, WindowListSkip, CirculateSkip

and your mozilla window happens to have the title named "FvwmButtons".
(This is only an example.)
