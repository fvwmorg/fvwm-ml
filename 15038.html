I think you mean win95, right?

Bowman (http://www-acs.ucsd.edu/~byang/bowman/) is a hack of fvwm 1.xx
to make it look like the Nexstep window manager, and then there is a
hack to bowman to make it look like (shudder) Windows95.

Todd> I forget who did it, but it is on my web page..

Todd> http://www.cs.umr.edu/~tfries/html.cgi?file=fvwm or =linuxurls
Todd> I forget...

Bowman itself is in:

               ftp://sdphu1.ucsd.edu/pub/byang/bowman/

The win95 patch for bowman is in:

               ftp://ftp.netppl.fi/pub/linux/win95look/

Eventually, changes may be merged into fvwm2 which will allow pixmaps
for buttons and title style changes which will allow you to create a
win95 look with it.

But until then, you'll have to go w/ the hacked bowman if you really
want the win95 look.
