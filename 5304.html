On Mon, Jun 10, 2002 at 11:12:35AM +0200, Riswick, J.G.A. van wrote:
> HI
>
> is there some way, for applications that don't have an icon
> defined by the 'Style icon ...' command, to find out the name
> of the icon that is displayed?

Not with fvwm. The icon is provided by the application and may
be even hardcoded into the executable without any name. Even if
it came from a separate file, the application doesn't inform the
X server about the file name. All you can do is to search your
disk for the iconf file.
