The method I use for xpm's is
xpmroot it comes with fv
inyour init function add
+ "I" Exec xpmroot <filename>


or for other image types
+ "I" Exec xv -root -quit <filename>

with xv there are various switches that go with -root that do things like
tile or maximize or mirror or ...

hope that helps,
Brad

P.S.
as for the man page try
man fvwm2
or
rpm -ql fvwm2 and you should see the path to the man page in there
