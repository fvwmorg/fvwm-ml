> Hello,
>
> I'm using fvwm for a long time and my questions are in my head for a
> long time:
> 1. Is it possible to include in further versions of FVWM the support for
> png (so that I can use png icons)?

Someday fvwm may have support for png, but,
a png icon wouldn't look any different than the same icon converted
to XPM. Wouldn't you rather see fvwm conserve its memory use
rather than add support for more image types?

You should be able to convert your png icons to xpm icons.

> 2. Why there isn't an offcial site for fvwm theming (on fvwm.org or
> themes.org)?

At fvwm.org on our "Links" page there is a link to the official
FVWM Themes home page.

> 3. I would like to use fvwm2.4 to imitate fvwm95. Have you got tips?

In our latest betas, there is a built in menu to create a 95 look.
