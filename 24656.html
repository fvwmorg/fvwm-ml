Hi,
  I decided to redo my .fvwm2rc file with FormFvwmSetup, and have found
a slight flaw with it. It creates module configurations in your home dir
called FvwmButtons, FvwmPager, etc.. but in the .fvwm2rc file it puts
this:

# Read config files for modules:
read ConfigFvwmButtons
read ConfigFvwmPager
etc...

so fvwm looks for ConfigFvwmButtons at starup but cant find it(because
its been named FvwmButtons). It was easy enough to fix, I just thought I
would let you all know.

thanks,
sa
