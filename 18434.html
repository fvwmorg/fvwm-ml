Hi all!
 
   Apart the other nasty little bug (dotted cursor hanging around
sometime) I have found a little problem with focus/raise (I don't
know if autoraise related). To be fast and effective: i start a
terminal via "InitFunction", and this and _only_ this window show
marked attitude to stay in the foreground: if I open and position
a window partially overlapped with it, the new window got IMMEDIATLY
lowered and the first one got focus. This happens only with this
first window.

   Data: fvwm 2.0.43 on a Linux RedHat 3.0.3 (kernel 2.0.25 libc 5.2
Xfree 3.1).

   The starting sequence is:

AddToFunc "InitFunction" "I" Module FvwmAuto 1500
+ "I" Exec BDROP_COMMAND BDROP &
+ "I" Exec rxvt -geometry +100+100 &
+ "I" Exec MYBIN/myxwatch &
+ "I" Module FvwmButtons

Any hints? Is the message understandable at all :-)?
Bye, happy fvwming,
