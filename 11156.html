 --- Alexander Fitterling <alexander.fitterling_at_gmx.net> wrote:

> Hello to everyone.
>
> I wonder if there is any clock applet available to use it with
> FvwmButtons. It should be capable of xft as fvwm 2.5.8 does.
>
> Xft works fine with my configuration, but so far the clock (I use a
> swallowed xclock) does get displayed the old style.

You can set the font style of xclock in ~/.Xdefaults (often a symlink to
~/.Xresources):

xclock*font: my_font_name

Then do:

xdrb -merge ~/.Xdefaults

And load the clock, see if that does it.
