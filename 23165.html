You need to set a "Reply-To:" field if you want replies to go to the
list rather than yourself. This is documented in RFC 822 section
4.4.3. I see that you have a "Mail-Followup-To:" but this is non-
standard.

Perhaps the majordomo could be configured to add a reply-to if none
has been set by the original poster.
