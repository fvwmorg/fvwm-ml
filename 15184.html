I am using fvwm-2.0.40 on a Sun (Solaris 2.3) and have a problem when
moving a window, usually by clicking the mouse on the _title bar_ and
dragging the window. When the move cursor hits one of the screen edges,
the move gets frozen/paused at this location for a little while; you can
still move the 4-arrow cursor away from the screen edge with the mouse,
but it takes a while before the actual window starts moving again.

Turns out that this is related to the EdgeResistance setting (I'm using
1500 50). I have it set to 1500 because I dislike all the inadvertant
page changes I end up making! If I reduce this time (to, say, 50), I
don't get this pause on window moves to the screen edge, but I'm always
changing pages!!

Has this problem been seen before? Is there a possible or known fix? I'm
going to look at the code tonight, but I've never done _any_ X
programming, so I don't count on finding anything. But, I'm interested in
learning it!

Thanks for the great window manager!!
