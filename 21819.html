Near 2000, Michael Hucka mumbled something about:

> Here's a stupid question.

Why would it be stupid ?

> I just managed to fling a window off the desktop entirely in fvwm2, so
> that it doesn't show up in the pager box at all.

C | N > K

> How can I get it back? I need a "find such-and-such window and put it back
> in the center of the screen" command, so I can get to it again.

I can't think of the "put it in the..." command, but you should be able to
access it using the FvwmWinList module.
You can bind functions to the mouse clicks there.
So you pop up the WinList and click on a mouse button to activate the
"get it back" function somebody else will have to give you.

I just thought of another *really freaky* idea:
Create a new buttonbar with only one button in there, swallowing the lost
window. Then KillModule that buttonbar. The window should be released somewhere
within the current viewport, then :-)

Hope this makes any sense,
