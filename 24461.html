> Also, is there a way to make all the icons the same size so that they
> have a uniform look? Is there an option I can specify?

I wanted the same uniform look so I've specified Icons for programs which
would normally provide their own as well as a default:


Style "NCDterm" Icon Terminal.xpm, Sticky
Style "NCDconsole" Icon Terminal.xpm, Sticky
Style "xlsbatch" Icon Database.xpm, Sticky
Style "Crisp" Icon TextEditor.xpm
Style "Netscape*" Icon Multiview.xpm
Style "pine" Icon Mail.xpm, Sticky
Style "*" Icon Shell2.xpm, Color lightgrey/dimgrey


My specified icons were choosen in such that they are the same size.
