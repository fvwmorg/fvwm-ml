Make up your mind, fvwm2, or fvwm95?

On my type 4 Sun keyboard under Solaris 2.6, the keys F25,6,and 7
are keypad divide, keypad multiply, and there is no F27.

Anyway, fvwm2 will use them if they exist. Use xkeycaps to verify
that they exist and to check which keys they are on.

> -------
>
> Is there a way to bind something to the print-screen, Pause and
> Scoll-lock keys on a PC keyboard?

Should be no problem. Find out which keysym they generate, bind
the key. If they don't currently generate a keysym, find the keycode
and use xmodmap to assign a keysym.

> How would I bind a key to focus the FvwmButtons? Sometimes this gets
> hidden under windows and it'd be nice to hit something to pop it to the
> top.

Write a function like this:

AddToFunc Warp "I" Next [$0] Iconify -1
+ "I" Next [$0] WarpToWindow 24p 24p
+ "I" None [$0] Beep

then bind a key like this:

Key Return R N Warp FvwmButtons
