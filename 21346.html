        Sorry, (again) I will repost the fvwm2.man diffs but for now it's
as easy as:

        MenuStyle .... next menu_style title_style

where:
        .... is the normal MenuStyle command (colors, fonts, etc)

        menu_style and title style are Style Definitions of the form:

        gradient_style start_color end_color #_of_steps

with gradient_style one of NWSE, NS, NSN, WE, WEW

if you put None as gradient_style you don't need to specify any more
parameters ( example: MenuStyle ... next None None )
