I think Dedeco wants to change what FvwmPager does
on a button press.

The closest thing I see is the SloppyFocus option
for FvwmPager.
That causes FvwmPager to run this:

FlipFocus NoWarp

Other than that, I don't see where FvwmPager allows
any customization of the command run after a click.
