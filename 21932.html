>>>>> On Mon, 6 Apr 1998, Garvey Paul said:
Garvey> I was just wondering if it was at all possible to restart my
Garvey> .fvwm2rc file from the command prompt instead of using the
Garvey> builtin menu option?!

>>>>> Kai Grossjohann replies:
Kai> There is FvwmTalk which opens a window where you can type Fvwm
Kai> commands. I think there is also FvwmConsole which might or might
Kai> not be nicer, but in principle it does the same thing.

There's also FvwmCommand which lets you run FvwmCommands from the
command line. Be aware it's in the "unsupported" extras directory
(but then, I think FvwmConsole is too) so you may have to do some work
to get hold of it. To use it you run FvwmCommandS module from Init
and Restart functions and FvwmCommand program at the command line. I
use it to do strange things at startup such as force my initial
Netscape window to start on Desk 5 but once Netscape is up and running
reset the style to StartsAnywhere so when emacs' browse-url-netscape
causes a url to be loaded in a new window, it pops up on whatever desk
I'm running emacs in.
