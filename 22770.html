Hi,

It would be nice if FvwmButtons had the following feature:

Let me describe it...

In HTML you can create a clickable image just like you can have a "clickable

pixmap/bitmap" in FvwmButtons. If you use

<img src="foo-bar.gif" alt="[ this picture says foo-bar ]"> in the HTML file

then pointing at the image with the mouse on the browser causes the sign

"[ this picture says foo-bar ]" to show up

(at least this is true with Netscape).

So, can we produce a similar effect in FvwmButtons, that is, I point at an

icon and FvwmButtons gives me a brief description of the Icon?

I think this would be a nice feature.
