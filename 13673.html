On 11 Jan 2005 11:19:58 +0800, jianqiu wrote:
>
> I refer to man page and got this
>
> Key [(window)] Keyname Context Modifiers Function
> Key F11 A SCM WindowList
>
> now how to bind other key rather than "Alt Shift Shift" ,
> I try Key F1 A Super_L WindowList but fail.

Someone suggested xev, it is not the best tool to run in this situation.
Run xmodmap to see which modifier is Super_L. It is mod4 for me, so:

  Key F1 A 4 WindowList SortByResource, NoDeskSort, NoGeometry, NoHotkeys
