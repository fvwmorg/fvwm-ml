>>>>> "Kai" == Kai Grossjohann <grossjoh_at_charly.informatik.uni-dortmund.de> writes:

Kai> Currently, FvwmPager uses about three colors: One for the background,
Kai> one for the foreground, and one for the highlighting. Is this
Kai> correct?

Kai> In a non highlighted desk/page, the little windows can be seen only
Kai> because they have a border round them. Is there any way of making
Kai> them have a different color than the little root windows?

Kai> To illustrate:

Kai> Desk 1
Kai> +-------------+
Kai> | |
Kai> | +-------+ |
Kai> | | (*) | |
Kai> | +-------+ |
Kai> | (*) |
Kai> +-------------+

Kai> (*) These two areas should have different colors.

The background color of the miniature windows are grabbed from the
window itself. For instance, typing this into FvwmTalk:

        Style "*" Color black/yellow
        Recapture

would cause all of the miniture windows in the pager to show up
yellow.

So, you should just be able to change the Background of FvwmPager or
the background of your non active windows to have it display as
different colors.
