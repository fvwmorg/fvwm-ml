Hi,

I've finally decided to move from fvwm-1.24r to latest
fvwm2. I've already converted my config file
succesfully, but two minor problems remain. Could any
one help with these?

1) Keyboard accelerators F11 and F12 seem to "steal"
bindings from F1 and F2, respectively. I have in my
.fvwm2rc:

Key F1 A M RaiseLower
Key F2 A M Popup "Window Ops"
...
Key F11 A M Maximize 100 100
Key F12 A M Close

If I use Meta+F1, it maximizes the window! If F11 line
is commented out, Meta+F1 does RaiseLower. What's
going on here?

2) I have a menu containing few Popups:

AddToMenu "Utilities"
+ "Program Menu" Title
+ "xterm" Exec exec xterm &
+ "Mozilla" Exec exec mozilla &
+ "" Nop
+ "Applications" Popup Apps
+ "Utilities" Popup Utils
...

There seems to be short delay before a Popup is shown
when I move pointer there. How can I disable this
delay?

My system is RHL7.3.

PS. Is there any plans for fvwm3? If so, does it
understand my new fvwm2 config files?

Thanks in advance!
