Greetings,

Last week I made the decision to move from fvwm 1.xx to 2.xx.
I dowloaded the latest version fvwm-2.0.45 and I upgraded
my config file as explained in the doc.

Everything works fine except the FvwmButtons. I had
a couple of GoodStuff buttons which I want to convert
into FvwmButtons. But whatever I try in the .fvwm2rc
I cannot get any buttons displayed on my screen.

Looking at the source directory in modules/FvwmButtons,
I recompiled the module with the various debug flags
set. The output is as follow :

FvwmButtons: Parsing...OK
FvwmButtons: Shuffling...OK
FvwmButtons: Loading data...
FvwmButtons: Loading: Button 0x804bf00: colors, colors2

Well, it seems correct !

After having launched the FvwmTalk I tried the
example samplebuttonrc given with the module.
The only thing I got was a xclock located
at a strange position in a right hand desktop.
If I moved into this desktop, the clock escaped
further to the right !

Similarly, if I make use of the system.fvwm2rc provided
with the package, I got a FvwmTalk window when I fired
up the Buttons module. It's very confusing !

I have tried to check out the page at the specified URL in
the README : http://www.fm.unit.no/~jatotal/fvwm.html.
Unfortunately the link is out of date :-(

I browsed the archive of the mailing list looking for
articles about Buttons but the output of the
search engine is not easy to manage.

Here are the spec of my configuration :

Linux 2.1.26
gcc version 2.7.2
XFree86 11.0 vendor release number: 3120

And last, the paragraph in my .fvwm2rc dealing with
the buttons :

# FvwmButtons button-bar

*FvwmButtonsFore Black
*FvwmButtonsBack Red
*FvwmButtonsFont -adobe-helvetica-bold-r-*-*-10-*-*-*-*-*-*-*
*FvwmButtonsGeometry -5-10
*FvwmButtonsColumns 1

*FvwmButtonsGeometry -135-5
*FvwmButtonsRows 1
*FvwmButtonsFont -*-helvetica-medium-r-*-*-12-*
*FvwmButtonsPadding 2 2

*FvwmButtons(Title Kill, Icon rbomb.xpm, Action Destroy)

*FvwmButtons(Title Mail, Icon mail.xpm , Action \
        'Exec "Mail" /usr/local/bin/ml &')
*FvwmButtons(Title Netscape, Icon spider.xbm, Action \
           'Exec "Netscape" netbrowser &')
*FvwmButtons(Title Xcalc, Icon rcalc.xpm, Action \
        'Exec "Calculator" xcalc &')
*FvwmButtons(Title Xpal, Icon xpaint.xpm, Action \
          'Exec "Palette" xpal &')

Thanks in advance for your help,
