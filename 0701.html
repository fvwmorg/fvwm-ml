Johannes Zellner wrote:
> is it possible to mix colors/fonts in window titles.
> e.g. in from my shell I can write on the window title

  I don't believe so. You might achieve that sort of thing by having a
script to call gimp/image-magick to generate a correctly-sized XPM with
the required coloured text in it, then setting that as the title-pixmap
style for a given window ID, but you'd *really* have to want that coloured
text!
