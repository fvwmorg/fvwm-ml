Hi!

I have set up my desktop with two StaysOnTop windows: FvwmButtons and
FwvmIconMan. I use SmartPlacementIsReallySmart. New windows should
avoid overlapping with StaysOnTop windows and icons, that works very
well except for FwvmIconMan, that apparently is completely invisibel to
the SmartPlacement routine. Not only is it not avoided as a StaysOnTop
window, overlap with FvwmIconMan is not at all taken into account when
placing new windows. Can anybody tell me if I can change this? I have
appended the relevant (I hope) lines in my .fvwm2rc below.

Versions: fvwm 2.0.45, FvwmIconMan 0.9.8

Thank you!

Jakob


>From my .fvwm2rc:

                [ snip ]
Style "*" SloppyFocus
Style "xbiff" ClickToFocus
Style "xclock" ClickToFocus
Style "xload" ClickToFocus
Style "top" ClickToFocus
Style "FvwmPager" ClickToFocus

Style "*" MWMFunctions, MWMDecor, MWMButtons

                [ snip ]

############################################################################
# WINDOW Placement
#
# SmartPlacement makes new windows pop-up in blank regions of screen
Style "*" SmartPlacement
GlobalOpts SmartPlacementIsReallySmart

# If SmartPlacement fails, this places it randomly instead of making you
do it
Style "*" RandomPlacement

                [snip ]

############################################################################
# STYLE Flag Setup
#
# (decorations and window options)
# Note: Order is important!! If compatible styles are set, styles are
ORed
# together. If conflicting styles are set, the last one specified is
used.

# change the default width, set a global icon, and give borders to
popups
Style "*" BorderWidth 5, HandleWidth 5, Icon unknown.xpm,
DecorateTransient, NoPPosition

Style "Fvwm*" NoTitle, Sticky, WindowListSkip
Style "FvwmButtons" HandleWidth 0
Style "FvwmIconMan" HandleWidth 0, ClickToFocus, StaysOnTop,
CirculateSkip
Style "FvwmButtons" ClickToFocus, StaysOnTop, CirculateSkip

                [ big snip ]

*FvwmIconMan*nummanagers 1
*FvwmIconMan*background #c0c0c0
*FvwmIconMan*foreground black
#*FvwmIconMan*format "%t"
*FvwmIconMan*format "%i (%c)"
*FvwmIconMan*buttongeometry 320x0
#*FvwmIconMan*managergeometry 1024x0-0+0
*FvwmIconMan*managergeometry 3x1+0-0
*FvwmIconMan*drawicons false
*FvwmIconMan*resolution page
*FvwmIconMan*followfocus true
*FvwmIconMan*focusbutton down black #c0c0c0
*FvwmIconMan*focusandselectbutton down black #e0e0e0
*FvwmIconMan*selectbutton up black #e0e0e0
*FvwmIconMan*dontshow class=FvwmButtons resource=FvwmPager

*FvwmIconMan*action Mouse 1 N sendcommand Focus, \
            sendcommand "Iconify -1"
*FvwmIconMan*action Mouse 2 N sendcommand "Iconify 1"
*FvwmIconMan*action Mouse 3 N sendcommand "Iconify"
