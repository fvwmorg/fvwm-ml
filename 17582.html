    Right! Don't use cmdtool or shelltool, but if you need to paste
    something in a mailtool or so on, you can add the following line
    in your .Xdefaults:

XTerm*vt100.Translations: #override \n\
            <Key>L6:select-set(CLIPBOARD) \n\
            <Key>L8:insert-selection(CLIPBOARD) \n

    which will remap the copy and paste keys in a xterm.
