You begin with Linux by buying a `distribution', basically the OS
kernel bundled with applications. I think Red Hat is a good
distribution and I know that Debian is good (use it myself).

Fvwm is just one program among many. Beginning to use Linux because
of Fvwm is similar to deciding to switch from MacOS to Win95 (say)
because of the File Manager in Win95. You will probably get Fvwm with
your Linux distribution, but it might be an old version and you could
go to the Fvwm web site (look at the bottom of this message) to fetch
the newest version. Fvwm is labeled beta but it is very stable
(hasn't crashed for me once!).
