On 17 May 2000 13:41:56 -0700, Arya Behzad wrote:
>
> If I have two desktops defined (or more), how can I move a window from one
> to the other?

Using the second mouse button in a multidesk pager (Module FvwmPager 0 3)
or using a MoveToDesk command. For example, you can create a menu:

AddToMenu MenuMoveToDesk
+ "Move to desk 0" MoveToDesk 0 0
+ "Move to desk 1" MoveToDesk 0 1
+ "Move to desk 2" MoveToDesk 0 2
+ "Move to desk 3" MoveToDesk 0 3
