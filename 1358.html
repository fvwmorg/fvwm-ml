Hi Thomas,

On Fri, 1 Sep 2000, Thomas Adam wrote:

>My question on FVWM is, how do I start other programs
>on different virtual desktops?? I know that this has
>been documented many times, but each time I try
>something goes wrong.

Look a the man page for fvwm2 for StartsOnDesk. There are examples there
such as

xterm -xrm "*Desk:1"

>I am using Linux Mandrake 7.0 (air). All my fvwm2 init
>files are located in /etc/X11R6/Anotherlevel/ I did
>succeed in starting a program called R on a virtual
>desktop but the problem was, was that I was unable to
>switch back to my main desktop. How do I go about
>avoiding this??

Well, it sounds like you are using the AnotherLevel package written by
RedHat. In this case you are using FvwmM4 to proprocess the m4 files in
that directory. In this case I expect that you have a window manager
style such as Fvwm95. Not knowing exactly which style you are using,
there is probably a pager in the lower right hand corner. If you clik
with the left button on the pager it will take you to the other desk.
Furthermore if you look at the file fvwm2rc.keys.m4 you will see the key
bindings they have put together for you. There is one there for
switching desks.

Hope This Helps,

Tom
