>
> Would be nice if we could do
>
> Current/Next/WindowID/etc (x/y/width/height </> number) op

For complex expressions you may use shell/awk/expr/perl. Try in 2.3.19+:

Key P W CM Current (Netscape*) PipeRead `[ $[w.width] -lt 210 ] && echo Close`

Now press Ctrl-Alt-P in any window. Only if you are in a netscape window
and its width is less than 210, the window will be closed. That you want.
