Hey, all. I am running Mandrake 7.1, with Fvwm Version 2.2.4.

I have the following key mappings in .fvwm2rc:

# Alt-Tab simili-Windows behaviour
Key Tab A M Next [CurrentPage !iconic] Focus
Key Tab A MS Prev [CurrentPage !iconic] Focus
Key Tab A M3 Next [CurrentPage !iconic] Focus
Key Tab A MS3 Prev [CurrentPage !iconic] Focus

Key Tab A CM Next [CurrentPage] Deiconify-and-Focus
Key Tab A CMS Prev [CurrentPage] Deiconify-and-Focus
Key Tab A CM3 Next [CurrentPage] Deiconify-and-Focus
Key Tab A CMS3 Prev [CurrentPage] Deiconify-and-Focus

and Deiconify-and-Focus is defined as:
AddToFunc Deiconify-and-Focus "I" Focus
+ "I" Iconify -1


The alt-tab and ctrl-alt-tab stuff itself (with NumLock off) works fine.
But if I put the NumLock on (which I always use) it stops working. Is
there any way to find out what the modifier applied by NumLock is?

I don't know if it matters, but I am using a Micro$oft Natural, but
Mandrake does not seem to map the Window buttons (or do they?).

Thanks.
