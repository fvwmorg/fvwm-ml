Michael Kirby said:
> Not sure where the basename library call is supposed
> to be. Is that part of the standard C library?


basename() is not part of ANSI C, but is present in some libc's.
Here is a substitute:


#include <string.h>
char *
basename(char *path)
{
        char *c = strrchr(path, '/');

        if (c) return(c+1);
        return(path);
}
