On 30 Jun 2000 12:00:53 +0200, Eric Raijmakers wrote:
>
> I want to achieve the following:
>
> When the right mouse button is pressed over a button in FvwmIconMan,
> a menu pops up, giving access to operations on the window.
>
> Anyone has an idea of how to achieve this.
>
> I succeeded in popping up a menu, but the menu items did not apply to
> the window corresponding with the pressed FvwmIconMan button.

The syntax is (menu position hints, i.e. last 3 words, are optional):

*FvwmIconMan*action Mouse 3 N sendcommand "Popup WindowOps2 Mouse c -102m"

But there is a bug you described. It is fixed in the developement version.
