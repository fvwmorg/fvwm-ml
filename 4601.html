  A bit of administrivia stuff. I also hit that booby-trap when I first
subscribed. The list is set up such that by default, replies go only to
the original sender. Is this intentional on the list-admin's part ? I
subscribe to more than one list with this "feature". Here's a snippet
of the procmail code I use to write a proper "Reply-To:" line to cause
my replies to default to the list...

:0 fhw
* ^Return-Path: .owner-fvwm_at_hpc\.uh\.edu
* !^Reply-To:.*fvwm_at_hpc\.uh\.edu
  | formail -i "Reply-To: fvwm_at_fvwm.org (FVWM list)"
