Just to let you know, I can reproduce the core dump with the following:

Mouse 0 5 A DoWindowShade
 
DestroyFunc DoWindowShade
AddToFunc DoWindowShade "I" DoWindowShadeUp
 
DestroyFunc DoWindowShadeUp
AddToFunc DoWindowShadeUp
+ "I" DestroyFunc DoWindowShade
+ "I" AddToFunc DoWindowShade "I" DoWindowShadeDown
 
DestroyFunc DoWindowShadeDown
AddToFunc DoWindowShadeDown
+ "I" DestroyFunc DoWindowShade
+ "I" AddToFunc DoWindowShade "I" DoWindowShadeUp

I've examined the crash and there is some problem with ComplexFunction.
In other words, it has nothing to do with Decors or WindowShade (just to
let you know). I'll look into the problem further if I have some time.
