I seem to have some focus issues using SloppyFocus. Sometimes new
windows will not get the focus (it seems like they get it and lose it
again), this is most notable with vim, especially on new files.

I also have a problem with windows in fvwmtabs not getting the focus
when a window above them closes and when switching to fvwmtabs using
M-Tab. Even commands like maximize and hide title don't pass through
properly, instead giving me the cross-hair to point to the window I want
the action to happen on.

I believe the first problem can be solved by warping the pointer to the
window when new windows appear. How do I do that? I already implemented
that by overriding the WindowListFunc but couldn't find how to do it
for new windows.
