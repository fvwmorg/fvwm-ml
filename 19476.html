Hi All,

I believe I have found a new 16 bpp bug in fvwm2.0.45...

When I run fvwm2 with 16 bpp and resize with windows (grabbing the "L" at
the corner of the window), the outlined size of the new window does
not appear. The window does resize correctly.

This is not a problem when I run a 8 bpp.

(I am have a S3 Trio 64+ based card, if that matters)
