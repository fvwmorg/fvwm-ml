I was wondering if it would be possible to add an addition to the
maximize function built in to fvwm2. What I want fvwm2 to do is allow
maximization from the _bottom_ of the screen rather than the top of
the screen. I thought it might be easiest to allow negative numbers
as arguments to the Maximize function.

Another feature I wouldn't mind seeing in fvwm2 is adding a prefered
placement area. As I tend to have all my "needed" windows at the top of the
screen, I don't want any new windows to pop up in this area. I'd like to
make this area off limits to any new windows.
