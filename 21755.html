Jin Ko wrote:
> How can I set certain windows "not resizable"?
> i.e. I don't wanna allow some sticky windows -- like xclock, xload
> ... - to be resized.

I think you'd have to edit the source code for that particular
application. FVWM honors the maximum and minimum size requests that
applications make, but I don't think there's any way you can pull it off
from within FVWM.
