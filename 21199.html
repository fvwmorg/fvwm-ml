    Ville> this, do i have to modify all the progs source to make this

Check out OpaqueMoveSize in the man page for fvwm2. It defaults to
showing the window contents (moving opaquely) if the window is 5% of
the screen size. Larger windows display a rubber band.


Note to maintainers. The man page incorrectly refers to OpaqueMove
rather than OpaqueMoveSize in the paragraph describing OpaqueMoveSize.
