On Sun, 27 Jul 1997, cron wrote:

> I found an image I wash to use for the root background.
> I didn't find anything in the man page - did I miss it?
> Is this possible??

Look through the man page and look for "InitFunction" ... In that
function, you can place the command to place an image in the background.
If the image is an xpm, you can use the program xpmroot (should have came
with fvwm). Otherwise, you can use xv (xv -root -quit <filename>) ...
