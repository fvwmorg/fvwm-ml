I think a FAQ entry is warranted for 'autoraising' windows.

I found this at :
http://igloo.its.unimelb.edu.au/Webmail/tips/msg00565.html

"Autoraise is not implemented now anymore through the statement
Autoraise ..., but you must add the following line explicitly in the
functions InitFunction and RestartFunction :

+ "I" Module FvwmAuto 200

With 200 the reaction time is set to 200ms.

Analogous this is valid for other modules. "
