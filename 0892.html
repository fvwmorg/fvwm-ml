On Tue, May 23, 2000 at 12:44:31PM +0400, Victor V. Evtushenko wrote:
> Hi,
> I'm novice with FVWM, and I have a small problem. I'm running LICQ
> 0.76 with QT GUI plugin 0.71 with appearance option "Use Dock Icon"
> checked. Dock icon runs in separate window named "LicqWharf". At this
> moment I'm trying to swallow this window in FvwmGoodStuff. I have
> following line in my .fvwm2rc :
>
> *FvwmGoodStuff licq licqIcon-sun.xpm Swallow "LicqWharf"
>
> but it does not work. Should I add Exec licq to that line?

1) You seem to use a *very* old version of fvwm. You should
   really consider updating to the stable version 2.2.4. The
   FvwmGoodStuff module has been renamed to FvwmButtons a long
   time ago. I'm not sure if the syntax is still the same,
   but if it is ...
2) ... yes, you have to add "Exec licq" to this line:

     *FvwmButtons licq licqIcon-sun.xpm Swallow "LicqWharf" "Exec licq"

Well, and of course you have to make sure that the new window
really has "LicqWharf" as it's title. Otherwise FvwmButtons
won't find it.
