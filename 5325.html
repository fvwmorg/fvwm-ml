Mikhael Goikhman wrote:
> On 16 Jun 2002 01:39:17 -0700, Tim Hanson wrote:
>
> Two issues. 1) If you want to simply create a static menu, just write a
> script that prints "DestroyMenu MyMenu", "AddToMenun MyMenu" lines and run
> it using "PipeRead your_script" command.
>
> 2) If you want to have dynamical menus, upgrade your FVWM version to
> 2.4.8. The one that is shipped with SuSE 7.1 is too old (2.2.4).
>
> These 2 shell scripts that create dynamical menus are in:
>
> ftp://ftp.fvwm.org/pub/fvwm/devel/sources/utils/
>
> But check whether one of the 4 fvwm-menu-* scripts that are installed with
> the rest of FVWM (2.4.x or 2.5.x) do the job you need. They have man pages:
>
> % man fvwm-menu-desktop
> % man fvwm-menu-directory
> % man fvwm-menu-headlines
> % man fvwm-menu-xlock
