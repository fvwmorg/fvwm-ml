Russell <rjshaw_at_iprimus.com.au> writes:
> Hi,
>
> I'm using fvwm 2.4.6 on debian. The fvwm manual mentions
> a .fvwm2rc example, but i can't find it. Where can i get
> one?

Gee, Debian could put it just about anywhere.

On my Mandrake box, I typed:

locate system.fvwm2rc

and found this:

/usr/share/doc/menu-2.1.5/examples/system.fvwm2rc

If you are trying to create a starting config file, the
best technique is to start up fvwm without any config file
at all, then press any mouse button on the root window
(or press F1), and use one of the setup procedures.
