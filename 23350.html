Will French <wfrench_at_interport.net> writes:
> Hi. I'm working on extending fvwm via a system of scripts, using the
> FvwmCommand module as the interface; and I'm having trouble getting the
> scripts all the info they need. I use "send_windowlist", but I've
> never seen any documentation for it (is there any?) and I don't know if
> I'm making full use of it. Are there any other undocumented commands?

Send_windowlist is documented:

http://fvwm.math.uh.edu/mod_m2f_communication.html

If you find any other commands that you think are undocumented,
let us know.

> Also, my scripts need info about the state of fvwm itself (as opposed
> to the windows it's managing) such as "What key bindings are currently
> in effect?" or "What is EdgeScroll set to?". Is there a way to ask
> fvwm such questions?

As the need to retrieve this kind of information appears, fvwm has
been modified to send it. You would need to change fvwm2, and submit
a patch.
