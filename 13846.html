On Thu, Feb 17, 2005 at 06:43:04PM +0530, Srivathsa Rao U wrote:
> Thomas,

Hello -

> One more small query. You have mentioned ("some_window_name") is the
> code. Is that mean this will work only for some kind of windows, What
> needs to be done for any window I choose ?

You replace "some_window_name" with the name of whatever window you want
to change. So for example, let's say I have an rxvt window called
"rxvta" that I always want sticky on desks 3,4,5. I would change the
example I gace you to be:

+ I All (CurrentDesk AcceptsFocus) ThisWindow \
  ("rxvta") WindowStyle !Sticky

In both the functions where it is defined.

Does that make sense?
