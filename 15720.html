>>>>> "Markus" == Markus Wasserschaff <Markus.Wasserschaff_at_gmd.de> writes:

Markus> Hi!

Howdy.

Markus> I really like FVWM2,

Glad to hear it.

Markus> but there are some features, that exist in olvwm, that I would
Markus> like to see in fvwm2 too.

Of course.

Markus> 1. A zooming effect, every time a window is iconified for
Markus> de-iconified (i.e. in Mac Finder or olwm)

This will most likely never happen. Adding too many 'cute' effects
like this will only serve to bloat and slow down fvwm, which is
something that I am trying really hard to avoid.

And I don't know if this would be possible via a module (I think not),
which would be my typical response after I make a statement like
that...

Markus> 2. What I like about the olvwm's virtual desktop is that you
Markus> can drag a window from the desktop and drop it over the
Markus> virtual desktop manager, thus moving it to another desktop

I don't think that this will be possible in the current setup. The
pager in 2.xx is no longer built into fvwm, but rather an external
module. This makes interaction of the type you would like rather more
difficult, I believe. Although slight changes (like the patch that
Henrique sent to show windows moving in the pager in real time) might
make it easier. We'll have to see. I'll add it to the TO-DO list
with a question mark to remind me to investigate it when I get a
chance.

Chuck
