from http://www.fvwm.org/documentation/manpages/unstable/FvwmPager.php:
<snip>
To use FvwmPager with several different configurations, you can invoke
FvwmPager with an optional parameter, which it will use as its name
instead (e.g "Module FvwmPager OtherPager"). OtherPager will then read
only the lines in the configuration file starting with "*OtherPager", and
not the lines belonging to FvwmPager. This way multiple pager instances
may be used.
