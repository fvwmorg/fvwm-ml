I don't see it.

I tried Mozilla 1.1 from Linux to Linux over a cable modem for
about 10 minutes.

While Mozilla has focus, the blinking cursor causes constant
network traffic.

During page changes the various animations cause heavy X traffic.
During all this, Fvwm still changes frame colors on focus change
or pops up menus instantly.

I don't think Mozilla should be doing anything that would interfere
with Fvwm.

I'm not sure what you mean about giving control back to Fvwm.
