On Thu, Nov 07, 2002 at 09:42:32AM -0500, Song Koh wrote:
>
> If from a Form, I create a button which has commands:
>
> *FvwmForm-Tasks: Command *FvwmPagerSubPanel: Label 0 newLabel
>
> The label in the pager is not updated until:
>
> *FvwmForm-Tasks: Command KillModule FvwmPager FvwmPagerSubPanel
>
> is invoked, and then the Pager is restarted.
>
> Is there a less intrusive way to get the Pager to update the labels?
>
> I am running fvwm 2.4.11

Not in 2.4.11, but with 2.5.x you can use the "DesktopName"
command. Just issue

  DesktopName 0 newLabel

However, I will change the name of the command to "DeskName" or
"DeskLabel" shortly, so be prepared to edit your configuration
when that happens.
