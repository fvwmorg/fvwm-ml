I've been using SloppyFocus so long I'm amazed I never noticed before,
but if a sloppyfocus window is focused but the mouse is completely
outside the window, including the decorations, then apparently either
the side or frame context (I always combine them so I can't
distinguish) is used. This should be documented. Although I think
it's correct, it did take me by surprise.
