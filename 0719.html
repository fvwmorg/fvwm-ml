> Hi,
>
> If when configure I get that I don't have XPM support, then how
> to have it and where can I find i?

Looking back at your previous question, I see that my last answer
assumed you got past this step. Sorry about that.

The file "INSTALL.fvwm" in the fvwm distribution explains what you
need. You need to get and install the XPM library, and then tell
configure where it is.

> I also would like to know how to
> change the window resolution in such a way that all X application
> show small?

Your previous mails indicate that you are running Solaris 2.6 on an
UltraSparc. Usually Solaris picks the highest resolution.

What Window resolution do you have? For example, I get:

fork> xdpyinfo | grep dimen
  dimensions: 1280x1024 pixels (361x289 millimeters)

You can make individual windows smaller by using smaller fonts.
For example:

xterm -fn 6x13bold
