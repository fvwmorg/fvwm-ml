Ok, I see it now.
I did some Google searches.

The bundled compiler apparently only supports K&R C.
I don't think you are going to be able to build Fvwm
with it.

I wasn't able to find any documentation on the compiler
but I do see lots of other programs won't compile with it.

Perhaps you have another compiler installed somewhere else.
On my machine, I have C compilers installed thusly:

 /opt/ansic/bin/cc
 /bin/cc -> /opt/ansic/bin/cc
 /usr/softbench/bin/cc
 /usr/bin/cc -> /opt/ansic/bin/cc
 /usr/ccs/bin/cc -> ./cc_bundled

If you don't have any other HP compiler, I recommend you
install gcc.
