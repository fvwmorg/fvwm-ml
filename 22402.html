> I'm running fvwm 2.0.46 on an SGI Indy (IRIX5.3). I can't run some
> programs from either the fvwm menus or FvwmButtons.

[snip]

> They all work from the command line. I've tried different permissions and program
> locations - nothing helps.

I have had the same problem on an SGI.
The only things I can think of are VERY ugly solutions, but at least they work...

First, try not doing an exec command directly but start a function instead,
in which you start the program with an exec.

If that doesn't help, make a script file in your home directory (I told you it
was going to be ugly !) that starts the program and call this from within Fvwm.

VERY strange problem, though.
On my machine, it just disappeared after a while.
Does anybody know what's happening here ???
