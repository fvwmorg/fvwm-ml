The solution posted has effectively saved my life. I am using the
default focus style for firefox, and have the following key bindings:

Key Tab A M Next (CurrentGlobbalPageAnyDesk, CurrentDesk, !iconic) SelectWindow
DestroyFunc SelectWindow
AddToFunc SelectWindow
+ "I" Focus
+ "I" Iconify false
+ "I" Raise
+ "I" WarpToWindow 50 50

I can reliably but only randomly reproduce this problem everyday. I have
two firefox windows open side by side. And it seems to happen more
often when I highlight in one window(text area or url bar), then use
Alt-Tab to switch focus. Hopefully this helps.
