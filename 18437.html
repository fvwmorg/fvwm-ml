On 20 Nov 1996, Kai Grossjohann wrote:
> S> the windows place themselves, but i want to be able to place them
> S> with the pointer... what modification to the system.fvwm2rc file
> S> do i need to make?
>
> Style "*" ActivePlacement, DumbPlacement

You might also try

Style "*" NoPPosition

Some programs (like netscape) request a certain position on the screen.
This instructs FVWM to ignore that and let you place them.


Speaking of PPosition, doesn't TWM have this kludge where you can tell it
to honor PPostion except when the program requests 0,0? I found this did
the right thing 99.9% of the time.
