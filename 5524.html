Ok, I have whipped together an extremely alpha-quality syntax file for
Vim. You can get it from here:

http://members.shaw.ca/feztaa/fvwm.vim

Probably save it to here: /usr/share/vim/vim61/syntax/ (if you are using
vim61). The location could be different depending on your distro. Just
do a 'locate fvwm.vim' if you're not sure. You might want to backup your
old one before overwriting it with this.

Anyway, all I know is that this highlights the syntax of my own fvwmrc
perfectly. I went through the manpage for 2.5.2 and set it up to match
pretty much everything that's there. There's probably a ton of stuff
that I got wrong because it wasn't in my own fvwmrc and I couldn't test
it properly.

So if you guys see any problems with it, send me your fvwmrc (so I can
see the problem firsthand), tell me whats wrong, and I'll try to fix it.
I'm not terribly interested in backwards compatability with old config
syntax, but if there's enough demand I'll put it in.
