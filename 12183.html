> 1. What is the best way to achieve Windows like alt-tab behavior
> (CirculateUp/Next/WindowList) ?

Try this:

    Key Tab A M WindowList Root c c CurrentDesk, NoGeometry,
CurrentAtEnd, IconifiedAtEnd

> 2. How can I get WindowMaker-like 'move to' feature : moving window to
> certain desk ? (Dragging windows on the pager is an unsatisfying
> compromise)

Being a big fan of the keyboard, this is how I move my windows between
Pages (MoveToDesk might suite you better:

    Key F1 A S MoveToPage 0 0
    Key F2 A S MoveToPage 1 0
    Key F3 A S MoveToPage 2 0
    Key F4 A S MoveToPage 3 0
    Key F5 A S MoveToPage 4 0
    Key F6 A S MoveToPage 5 0
