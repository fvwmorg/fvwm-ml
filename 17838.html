>How the heck to I get the frame or border color to change to what

>and can't find anything there to change it. I also have the FVWM
>faq and am unable to find what I need there. Remember, this is

Okay, there is one of two things (really just one) that is going on
here. There is a command called Style. When you see Style "*", then
all the command apply to all windows by default. Except if the is a
Style "whatever" in which case all command here will be run instead of
the defaults. Color, and every other interface type configuration
runs via Style.
