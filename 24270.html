I am having a really weird problem with xconsole and fvwm placing it at
startup.

I start it with the following:

xconsole -file /dev/xconsole -bg black -fg green -font 5x7 -geometry 743x69+0+258 -xrm "*Page:3"

Yet when it starts it does not show on page 3. If I bring up the
windowlist it list it with the geometry of: 743x69+0+198402.
                                                    ^^^^^^^^
                                                     yikes!!!
                                                     
If I select the window from the window list it brings the window to
+0+0. At which point I can then drag it to the appropiate place in the
window(+0+258).

I am running at 1024x768 on Debian Linux 2.1 with X 3.3.3.1, SVGA server
and fvwm 2.2.

This is the only window that does this. Any clues as why this is
happening?
