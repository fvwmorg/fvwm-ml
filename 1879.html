%% Thomas Pudil <Thomas.Pudil_at_tuwien.ac.at> writes:

  tp> ... but I'd like to set up a ScreenSaver under FVWM2. Where can I
  tp> configure this?

FVWM is a window manager; window manager != screen saver. There is no
screen saver in FVWM.

Go get xscreensaver or xlockmore or similar; these are screen savers.
But, they are not window managers :).

Personally, I like xscreensaver but YMMV.
