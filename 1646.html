You didn't say what exactly you want to translate. I don't think that
people which do not know Turkish, can help you with an actual translation.

If the problem is that you can't see Turkish, you should redifine fonts
used in window titles, menus, modules. Searching man pages for word 'font'
should help. For fonts used inside applications (something that fvwm can't
control), say fonts used in xterm, refer to the application man pages.
