Neil Zanella <nzanella_at_cs.mun.ca> writes:
> Hello,
>
> I really would like to have more popup meus without having to use
> modifiers from my keyboard. My menus are getting cluttered.
> So, it would be nice if it were possible to bind menus not only to
> mouse buttons 1, 2, and 3, but also to combinations of them, ie., also
> to mouse buttons: 1&2, 1&3, 2&3, and 1&2&3
>
> Wouldn't that be just great?

Well, not for me. I'm still convinced that the only way to operate
menus is thru the keyboard.

If you look at the way X reports mouse events, you will see that this
is not an fvwm2 issue. Fvwm2 will always see the the button presses
as separate events.

I believe that some of the X servers have a way of reporting multiple
button pushes as button 4, and button 5, in which case Fvwm2 would be
able to accept more button bindings.
