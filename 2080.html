"Louis LeBlanc" <leblanc_at_acadia.ne.mediaone.net> writes:
> I see. Somehow, I missed that my first time through the FAQ. Thanks.
>
> That does lead to another question, though.
>
> When I used Fvwm2 on Solaris, I was able to use the following Key mapping
> to provide a screensaver 'hotbutton':
>
> Key Scroll_Lock A A Function Sleep
>
> And Sleep is set to exec xscreensaver-command -lock
>
> I don't remember having to do any modifications to this part of the
> keyboard mapping there, how can I do this with Mandrake?

That odd, I have this in my .fvwm2rc:

#F23 is Scroll Lock key on Sun type 4 keyboard
Key F23 A N LockTerm
Key Scroll_Lock A N LockTerm # Linux definition

The Linux this refers to is Mandrake 7.0.

One way to tell what keysym or modifier is on a key is by using
"xev".

If you have a keyboard that JWZ knows about, xkeycaps is easier to
use.

Seems we agree on the correct usage of the "Lock" key.
