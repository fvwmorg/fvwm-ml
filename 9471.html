fvwm 2.5.7 on sparc solaris 5.8

< Colorset 4 fg black, bg orange
< Style cmdtool Neverfocus
<
< *FvwmEventPassID
< *FvwmEvent window_name winame_event
<
< AddToFunc winame_event
< + I WindowID $0 (cmdtool) Style cmdtool HilightColorset 4
<
<
< When cmdtool name changes, (such as using echo -n "]l `date +%l:%M%p`\")
< the cmdtool loses focus.
<
< Also...
<
< AddToFunc winame_event
< + I WindowID $0 (cmdtool) Style cmdtool MiniIcon textedit.xpm
<
<
< causes the titlebar to temporarily flash another color
<
< Thanx
<
