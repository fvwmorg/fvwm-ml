Hi Andrew,

> I have desktop icons made using FvwmButtons with transparent colorset and
> wanted the icon under mouse cursor to be highlighted (e.g. with increased
> IconAlpha or something like that).

If you want the icon to be highlighted, the easiest way to accomplish
this is to use 'Icon' & 'ActiveIcon'.

ie.

*Buttonbar(Icon redApple.png, ActiveIcon greenApple.png)

If you only want the background to change, use:

*Buttonbar: ActiveColorset 21

SCoTT. :)
