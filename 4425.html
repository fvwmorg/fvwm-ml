> Just remember that issuing this command crashes the gnome pager that is

Yeah, I remember the discussion about negative desktop numbers on
the ewmh mailing list. Nobody seemed to care about compatibility
with existing window managers. Anyway, you can think of this as
being very large positive numbers, so dumping core is still a bug
in these applications. If you like, report it to the GNOME/KDE
people.
