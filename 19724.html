here is my problem, right out of the FAQ:

33) I'm compiling under SunOS/Solaris with gcc, but it's complaining
    that "Language 'F' is unknown". What's the problem?

and here is the FAQ answer:

A: Your Imake setup is not right for using gcc. It's trying to use
   some flags for the native compilers that gcc interprets
   differently. Get your system admin to "fix" your setup for use
   with gcc, or try using the FvwmCompiler and FvwmCompilerFlags
   defines in the Fvwm.tmpl file to get it to work for you.

i tried FvwmCompiler and FvwmCompilerFlags and that didn't fix anything.
i can't exactly ask my sysadmin to "fix" the problem since this is my
personal SPARCstation 5 at home. i'm the sysadmin there, and i'll be
damned if i know how to fix it. i looked through the mailing list archives
and didn't find anything that helped me. if there is a web page that someone
has with the solution that would be great. if not, i'll put the instructions
for getting it to work on Solaris on my web page (with the how to get Doug
Jones' PDP-8 emulator working under Solaris) and you can point the FAQ or the
FVWM home page to there.

thanks for the help...

oh, i'm a long time FVWM 1.x user (linux) and am curious what major changes
i'll have to make to my lifestyle to start using 2.x

thanks again,
