I have some additional information concerning the huge window and FVWM2.

If I start my X-session with FVWM2, restarting with any other
window manager results with the same problem; i.e. afterstep or twm
also display the window incorrectly.

On the other hand, if I start with afterstep, restart with twm,
everything goes fine but switching to FVWM2 results with the problem
again.

Here's the relevant part of the problem, I believe:

user specified size: 533 by 399 (by xprop)
-geometry 1596x6384+0+768 (by xwininfo)

It appears that the user specified size is somehow overridden by FVWM2.

Here's a chance for you gurus out there to make a FVWM2-newbie happy :)
