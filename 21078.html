> Ric> I've noticed that one question which gets asked here quite a
> Ric> lot is: "How do I get my xterm to show {username, hostname,
> Ric> current directory} in the titlebar?"
>
> While the escape sequence thingy is a Good Thing (tm) for xterm
> windows, it doesn't work for Ghostview or GV windows :-( [*]

gv shows the title of the document if it is contained in the postscript
(using %%Title:).

It should be a very easy hack of the ghostview source to make it
show the filename in the title.

> Back when I used to use gwm I used its ability to change window titles
> on the fly a lot: you could double click the window title and you were
> then able to input a new title. Very useful. WIBNI one could do this
> with fvwm, too?

This is an interesting idea. It would probably be fairly straightforward
to implement a builtin ChangeWindowTitle function, and then maybe
use FvwmForm to type in the string and call the function.
