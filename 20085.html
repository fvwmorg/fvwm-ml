> Is there a way to redirect FVWM (95 for the matter) 's error
> output so that it appears in a window? If I log on a console, I can use
> xconsole. But I more often log from workstations.

One program I like is 'xsession'. Run it in foreground as the last client in
your X init file (.xinitrc, .xsession, or whatever) and it manages
your session, including startup of the window manager, e.g.:
  xsession -defaultWM fvwm2

This has a few benefits: it allows you to switch window manager on the
fly; if your WM fails (.fvwm2rc error, for example) you still get
your X clients and can switch into another WM (say twm) while you
fix the problem; it has a text pane that shows error messages, including
those from the WM.

It's probably at somewhere like ftp://ftp.x.org/pub/contrib/applications/
or your local X mirror.
