On Sun, Aug 11, 2002 at 06:57:45PM +0530, Rahul Rekapalli wrote:
> Hi,
> earlier I was using 2.4.6 about 4 months back. Then I migrated to the
> devel series 2.5.x
> Whenever I start FVWM, and then do Ctrl+Alt+F1 to go to the tty where I
> ran the command
> $ startx
> I get about ten warnings from Fvwm in the following format.
>
> [FVWM][ewmh_HandleDesktop]: <<WARNING>> An Desktop application
> <0xe00072> already run! This can cause problem
>
> Please tell me how I can get rid of these messages.

This means that you have several applications running that claim
to manage the desktop real estate. It's probably some application
like nautilus that manages the icons sticking on the desktop
background. Most likely, this has happened by accident. Find out
which applications are competing for the desktop and prevent one
of them from starting.
