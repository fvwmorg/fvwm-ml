Why not just bind the menu key?

If the config. file /etc/X11/XF86Config has the keyboard set for

   XkbModel "pc105"

(My point for mentioning this is that at work I have a 105 key keyboard
but the config file had "pc101" and the Menu key was not seen.)

then this works for me...

Key Menu A N Popup "MenuFvwmRoot"

though I guess you'd want Exec ...
