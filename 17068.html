Jim asked:
> I am looking for a mouseless way to switch between screens in a
> multiscreen environment within fvwm.
>
> Is there a way to do this?

Yup.
In my .fvwmrc file I have this section:
        
# Use F keys to jump about pages too.
# 0,0 1,0
# 0,1 1,1 for my 2x2 desktop.
#
# Modifier 3 = numlock
#
Key F1 A N GotoPage 0 0
Key F1 A 3 GotoPage 0 0
Key F2 A N GotoPage 1 0
Key F2 A 3 GotoPage 1 0
Key F3 A N GotoPage 0 1
Key F3 A 3 GotoPage 0 1
Key F4 A N GotoPage 1 1
Key F4 A 3 GotoPage 1 1


Hope this helps.
