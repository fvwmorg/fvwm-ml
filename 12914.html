I put the latest version of my hacks to automate the configuration of
fvwm2 here:

http://bhepple.freeshell.org/fvwm2/index.html

The general idea is to use m4, awk etc to build menus & launcher buttons
based on what's actually installed on the current system instead of trying
to keep it up to date manually. gnome-1 and gnome-2 menus are also built.
The result can be heavily customised to taste.

No more dead menu items, thank you very much!

Note that this time, I separated the code from the graphics so that future
downloads will be smaller.
