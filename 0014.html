  I am not sure if it will work but you might want to try this:

dtsession*wmStartupCommand: /usr/local/bin/fvwm > fvwm.log 2>&1

  and define NO_CONSOLE (that is probably not neccessary though).

  of course you might want to specify path to fvwm.log too since it might
try to create file in a directory where you do not have right to reate files
(I do not know what is current directory when dtsession starts fvwm).
