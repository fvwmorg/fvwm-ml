"Bawcom, Bruce" <bruce.bawcom_at_intel.com> writes:
> I can't find info on how to set up in my .fvwm2rc file so that my PC
> keyboard keys to map to Cut, Paste, and Copy keys that are otherwise
> available on Sun workstation keyboards. Can you explain how to do this or
> point me to where I can find out?

The mapping of your keyboard keys is not controlled by FVWM.
Instead the X emulator controls this.

If you are using Exceed, go to Xconfig, pick Input.
If you need help beyond this or are not using Exceed,
ask again with more information.
