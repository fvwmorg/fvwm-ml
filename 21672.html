WL> Hi, I am running fvwm2 on RedHat 5.0 with MetroX and I have problem
WL> trying to disable the edge scrolling feature.

If you mean the smooth scrolling that happens because you have the X server
configured to use a larger virtual size than display size, then you won't
be able to turn it off from FVWM because it has nothing to do with FVWM.
read the documentation for your X server and figure out how to turn it off
there.
