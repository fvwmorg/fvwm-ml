On Tue, Mar 04, 2003 at 10:37:29AM -0000, dermotwhelan_at_e-cog.biz wrote:
> Hi,
>
> I was wondering if you could help to override a webpage setting the window placement
> to 0, 0 (ie., <script> move.window(0,0) ) as it hides the title bar and prevents you from controlling the window.
>
> I have tried using NoPPosition and geometry but these don't work, though the twm window manager does handle it properly.
>
> If you can give me any suggestion to resolving this problem it really would
> be most .

The best way is to turn off JavaScript in your browser. Mozilla
can fine tune this to only forbid certain actions. In case you
can not live without full JS functionality, try

  Style <browser-stylename> FixedPPosition, FixedUSPosition

(you need the latest 2.5.x unstable version for this).
