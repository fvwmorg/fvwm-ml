On Thu, 22 Apr 1999, Sam Krishnabhat wrote:

> I am using latest fvwm2 2.2.
> I want to change the size of the title bar for any window.
> How do I do it ?
>

I assume you mean something like the height. The height is usually set
from the window title font size, however you can set it yourself with:

TitleStyle Height 24

Or any numbers you want, I prefer something like 15 myself since I only
have a 14" monitor.
