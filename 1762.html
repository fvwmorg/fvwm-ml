Good morning Neil (my time). First let me say, we really do
appreciate bug reports.

If this file crashes your X Server, you should send a bug report and
stack trace to the X Server folks.

I'm not running Linux right now, so I was unable to test with your file.
I don't think you should expect others to run a file that does "xhost +"
and executes a bunch of shell commands in its InitFunction. If you meant
to say "crashes fvwm" you should trim down this file a bit but start with
sending us a stack trace.
