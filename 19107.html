It is possible, though you cannot really get an exact match. If you
want to look and act just like Windows 95, it isn't for you (fvwm95
does a better job). If you are willing to spend time learning how to
configure and play with things, then it might be for you, since you
can then make your own improvements on the look and feel and
incorporate some other new features that fvwm2 has. There is an
examples file which contains some Win95 type examples from a few
people on this list in the sample rc directory in fvwm 2.0.45.
