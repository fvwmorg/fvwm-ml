>
> For the life of me, I can't figure out how to use the HideGeometryWindow
> function. Does anyone know how?

The man page on your disk should list all commands of your FVWM version.

  % man fvwm2
  /HideGeometryWindow

gives:

       HideGeometryWindow [Never|Move|Resize]
              Hides the position or size window that is usually
              shown when a window is moved or resized interac-
              tively. To switch it off only for move or resize
              operations the optional parameters Move and Resize
              can be used respectively. To switch both on again
              use the Never option.

This command is only available in the fvwm-2.3 series.
