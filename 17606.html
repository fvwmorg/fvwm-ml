>>>>> John Dawkins writes:

J> everytime i restart fvwm2, i keep getting another running copy of swallowed
J> apps. i am using the UseOld and NoClose options in my configuration file.
J> what am i doing wrong?? i forgot to include some info yesterday. i'm using a
J> Sun SparcStation 5. i'm using fvwm2.0.43

J> *FvwmButtons(6x2, Swallow "FvwmPager" "FvwmPager 0 3")
J> *FvwmButtons(3x1, Swallow(UseOld,NoClose) "xdaliclock" `Exec
J> /afs/eos/project/goodies/bin/xdaliclock &`)
J> *FvwmButtons(3x1, Swallow(UseOld,NoClose) "xload" `Exec xload -fg white -bg
J> gray55 -nolabel -scale 3 -update 5 &`)

try this:

AddToFunc ExitFunction "I" KillModule FvwmButtons
+ "I" FvwmBanner
+ "I" Wait FvwmBanner

this provides time to the FvwmButton module to unswallow the windows,
before the new one starts up.
