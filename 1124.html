Thanks for the tip! I got the message that libXpm.so.4.11 was not found, and
I realized that it was linking to /usr/lib/libXpm.so.4.11 rather than
/usr/lib/libXpm.so. Apparently I did a bad thing and removed the
(supposedly) older versions from everywhere, and only linked back into *.so.

Fixed. Now to start reading the config info!

Lou
