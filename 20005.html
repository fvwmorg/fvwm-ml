I have added a bit to the functionality of fvwm2, both to the icon code
and the menu code. I have written them as patches yet ;-) but if there's
any interest then I'll get patches written and post them somewhere.

I haven't been subscribed to this news group for very long, so for all I
know there could already be patches to do this. If there are, there is
every chance that they will have been better put together than mine,
since my X11 programming experience isn't great. But if not....

Basically I have added these new commands:
        
        *IconStyle flat | raised | sunk
        *IconColor foregroundcolor backgroundcolor

        *MenuTitleStyle forecolor backcolor font

To see what they look like in action, point your browser at
        
        http://www.ecs.soton.ac.uk/~jaf195/desktop.htm
