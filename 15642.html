Sometimes I should sit tight before posting to mailing lists. I usually find
an answer if I tinker long enough. I'm sorry for the multiposts, but this
post will probably be received well by those with ms$ keyboards.

The following binds the ms$ left window key to the Kanji keysym. Any keysym
will do (though it should be unused by your system -- look at keysymdef.h) as
long as the same names are used on the command line and in .fvwmrc.
 
         xmodmap -ev "keycode 115 = Kanji"
 
Then the following line allows the left window key (without having to press
any other keys) to call up the menu Utilities (main menu on my system).
 
         Key Kanji A A Popup "Utilities"
