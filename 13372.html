Hm, are you aware that such a script is included in the fvwm
tarball? Look for fvwm-menu-directory. It even has a man page.
If you instist on writing your own script, take utmost care that
file and directory names with embedded spaces, tabs and newlines
are handled correctly. Otherwise you might end up running
commands embedded in a file name, e.g. if the name is

 "foo<newline>rm -rf *"

Your script seems to work fine for me. Are you sure it it
executed at all? Is it in the PATH? Hm, nor that I run it again,
