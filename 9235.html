On 14 Jun 2003 00:52:23 +0100, Ray wrote:
>
> what is the correct config option to ensure that the full title of a
> window is displayable when that window is iconified?
>
> i have a netscape window which has a title "FVWM - Mailing List" and i
> would like the icon title to be the same -- currently it just says
> 'netscape'

I can't reproduce this with netscape (but maybe because I have 3.04).

If you mean real iconified windows, the icon title is not configurable
currently, it just displays what the application set for the icon title.

If you mean a module like FvwmTaskBar or FvwmIconMan, there are
configuration options to do this, see their man pages.
