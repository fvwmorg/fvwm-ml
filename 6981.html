On 20 Nov 2002 18:00:18 -0500, Rick Jones wrote:
>
> FvwmCommand sounds perfect.. I'm assuming any command
> or system call from a program will pass the command to
> fvwm. Is there a way to restart fvwm and specify
> different fvwm config file.. or will I have to have
> the program copy the desirable config to .fvwm2rc
> before telling fvwm to restart?

Just execute this commands in one or another way (from keyboard/mouse/menu
binding, from FvwmConsole or FvwmCommand):

  Restart fvwm -f another-rc-file # put another-rc-file to ~/.fvwm/

Or if you want to keep exactly the same command line arguments as the
currently running executable, just:

  Restart
