On Wed, Nov 04, 1998 at 02:27:42AM -0330, nzanella_at_cs.mun.ca wrote:
> [FVWM][executeModule]: <<ERROR>> No such module 'I' in ModulePath
> '/usr/X11R6/lib/X11/fvwm2'
> Warning: Cannot convert string "nil2" to type
> FontStruct

Probably some configuration error in your .fvwm2rc. My guess
is that Some '+'-line to AddToMenu, AddToFunc etc. is not
in correct format. Perhaps the + is missing or another
word comes before the "I" (=immediate in functions).

This error occurs when fvwm finds some word in the config
file it doesn't know. It then tries to execute it as a module.
