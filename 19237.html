> Does fvwm have a 'configure' file? I compiled 2.0.45 last week and the
> process was:
>
> edit Fvwm.tmpl
> make Makefiles

Sigh. And you missed out the 'xmkmf' stage.
The process is:
        gunzip package.tar.gz
        tar xvf package.tar
        cd package
        more README
        more INSTALL
        [do what those files say!]
This is the foolproof method!
