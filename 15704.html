Just a few ideas comments.
If this little algorithm for resizing pixmaps works could we have a
standardized size for pixmaps say 32x32 or 50x50(to provide some
detail). Also is it possible to use one pixmap as the background to
another to give that NeXT look?

Secondly a little idea for my win95 obsessed workmates; I reckon that it
would be relatively simple to write a module(a modified GoodStuff) that
emulates win95 shortcuts. I'm going to try it myself but since I have to
write up my project soon, I may have little time. I think all that is needed
is a module to write icons to the screen and accept 1 or 2(perhaps 3) button
inputs. the button 2 would be for editing the config of the module.

if any xpm gurus out there like this idea and think that it is easy to
do, have a go at it. I think it is one way of getting windows people
away from microsoft and on to a proper open operating system.

Graeme McCaffery
