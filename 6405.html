On 04 Oct 2002 16:08:31 +0000, leegold wrote:
>
> New user question, FreeBSD 4.4
> I put fvwm2 & in ~/.xinitrc and fvwm2 works OK.
>
> I now see ~/.fvwm, it's an empty directory. Is it correct
> to put the local configuration file .fvwm2rc inside this
> ~/.fvwm directory ?

Yes, it is preferable to keep all config files in ~/.fvwm, it is also
in your "Read" search path and some modules/scripts use this directory.

> Should I copy my system.fvwm2rc to the ~/.fvwm directory, rename
> it .fvwm2rc, and make my personal changes to this .fvwm2rc file?

You may.

> Would it be OK to only have the config lines I want to change in
> .fvwm2rc and not the whole file?

It should be the whole file, but you may achieve the same effect by having
the first line in ~/.fvwm/.fvwm2rc:

  Read /your/system.fvwm2rc
