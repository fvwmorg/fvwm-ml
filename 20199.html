I'm guessing that you're misunderstanding the difference between pages
and desks. The

*FvwmPagerLabel

directive lets you label a desk, not a page. If your fvwm desktop
(btw, DeskTopSize is in the fvwm man page, not FvwmPager, since its
part of the wm itself) is 6x2 then it sounds like you might actually
want to set DeskTopSize to 1x1 and then run FvwmPager 0 11 and set

*FvwmPagerRows 2

The desks that show up should be labeled. Also make sure you set the
font. Note that if you do this you lose the ability to use
edgescrolling, which only moves between pages, not desks.

All the above refers to fvwm2. I don't remember if it worked the same
way in fvwm1.
