DE> 2 commands to an Exec stopped working in 2.0.42. The workaround
DE> is to enclose the command in a '/bin/sh -c <cmd>' like this:

DE> + "Pine" Exec /bin/sh -c '/usr/bin/popclient -3 -u XXXXX -p XXXXXXX \

DE> -o /var/spool/mail/ray;xterm -fg black -bg white -fn 10x20\
DE> -geometry 82x27+16+32 -ls -e pine'

commands with parenthesis:

+ "Something" Exec (first_command; second_command)&
