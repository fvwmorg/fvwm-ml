  that's outside of scope of fvwm (AFAIK).

  you can reconfigure the X server (at least some of them) to have
virtual screen larger then your physical screen, for Xfree86 change the
XF86Config, look for the Screen definitions at the end (check the docs
for exact syntax, there is something about virtual resolution, I don't
have docs handy)

  you can achieve something similar with fvwm by setting EdgeScroll and
EdgeResistance in fvwm configuration file.

        erik
