it works as expected for me. Maybe some other style overrides the

.fvwm2rc as much as possible so that it still happens, and send it
to the mailing list.

> > window title is set only after mapping the window. Or the window
> > takes the focus itself when it maps. Or you have the "Lenience"
> > style set and the window asks fvwm to give it the focus. Without
> > knowing which application has the problem, I can't be more
> > specific.
