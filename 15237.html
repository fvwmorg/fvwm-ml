I'm about to release a new version of FvwmIconMan, and it now has the ability
to follow which window has the focus. (For those who don't know, which I
assume is most of you, FvwmIconMan is an icon manager similar to TWM's icon
manager. You can find it on hpc.uh.edu). It would be nice if the fvwm Focus
function wouldn't have to always raise and warp the cursor to the target
window. Probably the best way to do this is to have the Focus function take a
few paramters.

Is this on anybody's todo list, or should I do it myself?
