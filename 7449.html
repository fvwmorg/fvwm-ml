I can only speak for myself, for me the current default is convenient and
logical, probably because I use Alt-Tab mainly to see the list of all
windows and not mainly to switch to the previous window. For someone who
wants to see the listing the current default is better, because it does
not switch to a random window on a single Alt-Tab press and release.
By random I mean a liberal mouse usage and SloppyFocus/MouseFocus.

Additionally, CurrentAtEnd is only a hack to achieve a single click
functionality for switching to the previous window. In another OS the
current window is first in the list, not the last like with this option.

It is a habit issue. One with the Windows background may expect one
behaviour for Alt-Tab, one without such background may only expect a
good window list starting from the current focused window. Anyway,
this is only an opinion and not a vote against changing the default.
