On Wed, Dec 09, 1998 at 10:34:40AM +0000, a11576_at_aaual.ualg.pt wrote:
> I've just made some cool changes to fvwm2 (mainly cosmetics, it looks
> nicer), and i would like to send a patch, but, how the hell can I make a
> patch! (what a lame I am)

For each file you changed, do this:

  diff -u file.orig file

Where 'file' is the filename. If you diff does not have unified
diffs (-u) you can use contextual diffs (-c). This should be in the
FAQ too.

> PS - I've had the menu problem too! (fvwm crashes when you hit a cursor
> key while you have a menu open!)

That is fixed in the CVS tree.
