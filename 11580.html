On 01 Mar 2004 11:30:56 +0200, Micha Feigin wrote:
>
> I would like to pop up fvwm pager using a key binding to show the
> desktop layout.
>
> I don't have the desktop space to keep it open all the time (14" laptop
> screen ;-( and having it dock away is kind of a waste of memory as I
> don't use that much (most of the time I'm on one desktop).
>
> Is it possible to make it just appear in the middle of the screen or
> something one a key is pressed and then make it go away again later?

Lines from fvwm-themes:

  Mouse 2 R C Module FvwmPager -transient FvwmPager-Single * *
  Mouse 2 R SC Module FvwmPager -transient FvwmPager-Desker 0 3

Bind it to a key too if you want. "FvwmPager -transient" disappears after
any key press or mouse click.
