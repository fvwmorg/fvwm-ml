Hello:

  This is to (yell about) my latest programming project, FvwmCDE. I'm
working on a replacement for the CDE front panel that works with Fvwm. The
current version is pre-alpha (i.e. it doesn't work). You can take a look
at my preliminary work at:

  http://gibson.eee.upd.edu.ph/~orly/FvwmCDE/

  There's more information on that page, but to be quick: FvwmCDE is not
runtime customizable (i.e. drag icons to its applist and they get added to
the list automagically). All FvwmCDE customizations are in the .fvwm2rc
file, just like a good module should :)

  I'm not an FVWM worker or anything, so this isn't official. If you like
what you see there, I'll have a functional release in a few days, so check
back on the page listed above then.

  Cheers!
