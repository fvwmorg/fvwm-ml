Has there been any work done towards making background pixmaps more
customizable (sorta like what Sawmill does). By this, I mean the ability
to have a pixmap specifically for the text of the title bar, then another
pixmap for the remaining space, and pixmaps up and down the different
directions of the borders?

Anyone who's used the default theme for Sawmill on a recent Ximian Gnome
install should understand what I'm talking about. I thought that since
Enlightenment came out, some of this functionality would get worked into
fvwm2, but I don't even see that any work has started yet. I could be
wrong, I haven't looked at the code itself, but there's nothing in the man
pages :-)
