On Sun, Feb 11, 2001 at 12:09:45PM -0500, Elizabeth Schwartz wrote:
>
> >When I use my righthand clicker to get a list of my windows, the windows
> >are sorted in order of how recently I have looked at them. Can you tell
> >me how I make it so they're sorted in order of when they were started?
> >That's how it was before, but I can't figure out where it is in the
> >.fvwm2rc

That depends on the action that is performed. If it's the
WindowList command, then the order of creation is not stored. If
one of the modules is brought up, it depends on the module. If it
is FvwmWinList, this behaviour is controlled by the

  *FvwmWinListFollowWindowList

option. This has to be removed from the config file. For further
detail refer to the FvwmWinList man page. (I did not test any of
this, it comes directly from the man pages).
