Okay, there were some requests, so the UseTitleStyle flag is back.
Also, more examples (and some fixes) are now in the man page. The
ButtonStyle syntax has been made consistent with the TitleStyle syntax
(i.e., you can specify multiple definitions on a single line).
Explained in the man page, a quick example would be:

ButtonStyle 1 ActiveUp Pixmap led.xpm -- Top Left UseTitleStyle
ButtonStyle 1 ActiveDown HGradient 8 grey black
AddButtonStyle 1 ActiveUp (Pixmap a.xpm) ActiveDown (Pixmap b.xpm -- Top)
AddButtonStyle 1 Vector 4 50x30_at_1 70x70_at_0 30x70_at_0 50x30_at_1

As usual, it is at:
http://www.rpi.edu/~veliaa/projects/fvwm_patches/fvwm_border_picture.html
