All the problems you describe should be gone now. There is one
incompatibility however. The ClickToFocus/MouseFocus/SloppyFocus/
NeverFocus style must be set as *first* of all the focus related
styles. These for reset almost all others when issued, e.g. in

  Style * ClickToFocusRaisesOff, ClickToFocusPassesClickOff
  Style * ClickToFocus

The first line is completely overridden by the second. Instead,

  Style * ClickToFocus
  Style * ClickToFocusRaisesOff, ClickToFocusPassesClickOff

must be used.
