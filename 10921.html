On 01 Jan 2004 21:52:51 +0200, Micha Feigin wrote:
>
> I would like to make a shortcut to switch between the previous and
> current desk/page. Something like Alt-tab does for windows (I would
> settle for just the current and previous ones on the stack, but it
> would be nice to have the whole history actually).

To flip pages/desks use one of:

  GotoDesk prev
  GotoPage prev
  GotoDeskAndPage prev

If you want a whole history, setup your own recorder/player, for example
using FvwmEvent listenning to new_page/new_desk, or by writting a short
fvwm module in Perl working with the PageInfo tracker.
