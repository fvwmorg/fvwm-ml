Hi,
I've just noticed something strange, my key bindings to jump between screens
doesn't work after I restart fvwm! (fvwm-2-4.0, on a Solaris 2.4 machine).


My (rather short) list of bindings looks thusly:
Key Left R N Scroll -100 0
Key Right R N Scroll 100 0
Key Up R N Scroll 0 -100
Key Down R N Scroll 0 100
Key q FITDWS M Close
Key q FITDWS M3 Close
Key w FITDWS M Iconify
Key w FITDWS M3 Iconify
Mouse 1 R A Menu Utilities Nop
Mouse 2 R A Menu Window-Ops Nop
Mouse 3 R A WindowList
Mouse 0 1 A Menu Window-Ops2 Close
Mouse 0 2 A Maximize
Mouse 0 4 A Iconify

The <>-Q and <>-W etc all work fine, its just my arrow keys that are
rendered impotent by a restart. Any ideas?
