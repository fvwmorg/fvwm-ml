Hi,

I've successfully built fvwm 2.2.2 on AIX 4.3. It then took some time of
trawling through various CDE help to find that to get fvwm used after login one
should add the following line to .Xdefaults

Dtsession*wmStartupCommand: /usr/local/bin/fvwm2

Or where ever fvwm is installed. Contrast this with Solaris where you simply
add the following line to .xinitrc

/usr/local/bin/fvwm2

and then use the options on the login screen to use openwin rather than CDE.

Would it be usefull to add these kind of instructions for all the main flavours
in the release?

One other thing, although I did the above for my AIX system there is this
strange one minute pause after login during which I can't do anything and I get
the hourglass mouse pointer.
I suspect it is something CDE still going on. Does anyone know what this
actually is and how I can fix it?

TIA for any help

Mike.
