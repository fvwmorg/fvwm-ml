> I still have the problem that clicking Start warps me to the top of that menu.
>
> How do I stop that? Is there a hidden config file for that or is it buried

Warping depends on the type of menus you use. I don't know which
version you are using, so I can't tell you how exactly to do this.
You are probably using FVWM menus at the moment. MWM-style menus
don't warp, FVWM menus do. So changing the menu style should help
(I think the command is called MenuStyle, but I don't know much
about 'history').
