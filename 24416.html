%% Regarding FVWM: Icon area for fvwm2?;
%% Bruce Burden <bruceb_at_austx.tandem.com> writes:

  bb> What is the directive to create an icon box for the various
  bb> windows in fvwm2?

You can do it either through the builtin icon box, which really just
provides predefined space on the desktop to put icons; there's no
graphical box with scroll bars, etc.; see the fvwm2 man page and look
for IconBox.

Or you can do it with the FvwmIconBox module, which gives a real,
graphical box for icons. See the FvwmIconBox man page.
