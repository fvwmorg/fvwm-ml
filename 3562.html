On Tue, Nov 06, 2001 at 08:14:27PM +0100, Marcus Lundblad wrote:
> Is possible to "force" FvwmButtons and FvwmPager to reread setting "on the
> the fly"?

Yes and no. Most modules don't react to configuration changes on
the fly in general. But since the "colour set" concept was
implemented (2.4), all colours can be controlled via a central
module called FvwmTheme. You can basically specify sets of four
colours each (foreground, background, shadow and hilight) and
assign them to the various colours used in the modules (and fvwm
itself). These can be changed at any time and take effect
immediately. To learn about colour sets look for the keyword
"colorset" in the fvwm2 man page and read the FvwmTheme man page.
