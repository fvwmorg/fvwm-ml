In japan, we use i18n-patch of fvwm. It enables fvwm to handle
compound text in the titlebar and icon, to designate fontset instead
of font, and of course, to use non-ASCII characters in the context of
character (button name of FvwmButtons, FvwmForm, and so on).

This patch uses X's i18n feature ONLY, so any charsets which can be
handled by i18n framework will be displayed (I think). Patch is made
in two years ago (when fvwm version is 1.20 :-), and supported by the
people in "Fvwm Users' Mailing List in Japan". Patch for the latest
version of fvwm (2.0.42) is also available.

Anybody interested in this patch may get these patches freely by ftp.
Check keyword 'fvwm-users-jp-kits' by archie. I18n patch of fvwm is
included in fvwm-users-jp-kits-VERSION.tar.gz (VERSION is a string of
the release date). Be aware that most of documents are written in
japanese (very sorry).
