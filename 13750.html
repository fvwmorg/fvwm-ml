On Wed, Feb 02, 2005 at 11:11:00AM +0100, "Kai Kühne" wrote:
> Hi list.
>
> I have a problem with my key binding. I've found a config using "3S" as key.
> I know the meaning of C (Control), S (Shift) and so on...
> What does 3S stands for?

It stands for Mod3+Shift. In my case I define Mod3 to be the left
"windows" key via xmodmap:

    xmodmap -e 'add mod3 = Super_L'
