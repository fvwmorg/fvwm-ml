I do not have the problem in 2.5.6, I have not tried current/CVS.

However there seems to be another problem. This is not analyzed
and I have moved back to 2.4.14.

When setting 'Resolution Page' or 'Resolution Screen' the
FvwmIconMan was not updated until switch number to.

Same setup as before one xterm and one iconified xconsole.

[ xterm ]
[X xconsole]

Switch to virtual desktop number X.

[ xterm ]
[X xconsole]

Switch to virtual desktop number Y.

[X xconsole]

Switch back to starting position.

[ xterm ]
[X xconsole]

According to _ME_ the correct sequence should be.

[ xterm ]
[X xconsole]

Switch to virtual desktop number X.

[X xconsole]

Switch to virtual desktop number Y.

[X xconsole]

Switch back to starting position.

[ xterm ]
[X xconsole]
