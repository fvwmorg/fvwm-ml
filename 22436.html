Hi,

Has anyone taken up the task of developing a follow-mode like feature for
the window manager?

<http://www.csd.uu.se/~andersl/follow.shtml>

To summarize, the feature should work as follows:

A single command opens up multiple windows (say n windows).

The windows are all related to each other as they all show the same file

The very top line of window k follows immediately from window k-1 and

scrolling one window k will scroll window k+1 as well. To select the n

related windows with the mouse it suffices to select one of them.

All related windows get the same HiForeColor and HiBackColor when the mouse

points to one of them whereas the other unrelated windows act as usual with

with respect to color.



Is anyone implementing this in fvwm? It would be a nice feature.

This would add a follow-mode like feature to every application running

within fvwm. That the feature won't have to be incorporated into every single

program that needs it (like the various editors, pagers, postscript viewers,

etc...)
