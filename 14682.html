I have recently switched to FVWM from MWM. One thing I really liked about
MWM was that through an Xresource I could specify a threshold for moves. If
the cursor did not move more than this threshold then the button down was
considered a click, not a drag.

Is there anything like this for FVWM? I find that when I bring my windows
to the front with the Move-Raise functions I always end up moving the
window as well.
