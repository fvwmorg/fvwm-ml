=> From: Mark Krischer <mkrisch_at_bbn.com>
=> Subject: FVWM: .fvwmrc customization questions and requests
=>
=> i've got a couple question for things i'd like to do in my .fvwm2rc
=>
=> 1) i'd like be able to bind actions of my mouse to buttons on a
=> specific window or to a specific icon. e.g. when i right click
=> on the netscape mail icon, i'd like to get a special menu with
=> Get New Mail
=>
=> 2) i guess being able to do 1) would let me do this, but...
=> another thing i'd like to be able to do is have an icon (the
=> netscape mail one) be sticky, but the opened window not be.
=> that is, i'd like to see the icon on every "page" but when i
=> open the window, i don't want that following me around.
=>

Well I can't help with 1) but 2) is easy enough. You just need to specify
StickyIcon in a Style line in your .fvwmrc file. Something like;

Style "Netscape Mail" StickyIcon, Slippery

will ensure that the icon follows you around but the window doesn't.
