It is not very clear for me whether you have response problems while
everything is still correct (just slow), or "menu bars loses all
decorations, text within the title bars, everything". Do you see the same
window decorations (borders, titlebar with buttons) and root menu colors
that you used to see before starting pan? Please try to describe this.

I just installed pan (I suppose you speak about a GNOME news reader),
downloaded some Mb of binaries during one minute (that is what you seem to
do) and I don't see any problem. Do you still continue to have problems if
you exit pan? Which pan version do you use?

Just to be sure, do the following, Restart to fvwm without configuration,
for this open FvwmConsole from the Modules menu and type in it:

  Restart fvwm -f none

Then start xterm, it is first in the default menu and start pan in it.
Start more xterms and see whether you have any response problems.
To return to fvwm-themes start FvwmConsole from the menu again and type:

  Restart fvwm-themes-start
