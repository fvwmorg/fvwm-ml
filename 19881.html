I've been running Fvwm2 for several months now with no problems and have
recently found TkGoodstuff.
 
After looking at TkGoodstuff it appears that it does almost exactly the same
thing as FvwmButtons.
 
Can anyone give me a reason why I would want to use one utility over the other?
 
The only thing that I see different is that TkGoodstuff provides a preferences
manager instead of manually having to edit the .fvwm2rc file.
 
I have preferred so far to use FvwmButtons since it is a part of fvwm and
therefore I don't have to manage 2 programs and keep 2 config files etc...,
however, if there are advantages of using TkGoodstuff I'd be glad to use it.
 
A comparision between the 2 tools would be great!
