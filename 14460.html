  Kind of annoying. Why does it have to be that way? Other mailing lists
I've been on at least set the Reply-To: field to the mailing list so it's
easy to reply to the list. I guess I'm not so concerned about getting
the two copies of mail (although it would be nice not to) as the fact that
I always have to remember and then type the address of the list in my
note (I guess I should set up a nickname). Although when I do that, I
usually just change the To: tag so the person I'm "replying" to doesn't
get two copies.

  Although maybe I can see why that is. Whenever I send a note to the list,
I always get a few bounced mail messages. If it weren't for the lack of
the Reply-To, would the list get those messages and not be able to filter
them?
