  ja> Nah, that's exactly what I don't want. With one "alt-tab" press,
  ja> I want to flipfocus, with multiple ones in a row, I want to cycle
  ja> my focus.

You can't get FVWM to work exactly like Windows.

The latest 2.3.x pretests do it this way:

  * Alt-TAB brings up a window list.

  * More Alt-TAB's will cycle through the list.

  * You have to press RET (e.g.) to select the window you want; letting
    up the keys doesn't do anything.
