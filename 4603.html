Xmms seems to have an infinite supply of bugs related to the setup
of its windows. Recently I read about some problems releated with
xmms windows with a title being fixed. You should try out if this
works with 1.2.7 (or the latest pre release if it's not ready yet)
and report to the xmms people if it still causes problems.

Up to now, all the problems with xmms (and there were a lot of
them) were caused by xmms, not the window manager.
