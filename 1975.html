i'm wondering how well fvwm can deal with multiple screens.

i recently scored a second head for my sun workstation at work, so
that when i log in, fvwm is now managing two screens on a single
display.

right now the only "connection" between the screens is that my mouse
will move seamlessly from one to the other -- but each screen appears
to have a separate window manager running it. can i do better than
this?

of course, i'd love to be able to drag things from one screen to the
other, but i'd be equally happy with a shared set of virtual desktops,
and i'd even be thrilled with simply having a means of shifting focus
from one screen to the other via a keystroke (rather than exclusively
with the mouse). (since one of the reasons i chose fvwm in the first
place was the next/prev window ring capability.)

many thanks in advance (and in arrears, for all the hard work that's
gone into this thing!)
