>>>>> Fleish writes:

F> I am running fvwm-2.0.43, and am wondering if there is a way to make fvwm
F> turn messaging off on certain terminals that it opens (without having to
F> add mesg n lines before all programs I want to do this for) For instance,
F> when I open pine, messaging is turned off automatically it appears.
F> However, I would also like programs like irc and rlogin to have messaging
F> turned off. If anyone knows of a way to do this, please let me know.
F> Thanx
F> Todd

if you start xterm -e pine, you should add mesg n in front of that.
ugly, I know
but sth like
Function XtermApp "I" xterm -e /usr/bin/tcsh "mesg n;exec $1 $2 $3"

might help.

if you start a normal xterm and start the commands there, shell aliases
should do the trick.
