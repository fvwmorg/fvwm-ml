This is the first time I've written to this mail list, so I'd like to
begin by thanking EVERYONE who's contributed to make Fvwm2 "The Best"
window manager available that I've been able to find. And I've tried as
many as I could. Fvwm2 is the ONLY win manager that's practical on my
old 486dx33 linux box! It's relatively fast, lean and easy to use!

That having been said, I'd also like to say I've been mainly running
2.0.43b but recently upgraded to 2.1.4 then 2.1.5, and now 2.1.7!
Beautiful! It all works!
No problems so far that couldn't be sorted out by modifying the syntax
of the .rc except that DestroyModuleConfig doesn't seem to work for
FvwmButtons but DOES for FvwmButt*, and DOESN'T for FvwmEven* but DOES
for FvwmEv*. Go figure!
I 'especially' like the new MenuStyle options and discovered by accident
how to set WindowList a new font style!
My question is this....
I've played about with various cursor styles... and an animated cursor
proggy I found somewhere.... but nowhere in any of the documentation
I've read, is there a mention of root window context! The man page
refers to the cursor in windows and/or decorations, but I need to
restore the normal "X" cursor and nothing I've tried works except
restarting fvwm2. :( There's probably a 'trick' to it I don't
understand.
