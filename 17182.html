A few weeks ago I created the fvwm-workers mailing list to try and free
things up on the main list for more user questions. I want to reiterate
that we should try to move all development talk to that list, because it
seems the user questions are getting buried here under a flurry of patches
and internals discussions.

To subscribe to fvwm-workers, send a message containing the line

subscribe fvwm-workers

to majordomo_at_hpc.uh.edu and follow the instructions which will be returned
to you.
