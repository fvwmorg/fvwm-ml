Hi all,

on fvwm-2.2.2 on Digital UNIX I've got a small but annoying problem: Amaya
(the w3-consortiums html editor) doesn't get any window decorations. I've
got the usual

        Style "*" BorderWidth 5, HandleWidth 5
        Style "*" Icon unknown1.xpm, Color Black/#40a0c0
        Style "*" ActivePlacement, SmartPlacement
        Style "*" MWMFunctions,MWMDecor,HintOverride
        Style "*" DecorateTransient,IconBox 0 -10 -280 -1

and this works for all other windows. fvwm v2.0.46 exibited exactly the
same behaviour. Anyone got an idea?

I've noticed that the name of the window, as returned by xwininfo depends
on the document being edited, but the 'style "*" ...' directive shouldn't
be bothered by this, should it?

confused in Munich,

Rob Urban
