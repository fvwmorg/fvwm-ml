: When I try to start the FvwmTabs module it seems to start ok (got rid
: of the startup error messages and ps shows that it is running). However
: when pressing Alt-t or trying to open the tabs window using the fvwm
: console nothing happens.
:
: How can I find out what the problem is?

Do you override the Alt-t key binding in your .fvwm2rc?

Try executing this line from FvwmConsole:

SendToModule FvwmTabs createNewTabber

Does FvwmTabs output any error messages?

If all else fails, change the debug line to:

Debug => 2

in the FvwmTabs file & restart.

What output do you get now?
