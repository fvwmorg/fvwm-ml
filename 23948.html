To fvwm list:

I have recently installed Red Hat Linux. My previous experience is with
Ultrix and Digital Unix.

With those products there was a way to allow other hosts to use the local
display.

  i.e. # rlogin central
        # xv

The xv window would appear on my local display if I gave permission to
central. If I didn't I got the standard message:

  Xlib: connection to "local" refused by server
  Xlib: Client is not authorized to connect to Server
  xv: Can't open display

which is the message I am currently getting using Linux and fvwm.

I know this operation should be pretty basic but I can find no documentation
on how to grant display permission using fvwm. I would really appreciate if
someone could point me in the right direction.

Thanks in advance.

Richard
