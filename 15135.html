Howdy all,

Well, I haven't had a lot of time lately to do much with FVWM (work
and other responsibilites have kept me VERY busy lately), but in a
burst of post-lunch activity version 2.0.40 is finially ready and
available at ftp.hpc.uh.edu in /pub/fvwm/version-2 (and in keeping
with my 1 release per month average lately).

-rw-rw-r-- 1 chines users 477303 Dec 20 14:28 fvwm-2.0.40.tar.gz

>From the ChangeLog:

2.0.40
       1. Removed 'basename' reference
       2. More small Imake fixes
       3. Renamed FvwmConfig .cc files to .C, because some C++ compilers
          can't handle the .cc extension (AIX's, for instance)
       4. FvwmForm now gets config info in standard way
       5. FvwmBanner now dissappears if you click on it (like in 1.xx)
       6. Renamed FvwmConfig's makefile to Makefile.std and cleaned it up
       7. More small man page cleanups
       8. Improved handling of nonexistant fonts (added GetFontOrFixed)
       9. More debug messges, and central fvwm_msg function for printing
          msgs (started converting fvwm_err and fprintf calls to use this
          instead, as it has variable args and allows for a consistent
          message feel and I'll use this to add -outfile <file> option
          eventually to capture all fvwm output to a file as well as
          stderr)
       10. Fixed function parm $9 coredump
       11. Added basic TitleStyle (for justfication & edge appearance)

Some notes on these:

        #1 Hopefully I won't mistakenly put anything like that in again.

        #2 Please let this be the last of the Imake problems...

        #4 It should now work ok for those of you who use FvwmM4 and Cpp.

        #5 I kinda missed that feature.

        #9 I used stdarg.h vararg stuff. Hopefully it'll compile ok for
           everyone (I don't see why it wouldn't). I still have to
           finish converting to using it and work on the module
           interaction with it.

        #11 I added info about this in the man page. Try a few combinations:

              TitleStyle Centered Raised
              TitleStyle LeftJustified Flat
              TitleStyle RightJustified Sunk

            It probably isn't complete, but it wasn't much code to implement
            so I threw it in there (finally). Admittedly, I still
            much prefer the default...

There were a few patches that people sent me that I didn't have time
to throw in. Sorry if yours didn't make it this time.

I'm going on vacation this afternoon until about Jan 2nd. So I
probably won't be too responsive to notes during that time, although I
will try to dial in occasionally to check them.

During that time I hope to do a fair amount of work at home on fvwm.
I plan on putting in a few more of the patches sent to me, and
cleaning up the docs a bit.

I'd like to release the next version as the first official 2.xx
release (2.1.0) shortly after I come back as my way of saying Happy
New Year to the Internet.

Then I can finially do some bigger cleanup, bugfixes, and enhancements
without feeling too bad... ;^)

Happy Holidays,
Chuck
