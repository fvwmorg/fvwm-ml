Hello all,

I am using fvwm 2.5.7 and am trying to make all icons of all
applications look the same save for the pixmap. I have the following
in my .fvwmrc:

Style * IconSize 32 32
Style * IconBackgroundColorset grey60
Style * IconBackgroundRelief 3
Style * IconBackgroundPadding 0
Style * IconTitleRelief 3

However, some applications seemt o ignore these styles, most notably,
java applications.

Another problem I have is when I start the thunderbird mail
application, the icon for it doesn't show up until I restart fvwm. The
Icon style for thunderbird is as follows:

Style Inbox* Icon Mailbox-Summer.dir.xpm

Does anybody have any idea on how to solve these problems? I'd be very
grateful,

thank you,

Ian.
