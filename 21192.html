->2) I cannot use icons or binaries from my own account, unless they are

It sounds to me as though you have some problems with the
configuration file .fvwmrc, but I can't really tell without seeing it.
I do know that FvwmButtons is very choosy about how it lays out
buttons so you have to be very careful when deciding on a
configuration. Sometimes it helps to specify a geometry for the Button
bar window and watch how FVWM tries to set up the buttons.
