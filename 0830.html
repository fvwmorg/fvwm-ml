> O, where did you get the cde like front panel, i'd be interested.

        The link that Mikhael referenced -

  http://www.go.dlr.de/linux/src/.warix/Panel-CDEm-v3.0.tgz.html


- points to an old version of Matt Baron's Panel. The latest link I have
to his Panel page is

  http://www.visi.com/~mbaron//panel/


The link is still alive, but I don't think Matt's been actively working
on it for a long time. As I recall, Panel is supposed to be a generic
CDE panel look-alike that should work with any window manager, but had
some special extensions for working with FVWM - I don't remember just
what. If it used the module interface, I doubt that that part will be
anywhere near up to date.

        I used it for a while a few years ago, and it seemed pretty slick. But
I like a clean desktop, so I tend to stop using things like task bars
and control panels as soon as the new wears off.

        Writing CDE clones was a popular activity some 3 or 4 years ago, but
interest seems to have waned. Another popular one was Orlando Andico's
FvwmCDE, but it and he seem to have disappeared; I gave up trying to
locate him a couple of years ago. I don't know anything about the
current status of xfce.
