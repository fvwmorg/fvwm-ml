"Horv th Zolt n" <horzol14_at_hotmail.com> writes:
> The fvwm2 man page says: during initialisation fvwm2 searches for
> .fvwm2rc configuration file in the home dir ectory of the user, if
> doesn't find, then in /usr/lib/X11/fvwm. If can't find any of tehm,
> fvwm2 will be "basically useless". Now, I've got none of them, my fvwm2
> still works pretty well. How is it possible?
> Ver.: 2.0.46.

As Michael Han hinted at, but I don't think explicitly stated,
if fvwm2 can't find the .fvwm2rc in the 2 places it looks, it
then looks for system.fvwmrc.

It may work "pretty well" that way, but I would expect that you would
not want to leave it that way.

The system.fvwmrc stuff is documented in the installation instructions.
If the installer does his thing to the system.fvwmrc, fvwm2 could work
"very well" that way.
