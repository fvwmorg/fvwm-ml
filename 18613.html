Conrad Sabatier wrote:
>
> Just wondering if anyone is doing any work in this area. I think it would
> be terrific to have some CDE-like visual configuration tools as part of
> fvwm(95). Online menu editing would be a real boon (maybe something like
> Netscape's bookmark management). Button configuration, etc., anything to
> make using and configuring fvwm easier, relieving the user of manually
> editing rc's, etc., would be such a blessing.

I'm running Slackware 3.10 and when I use the default installation of X,
fvwm95 is set up as the window manager. If you bring up the utilities
menu you will find a submenu called "Modules". Under the Modules menu,
you find "Configure" with a small hammer icon. When clicking on
"Configure" nothing happens. When looking at the VC that X was started
from you get:

fvwm: No such module /usr/X11.../FvwmConfig

It seems as though the person who prepared the default .fvwm95rc either:

1) Forgot to include FvwmConfig with the
   distribution.
2) Put it on the menu in anticipation.
3) Has a very bad sense of humor.

I'm hoping for #2. The ability to edit .fvwmrc and configure FVWM is
fantastic, however something like the Windows Control Panel would be
MUCH better! (Before the flames come: I HATE MS-WINDOWS of any flavor or
version (Including '95))

Does anyone know the scoop on fvwmconfig?????
