> Hi. I tried to start fvwm, but there was an error message:;
> 'cannot open display'.
>
> I know it's not much, but I'm a new linux-user, so I would appreciate any help.
>
> I'm using redhat 7.1 with kernel 2.4.5
>

Possible problems:
- no X server started
  How did you log in?
  What did you execute in order to start fvwm?
- X server is started by another user, that user did not allow
  "actions" on the X server
  How did you log in?
  What did you execute in order to start fvwm?
  Are there other users logged in? If yes, how?
- DISPLAY environment variable with wrong contents
  Try echo $DISPLAY to check that
- ...
