If I may chime in, I think it's worth mentioning that the choice of
"where" to install the compiled code is worth thinking about, since they
can not be moved from there without recompiling... Therefore, I think a
different directory would be preferable, say without the dash...

configure --prefix=/home/gsm/u08/pushpak/fvwm/fvwm2.2.2 --without-cxx
