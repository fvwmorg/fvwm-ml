On Fri, Dec 14, 2001 at 07:50:29PM +0100, Marcus Lundblad wrote:
> I tried to turn on the option "FvwmPager: MiniIcons"
> But nothing happens... is this something new that has been added since
> 2.4.3 in the CVS?

So you use CVS and you've get mini icon in your pager with 2.4.3?
This is strange ... So first I assume that your are new with mini
icons, sorry if it is not the case.

Do you have lines of the form:

Style * MiniIcon a_default_small_icons
Style "match" MiniIcon a_small_icons

in your config files? If not add at least

Style * MiniIcon a_default_small_icons

If you use the current CVS some mini icons should appear
automatically (without MiniIcon config lines) in the pager
for KDE version 2 applications.
So yes there were changes in the CVS regarding MiniIcon
but it should not break old config (but for KDE2 and GNOME2
apps).

If all the above is stupid regarding your config, maybe you
can try a full rebuild (some fvwm2 to modules communication
types have changed recently and this may cause problems (??)
if you rebuild fvwm2 only).
