KG> You might want to check, using FvwmIdentify, what `name' xbuffy
KG> passes to the popup windows.

> I'm not sure how to accomplish this. The popup windows only popup
> when the left mouse button is pushed; when you release the button
> they disappear. Anyone know of a trick to do it?

You can set up xbuffy to pop up the window when mail arrives and keep
it up for some period of time using the headertime directive in
.xbuffy (there's probably a command line flag as well). Then run
FvwmIdent on it. I just tried it. Sure enough, the class is XBuffy,
but the window name is the name I assigned the mailbox with the title
directive in .xbuffy.
