Hi Uwe and others,

Happy to see the theme has pleased you.

Concerning the X server, I use Xorg 6.8 and its composite extension.
They are managed by xcompmgr that I run via the InitFunction. It
provides shadowing of app and real transparency (called opacity). The
shadowing effect really help to target the focused window. The opacity
is just eye candy.

The VT is MRXVT and it allows to configure its own opacity. To modify
the opacity for an already launched app, I use transset. I've seen
people using a modified transset able to assign a specific opacity
function to any app during its launch. I haven't set it caus' my distro
is based on sources and I don't like the idea to patch my code each time
a bug fix is available.

That only apply for the apps. For the menu, it's basic colorsets.
Transparency for the pixmap menu and transluency for the other: kinda
tricky.

CU, pem
