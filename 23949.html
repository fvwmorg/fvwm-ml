%% richard.abraham_at_mnr.gov.on.ca writes:

  ra> I know this operation should be pretty basic but I can find no
  ra> documentation on how to grant display permission using fvwm.

That's because you're looking in the wrong place.

Permissions to use your display are granted by your X server, not your
window manager. It's completely out of fvwm's control who has
permission to create windows on the display. So you won't find any
discussion of it in the fvwm docs :)

You want to look at the man page for the "xhost" command. For example,

  $ xhost +remotehost

will allow any user logged into remotehost to create windows on your
display. There are a number of security issues with this, so you should
check the manual.
