On Thu, 31 Oct 1996, Lance Heller wrote:
> I've installed fvwm 1.23b on a x86 box running Solaris 2.5. All
> seems to be sweetness and light except function keys F1-F12, which
> no longer produce the actions found on my AIX 4.1.4 machine running
> the same fvwm revision.

Try using xev to see what keysym they are making. They might not be
making F1 and F2 and so on, but Copy, Cut, Help or something. I'd
also make sure your meta key is really working as meta.
