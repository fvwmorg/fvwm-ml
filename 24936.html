> how to set a wallpaper?

Woo-hoo! (opportunity for a shameless plug)

I've got a script called "bkgd" which will randomly (or
non-randomly) choose and set a background/wallpaper image.

        http://www.vis.colostate.edu/~scriven/bkgd.php3

Just create a directory "/usr/local/backgrounds" and/or
"$HOME/backgrounds" with your images in it, then run "bkgd" to
set new wallpaper. I stuck this in my /etc/X11/Xsetup_0 to
make xdm look pretty.

You may also need to edit the options near the top of the
script, to configure various options.
