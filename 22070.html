Hi FVWM2 Users,

I've got strange problem with FVWM2-2.0.45 on my Linux computer. I have
the next two lines in my ~/.fvwm2rc file:


Style "*" SmartPlacement, RandomPlacement

and

Style "rxvt" ActivePlacement


These two lines are mean the next: i would like start rxvt application
with ActivePlacement mode and others will be RandomPlacement.

So, then i try start rxvt application nothing will happen with active
placement of rxvt's window. But i will get invitation to place rxvt if
rxvt application will be the third one on the screen. It doesn't matter
how i start rxvt application, from FVWM menu or from XTERM like
console. So if i have got more then two copy of rxvt application on the
screen, then ~/.fvwm2rc lines are worked correctly. This problem will
appear if i will try set for another application the same, for example
EMACS or XTERM. Random placement work for all another application (if
number of rxvt copies less then two, RandomPlacement is used).

The question is, is it bug or i missed something?


Oleg
