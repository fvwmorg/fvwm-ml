You mention ".fvwm95rc", if you are running fvwm95, you are on the
wrong mailing list.

You mention, "fvwm", there are 2 versions of fvwm around, if you are
running fvwm version 1, the config file is ".fvwmrc". If you are
running fvwm version 2, (fvwm2) the config file is ".fvwm2rc".

You seem to be using the word "run" for 2 different purposes. I think
you mean, I currently "run" vtwm which "reads" my .vtwmrc.

I know I am being picky, but I hope by helping you phrase your
question more clearly, you will get a better answer.
