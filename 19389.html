Hi,
I've been a very satisfied user of Fvwm 1 for a few years. Now I'm contem-
plating about switching to Fvwm 2 (mostly due to the improved flexibility).

However, there is one function I've been missing in Fvwm 1 and I wasn't able
to find in Fvwm 2 up to now: How can I program a hotkey for (repeated) toggling
between windows?
That means, if I select window B after having selected window A before, I'd
like the toggling function to pop me back into window A, and from there to
window B again (without having to look at the window list). I hoped this
could be achieved by the new `Prev'-Function in Fvwm 2, but it doesn't work
for me.

Did I misinterpret the manual? Could the kind of toggling I want be done
by an Fvwm function or would it take a new module?

As I am no subscriber of the Fvwm mailing lists, please put me on the CC
if you have any comments.

Thanks in advance,
