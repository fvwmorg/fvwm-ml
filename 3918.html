eug_at_post.tau.ac.il writes:
> Hi!
> Recently I started to use fvwm2 window manager
> after several years of fvwm usage. I have a .fvwmrc
> file and I want to convert it somehow to .fvwm2rc
> How can I do it? (Please don't suggest me to write
> a new one from the scratch )

OK.

To convert to fvwm2, use the fvwmrc_convert script that comes with
fvwm2.

If you picked up a current 2.4.x release, you need a second
conversion, run fvwm24_convert after running fvwmrc_convert.
