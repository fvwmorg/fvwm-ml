I think I have to make a little bit clearer what we would like to do:

We have made a set of motif applications to display medical greyscale images
with sizes in the range from about 256x256 up to more than 2048x2048 pixels.

The window contents of the displayed images may change very often (about 10
frames per second depending on the image size) due to brightness/contrast
adjustment.

Since there may be many images displayed on many virtual screens (eg. 32
virtual screens with two images each) displaying the images in the pager would
be a big aid in navigating the desktop, i.e. to quickly switch to the virtual
screen with the image I am looking for.

It should be possible to make an extended pager module which displays the
scaled down images (using a crude scaling algorithm this could be very fast)
with reduced colors (eg. 16 grey values) only when an application requests the
pager to do so (eg. by a new x-event).

So I am asking again if anybody has done this or is willing to this if he gets
some financial sponsorship?
