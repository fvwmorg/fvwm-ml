On Wed, 9 Feb 2005 17:52:47 +0100, Bjoern Steinbrink

> On 2005.02.09 16:35:27 +0000, Sebastian Rodriguez wrote:
> > I rewritte to the mailing list cause now i've a little problem :p
> > How can i do to dont put borders or anyting else to the xine bar? the
> > one you can make pause/play etc, cause now it has a border, and its
> > quite annoying :'(
> >
>
> Use the FvwmIdent module to get some information about the windows for
> which you want different styles. The fields in question are Name, Class
> and Resource IIRC (maybe Transient is useful, too). Then figure out, which
> of them are unique to those windows you want to change. Finally put some
> Style settings into your fvwm configuration. For xine I use:
> Style "Xine Window" NoTitle, !Borders
> Style "Xine Panel" NoTitle, !Borders
>
> HTH
