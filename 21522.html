You have to use

Style "*" NoIcon

And

*FvwmIconBox "XTerm" NeXTterm.xpm
*FvwmIconBox "Emacs" gnu.xpm
 [ and so on ]

If you do it like this it works. I know FvwmIconBox is supposed to
extract the Icon files from the Style commands as well, but for some
insane reason that doesn't work. If I have some time left during the
holidays, I'll take a peek at the code...
