     Mode_Switch+Open Key => SunOpen (Mode_Switch usually assigned to Alt
                                       Graph)
Under R6 and R6.1, it is assigned:
  'SunOpen NoSymbol F17':
    Open Key => SunOpen
    Shift+Open Key => SunOpen
    Mode_Switch+Open Key => F17

Note the difference. When you press the 'Open' key in OpenWindows or 'R5,
the keysym the client gets is F17, or L7. In 'R6, it gets SunOpen or
SunKX_Open (0x1005FF73, if you want the hex code, which is the same in all
implementations). The net result is that a client looking for some variation
of 'Sun*Open' will see it without any modifiers under 'R6, but won't see it
unless Alt Graph is also pressed under OpenWindows or 'R6.

Check the output of 'xmodmap -pk' for more details.

There were a lot of questions when 'R6 came out to the effect of 'why are my
OpenWindows clients broken?'. This was because the keysym mapping was
changed from 'R5 to 'R6.
