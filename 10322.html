You apparently start fvwm-themes-start, not fvwm.
Your ~/.fvwm/.fvwm2rc is ignored in this case.

Either start fvwm or use the fvwm-themes rules to define the personal
adjustments as described in its FAQ.

If you decide to continue with fvwm-themes, then you may create file
~/.fvwm/themes/personal/modules-extra with one "DeskTopSize 2 2" command
and follow the instructions in the fvwm-themes FAQ. Or choose modules
from other themes, some use DeskTopSize 2x2.
