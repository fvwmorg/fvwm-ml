>>>>> Ken Fuhr writes:

  Ken> I am trying to use fvwmiconman v 0.9.7a, but I have one small
  Ken> problem. FvwmIconMan, when compiling complains about not
  Ken> finding fvwm.tmpl.

  Ken> Care to tell me what fvwm.tmpl is and why I need it?

"Fvwm.tmpl" (note the capitalization) is a file needed to compile
fvwm. It comes in the fvwm-*.tar.gz file and unpacks into the main
directory (where you find README and INSTALL, too).

Did you unpack FvwmIconMan-*.tar.gz into the modules subdirectory of
the fvwm source tree? If not, try to move it there then just do a
recompilation of fvwm -- the new FvwmIconMan is a plugin replacement
for the old module that's already there.
