With this patch, fvwm can now understand Openlook [Virtual] Window
Manager hints. Programs compiled with XView or OLIT now have control
over which window decorations are displayed.

I have added a new Style option called OLDecor (and it's inverse
NoOLDecor) to use hints, approximately like MwmDecor.
