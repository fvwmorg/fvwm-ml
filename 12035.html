If you use fvwm-themes, then there is no fvwm2rc file. You should read
the documentation explaining how to use and change themes, available on
the fvwm-themes web site (that in turn is available from your root menu).

If you want to change the redmondxp (or redmond98?) theme, then make a
copy of the directory like /usr/local/share/fvwm/themes/redmondxp to
redmondxp2 and edit the copied files.

Another way to customize things in fvwm-themes is to create corresponding
files in your personal theme (create ~/.fvwm/themes/personal/ dir first).
For example, you may define your own Style Icon and MiniIcon lines in
file "styles-extra" (or better yet "iconstyles-extra") in this directory,
and then follow the procedure described in the fvwm-themes FAQ.

There is no GUI in fvwm-themes to create this iconstyles-extra file, if
this is what you want. But you may write such GUI, it is not very hard.
