> What purposes do the M4 & CPP pre-processor modules serve?? Since

I use m4 extensively in my .fvwmrc files. By keying on `hostname` I can use
the same file for multiple machines. This is useful for NFS mounted
filesystems as all machines will use the same rc file. I use m4 macros
to run unix commands, using the results for various things such as
retrieving my logname, etc.
