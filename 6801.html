  Did you, perchance use {F11} for "fullscreen mode"? In X, there isn't
a true fullscreen mode. The Mozilla people emulate it in X by increasing
the window so that the decorations/borders are off the screen, and you
see nothing but the web page. So far, so good. However, upon returning
to "normal" mode ({F11} again) it's offset a bit for me too. I define
the following key binding to maximize/restore a window with {ALT-M}

Key M A M Maximize 100 100
