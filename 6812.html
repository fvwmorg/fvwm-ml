Eduardo Gargiulo wrote:
> Hi all.
>
> I´m trying to set up FvwmWharf on my system, but I can't. I'd added some
> *FvwmWharf settings on system.fvwm2rc, but I can't load the module and I
> don't know where to look for logging errors. (I´d read FvwmWharf man
> page). Any help would be appreciated.
>
> TIA
>
> -ejg
>

fvwm prints its errors to stderr--try looking at the server where you
started the X server (if you started it from a terminal), or try looking
at ~/.xsession-errors or similar. If errors aren't being printed to
stderr and you can't seem to find where they're being redirected to, try
creating an error (by, say, restarting fvwm2 with the FvwmWharf module
failing) and doing 'ls -ltra ~' to see which files were recently changed.
