Near 2000, Rick Munday mumbled something about:
>
> What is the trick with the .fvwmrc file? I have placed a copy in my
> /home/username dir and it still uses the
> default system.fvwmrc!

You should include a reference to the rc file when invoking Fvwm.
This means that you should strart Fvwm by a command like:
fvwm2 -f /home/foobar/.fvwm2rc

Hope this helps,
