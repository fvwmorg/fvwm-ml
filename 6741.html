Gabriel Moll <gmoll_at_dcc.uchile.cl> writes:
>
> The error message that I get when I try to start the FVWMpager is the followi
> ng
>
> [FVWM][executeModule]: <<ERROR>> No such module 'FvwmPager' in ModulePath '/u
> sr/local/lib/X11/fvwm2'
>
> How might I know if the module is installed and if it is, how might I know
> where
> is it?

The above ModulePath doesn't look right.
If you are setting the ModulePath in your .fvwm2rc, remove the line
that sets the path and then restart Fvwm.

You should be able to find FvwmPager by typing the command

locate FvwmPager

but its not necessary to find it, Fvwm should know where it is
automatically.
