Hello,

As in the subject, I'm having trouble getting XPM, and thus colored icons
to work in fvwm. This server was recently given a clean install of Solaris 7
8/99, and then patched with the most recent Sun recommended patch set
for Solaris 7. The server itself is a Sun clone (please don't laugh). The
processor on a psrinfo -v says "sparcv9 processor running at 360 MHz"

Anyway, I compiled XPM 3.4K. I put the X11 includes in /usr/local/include
and the lib files in /usr/local/lib. I ran configure as follows:

LDFLAGS=-R/usr/local/lib
./configure --without-cxx --with-xpm-library=/usr/local/lib
--with-xpm-includes=/usr/local/include

The make ran with most gcc comands using the "-lXpm" flag.

The icons themsleves are in /usr/local/icons.

The configure.log file contains no references to errors involving Xpm.

If I am not providing enough information, please write back, and I'll
provide what
I can. If not, then thanks in advance for your assistance!
