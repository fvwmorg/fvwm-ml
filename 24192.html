Hi all,

I use fvwm 2.2 with linux-2.2.x and I have experiment some well
known bugs (the TaskBar startmenu always pressed bug, the many open
windows TaskBar bug, the FvwmCommandS kernel 2.2.x bug, the WindowShade
on maximized window bug).

I have tried a devel-snap and all this bugs seems to be fixed !
However, I prefer to use a stable version ... even if I use now
the FvwmCommandS of the devel-snap (the devel-snap TaskBar does not
work with 2.2).

So my question: Is there any plan for a 2.2.1 version which will be
a stabilized version of 2.2 (no more feature but fix for easy fixed
bug) ? This will be great !

Many Thanks for all the fvwm devellopers !
