Until now we didn't have rpms for 2.5.x.

I don't know which features to include. I want to have new features:

  * bidi (Arabic and Hebrew are broken without this, requires fribidi)
  * png (requires libpng)
  * xinerama (can't easily work on RH 6.x because of too old XFree)
  * xft (can't work on RH 6.x because of too old XFree, I think)
  * xrender (can't work on RH 6.x because of too old XFree, I think)

Probably it is a time to move to RH 7.x rpms for 2.5.x.

I guess there are users that badly need these features and the ones that
don't like to add additional requirements. Too bad.
