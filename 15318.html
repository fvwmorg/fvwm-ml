Sounds like you probably need a keymap file. Unfortunately, I have a type 5
keyboard and not a type 4, so I'm not sure if my keymap file will work for
you. I'll include it below anyway, just in case.

Just stick this file in your home directory and call it ".KEYMAP" or something
like that. Then, at the top of your .xinitrc file, put the following line:

xmodmap $HOME/.KEYMAP

If you figure out the right keycodes for your arrow keys, you should be able
to get them working this way.

Now, I have a question for you. Where did you get the version of DOOM that
runs on SunOS 4.1.4? The Solaris 2.4 version was the only DOOM that I knew
of that ran on SUNs.
