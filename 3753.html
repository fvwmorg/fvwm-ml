On Mon, Dec 03, 2001 at 03:30:46AM -0330, Neil Zanella wrote:
>
> Hello,
>
> I am running fvwm2 version 2.2.4. When I try to kill an icon on the
> desktop with the xkill utility the icon does not disappear. Is there
> a way that such behavior can be enabled with fvwm2?

Just bind a mouse action to icons, e.g.:

  Mouse 3 I N Delete

(or use Destroy/Close).
