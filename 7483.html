Hi,

Here's an snip of what ends up being my config:

TitleStyle ActiveUp MultiPixmap Main eazel_blue/title_active.xpm, \
UnderText eazel_blue/titleunder_active.xpm, LeftOfText \
eazel_blue/titlelt_active.xpm, RightOfText \
eazel_blue/titlert_active.xpm

TitleStyle ActiveDown MultiPixmap Main eazel_blue/title_active.xpm, \
UnderText eazel_blue/titleunder_active.xpm, LeftOfText \
eazel_blue/titlelt_active.xpm, RightOfText \
eazel_blue/titlert_active.xpm

TitleStyle Inactive MultiPixmap Main eazel_blue/title_inactive.xpm, \
UnderText eazel_blue/titleunder_inactive.xpm, LeftOfText \
eazel_blue/titlelt_inactive.xpm, RightOfText \
eazel_blue/titlert_inactive.xpm


... mind you this is merely what my "fvwm-theme" (not fvwm-themes)
script does. I grabbed this _at_:

http://www.igs.net/~tril/fvwm/configs

and give all props to Suzanne Britton who actually managed to create
something usable out-of-the-box.

Hope this helps,
Pete =)
