I use pixmaps from the Tile of the Day web page as backgrounds for my
FvwmButtons button bar. For the items that are just icons it's great,
but for the four (non-fvwm-module) applications I run in the buttons
(AfterStep apps: asclock, ascd, asmixer, asmodem), the part of the
background pixmap that's *not* under the smallest rectangle that
encloses the app doesn't show up until I do a screen refresh (or a
window refresh on the button bar).

It's just an annoyance, but is there a way to get the button bar to
refresh itself or show up correctly to start with?
