>>>>> "Grant" == Grant McDorman <grant_at_isgtec.com> writes:

    Grant> One problem is that the keysyms assigned to those keys vary
    Grant> depending on which server you are running.

    Grant> Under X11R6.1, which I'm currently running, you'll use
    Grant> "SunOpen". Under X11R5 (and OpenWindows), use "F17". ("L7"
    Grant> is a synonym for "F17".)

Wrong. It is the client's X11 librarary which decideds the names.
Xlib reads its own XKeysymdb then translating keys.

That means that OpenWindows 3.0 (which is X11R4) for example uses
SunXK_Open ,OpenWIndows 3.3 (Solaris 2.4 and X11R5) SunOpen
dito for X11R6 distribution.

Key translation is purely done by the client side.
