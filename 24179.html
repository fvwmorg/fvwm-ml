Hi,

  I don't know if this is a fvwm 2.2 issue or a Netscape issue, but I've
only noticed this problem with Netscape windows. (Bear with me, I think
it's an original report...)

  I run Netscape (actually, just Navigator and Messenger) at
1018x688+0+56, with "BorderWidth 3, HandleWidth 3", essentially expanding
it to fill the whole screen, minus the 56 pixel row at the top of the
screen. (Actually, to get this geometry, I just did a "Maximize 100 100",
and then resized the top down by 56 pixels.) From here on out, when I
mention Netscape windows, I'm referring to just Navigator and Messenger
windows, since both of these exhibit the problem. (Others, like Composer,
might too, but I don't run them.)

  The problem: sometimes, if I try to move my mouse pointer into another
page, I get "blocked" on the edges by the Netscape window. I just can't
"push" past the border of the window. It's not always the same edge -- it
can be the left, right, or bottom edge (never the top edge, since this
only occurs for Netscape borders against the edge of the current page).
When this happens, I see the arrow-pointing-to-the-edge icon, the one you
get for resizing edges, since the mouse pointer is "stuck" over Netscape's
border. This happens maybe once or twice a day, and I'm on the computer a
lot. (Or so says my fiancee :-)

  I run several other programs full-screen in a similar fashion, e.g.
xterm, xmaple, xemacs, etc. but I haven't seen anything exhibit this
behavior other than Netscape. I've found that it's usually pretty easy to
get around, either by using Ctrl-Arrow keys to switch pages, or to move
the mouse off of some other edge of the page, return back to the original
page, and attempt to move the mouse to the desired page again, which seems
to always cause it to work again. However, it is pretty annoying.

  Is this an fvwm issue, or a Netscape issue? I'll post what I think may
be relevent config info below. As always, please contact me if you need
more info.

Thanks,
-jason


EdgeScroll 100000 100000
EdgeResistance 0 10
EdgeThickness 1

Style "*" IconBox 30 90 -30 -30, IconFill right top, IconGrid 90 90, \
  ForeColor wheat, BackColor DarkOrange4, ActivePlacement, DumbPlacement, \
  NoPPosition, BorderWidth 3, HandleWidth 3, Icon linux-penguin.xpm, \
  DecorateTransient
