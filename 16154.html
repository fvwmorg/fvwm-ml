there should be a
        char * strerror(int errnum) for ANSI C
but many old headers just define the following:
---------------------------------
NAME
     perror, errno - system error messages

SYNOPSIS
     void perror(s)
     char *s;

     #include <errno.h>

     int sys_nerr;
     char *sys_errlist[];
     int errno;
--------------------------------
so a
        #define strerror(e) sys_errlist[e]
would be cool for those compilers (here it's gcc 2.6)
