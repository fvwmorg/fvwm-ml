On Thu, Feb 05, 2004 at 07:20:07PM -0500, Stephen R Dennison wrote:
> Anyone know of a good program to use in conjunction with FvwmBacker that
> caches background images?

Fvwm.

Fvwm caches images. Define colour sets for the backgrounds and
use them in FvwmBacker. The images can even be used in other
modules or the fvwm core without loading them again:

  colorset 77 tiledpixmap image1.xpm
  colorset 88 tiledpixmap image2.png
  *FvwmBacker: Command (Desk *, Page 0 *) Colorset 77
  *FvwmBacker: Command (Desk *, Page 1 *) Colorset 88
  FvwmBacker

The only drawback I see is that fvwm supports only png, xpm and
xbm. But there are plenty of programs around to convert your
images to a suitable format.
