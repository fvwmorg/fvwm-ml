jtsiao> I've just started to play around with fvwm 2.0 and noticed that
jtsiao> the TogglePage command is no longer valid. Is there an equivalent
jtsiao> command for doing the same thing (i.e. toggle page scrolling on/off)?

Use EdgeScroll. You can set 'EdgeScroll 0 0' or 'EdgeScroll 0 100' or
'EdgeScroll 100 100' or 'EdgeScroll 100 0' (or whatever) at any time
you want. Of course it's not TRUELY equivalent of TogglePage, but
this way is more flexible, and this is why TogglePage is disappeared.

(If you wanna patch which adds TogglePage to fvwm-2, email to me).
