On 12 Oct 2001 11:29:56 +0200, Marcus Lundblad wrote:
>
> Is it possible to remove the title of the FvwmButtons window?
> And is it also possible to turn of the "stipledness" for individual
> windows when they are "sticky" (in this case FvwmButtons).

No, you can't currently disable stipples for sticky windows, you can only
enable stipples for non-sticky windows.

> I did set the title font of FvwmButtons to a smaller one than the normal
> title font so that the window will get a thinner title bar that
> "normal" windows.
> Setting: Style "FvwmButtons" NoTitle
> removes the title bar entirely. I would still like it to have a title bar.
> Can this be done by creating a new decor?

It took me some time to understand that you want to disable a title text.
The title name of FvwmButtons is either "FvwmButtons" or the module alias
name. You may do some not nice tricks like creating a short alias like "_"
(you can only use alphanumeric chars in alias) or use the same color for
text foreground and background or use invisible font.

Probably a feature to specify a customized static title name for some
windows to display would be interesting, but it is not currently possible.

Or if you really want it now, find the line: list[0] = MyName;
in FvwmButtons.c and specify an empty string "" instead of MyName.
