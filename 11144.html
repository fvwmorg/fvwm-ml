 --- Al <mediadesk_at_linfe.it> wrote: > I would like to add the aqua theme
to my fvwmrc, but it is too hard
> to extract it from the FVWM-Themes package.
>
> Will anybody point me at an fvwmrc with aqua-like window style ?
>
> thank you

What I would do is to run the aqua theme as your default theme
temporarily, and then run:

touch ~/.fvwm2rc && mv ~/.fvwm2rc ~/.BACKUP-fvwm2rc && fvwm-themes-config
--expand-rc > ~/.fvwm2rc

This will then capture your current themes setting into a single file that
you can use. You'll have to copy any dependant icons across from the aqua
theme and tweak the location in the config files, etc...

HTH,
