On 05 Oct 2000 17:09:03 -0000, Les Martin wrote:
>
> Can I specify geometry with the Style command,
> so that, for example
>
> Style "*rocmeter*" Geometry 120x80+40+20
>
> would make it so if I run procmeter3 from the
> command line, it will come up though I had said
>
> procmeter3 -geometry 120x80+40+20

Not using a style. Just add the following line into your ~/.Xdefault:

  procmeter*geometry: 120x80+40+20

> If not via the Style command, is there another
> way I can tell the window manager to make sure
> when a new window comes up which satisfies some
> criteria it will be resized and repositioned to
> a particular geometry?

Yes it is possible too using FvwmEvent, see:

  for 2.2: http://www.hpc.uh.edu/fvwm/archive/0006/msg00393.html
  for 2.3: http://www.hpc.uh.edu/fvwm/archive/0006/msg00382.html
