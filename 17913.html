I am trying to select the bitmaps to use for the icons for several
windows. The man page for fvwm2 says that when selecting the

matched with name, class, or resource string with 'name' getting
preference, followed by class.

My .fvwm2rc contains: Style "*Weather*" Icon weather.xbm

The application is run in an xterm, but the window is renamed "LLNL

xterms, not the one for weather.

Is this a bug or am I mis-understanding something?
