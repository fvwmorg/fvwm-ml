Catherine Luedecker <luedchic_at_comcast.net> writes:

> I have a question about ImagePath. in the rc file do you need to put
> with complete path for icons or will it check in every file folder of a
> directory?
>
> for instace I have an icon directory with several file folders
>
> /usr/share/icons/ and this directory has a folder for each group of
> icons


Use the + !

If /usr/share/icons contains folder1, folder2, folder3 etc
then in .fvwm2rc use

ImagePath /usr/share/icons:+


You can then write
eg
Style "emacs" Icon folder1/emacs.xpm, etc
Style "gvim" Icon folder2/vim.xpm, etc
Style "xmcd" Icon folder3/cdrom.xpm, etc
