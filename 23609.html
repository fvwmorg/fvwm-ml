Hi -
        I am running 2.1.10 with the following pager relevant config:

DeskTopSize 1 8
*FvwmPagerBack #5c54c0
*FvwmPagerFore black
*FvwmPagerFont none
*FvwmPagerHilight #6b64b0
*FvwmPagerGeometry 70x500+0+0
*FvwmPagerLabel 0 Silk
*FvwmPagerSmallFont 5x8
*FvwmPagerBalloons


Titles do appear (and pop up when I rolloever). However, some windows say
"xterm" and some say "myspecialtitle".
        The two lines below (wrapped) are in my popup menu. the first
names the xterm "localhost" when I'm looking at it, but pager says it's
"xterm". the second says "grunt_at_eilio.com" in both cases.

"localhost" Exec xterm -ls -title localhost -sb -font 9x15 -geometry
80x40 -s -rw

+ "grunt_at_eilio.com" Exec xterm -ls -title grunt_at_eilio.com -sb -font
9x15 -geometry 80x40 -s -rw -e telnet login.eilio.com


At the command prompt: xterm -title foo doesn't give me foo in the pager,
but xterm -name foo does. (I don't want to use -name though because that
messes with Xdefaults...)

Is this a bug? It's not that big of a deal, but I'm in a tweaking mood
and it caught me :)
