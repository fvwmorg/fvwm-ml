> sticky buttons, and these work fine except I can't get the Toggled style to be
> flat. Normal buttons come up flat without any problem, but when toggled, the
> buttons always come out raised. I can't seem to find any references to them
> outside of the theme directory, and even tried using FvwmCommand to set things
> on the fly, but no luck. Typical example in the theme rc:
>

The problem here is that it is the windowlook component that decides
if the buttons are flat or not (technical reasons). So, which windowlook
do you use? (it seems to me that windowlook_at_osx gives flat buttons).
You can enforce flat buttons by adding AddButtonStyle All -- flat in
the file .fvwm/themes/personal/extra-windowlook (and refresh and then
select this component, see the FVWM themes FAQ and doc/* for more
details). If you a creating a complete themes use the windowlook
component. I Hope I am not wrong ...
