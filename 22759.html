(The following is with fvwm2 2.0.46)

Say I start a few xterms, and iconise them all. They automatically fill
the icon box I've defined, in the order I iconise them. But they don't
remember their places - it seems that fvwm2 will automatically "pack"
the icons into the box. What I want is for the icons to remember where
they started, and stay there when re-iconised (like olvwm).

It seems this is possible, as I noted fvwm2 will remember the icon's
placement if I physically position the icon where I want it - it always
returns to this position. (I don't want to have to specify an icon
position on the command line).

This is useful primarily for when I have a dozen editor windows open and
I can keep track of them by simply remembering where the icons stay.

Any ideas?
