  Patrick> Hi there, this has probably been asked/answered before, but
  Patrick> I have looked through the archives and can't seem to find
  Patrick> it. I have a sparc5 running SunOS 5.5.1 (solaris 2.4
  Patrick> whatever) and the install (the MakeMakefiles and/or xmkmf)
  Patrick> pukes due to it trying to do a imake -I/usr/lib/X11/config
  Patrick> [...]

Look at the /usr/openwin/bin/xmkmf shell script and you'll see that it
uses $OPENWINHOME/lib/X11/config if that environment variable is set.
Thus, setting the environment variable OPENWINHOME to /usr/openwin
then running xmkmf should do what you want.
