> Thankyou for your suggestions about kedit.... it seems

> would like your help with....
>
> 1. Is it possible to make that Function generic for
> any K app that is loaded, i.e. maybe by using
> wildcards?????
>
> 2. Secondly, because Kedit (with your function you
> supplied) loads at startup, if I shut down that
> program, and then load it up again, it starts on the
> page it is meant to, but it does not load as an icon,
> instead it loads up as a window. So my question is,
> once I shutdown kedit, how do I re-envoke the Function

OK, none of this is tested, but I think you will get the idea:

# Start program iconic, first arg is program name, class/resource name
DestroyFunc ksicon
AddToFunc ksicon
# I Style "$0" StartIconic
+ I Exec $0
+ I Wait $0
+ I Style "$0" StartNormal


AddToFunc InitFunction I ksicon kedit
AddToMenu Utilities "Start kedit" ksicon kedit

I made the function I gave you earlier generic, used the function from
the InitFunction and from a menu.
