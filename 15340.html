I have this in my .fvwmrc:

# Button Context Mods Function
Mouse 1 FS A Function "Resize-or-Raise"

And this in my .fvwm2rc:

Mouse 1 FS A Resize-or-Raise

If you're only able to resize when grabbing the corners I suspect
your line says "F" where mine says "FS". I think I added the "S" to mine
when I was making it work the way SGI's "4Dwm" works...
