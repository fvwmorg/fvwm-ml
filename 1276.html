On Fri, Aug 18, 2000 at 05:02:54PM +0200, robert_wilhelm_land wrote:
> How do I set other parameters like a smaller window width on a Xterm startup?
> the XF86Config file doesn't seem to control that. I looked at the fvwm2 config
> files and searched for everything containing a "width" or a "height" but found
> nothing responsable for these settings.

Well, you're referring to the window geometry. Look at the manpage for
xterm. There are standard toolkit options that most programs accept,
such as -bg, -fg and -geometry. A useful book is O'Reilly's X Window
System in a Nutshell.
