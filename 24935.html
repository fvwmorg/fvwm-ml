On Wed, Oct 20, 1999 at 12:34:15PM +1000, Shao Zhang wrote:
> Hi,
> When I hit the minimize button in one of the windows, the window
> just disappeared, and I have to retrieved it using the
> mouse(windows list).
>
> How do I configure my fvwmrc so that, it will put a icon on the
> current virtual desktop.

If the application doesn't provide an icon itself you can specify
one in your fvwm config file. Usually it is done by setting a
default icon for all applications:

  Style * Icon default_icon.xpm
