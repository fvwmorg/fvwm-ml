  Hi,

I run a lot of apps through XTerms, usualy opening about 5-10 xterms and
running lots of different apps on them throughout the day/week/month
(however long it is till something reboots me!). Anyway, as you can
imagine, the taskbar gets really cluttered (lots of XTerms plus lots of
other apps). I have most of my common tasks aliased so they change the
xterm title when they run, so I can more easily find them on the taskbar
(ie, when you run pine it changes the title to "Pine", when you exit, back
to "XTerm"). But what I'd really like is the miniicon on the taskbar and
the window to change.

I've setup different miniicons for just about everything I run, but so far
the only way I've found to change them after fvwm has started up is to
restart fvwm... which takes time (it's an old computer) and, heh, messes
up the order windows are listed on the taskbar.

Does anyone have a better way of doing this? I'm using fvwm2.0.46, with
the FvwmTaskBar that comes in the extras dir (I've modified it so it sees
my mailbox in $HOME/Mailbox).

Really hope this makes sense, I'm writing this half asleep. (:
