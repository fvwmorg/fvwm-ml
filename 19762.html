Hello Fvwm Mailing List!,

        Sometimes, certain dialog boxes in Netscape (such as the Open Location
dialog, or Save As dialog) would only pop up with just the title bar
showing (as if the WindowShade function was in effect). I've noticed
this phenomenon with other FVWM derived WM's as well (i.e. AfterStep and
Fvwm '95). Since I now use FVWM 2.0.x., I figured I might as well post
about it here.

        I'm almost certain that this is a Motif related occurance, but since
I've only noticed this so far with Netscape (probably because the Motif
app I use the most is Netscape), I thought it would be better to post
this as a "Netscape problem" rather than a more general "Motif problem".

        Anybody have any ideas as to what could be the problem? And if so, a
fix to this problem? Thanks!
