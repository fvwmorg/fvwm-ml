On 11/29/04 08:24 AM, Louis LeBlanc sat at the `puter and typed:
> On 11/29/04 08:04 AM, Jay Dobies sat at the `puter and typed:
> > I've been using FVWM for about 3 months now, and have grown quite
> > accustomed to my keyboard shortcuts. In particular, I use the Windows
> > Key for a ton of different things; I don't like to tie up Ctrl or Alt
> > because they are used a lot by other programs I use (for instance
> > IntelliJ). But I wanted to use a more standard key than Windows, in
> > case I'm on a laptop and don't have one available. I don't use the
> > Caps Lock for it's actual purpose, so I wanted to disable it as far as
> > capital letters is concerned and instead use it as a modifier. I think
> > I have used it as a modifier before (I'd have to dig through backup
> > copies of .fvwm2rc), but how can I disable it such that it won't make
> > my letters capital?
>
> I think that needs to happen in the X config. I'm using X.org on one
> machine and XFree86 on another, and I am using the following line in
> my keyboard sections for both:
>
> Option "XkbOptions" "ctrl:nocaps"

Of course, you want to keep in mind that this (and the other
suggestions given in this thread) won't actually disable that key, it
will just convert it to another control key.

The X docs and xmodmap docs should contain details on giving it a
different function.
