I have written a patch for FvwmTaskBar which allows the following behavior:
  - Only windows on the current desk (including sticky windows)
    show up in the taskbar
  - When the desktop is switched, the taskbar updates to show only windows which
    are associated with the new desk

This allows the taskbar to be much less crowded, and I find this behavior to be
more useful than having taskbar entries for every window known to FVWM.

If there is any interest in this patch, I'd be happy to make the above behavior
optional and then submit the patch to the public domain. I have two questions:

1) Is there any interest?
2) Who would be the appropriate person to send such a submission to? I checked
   the source code, but none of the email addresses listed there exist anymore.
   I also checked the FAQ, which said that the mailing lists were the current
   forum for new code submissions.

Thanks for any info,
