I recently switched from twm to fvwm 2.2.4. I'm using the FvwmWinList
module to replace the twm Icon Manager. In twm I used the SortIconManager
command, is there any way to get the FvwmWinList module to keep its list
sorted in a predictable way? It seems to default to adding windows at the
end of the list, although if I close the module and restart it they end up
in a pretty random order.

The WindowList command has an Alphabetic option, but I want a permanent
list of windows, not a temporary pop-up.

Last month I asked this question in the comp.windows.x.apps newsgroup, and
someone sent me the suggestion:

AddToFunc WL-Func "I" WindowList Alphabetic

I tried this but it didn't do anything AFAICT.
