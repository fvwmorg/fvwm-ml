To follow-up, I'm not going to look into this further, but I think your
crashing it by destroying a function you are currently calling (i.e. your
mouse binding is bound to DoWindowShade which calls a function which
destroys DoWindowShade -- what does the function return to :-) !). This
to me seems like a faulty rc setup (in the same vein as if you call a
function recursively).
