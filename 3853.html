> Sun workstations have help keys. Why all workstations don't is
> a mystery. The binding is built into FVWM. As Dominik said, ignore
> it for now, it will go away with the next release.

  you can remove it, it's in fvwm default configuration file, use strace
and search for open to find out where the file is, on my system (debian
unstable) it's:

/usr/share/fvwm/ConfigFvwmDefaults:Key Help R A Popup MenuFvwmRoot

  (at least I assume once you comment that out it will go away, it can
be also hardcoded in source, I didn't check for that)
