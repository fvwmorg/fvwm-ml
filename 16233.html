>
> The only thing I would like is for them to use actual named colors
> instead of #rrggbb type specifications.
>
I don't know about X11R6, but between X11R4 and X11R5 the list of named
colors changed significantly. (I know because here at work I run fvwm on
X11R4).

Here's a context diff showing the differences (+ -> only in X11R5,
- -> only in X11R4); you can see that most of the "numbered" color shades
(xxxx1, xxxx2 etc.) are not available in X11R4. It would be nice if we
could settle on a set of named colors which are available on all X
versions ;-)
