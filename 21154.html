Hello,

I'm using fvwm manager quite a long time (since 1.0.20 times) and I
still cannot get rid of a nasty problem: Sometimes, during havy word the
manager looses focus, it isn't possible to get active window with mouse
click. It's possible to get it through the win-list, but after clicking
another window it gets lost again. Neither restarting fvwm helps. The
only way to get things OK for a while is to quit fvwm and start again.
This problem appeared already in 1.0.x versions. I use version 2.0.46,
Xfree 3.3 and linux 2.0.30 now.
