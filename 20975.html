My X startup script is called .Xclients. I have always chosen my startup
programs by putting them in this file, and starting the window manager at
the end. It has always worked.

Now I am invoking fvwm2 with the command

exec fvwm2 -cmd "FvwmM4 -debug /usr/local.....

It works very well, except for one thing. The correct colors and buttons
(styles, actually) are not being applied to the windows that appeared
before fvwm2 started. Now if I take the .fvwm2rc file created by the
-debug option above, put it in my home directory, and change my fvwm2
invocation in .Xclients to

exec fvwm2

Those windows get the correct styles when fvwm2 starts.

Get it?

Now if I restart fvwm2 from the session where the correct styles were not
applied to the old windows, those styles are applied after the restart is
done.

Without putting the programs I want in my InitFunction can anybody tell me
how to get the correct styes to apply to existing windows when M4
preprocesses my startup file?
