[about changing the mouse1 ( mouse2 ? ) action in FvwmPager]

pleas make it

*FvwwmPagerMouse1Click any_function

and use SendInfo()

SendInfo(fd,fn,win);

fd is the pipe back to fvwm2,
fn is any fvwm2 command (read from .fvwm2rc)
win is the selected window.

If you want to make it really clean, you can make all the following
possibilities:

*FvwmPagerMouse[123](click|move|doubleclick) anyfunction
