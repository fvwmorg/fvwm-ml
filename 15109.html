Richard Guay said:
>
> I am still trying to learn how to do things on Fvwm2. It is a great
> desktop! My boss wants to be able to use the "Front" key on his Sparc
> Station 2 as it did work under olvwm (ie. if the window is in front, make
> it go to the back. If the window is in back, make it come to the front).
> Is there a way to toggle Raise/Lower commands?
>
> Richard A. Guay
> rag_at_asicint.com
> http://www.asicint.com


The "RaiseLower" command will do what you want.
I bind this command to Alt-F5. This is not as convenient as just plain F5,
but it leaves the function key free. To bind to plain F5, replace the 'M'
with an 'N'.

add to .fvwm2rc:
        Key F5 A M RaiseLower

While F5 is appropriate for my PC keyboard, it is probably *not* correct
for a Sparc keyboard. Who knows the correct keycode?
