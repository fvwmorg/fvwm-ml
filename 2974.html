If these keys do not generate KeyPress/KeyRelease events in xev, the
problem is either in X that does not recognize these keys, or in some X
application supposedly run in .xinitrc does something about these keys.

If you want to be sure that it's not an FVWM problem, create .xinitrc file
(or its equivalent) with one line "exec xterm -e xev" and rerun X.
