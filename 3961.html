> Dominik Vogt <dominik.vogt_at_gmx.de> writes:
>
> > BTW, what is the right way to disable icons but show icon titles?
> > I found no other way than specifying an illegal pixmap file. If I
> > have a default icon, I see no proper way to disable pixmaps for
> > certain applications.
>
> Although icon titles without icons work with FVWM,
> its not really supported in the sense that there is no clean way
> to turn it on. (I guess if you never define a default icon its on.)

I wrote a trivial hack for this in 1996, which allows:

  Icon NoPixmap

to turn off icons, even for motif apps that supply them (such as netscape).
It was in the code back them, but I switched to using FvwmIconMan a long
time ago, so I don't know if its still there in the latest. grep icons.c
for "NoPixmap".

There was no option to limit title width, but in those days people
generally set more reasonable titles on their web pages :-)
