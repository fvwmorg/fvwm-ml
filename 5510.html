On 08 Jul 2002 18:37:02 +0200, Johan Svedberg wrote:
>
> I have the following rows in my rc:
>
> # Sets Alt + Page Up/Page Down to run Maximize-Func.
> Key Prior A M Maximize-Func
> Key Next A M Maximize-Func
>
> and they work great, there is just this one thing; I like to use the
> keyboard alot and when I run Maximize-Func I need to touch the mouse a
> bit in order to make it work (I guess it has something to do with that
> the screen has to be redrawn or something). Anyway, I was just wondering
> if anyone had some small tip && trick to make this happend whitout the
> involvment of a mouse... :-)

Take a look at your Maximize-Func, it requires a mouse. Different
combinations of "C", "D", "M" or "H" require a mouse differently.

Write "Maximize" instead of "Maximize-Func" or use a function that only
has "I" commands if you want to bind it to a key.
