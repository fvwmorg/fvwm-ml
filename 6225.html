On Fri, Sep 13, 2002 at 01:58:33PM +0200, Klaus Zeitler wrote:
>
> When I press Alt-Tab a fvwm window list appears
> even though I have the following line in my .fvwm2rc file:
>
> Key Tab A M -
>
> How can I disable this key setting (FVWM version 2.4.3)?

In principle, that's the correct command. If it doesn't work, a
different binding must be somewhere in the config files. Look for
bindings with the tab key in your config file and the
ConfigFvwmDefaults that comes with the distribution. You have to
negate all bindings by issuing the same command to "-" as the
action.
