On 14 Jan 2001 21:02:43 +0100, philipp wrote:
>
> i would like to open a new xterm and have the mouse pointer placed above
> it immediatey. i am currently using a simple exec command triggered from
> a menu to open the xterm, with random and smart placement, which is
> fine, but i can't figure out how to move the pointer (causing the window
> to have focus and be raised)

Please see FVWM FAQ, question 3.18, which deals with the "I want this and
this when a new window pops up" problem. The command to warp the pointer
to a window is WarpToWindow, try it with "50 50" or "30p 10p" parameters.
