Yep, Gentoo Linux, All users who have the `perl` and `tcltk` USE flags
set when they install fvwm automatically get FvwmTabs for no extra
charge :)

USE flags are basically configuration options or settings (Gentoo
installs all applications from source, so we can make compile-time
features optional)

To guarantee they get FvwmTabs, a user might do this

# USE="perl tcltk" emerge fvwm
