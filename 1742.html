On Thu, Nov 30, 2000 at 03:28:37PM -0800, Abbas Amiri wrote:

> I have a SUN machine with OpenWindows window manger coming up by default.
> How can I change it so that it opens fvwm instead?

The easiest way is to set the OW_WINDOW_MANAGER environment variable from
olwm to fvwm2 or fvwm-themes-start if you use the fvwm-themes.
