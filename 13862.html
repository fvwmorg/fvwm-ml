On 19 Feb 2005 at 13:13:32 +0100, sastan wrote:

> i am knew to fvwm and very impressed about the
> configuration option. After played a while with it i
> tryed to swallow my lm-sensors like the clock example.
>
> Swallow FvwmScript-LMSensors `FvwmScript
> $[fvwm_script_path]/FvwmScript-LMSensors`
>
> FvwmScript-LMSensors:
> WindowTitle {FvwmScript-LMSensors}
> ...
>
> I tested the code in by replacing the relevant parts in
> my DigitalClock-script. It is ok. When i use the above
> swallow stuff nothing shows up.

You can check if your script works correctly by typing

FvwmScript $[fvwm_script_path]/FvwmScript-LMSensors

in FvwmConsole. If your scripts is shown as a window which
is not swallowed in the button bar you can use FvwmIdent to
unsure that the window name is set properly.

If your script starts up and the name is correct it should
work - actually ;-)
