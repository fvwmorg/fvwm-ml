> That reminds me, though...
> I guess I should go ahead and ask the mailing list the stupid question of
> How can I make this particular instance of Eterm appear with no titles?
> I want the other Eterms to have the normal decorations, but I'd like THIS
> instance to be NoTitle, NoHandles, etc.

Use Eterm's -title switch to uniquely identify the particular window, then use
a style line like:

Style "foo" NoTitle, Sticky, NoHandles, etc...

Line to specify that one term.

Have fun.
