ftp://ftp.cs.arizona.edu/people/bradym/FvwmIconMan/FvwmIconMan-0.9.4.tar.gz

Two major changes: fixed (hopefully) bug which caused the manager windows to
teleport around the window when changing pages, and added branch instructions
to the list of commands you can run in actions. Can't do backwards jumps
though, so no loops. With these commands you should be able to navigate the
manager windows quite nicely without the mouse.
