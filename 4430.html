On 2002-Mar-07 (Thu) at 13:54 (+0000), Ripalda-Marin, Miguel-Angel wrote:

>> I have noticed something strange (I think it's strange, but maybe not ;-). I
>> change the value of the LD_LIBRARY_PATH environment variable when entergin
>> with my user (in .bash_profile), but when I type startx to work with my
>> windowed tools, the LD_LIBRARY_PATH variable is cleared... I wonder whether
>> it is normal or not... have I done something wrong? Maybe it has nothing to
>> do with FVWM... apoligizes if it has nothing to do with FVWM...

Miguel,

   Try putting into your '.bash_profile' file the line:

      export LD_LIBRARY_PATH

   Also, check to make sure that your '.bashrc' file
   is not re-setting the 'LD_LIBRARY_PATH' variable.
