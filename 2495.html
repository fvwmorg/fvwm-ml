I think the -lXpm before -L/usr/local/xpm/lib is the problem.

Since you are using Solaris 8, try this:

  --x-libraries=/usr/openwin/lib:/usr/local/xpm/lib

> ocal/xpm/lib -lXpm -L/usr/openwin/lib -R/usr/openwin/l
> ib -lSM -lICE -lX11 -lsocket -lnsl 1>&5
> ld: fatal: library -lXpm: not found
> ld: fatal: File processing errors. No output written to conftest
