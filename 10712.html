On 07 Dec 2003 11:11:20 -0200, papa wrote:
>
> How do I get fvwm to run when my system is booted I have Motif 2.2
> installed and now gnome does not work any more so I want to use fvwm on
> my RedHat 9.0 OS.

  * install our rpms
  * create ~/.Xclients with one line that is either "exec fvwm"
    or "exec fvwm-themes-start"
  * chmod a+x ~/.Xclients

  * restart X
