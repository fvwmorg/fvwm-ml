Hello,


> > Some programs have more than two windows, like Gimp and
> > MPlayer (one for GUI, another the movie window).
> > Take MPlayer as an example, I want to decorate only the movie
> > window and not the main GUI window since it already has a
> > titlebar. How to do this? Thanks!
>
> Use FvwmIdent, it will show you the window Name, Class, and
> Resource.
>
> Fvwm style commands match on any of these 3 names.
> You will probably find that the windows differ on one of
> these names. Use that name to set a unique style for the window.

I had this problem too and I "solved" it adding each window in
the .fvwm2rc. Though I consider it is just a work around, and I
was wondering if there was some real solution, like applying a
style to all windows created by a specific application...

For instance, how would you deal with an application creating
windows whose names/etc. are decided at runtime?
