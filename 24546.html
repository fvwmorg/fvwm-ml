First make sure you have FvwmAnimate running:

ps -ef | grep FvwmAnimate

should give you a result like this:

xxxx> ps -ef | grep FvwmAnimate
    dane 11613 2075 0 09:33:13 ttyp0 0:00 grep FvwmAnimate
    dane 26666 26660 0 Jul 08 console 0:00 /usr/local/fvwm/2.3.6//FvwmAnimate 9 4 none 0 8

Also note that FvwmAnimate currently only animates desktop icons.
If you are using FvwmTaskbar, FvwmIconMan, or FvwmIconBox and have
"Style * NoIcon" set, you won't see any animation.

I can't predict what putting the "Save" command in the file is going to
do. (I've never tested it that way.)
