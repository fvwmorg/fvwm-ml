My personal favorite is auplay that is part of NAS, Network Audio
System. The whole package might be a bit too big for you, but, it has
one major advantage over rplay: if two sounds are played at almost the
same time, the sounds are added together so that you hear both. With
rplay, I think the second sound is blocked until the first one is done
playing. This is rather annoying if you use sounds a lot.

NAS runs on builds on linux without problems.
