> Does fvwmconsole have command substitution, like that of tcsh?
> I reckon that would be very useful especially as most of the modules are
> called Fvwm something
>

Yes it does. If you create ~/.inputrc, such as,

        Rubout: delete-char
        "\e[214z": beginning-of-line
        "\e[220z": end-of-line
        "\C-i": complete

it will bind tab key to complete command and such. But I don't know exactly
how to specify the directory to search files to complete file name and
extract basename by just configuring this .inputrc file.



It is much better off if you return the mail to fvwm mailist instead of
just to me. Like I said, I have very limited knowledge on other machines.
Also, I personally use FvwmConsoleC.pl but not GNU readline.
