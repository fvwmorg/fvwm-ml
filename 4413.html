On Wed, Mar 06, 2002 at 11:17:38AM +0100, Bob Vloon wrote:
> Hi all,
>
> I was wondering: is it possible to draw flat borders around menus? I've
> looked intensively into the docs, but it seems quite impossible ;)
> What I want: I've "flat windows", and this requires (yeah) a flat
> menu! So I got rid of the standard border, the 3D-look, but then,
> well, the menus are quite borderless.
> Am I overlooking something, and if not, could this be done in future
> versions of Fvwm?
> Next to that, I want to assign a color to the inner bevel myself, otherwise,
> my windows sometimes get this "borderless look" also. With two colors, this
> problem gets out of the way (being able to use different pixmaps for all the
> four borders would help also =)

The menu border can be controlled with a colour set. An example
from my configuration file (except the colours).

  MenuStyle * MenuColorset 3
  ...
  *FvwmThemeColorset 3 fg red, bg green, hi blue, sh yellow
  ...
  # run FvwmTheme before anything else is done
  ModuleSynchronous Timeout 5 FvwmTheme

Use the same shadow (sh) and hilight (hi) colours to get flat
menu borders. Unfortunately, menu separators use the same
colours and look a bit blocky without the 3d effect.

For more details, read the FvwmTheme man page.
