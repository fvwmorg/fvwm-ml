Agreed. Ideally I think the user should select 'Add' from the
FvwmTabs window when they want to add an existing window. When this
happens, FvwmTabs needs some way to ask FVWM to get the user to select
a window & have it pass back the window Id. That'd work nice - that
way you can have as many FvwmTabs running as you like.

I haven't worked out how to control FVWM (from a module) to select a
window with a mouse-click though - anyone?
