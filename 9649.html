Is anyone else having a problem with Mozilla Firebird 0.6.1 where the icon
doesn't appear when the window is minimized? I figure this is a glitch
with Firebird, so I added the path to where the Firebird icon was on my
system (it ships with Firebird) to my ImagePath, and I added the following
style line:

Style "Mozilla Firebird" Icon mozicon50.xpm

But that didn't work, so I tried:

Style "Mozilla Firebird" IconOverride, Icon mozicon50.xpm

And that didn't work either.

Anyone else seen this or have a fix?
