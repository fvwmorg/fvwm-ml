I was playing around adding various pixmaps to my Window button and
found a few bugs. One, the PixmapPath doesn't appear to be followed for
loading xpms into the buttons. I had to hard code the path to the
pixmaps to get them to load. Also, I was experimenting by adding new
pixmaps with the FvwmTalk command line module and I filled up my colour
map. The only way I could get any colours back was by quitting fvwm.
