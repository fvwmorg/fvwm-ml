->Hi,
->
->how can i use FvwmPager without a Label ??
->i'm using the pager as a ShallowModule in FvemButtons
->and the Label takes only some place.. i don't need it.

Put a line like this in your .fvwmrc file:

*FvwmPagerFont none

and the label will disappear.
