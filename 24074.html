On Mon, 26 Apr 1999, Sam Krishnabhat wrote:

> I am using fvwm2.2.
>
> Is it possible to place the cursor automatically on a window
> when it pops up, I mean to draw the FOCUS on that. For example,
> when I do netscape find, netscape will open a dialog box
> with title "Netscape: Find". I want to be able to give focus
> to this whenever it pops up, by automatically placing the
> cursor there. How can I do it ?
>
I'm not sure what netscape calls the find window off the top of my head.

As a matter of fact I'm not even sure if this is a GlobalOpts in 2.2 but
in 2.3.2 it's style "?netscape*" GrabFocus seems to work find for me.

This style command will make any netscape window grap focus when it comes
up.
