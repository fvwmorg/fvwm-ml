First,
I realy enjoy fvwm2. This is my first unix set up, it rocks...
I use fvwm2 with solaris7 x86 pent 266, 200mg ram, api xpert_at_play98 video card.
though the driver is for xpert_at_play or the 3d rage pro (i need 3d rage pro tubo)

1) netscape problem
I am having some problems though. Netscape comunicator 4.5 w/128 bit encryp
seems to crash the window manager occasionaly, mostly it seems when there is
graphicaly intense stuff, ie. www.gnome.org . When the crash occurs I just get
logged out of my session and brought back to the login screen.

I have tried to use netscape 4.08 (comunicator) and it crashed also. I don't
know if this is a problem with netscape or fvwm2. I have not been able to
reproduce the problem with CDE, but I don't use it much because I like fvwm2
much better.

2) Sometimes when I log out out fvwm2 and try to login in console mode my screen
gets filled with funky shapes and charachters which blink. the space each one
occupies is about 1.5cm x 0.5 cm. I can log in to the console, execute
commands, eventhough I can't see what I am typing because my screen is messed
up. I can exit and I will get brought back into sun's GUI login screen and
everything seems fine. I can loginto my window manager of my choice and code
away.

--I understand that this could be anumber of things
1st: I don't have the correct video driver
2cond: fvwm2 problem
3rd: sun problem
4th: netscape problem
5th: my problem
--not listed in any logical order.

--My fvwm2 set up is pretty simple. I just have a solid backround, a 3x3
desktop the pager to manage the desktop and the button box to swallow the pager
and a xclock.
