For what it's worth, I reported a but to Jarl after downloading the
latest version at his site. I assume that's what's in 2.0.42, but
haven't had time to upgrade to that just yet due to other problems
keeping me away from fun stuff. Anyway, I found that if I specified
*FvwmButtonsFont None, it would dump core. I don't know if would
happen with a different font spec or if I specified the font in a
container. I just removed the line and it works OK for me now. I
figured out where it was dumping with gdb but haven't yet had time to
come up with a fix. Unfortunately I don't remember any details other
than it was an attempted NULL dereference. Anyway, if you've got a
*FvwmButtonsFont line, try removing it and see if you get any further.
