Hi folks,

I've come across a problem with Netscape. I'm using fvwm2.2, Solaris 7 on
a Sun Ultra 1 and in my configuration file I had

Style "Netscape" StartsOnPage 3 0, SkipMapping

This worked fine with Netscape v3.x, but we have recently got Communicator
4.61. Here, the SkipMapping seems to kill the StartsOnPage and it always
starts on the current page. If I take SkipMapping out, then the
StartsOnPage works, but I can't stop the cursor from warping to every
instance of Netscape. So anyway, I'm not sure how much of this has to do
with fvwm, because it worked with the old version of Netscape that we had,
but does anyone have any ideas?

ferg

ps. this problem also occurs with mxaudio, and perhaps others....
