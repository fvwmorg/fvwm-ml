On 12/06/04 05:21 PM, Mathieu Jouan sat at the `puter and typed:
> May it be possible to consider the following functionality, create
> a mapping key function to launch main program (the most used) ?
> exemple in .fvwm2rc:
>
> map <F1> /usr/bin/my_browser_web

Already there, You're looking for the Key command:

Back when I used the MS Internet keyboard, this tied the HomePage button
directly to Netscape:

Key XF86HomePage A A Exec /usr/local/bin/netscape

Basically, regardless of where the focus is or what modifiers apply,
that key would kick off Netscape.

Nowadays, I'd probably do something different, like using modifiers to
cause that key to kick off different browsers, like Firefox, Mozilla,
Netscape, Galeon . . .
