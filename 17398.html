I have the problem with Linux too. Since 2.0.39 i believe. As posted
earlier i have made a minimal .fvwm2rc that shows the problem. It does
nothing but start FvwmPager. I found out that the bug only shows up when
i use a preprocessor.

Here is the minimal .fvwm2rc:

# This enables paging when called with:
# fvwm2 -f "Read /home/stefh/test.fvwm2rc"
# But *not* when preprocessed:
# fvwm2 -f "FvwmM4 /home/stefh/test.fvwm2rc"
# fvwm2 -f "FvwmCpp /home/stefh/test.fvwm2rc"

DeskTopSize 2x2

AddToFunc "InitFunction" "I" Module FvwmPager 0 0
