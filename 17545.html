> Despite not having
> SmartPlacement and ActivePlacement defined and having

> Style "*" NoPPosition

> All my windows are getting placed automatically instead of letting me
> place them myself.

> I can't find any documentation that this has changed so any ideas
> friends?

Change your Style command to be:
        Style "*" NoPPosition, ActivePlacement, DumbPlacement

The default placement styles are now SmartPlacement and RandomPlacement.
 I think I ended up looking at the code to figure this out for sure.
The help text isn't real clear on default values in a couple of places
and this maybe one of them. I believe Chuck has been updating the help
for 2.0.
