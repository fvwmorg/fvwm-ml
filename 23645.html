Sam Krishnabhat wrote:
>
> Hi,
>
> I did read somewhere that Taskbar does not work properly with AutoHide turned
> on. But, for me autohide does not work at all. Am I doing anything wrong here ?
> I have the latest code downloaded from the CVS tree.

It works for me (using your config lines). What part of Authohide
doesn't work for you?

The problems with AutoHide were that it was impossible to put the
pointer on the FvwmTaskBar window with a setup that had more than one
virtual page. The thin transparent windows around the edge of the screen
(the pan-frames) got in the way. The fix was to make autohide leave two
pixels visible and to use the "EdgeThickness 1" command at the start of
your .fvwm2rc. Some X-servers have a bug that prevents page scrolling
with a pan-frame of one pixel.
