And, not only has tkgoodstuff's developer given up, the mailing lists have
gone too. (I'm not trying to blame the developer here, it's perfectly
reasonable to do this---it's just annoying!) Any chance of hpc.uh.edu hosting
a list? Perhaps tkgoodstuff being an official contributed fvwm module, so
development can continue by members of this list?

I've got it working much as I want, but there are some annoyances.

There doesn't seem to be a way of reordering items in preferences. For
example, if I decide I want the clock to go after the pager, instead of
before. Not too bad, since I always have my trusty text editor, but I ought to
be able to drag it.

I'd quite like to use the WindowList client, since it looks more fun than
fvwm's, but I haven't found a way of doing it nicely yet. I don't want it in
the main tkgoodstuff, because I've got too much else there. A button which
popped up a client would be OK, but that doesn't seem to work (I just get an
empty window); perhaps I've just got the syntax wrong?

Similarly, a dedicated tkgoodstuff would be OK (possibly better), but I can't
seem to persuade it to work properly. If I have it auto-hidden on the right
hand side of the screen, it's so narrow that I can't read much of the window
names, and if I have it at the top of the screen, then it's so thin that it
only gives one line of windows. If I explicitly resize it to about three
lines, then it looks great; so how on earth can I get it this size by default?
 None of the preference options seem to do the right thing: the WindowLists's
geometry doesn't seem to do anything, and the main tkgoodstuff geometry is
ignored, since I want it in edge mode. *sigh* So what *do* I need to set?
