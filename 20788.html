On Mon, 25 Aug 1997, Stephen Riehm wrote:

> Hi all,
>
> Firstly, I don't use / like desktops, because you can't jump between
> them with the mouse AND the keyboard - thus I only have one desk with
> 6 screens. When I start Netscape, I normally find an empty screen,
> start netscape, and then have to wait the 5 hours that it takes
> netscape to get going. What would be really neat is to have fvwm
> remember that I started netscape (or any other ap for that matter) on
> the screen I was in when I selected the menu item (effectively running
> a function),

I agree it is annoying the first time you start Netscape, but I actually
take advantage of this slowness when I start an additional Netscape
window, because I have time to switch to a different page before it
appears.

If your suggestion were in force, any additional Netscape windows would
always appear on the same screen as the first.

What I'd like more is a way to specify what page to start an app on, not
just which desk.
