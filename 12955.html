On Tue, Jul 27, 2004 at 10:11:05PM +0200, Anders Persson wrote:
> I have now ripped the last hair of my head...
>
> How do I set up my files to NOT show the title/label on the active
> (currently viewed) desktop? I use fvwm 2.5.10. I had a $HOME/.fvwm2rc but
> I removed that when I saw that during the installation of fvwm a simular
> file was created but in a fvwm-directory, eg. $HOME/.fvwm/.fvwm2rc.

Assuming that you mean the desktop labels in FvwmPager:

  *FvwmPager: Font none

> Maybe this it set in another file? If I have messed up my all my files (I
> have alot ot themes installed), what text can I search for?

fvwm looks for $HOME/.fvwm/.fvwm2rc, but you can force using a
specific file with the -f option:

  $ fvwm -f /foo/bar/myfvwmwrc
