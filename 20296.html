Quoc Duong wrote:
>
> Hi all,
>
> I would like to know if it's possible to configure the button style so that
> each APPLICATION window will have specific button style on it.
>
> i.e. button style is not the same for all windows.

You should use the "Recapture" Command after changing the styles.
Then all applications will get the latest style commands. Try

AddToFunc My_function "I" Style [$0] BorderWidth 1, HandleWidth 1
                            "I" Recapture

that will work. Also, somewhere in the in the fvwm2 distribution you
will find a file called "decor_examples" which explains many jet
undocumented features of fvwm2. Might be an answer for your first
posting "FVWM: specific button style for each window ?" as well.
