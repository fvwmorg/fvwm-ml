Hi everybody,
In FVWM2, is there any way to restrict the size of all windows produced by
any application to a certain size and position? You'll tell me "use a
geometry setting." Well, certainly geometry works but only for the initial window.
What about all the other windows further produced by an application? My card
supports only 640x480 and if I, for example, use Netscape in Linux ( with
any window manager), the initial window obeys to the geometry I specify, but
the rest are somtimes so huge that they don't fit in my house:-) Same about
Word Perfect. Same about other stuff. In twm there is an option called
MaxWIndowSize, but what about FVWM? I've also tried KDE and I had the same
problem. It's especially annoying when I can't see "O.K" "Cancel" buttons at
the bottom of a window because it's too huge or because the task bar covers
that portion. You'll tell me "Use autohide", but I don't want to use
autohide and I don't want to use virtual screens. I just want my windows to
fit in one screen above the task bar.
Thanx,
    Sergei
