Fvwm2 will terminate if I issue the following command several times
from FvwmTalk or FvwmConsole:

        Style ttt UseStyle xterm

The first time the command is run there is no problem.
Sometimes I can enter it a couple of times but usually on the third or
forth attempt fvwm2 just dies.

This is repeatable with both 2.3.11 and 2.3.12 under linux.

In order to rule out something funny with my .fvwm2rc file I replaced it
with a
one line file as follows:
 #########################################
 # .fvwm2rc
 Style xterm ForeColor black
 #########################################
I launched FvwmConsole from the default mouse menu and issued
the same command (Style ttt UseStyle xterm).
After about 4 tries, fvwm2 hung again.

This may seem like a minor problem but I run into the same issue with
more complex styles and decors.
