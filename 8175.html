Hello,

I've just installed v2.5.6 and I'm seeing some strange things with
MPlayer. Not sure whether this is actually FVWM's fault or MPlayer's,
but I wanted to see if anyone else is seeing a problem.

I have the following lines in my styles files:
Style MPlayer StaysOnTop, StartsOnScreen 1
Style "*MPlayer - Video*" NoTitle, StaysOnBottom

Prior to this version this had the desired effect of keeping the video
on the background layer with the controls on the top, but this doesn't
seem to work any more, they both move layers as if those lines were not
there.

Does anyone have any idea why this might be?
