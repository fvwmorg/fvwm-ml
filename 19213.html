It is strange, and I don't know where the problem comes from: I can
select almost any window I like with the function GoThere, except for
FvwmIconMan.

,-----
| AddToFunc GoThere "I" Iconify -1
| + "I" WarpToWindow 50 50
| + "I" Focus
`-----

I use it like this:

,-----
| Key i A 4 Next [--Mail--] Function GoThere
| Key y A 4 Next [CurrentDesk FvwmIconMan] Function GoThere
`-----

I have more lines like this, and all of them work (including others
with "CurrentDesk"), *except* the FvwmIconMan line. When I hit Alt-y,
nothing happens.

What's the story?
