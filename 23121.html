The new page has only been publicly available for about a week, the same
time as the 2.1.4 beta announcement.

Things are in transition, so there are still issues. I'm not sure if
the plan is to leave the old page as-is until the official 2.2 release
is available, or whether we were just waiting to make sure the new page
didn't fall down before completely switching, with a pointer or
something at the old location.

The best thing you can do to help is continue to note the problems and
report them to the fvwm-workers_at_hpc.uh.edu list. If you want to help
more you can edit the files in the docs subdirectory of the latest beta
(which are the same things on the new web site) and submit patches to
the workers list.

Note we're aware already that the Links page on the new site is very
badly out of date :)
