I just completed upgrading to 2.0.44 from 2.0.43. FvwmAuto only works
once.

here is the relevant section of .fvwm2rc:

DeskTopSize 2x2
MenuStyle Grey80 #505050 Grey50 -adobe-times-bold-r-*-*-14-*-*-*-*-*-*-*
fvwm
ClickTime 750
OpaqueMoveSize 100
FvwmAuto 750
ColormapFocus FollowsMouse

When first started or restarted, the module seems to work. I restarted
with several different argument values and got a reasonable response.
Howver once the module pops up a window the timout becomes very short,
or the module unloads (how do I tell?)

I diffed the code from FvwmAuto 2.0.44 with the version from 2.0.43, but
the code is way over my head. I tried to recompile the the new FvwmAuto
with the old libfvwm2.a and the new FvwmAuto with the old library, but
neither would compile.

The .fvwm2rc worked fine under 2.0.43 for several months.
