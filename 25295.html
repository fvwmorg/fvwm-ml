> Hello
>
> Does anybody know how to disable the Builtin Exit FVWM function?
> The built in menu appears when you click mouse button #1 in an open
> space (where a window does not exsist). Any help is appreciated.

Mouse 1 R N -

but actually, the built-in menu is bound to all mouse buttons, so
if you really wanted to remove all mouse bindings for it:

Mouse 0 R N -
