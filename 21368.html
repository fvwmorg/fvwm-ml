If I remember wright solaris on install, by default, wont let you start up
your own X server you have to let xdm do it for you

so what you do is change the line in your .xinitrc
that probably starts up openwindows to fvwm instead

Then when you are logining in the default is to cde you just switch to
openwindows and it runs xinit on your copy of .xinitrc

Hope this helps, the other option is to change the permissions on
/dev/console and a couple of others (as you fix one the next one pops up)
and then anyone can start up an X server from the console
