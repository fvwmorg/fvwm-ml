I've installed fvwm 1.23b on a x86 box running Solaris 2.5. All
seems to be sweetness and light except function keys F1-F12, which
no longer produce the actions found on my AIX 4.1.4 machine running
the same fvwm revision.

The keys are assigned in my .fvwmrc as follows. The functions
referenced, e.g. RaiseLower, work as expected when called via
other pop-up menus.

# Keyboard accelerators
Key F1 A M Popup "Window Ops"
Key F2 A M Popup "Utilities"
Key F3 A M RaiseLower
Key F4 A M Destroy
Key F5 A M CirculateUp
Key F6 A M CirculateDown
Key F7 A M Move
Key F8 A M Resize
Key F9 A M Iconify
Key F10 A M Maximize

There are no errors or warning during the build and so far as I can
see keysymdef.h is correct.


Any suggestions will be gladly accepted.

Lance
