This is a new builtin function I wrote back for fvwm-2.0.43. I modified it so
it would work with the new fvwms and thought I would re-submit it. This
patches applies to fvwm-2.1.6.

Here is the man page entry for the function:

       Replace [ Anim ]
              Causes the current window's position to be re-com-
              puted using the initial window placement logic.
              Basically, the window is moved to where it would
              have been if it were a new window that had just
              appeared. Most useful with Smart or Clever
              (ReallySmart) placement. If the optional argument
              Anim is specified, an animated move is used to
              place the window in its new position.
