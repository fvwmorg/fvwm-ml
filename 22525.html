        If it's Fvwm version 1, the command is

              DeskTopScale <divisor>

and for version 2 it's
 
             *FvwmPagerDeskTopScale <divisor>

where the reduction factor is 1/divisor. In other words, to show
things in the pager at 1/36 the size of the actual desktop, you'd
specify "DeskTopScale 36".

        Specifying the width and height in the geometry command
is rather iffy, and the man page recommends against it if you want
to keep the aspect ratio undisturbed. This is because the pager has
no fixed width - it shows as many desktops as you specify in the
pager invocation. (I'm not sure you could specify the width+height
in FvwmPagerGeometry in version 1.)
