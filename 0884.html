On Wed, May 24, 2000 at 11:33:37AM -0700, Arya Behzad wrote:
> Hi Dominik.
>
> THanks again for your help below. Just one problem...
> WindowList (CurrentPage) works just fine
> WindowList (xterm) also works fine
>
> However,
> WindowList (xterm, CurrentPage)
> does not work (returns an empty list). Am I doing something wrong?

It's because of the comma (',') after the string "xterm" for some
obscure reason it is interpreted as part of the name. This does
work:

  WindowList (xterm CurrentPage)
                  ^^

> Also, where is the most uptodate man pages for these features. The man
> pages on the fvwm web site does not talk about these as I believe it is for
> an older version of fvwm.

The man page you have should always match the executable, i.e. if
something is not documented it is not there. Well, I know that in
the past some features were not documented at all. The man page on
the web site always refers to the latest stable release (i.e. 2.2.4
at the moment). The very latest code and man page is in the current
beta release (2.3.17) or development snapshot. You can find the
instructions how to download either on our 'downloads' page.
