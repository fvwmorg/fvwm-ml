Robert Kim writes:

> But I get the following error afterwards:
> ClientMsg.c:14: X11/Xlib.h: No such file or directory
> ClientMsg.c:15: X11/Xatom.h: No such file or directory
> ClientMsg.c:16: X11/Xutil.h: No such file or directory
> Am I missing some library? Thanks for your help.

Try "locate Xatom.h". Normally those files are in directory
"/usr/X11R6/include/X11/". If you do not have those files, you have
not probably installed package for X-developement. If you are using
Debian, the needed package is xlib6g-dev. Other distributions
probably have a similar package.
