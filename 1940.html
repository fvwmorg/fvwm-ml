Its not automatic.
After you've installed 2.3.x, you can pull up the documentation
by doing:

man fvwm24_convert

Basically you should do:

mkdir $HOME/.fvwm
fvwm24_convert

It reads $HOME/.fvwm2rc and writes $HOME/.fvwm/.fvwm2rc.

This utility assumes you were previously using 2.2.x and changes
the 2.2.x commands to 2.4.x syntax.

If you aren't starting from 2.2.x, I don't know how useful it will
be to you. You could just read thru the man page, the man page
describes the changes it will make.
