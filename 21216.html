> > It's also possible to have a sound played when you receive new mail, but I

I figured out already :)

> I would like to do this, if anyone knows how let me know.

If you use the program called "mailbox", just add a parameter
-B "command..." (the command string needs to be quoted !).
Then, take a sound player for your command. So, you'd get something like:
mailbox -B "/usr/sbin/sfplay /home/`whoami`/mail.wav"

Note that mailbox is quite sensitive about the order in which parameters are given.
For more info, refer to the man pages.
