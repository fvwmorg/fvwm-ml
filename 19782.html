Hi,

I've got my FvwmButtons set up so that it swallows xbiff in one
button. I'd like to have something happen when I hit a button in the
xbiff, so I used the following config line:

,-----
| # continuation line for easier reading only
| *FvwmButtons(1x1, Action "Exec exec xfrom", \
| Swallow XBiff "Exec exec xbiff -geometry =48x48+0+0")
`-----

But, alas, it doesn't work: a mouse click is passed on to the xbiff.

I would also be happy if you told me of a program that can produce a
list of messages when you click on it. I know of xmultibiff, but it
only shows the number of messages, no nice icon like xbiff does.

Also, if you had an xbiff that could ignore certain mails (such as
those from the Debian mailing list) I'd be very glad.
