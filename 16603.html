I am trying to set up some generic styles so I can use UseStyle for
many windows. It seems to work except for Color.

For example:

Style "Highlight" Color Black/Red

Style "Window" UseStyle Highlight

Window will come out with color Black/gray, even though I don't use
Black/Gray anywhere. If I change it to:

Style "Window" Color Black/Red

it works fine. I believe there is a bug (or undocumented feature) in
UseStyle.
