Using the following .fvwm2rc configuration, all the keys perform as
expected except for the Left Arrow. No leftward motion whatsoever
from anywhere is possible. ???

Key Left A C3 CursorMove -10 +0
Key Right A C3 CursorMove +10 +0
Key Up A C3 CursorMove +0 -10
Key Down A C3 CursorMove +0 +10

They all function correctly with just the 3 modifier, or just the C
modifier, of if I use SC instead of C3. And, it remains broken if I
put the Key Right configuration above the Key Left, for example. The
3 represents a Mode_switch placed at the Caps_Lock key, in attempt to
replicate in Xfree86 the GNU/linux Hypermap.m4 keyboard configuration.
Interesting possibilities there are, such a big and normally underused
key that Caps_lock!

System is GNU/Linux i586 2.0.36, Xfree86 version 3.3.2.3a-7.deb, S3
server.
