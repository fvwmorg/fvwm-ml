For those who wants to take a look at me rc file
please download this

http://student.uci.agh.edu.pl/~artursc/artist-xp-theme.tar.gz

This file contains also graphics for window decorations and
taskbar.
I use Fvwm 2.5.7
I compiled it from sources.
I don't use fvwm-themes because I like everything to be done simply
without superflous programs. Whole my configuration depends
only on .fvwm2rc file and graphic files of course.

This creation was inspired by screenshot of Mikhael Goikhman's desktop.
I used Fvwm four years ago for few days :) but I didn't like it
and removed it. I installed Window Maker. Not long ago I visited
fvwm.org for the first time and I was nicely surprized about Fvwm's
possibilities ... soon I became large Fvwm's fan and here you can take
a look at my first creation.

I think my rc file may have some bugs. I'm not quite sure about
configuration of FvwmButtons, but first of all ... this configuration
seems to work ok ... second, I will tune it soon.
