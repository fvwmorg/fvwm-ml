>
> > On 03 Dec 2003 02:04:32 +0000, Ukpong Ukpong wrote:
> > >
> > > I have written a function to make any window stayOnTop, but i want
> > > the video out window from MPlayer to always StayOnTop. How do i apply this
> > > functionality to mplayer by default ......
> > >
> > >
> > > AddToFunc ToggleOnTop
> > > + "I" Layer 0 6
> >
> > Either:
> >
> > Style MPlayer Layer 6
> >
> > or the same:
> >
> > Style Mplayer StaysOnTop
>
> is it also possible make the current active window stay on top using
> a keybinding ?

  Key T A CM Layer 0 6 # Ctrl-Alt-T to put the current window on top
  Key P A CM Layer 0 4 # Ctrl-Alt-P to restore the default layer
  Key B A CM Layer 0 2 # Ctrl-Alt-B to put the current window on bottom
