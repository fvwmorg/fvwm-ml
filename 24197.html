 When I set up Exceed, I didn't set it up to use CDE, even though I had it
available. I just set Exceed up in passive mode, then wrote an xstart file
to launch the window manager itself, with an appropriate -display setting.
(Well, I wrote a shell script wrapper to set some environment variables
first.) The menus were all generated from the window manager, so it all
worked pretty well. The users essentially get a unix "desktop in a window"
on thier NT stations. This concept was familar to them, having used WinDD
and WTS on thier old unix stations before.

 One caveate is that you lose access to /dev/console. We had some old
scripts that tried to write to there and had to be re-written.
