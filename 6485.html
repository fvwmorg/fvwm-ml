> * What module do you use to swallow these applications?
> * Are you sure you have xeyes and asclock installed? Can you

Yes. I see xeyes and xclock briefly on the screen before they disappear.
asclock finds a home in the upper left corner of my screen.


> * What config lines do you have to swallow these modules?

*FvwmButtonsFont 6x13
*FvwmButtonsFore Black
*FvwmButtonsBack grey67
*FvwmButtonsGeometry +300+30
*FvwmButtonsRows 2
*FvwmButtons (2x2 Frame 0 Swallow(UseOld) "FvwmPager" "Module FvwmPager 0 0")
*FvwmButtons (3x2 Swallow "xosview" `Exec xosview`)
*FvwmButtons - - Swallow "FvwmXeyes" Exec xeyes -name "FvwmXeyes" -geometry +0+0
-bg grey67 &
*FvwmButtons - - Swallow "FvwmXclock" Exec xclock -name "FvwmXclock" -update 1
-geometry +0+0 -padding 1 -bg grey67 &
*FvwmButtons - - Swallow "Fvwmasclock" Exec asclock "Fvwmasclock" -geometry
+0+0 -padding 1 -bg grey67 &
*FvwmButtons Fvwm2 window3d.xpm Function barthiswmpopupfunc
*FvwmButtons Help question2_3d.xpm Function barrcpopupfunc
*FvwmButtons Internet Terminal.xpm Function barinetpopupfunc
*FvwmButtons Graphics palette3_3d.xpm Function bargraphicspopupfunc
*FvwmButtons Games calvin_3d.xpm Function bargamespopupfunc
*FvwmButtons Tools box_full_3d.xpm Function bartoolspopupfunc
*FvwmButtons Shells Shell.xpm Function barshellspopupfunc
*FvwmButtons Files drawers_3d.xpm Function barfilesspopupfunc
*FvwmButtons Documents butterfly2_3d.xpm Function bardocspopupfunc
