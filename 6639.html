The point of xvirtualroot is to avoid having to use FvwmBacker. Start
one xvirtualroot window per desktop at startup, with the appropriate
-xrm '*Desk: n' parameter and the appropriate FVWM style settings, and
then the background will be different for each desktop automatically.

Or, just start one xvirtualroot window for all desktops, if you want
the same background on all desktops.

Use "xv -root ..." or equivilent to change the background image if
desired.

Since xvirtualroot is just another X window, the background image that
it provides appears much more quickly than a FvwmBacker-invoked
background when changing desktops. Typically FvwmBacker causes the
image to be reread/recalculated/redrawn on each desktop change, which
makes for a less-than-smooth transition.
