Adrian Llahana <ladrian_at_cs.sunysb.edu> wrote:
>
> For example, when I need to press ALT-TAB keys, to focus and raise another
> window, but it just focusses.

I had a lot of trouble with this, too, because it "just worked" in
fvwm1. One solution is to use FvwmAuto to raise the focused window.
The other is this one that I like better:

    Key Tab A M Function NextWindow
    Key Tab A SM Function PrevWindow

    AddToFunc NextWindow "I" Next (!Iconic CurrentDesk) Focus
    + "I" Current Raise

    AddToFunc PrevWindow "I" Prev (!Iconic CurrentDesk) Focus
    + "I" Current Raise
