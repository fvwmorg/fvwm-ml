It's possible to scale icons manually:

You can extract EWMH icons from an app by using:

http://users.tpg.com.au/users/scottie7/tmp/ewmhIcon.pl

Resize them to whatever size you'd like using gimp, ImageMagik, etc.
& then use EWMHMiniIconOverride as Liu suggested.

But, there is no way to get FVWM to do this for you automatically
(at the moment).
