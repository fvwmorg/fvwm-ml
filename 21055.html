My idea is to be able to have a "balloon" popup with the name of the
tool or action this button would activate when it received focus.

This idea came from the way I have my FvwmButtons set up. I have one
button split into three rows of three columns. These buttons are very
small and can just fit a mini pixmap so no title will fit.

I have read about a "balloon" popup for the pager, would something like
this be possible for the FvwmButtons module?
