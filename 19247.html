>>>>> Sharad Gaur writes:

  Sharad> Q.1) How can I add a single application to all the desks??
  Sharad> (Like FWM Pager is there on all the desks)

Make it sticky. There is a function for that. RTFM and search for
"sticky".

  Sharad> Q.2) When ever I click mouse on any of the windows it is
  Sharad> brought to the top but I have to click again to get it
  Sharad> in context?? What I want is that as I move my mouse on
  Sharad> a xterm it should become active and my keyboard input
  Sharad> should go to that xterm. Rightnow I have click once to
  Sharad> make that xterm active.

RTFM and search for ClickToFocus and for FocusFollowsMouse and for
SloppyFocus. I use SloppyFocus and I'm very happy with it.
