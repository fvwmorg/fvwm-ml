On Mon, Oct 11, 2004 at 03:27:22PM +0800, David Sun wrote:
> Hi, all
> I am new to fvwm, sorry if I pose a old question here.
>
> I am using fvwm 2.4.18, and when I running some application in
> fullscreen mode, the taskbar becomes unavailable.
>
> My question is, is it possible to wake up the taskbar when it's covered
> by some other fullscreen applications? say something like
> autohide-autoshowup, but the simple autohide option of FvwmTaskBar does
> not work here because the taskbar becomes completely invisible. or some
> key binding?

I recommend to use a key binding:

  key f1 a n All (FvwmTaskBar) Layer 0 99

puts the TaskBar above all other windows, and

  key f2 a n All (FvwmTaskBar) Layer 0 4

puts it back where is belongs (upon pressing f1/f2).
