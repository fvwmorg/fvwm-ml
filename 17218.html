>>>>> "EK" == Elmar Kurgpold <ekurgpol_at_WR.BBNPLANET.COM> wrote:

EK> Does anyone know of a way I can change an icon on the fly? Specifically,
EK> I've got a shell script that does certain things depending upon how many
EK> new mail messages I have, and I'd like to change the icon on the xterm
EK> which my mail program is running under so the cute little flag on the
EK> mailbox flips up when there is new mail. It just looks wrong otherwise.

I think you are looking for xmailbox. xmailbox's icon will change when
new mail comes (can even do some animation and sound) and it will open
an user-defined mail program when you click on it.

Hope this helps.
