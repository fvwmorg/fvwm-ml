>>>>> Mick Morrow (MCSC) writes:

  Mick> I'm trying to install fvwm for the first time - but I'm not
  Mick> sysadmin or root. I've been through th FAQ's but I couldn't
  Mick> find any answers that look a basic as my problems !

First, did you run "xmkmf" before doing "make install"?

Second, have you set the OPENWINHOME environment variable (to
/usr/openwin, I guess)? Seems that your Makefile includes directories
to search that do not contain the /usr/openwin prefix. For example,

  Mick> mv Makefile Makefile.bak
  Mick> imake -I/usr/lib/X11/config
  Mick> makedepend -s "# DO NOT DELETE" -- -I. -I/libs -I/include
                                                       ^^^^^^^^^^

This should probably be "-I/usr/openwin/include".


  Mick> -DSVR4 -DSYSV -DSHAPE -DACTIVEDOWN_BTNS -DINACTIVE_BTNS
  Mick> -DMINI_ICONS -DVECTOR_BUTTONS Mick> -DPIXMAP_BUTTONS
  Mick> -DGRADIENT_BUTTONS -DMULTISTYLE -DEXTENDED_TITLESTYLE Mick>
  Mick> -DBORDERSTYLE -DUSEDECOR -DWINDOWSHADE -DXPM -- ClientMsg.c
  Mick> GetFont.c Grab.c Mick> Module.c Parse.c Picture.c Strings.c
  Mick> System.c envvar.c safemalloc.c Mick> sleep.c wild.c
