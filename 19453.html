> Heres the error message I'm stuck on.
>
> make: Fatal error: Don't know how to make target `../libs/libfvwm2.a'
> Current working directory /home/jamesl/fvwm/fvwm-2.0.45/fvwm
> *** Error code 1
> make: Fatal error: Command failed for target `install'

I have a similar setup, hope I can help.

It doesn't look like it's trying to install the lib, its in the fvwm
subdirectory trying to install fvwm. Before it can do this, it knows
that the library must already be built in a parallel directory:
../lib/libfvwm2.a.

Check that the lib is there. If not, retry your make from the top
level directory, and make sure that the make in the lib directory is
working. (Ie. creating libfvwm2.a.)
