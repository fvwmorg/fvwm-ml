Justin Ashworth wrote:
> How would I create a key binding to make ALT-TAB cycle through the
> windows on the current desktop?
> Any help will be greatly appreciated...thanks!

Put a line like this in your .fvwm2rc (I assume you're using fvwm 2.xx here):

Key Tab A M Next [*] Focus

In fvwm 1.xx, put a line like this in your .fvwmrc:

Key Tab A M CirculateUp
