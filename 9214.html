Hi folks,
I just installed FVWM-2.4.16 after using FVWM-1.2.4 since .... oh,
about the late Neolithic. (Well, OK, since 1993.) I have settings that
are intended to produce the same visual results in both -- same border
width, same colors, etc. My .fvwmrc and .fvwm2rc both say:

Style "*" BorderWidth 5, HandleWidth 5, Color grey80/grey20,Icon unknown1.xpm

Inactive windows do indeed look just about as close to exactly the same
as I can tell without being able to run both versions at once, but
active windows ... well .... the best I can describe it is that the
window border and decorations look "blurry".

(See the attached gif file.)

After comparing the two closely, it looks to me as though while fvwm1
and fvwm2 are rendering the window decorations at the same specified
width, fvwm2 is rendering the raised shading very differently.

Is there any way to configure this? I like the way fvwm1 renders the
border. fvwm2's rendering makes me squint trying to bring it into
focus.
