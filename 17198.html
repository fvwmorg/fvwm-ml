Hello,

I'm relatively new to the list & have been lurking about for a while.
But, it just so happens I've been icon-hunting myself.

I've been ripping off icons left and right. :-) Before trying fvwm
2.0.42, I tried fvwm95, but decided it was too little of a change to
bother right now. But, I like their mini-icons for menus.

It can be found at: ftp://mitac11.uia.ac.be/pub/fvwm95-2.0.41f.tgz
or on the web site at: http://ltiwww.epfl.ch/~barth/fvwm95.html

I've also played around with AfterStep, and I'm using quite a few icons
snagged from there. The web page is located at:

  http://woodward-52.rh.uchicago.edu/~frank/afterstep/

Although, I haven't been able to get through lately. I seem to recall
there's a few icons available through the web page and a link to more
photo-like icons.

I've also done some icon-hunting through the Yahoo icon page. Although
most of the emphasis is on WWW-oriented stuff, there's some interesting
things to be found if you're persistent:

  http://www.yahoo.com/Computers_and_Internet/Internet/World_Wide_Web/
  Programming/Icons/

One day I'd like to compile all of my favorites with proper credits and
offer them for distribution. If/when I do that, I'll post here.
