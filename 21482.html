shendrix_at_gypsy.atlantic.infi.net said:
> Still, I wish something like that existed... maybe something of a
> theme-editor for X that understand the differenent files and how they
> related. I have seen anything written about this in any kind of
> serious detail.

RedHat 5.0 (and possibly earlier versions, although I don't think it was in
4.1) has wmconfig, which seems to be somewhat along those lines. I haven't
really tried it yet, but the documentation seems to suggest that you can write
one file defining menus, icons and so on, and get it translated for a number
of different window managers. AnotherStep (another RedHat thing), apparently
uses this, and lets users change between a couple of styles of desktop.

As I say, I don't know how much these do, but they're probably worth a look.
Both are available in source from a suitable RedHat mirror (see
<URL:http://www.redhat.com/mirrors.html>).
