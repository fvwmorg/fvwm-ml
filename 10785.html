Yes cool screenshot. I am happy to see a transparent theme in fvwm-themes.
I've noted that Xft fonts are used too, I suggest you use Xft font in
the xp panel too :o).

For those interested by Translucency I've uploaded a patch

http://fvwm-themes.sourceforge.net/patch/fvwm-mettise-2.5.8.patch.bz2

against 2.5.8. This patch gives Translucent menus (for an usual X
server), translucent menu and windows with the XDirectFB X server and
also some stuff I need. The patch should be applied in the 2.5.8 as
patch -p0 < the_patch. Then, read the README.PATCH ...
