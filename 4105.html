Hi,
        We have an X-application which has several windows each of which
has buttons to be clicked. The application is written in such a way that
only one button can be selected at a time.

In one of the windows (and sometimes after doing reload of that window),
among the left pane radio buttons more than one button were seen to be
pressed (which is wrong).
So, if we want to click one button among that set (which are seen
to be pressed), that option won't be selected. However, if we select any
other button (which are in unselected state), then this problem goes off.
Everything returns to normalcy.

Only with the XWindow manager (mwm) this problem is seen
(that too intermittently). This problem does not occur in case of fvwm.
It would be helpful if anyone could let me know their comments on this.
Thanks,
Mala.
