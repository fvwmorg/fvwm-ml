Yes - GoodStuff is the button bar in 1.xx. Try something like the
following in your .fvwmrc file:

######################################################################
Function "InitFunction"
  Module "I" FvwmBanner
  Module "I" GoodStuff
EndFunction

*GoodStuffFore Black
*GoodStuffBack dimgrey
*GoodStuffFont fixed
*GoodStuffGeometry -1-125
*GoodStuffRows 1
*GoodStuff - clock.xpm Swallow "xclock" xclock -bg \#908090 -geometry -1500-1500 -padding 0 -bw 0
*GoodStuff Xeyes untitled.xpm Swallow "xeyes" xeyes -bg \#908090 -geometry -1500-1500
*GoodStuff - load.xpm Swallow "xload" xload -bg \#908090 -geometry -1500-1500 -fn 5x8 -label xload -update 1
*GoodStuff - mail.xpm Swallow "xbiff" xbiff -bg \#908090
*GoodStuff XTerms xterm.xpm Popup "XtermMenu"
*GoodStuff WinOps 3D_Menu15.xpm Popup "Window Ops"
*GoodStuff Paging clamp.xpm TogglePage
*GoodStuff(2x1) - whatever SwallowModule "Fvwm Pager" FvwmPager 0 1
######################################################################

You can find pointers to other 1.xx examples on some of the WWW pages
pointed to from the official FVWM homepage:

                     http://www.hpc.uh.edu/fvwm/
