I'm new to the fvwm bandwagon....running 224....here's what i'm finding.

1) mailtool starting msg iconic
   Well, i can get around the iconic problem via FVWMEvent....it
   opens....but then sinks to the bottom. Raising it doesnt help. It just
   wants to be at the bottom. Clicking the msg in the header window will
   then bring it to the top.

2) Using click-to-focus....i lose focus when mouse is over root window.
   Doesnt lose focus if i go to another window w/o crossing over root.

3) I wish there was a mouse-over-window (say, called "O"?) so i can use the
   sun's Open/Front keys on it w/o having to click the window first....ala
   olwm.

4) Xview testedit windows dont have a label in pager.

5) What does afterstep support that fvwm2 doesnt?

6) I love watching the netscape popups coming up and then getting
   auto-zapped....ala FVWMEvent. "No i dont want the plug-in....Zap".
