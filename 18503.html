>>>>> Jamie Sparks writes:

J> What's a swallowed xterm?

an xterm that has been swallowed by FvwmButtons.
like
*FvwmButtons(10x2, Swallow (UseOld, NoClose, Respawn) "Console" "X console -display $HOSTDISPLAY -geometry 70x5-1500-1500")

(Console is a local script that starts
        xterm -C if i log in on the display of my main computing host
        and plain xterm if I log in from a remote display.
