On 24 Jan 2002 11:08:20 -0800, Marc Greene wrote:
>
> Hello, I've been having a bit of difficulty with an fvwm 'keyboard
> accelerator'. My problem is that when I press <Alt-F1> to bring up the
> WindowList, it goes away as soon as I release <Alt>, and before I've had a
> chance to key in my selection. This behavior only effects the left <Alt>
> button. The right <Alt> button performs correctly, as does bringing up
> WindowList with mouse button 3. Oddly both <Alt> buttons function
> correctly when using accelerators defined for <F2>, <F3> etc. I'm
> connecting to a remote Xserver (XwinPro5.3) running on windows98 if that's
> relevant.
>
> # Keyboard accelerators
> Key F3 A M Popup "Utilities"
> Key F2 A M Menu "Window Ops"
> Key F1 A M WindowList

To disable the ALT-auto-selects-on-release behavior, use:

  Key F1 A M WindowList SelectOnRelease
