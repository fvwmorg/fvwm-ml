Well, if it were at the application level then every single application would
have to write code to do the same stuff. The philosophy of Unix was:
"small is beautiful" so in view of that it would be nice to have a feature in
the window manager that filters the output of whatever application one wishes.
Even if it is not a window manager feature there should be a program that lets
you do it with any window. And the window manager is what deals with windows.
Analogously, just cause "less" is there that doesn't mean that it is reasonable
to pipe the output of every single program through "less".

Neil
