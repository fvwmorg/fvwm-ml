Hello,

I am running Fvwm 2.3.24 and iconifying crashes the X Window System. To
reproduce this bug:

Step 1: Grab http://www.cs.mun.ca/~nzanella/computing/fvwm2/system.fvwm2rc
Step 2: Run fvwm2 with this file
Step 3: type "xterm -n abc" in an xterm
Step 4: from the left hand side of the titlebar select the menu by
        clicking on the leftmost button and select iconify...
        the X Window System will crash.

Another thing I have noticed is if I iconify the window named abc
by clicking on the rightmost button no icon appears but I don't
have many "Style xyz NoIcon" in my config file.
