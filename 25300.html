On Wed, Dec 29, 1999 at 11:54:53AM -0800, Elliot Sowadsky wrote:
>
> I'm new to the fvwm bandwagon....running 224....here's what i'm finding.
>
> 1) mailtool starting msg iconic
> Well, i can get around the iconic problem via FVWMEvent....it
> opens....but then sinks to the bottom. Raising it doesnt help. It just
> wants to be at the bottom. Clicking the msg in the header window will
> then bring it to the top.

Perhaps it asks fvwm to be at the bottom when it starts up. If so
you should be able to raise it if you wait a second or two.

> 2) Using click-to-focus....i lose focus when mouse is over root window.
> Doesnt lose focus if i go to another window w/o crossing over root.

Does this happen with any application or just with certain windows?
My guess is that you have either some windows with MouseFocus or
the application reverts the focus back to the root window itself.

> 4) Xview testedit windows dont have a label in pager.

Does the window have a title? Does it have a title when it's
iconified? Probably the application doesn't set an icon title
and your pager is configured to show the icon label. Try

  *FvwmPagerWindowLabelFormat %r

or

  *FvwmPagerWindowLabelFormat %t

> 5) What does afterstep support that fvwm2 doesnt?

Better ask the Afterstep folks :-) If there's something worthwhile,
just give us a hint so we can copy it.

> 6) I love watching the netscape popups coming up and then getting
> auto-zapped....ala FVWMEvent. "No i dont want the plug-in....Zap".
