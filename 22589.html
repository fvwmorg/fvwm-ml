To change desktops using keys, try something like the following in your
~/.fvwmrc or system.fvwmrc :


###################### Key and mouse bindings ##########################
# Format:
# Key keyname context modifiers function
# Mouse button context modifiers function
# ------- ------- --------- --------
# | |
# | |- A = Any
# | |- C = Control
# | |- M = Meta (Esc or Alt)
# | |- N = None
# | |- S = Shift
# |
# |- 0-9 = Title bar buttons [1 3 5 7 9 0 8 6 4 2]
# |- A = Any except buttons
# |- F = Frame corners
# |- I = Icon
# |- R = Root window
# |- S = Sides, top, bottom
# |- T = Title bar
# |- W = Application window

# Alt-tab for next window [like CDE]
Key Tab A M Next [*] Focus
Key Tab A SM Prev [*] Focus

# Alt-F4 to close current window [like CDE]
Key F4 A M Close

# Alt-1 through Alt-4 for changing desktops
Key 1 A M Desk 0 0
Key 2 A M Desk 0 1
Key 3 A M Desk 0 2
Key 4 A M Desk 0 3
