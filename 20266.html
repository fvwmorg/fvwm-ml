OK, I've RT'd the FM :-), checked the archives, sacrificed a black
chicken at midnight on the night of a full moon...

And I still can't get ClickToFocus to work with the num lock key or the
scroll lock key on.

The archives mention a problem when a mouse button is bound with no
modifiers (N) but all of my mouse binding accept any modifier (A). So I
don't think that is the problem.

The windows will raise, lower, resize, move, etc ust like they're
supposed to. Menus pop-up as expected. The only thing missing is the
focus!

If I change the setting to SloppyFocus or MouseFocus the focus changes
as expected.

I am using

        Linux 2.0.27 (i386)
        XFree86-3.2 (XF86_SVGA)
        fvwm 2.0.45 (From the RedHat 4.1 distribution)
