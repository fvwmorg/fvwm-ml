I'm running fvwm 2.039, and there's a problem with FVWMButtons that's been
there since 1.24. Whenever I want to specify a "text-only" button bar,

string of text. Instead, I have to force a geometry size on it, which is a

know if FvwmButtons, or any toolbar out there, that can have a pixmap + text
