A simple suggestion for those, like me, who compile FVWM
for personal use only. Note that I've no idea about
syntax etc, I'm making a general suggestion.

Could we have a line in the Fvwm.tmpl file like:

#define USER_DIR = /home/dcjk/

Allowing something like
#ifdef USER_DIR
#define FvwmBinDir $USER_DIR/bin
#define FvwmDir $USER_DIR/bin/modules
etc
else
normal default setup for system wide installation.

Then I wouldn't have to define the same path over and over...
and I always spell one of them wrongly!
