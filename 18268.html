Sure. here is the relevant section from my .fvwm2rc file. It is set up
so that the left mouse button opens an 80x25 xterm, the middle button
opens an 160x25
 window, and the right button opens an 80x60 xterm.


(^8= Eric T. Sylvain =8^) wrote:*FvwmButtons(1x1, Container(Rows 2,
Columns 2))
*FvwmButtons(Title Peb, \
    Action(Mouse 1) 'Exec "Aixterm" $SHLPEBB aixterm -g 80x25 -d
$DISPLAY -T pebbles', \
    Action(Mouse 2) 'Exec "Aixterm" $SHLPEBB aixterm -g 160x25 -d
$DISPLAY -T pebblesw', \
    Action(Mouse 3) 'Exec "Aixterm" $SHLPEBB aixterm -g 80x60 -d
$DISPLAY -T pebblest')
*FvwmButtons(Title Rob, \
    Action(Mouse 1) 'Exec "Aixterm" $SHLROBN aixterm -g 80x25 -d
$DISPLAY -T robin', \
    Action(Mouse 2) 'Exec "Aixterm" $SHLROBN aixterm -g 160x25 -d
$DISPLAY -T robinw', \
    Action(Mouse 3) 'Exec "Aixterm" $SHLROBN aixterm -g 80x60 -d
$DISPLAY -T robint')
*FvwmButtons(Title Bet, \
    Action(Mouse 1) 'Exec "Aixterm" $SHLBETT aixterm -g 80x25 -d
$DISPLAY -T betty', \
    Action(Mouse 2) 'Exec "Aixterm" $SHLBETT aixterm -g 160x25 -d
$DISPLAY -T bettyw', \
    Action(Mouse 3) 'Exec "Aixterm" $SHLBETT aixterm -g 80x60 -d
$DISPLAY -T bettyt')
*FvwmButtons(Title Bam, \
    Action(Mouse 1) 'Exec "Aixterm" $SHLBAMB aixterm -g 80x25 -d
$DISPLAY -T bambam', \
    Action(Mouse 2) 'Exec "Aixterm" $SHLBAMB aixterm -g 160x25 -d
$DISPLAY -T bambamw', \
    Action(Mouse 3) 'Exec "Aixterm" $SHLBAMB aixterm -g 80x60 -d
$DISPLAY -T bambamt')
*FvwmButtons(End)
