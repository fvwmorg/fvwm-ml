    Alan> tim werner wrote:
>>
>> I have escaped from X to a virtual console under Linux, but never made
>> it back. Sorry for the off-topic, but what keystrokes does one use to
>> switch back and forth?

    Alan> The defaults for XFree86 Are:

    Alan> Ctrl-Alt +F?

    Alan> Where '?' is the number of the consoel you want to use. F7 is used for
    Alan> X. I believe if you are using xdm, there is a virtual console attached
    Alan> to 6, as well.

That only gets you from X to some other console. To get back to X use
Alt F? where '?' is the first virtual console that doesn't
have a getty running on it. For me, that's 5---getty is running on
the first 4.

Of course, there are probably other strange setups where this is not
true. I'd just try pressing Alt F? for all values of ? until X comes
back.
