>
> Is there any way to get rid of the geometry column in the popup window

You can add the "NoGeometry" argument to the windowlist command.

If you are having trouble finding the WindowList command in your
.fvwm2rc, tell use which release you are using and the keybinding or
mouse button you are using to invoke the windowlist.
