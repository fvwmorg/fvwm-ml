Hello,

I too use AnotherLevel and it is an extrememly
efficient M4 pre-processor. Which style are you
using?? I like to use the MWM (Lestiff) style the
most.

To answer your question, you have to obtain the
original AnotherLevel rpm from your RedHat
distribution and install it on your machine.

If you already have customised "fvwm2.*****.rc" files
then make sure that these appear in your home
directory.

Then to use it, you have to use:

startx

making sure that startx loads the following:

RunWM.MWM

(edit startx as appropriate).

Furthermore, you should change the content of the
following file:

"/etc/sysconfig/desktop"

and delete whatever it says in there and put

MWM
