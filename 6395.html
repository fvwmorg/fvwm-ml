On 03 Oct 2002 17:37:01 +0700, Andrey R. Urazov wrote:
>
> I'm using fvwm 2.5.3 and want to get alt-tab behaviour like the one in
> another GUI :). I.e. if I'm using two applications, I would expect that
> alt-tab switches between them. This is what I cannot get now.
>
> Windows in the WindowList are shifted cyclically and that means that
> when I press alt-tab, the current window goes to the end of the list
> (assume CurrentAtEnd is on) and then if I want to switch to it again,
> it's not sufficient to just press alt-tab, I have to press it many times
> so that to reach the next to last entry at which the previously used
> application resides.

This is a bug. I believe it was introduced in 2.5.3 after FP* style
additions. The source problem is that FlipFocus works just like Focus.
I think that when this is fixed, WindowList will work as usual.
