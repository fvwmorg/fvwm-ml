>
> Does setting some windows to "StaysOnBottom" while just using FvwmAuto
> module not work? ;)

1) Marking a window StaysOnBottom just so it will not be auto raised is
bad, it should be raised, just not automatically.

2) FvwmAuto in all versions except for the current cvs works on focused
windows only.

This means that FvwmAuto does not work on NeverFocus or ClickToFocus
windows (most of persistent modules like FvwmButtons, FvwmPager,
FvwmTaskBar, FvwmIconMan and so on are best when marked NeverFocus).

In the latest snapshot, FvwmAuto may work on a enter-leave basis, not
on a new focus basis, this solves a problem of autoraising NeverFocus
or ClickToFocus windows.
