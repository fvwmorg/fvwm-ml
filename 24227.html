Quoting Boris Alemi (alemi_at_umbc.edu):
> I am sorry if this is a repeat but I just subscribed to this list.
>
> I have installed the new RedHat 6.0 and being an avid supporter of fvwm I
> would like to use it under the new operating system. I previously used it
> under RH5.2 without any problems. However under 6.0 I can only start fvwm
> using startx and cannot start fvwm from xdm screen. Everytime I start from
> xdm it seems to want to force me to use a gnome compliant window manager
> and ...
>
> Any suggestions.

Okay, perhaps this is too obvious, but I believe that xdm reads .xsession
whereas startx reads .xinitrc. If you don't have a .xsession, try copying
your .xinitrc and see if that helps. If you do, sorry for wasting your time.
(I haven't used RedHat 6.0 myself.)
