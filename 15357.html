On Wed, 31 Jan 1996, Gregory Margo wrote:
> I found another problem in this process. Why does this work:
> Mouse 1 S N Move-or-Raise
                        ^^^^^ This means No modifier keys

> Mouse 1 S C Move-or-Raise
                        ^^^^^ This means Control key modifier

> Mouse 1 S M Move-or-Raise
                        ^^^^^ This means Meta key modifier

>
> But this does not work:
> Mouse 1 S NCM Move-or-Raise
                         ^^^^^ None _and_ Control _and_ Meta modifiers

                        All 3 of these modifiers have to be pressed at
                        the same time. Kind of difficult when one of the
                        modifiers is _None_ ;-) . The modifiers are
                        all and'ed together, not or'ed.

It's great that fvwm has the flexibility to emulate the OpenWin feature you
wanted.
