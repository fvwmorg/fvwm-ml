Sounds like you have plain 'fvwm', either versions 1 or 2.

The version with the task bar is fvwm95 which provides a windows 95 type
interface with a taskbar and start menu etc.

It may well be on the same CD (I seem to remember my friends Slackware
compilation had fvwm1, fvwm2, AND fvwm95 on). Personally I found fvwm95
a little unstable though, and if you want the latest fvwm functionality
its best to stick with version 2.
