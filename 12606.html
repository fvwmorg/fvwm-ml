Ruslan> Question: how to bind action to key release?

If you're really, really desperate, and no other solution works, you
can try the following nasty hack:

In .Xmodmap define your key as a modifier, say Mod5. Configure fvwm
in such a way that every time your key is pressed you simulate
pressing another key, say F11, using the FakePress command. Then,
define two bindings for F11: one with Mod5 active and the other
without. If Mod5 is active, then use the Schedule command to simulate
pressing F11 again in, say, 50ms. If not, then execute the command
you want to execute on the key release. Perhaps F11 and Mod5 can be
the same key, but I haven't tested this.
