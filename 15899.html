The file must be named .fvwm2rc and it must be an fvwm2.0.x specific
file. You cannot use your old 1.x files with fvwm2. Although you can
use the conversion script that comes distribute with fvwm2 to translate
your old file to the new syntax/

Personally, I would rather code a new file. There are so many new
features and options that fvwm 1.x based setup seems rather obsolete to
me.

Especially now that I have started to play with Jarl's new FvwmButtons.
It's really impressive.

Too bad that fvwm warp's your pointer to the top of menus. Chuck? :)
