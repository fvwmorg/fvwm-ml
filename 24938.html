io <iroiimagay_at_hotmail.com> writes:
> i am a unic user ( not root)
> and i want to set a wallpaper by using fvwm
> but when i try to insert the "xv............" code to .fvwmrc
> and reload it ....it become "hang"....
> and where can i find more ( or all) "command" of fvwm?

Please only send questions to fvwm_at_fvwm.org.

Make sure you add "-q" to the "xv" command line, or fvwm will
hang as you describe.

You should be able to find all the commands for fvwm in the fvwm
man page.

If you are running fvwm version 1, in an xterm, type in:

man fvwm

if you are running the only supported version, version 2, type in:

man fvwm2.

Or you can read our version 2 man pages at

http://www.fvwm.org/man-pages.html
