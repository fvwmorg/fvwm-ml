On Wed, Jun 20, 2001 at 09:09:25AM -0700, Erik Steffl wrote:
> using fvwm 2.2.5
>
> is there an action to StaysOnTop style?
>
> there is Stycky style and Stick actio0n/command, there is StaysOnTop
> style but I can't find any command/action to make particular (focused)
> window to stay on top. Is it possible?

Using 2.2.x you have to change the style of the window and
recapture it with RecaptureWindow. In 2.3.x you can simply use
the Layer command. The normal window layer is 4 and the
StaysOnTop layer is 6:

  Layer 0 4

or

  Layer 0 6
