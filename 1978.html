"Kahsay, Fessehazion H (Festus)" <Festus.Kahsay_at_NAV-INTERNATIONAL.com> writes:
> I am having a problem trying to have the GoodStuff work on HP-UX 10.20. It
> works good with IBM AIX. I am using fvwmrc which spawns the GoodStuff. I
> couldn't find the source code GoodStuff.c ( if there is one at all) to
> compile it on the HP-UX. Need some help.

Goodstuff was part of fvwm1. We no longer support fvwm1,
believe me, its for your own good.

However, if you go to www.fvwm.org, and then go to downloads, you
will find fvwm1.24r which does contain Goodstuff.c.

Please consider upgrading to fvwm2. Fvwm2 contains a script that
will do most of the .fvwmrc conversion you might need.
In fvwm2, Goodstuff was replaced by FvwmButtons.
