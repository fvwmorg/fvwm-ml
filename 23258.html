Hello!

As I read from doc modifiers are: control, shift, lock, meta, mod1-mod5
or all (CSML12345A).
So if we want to obtain the same behaviour with or without Num etc.
we must define additional bindings i.e. :

# without additional modifiers
Key Escape A S PopUp Window-Ops

# with NumLock
Key Escape A S2 PopUp Window-Ops

#with CapsLock
Key Escape A SL PopUp Window-Ops

.. the same with mouse bindings.
