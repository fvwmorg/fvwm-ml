> Tim Phipps <timbo_at_computer.hpl.hp.com> writes:

> 3) Are you working on making the placement direction controllable?
> The above line is only there so I can have icons going upwards. If
> you are could you make it possible to bottom align the icons if
> they're going upwards?

I started work on it last night. Stay tuned.

The syntax is:

Style "*" IconFill arg1 arg2

The first arg can be:

b | bot | bottom | t | top

The second arg can be:

l | lft | left | r | rgt | right
