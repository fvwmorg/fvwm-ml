> PS: another feature which I like from the mac's was that if you threw
> the pointer to the bottom left corner of the screen, that the
> screensaver would startup right away - combine that with locking, it
> would be a nice way to leave your workstation - but the pointer has to
> be at +0-0 for the feature to start up. any takers?

xautolock does what you need. Find it at ftp.x.org in
/contrib/applications/xautolock.pl10.tz. You can specify which locker to
use (xlockmore in the same directory is a good choice) and when to lock
the screen: after inactivity or when the mouse pointer is in a certain
corner. Which corner(s) you want to use is also a preference.
I don't think this should be a feature of the window manager...
