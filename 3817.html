On Fri, Nov 30, 2001 at 09:05:45AM +0100, johannes karlsmyr wrote:
>
> Q1: I'm running fvwm2 with Xinerama on 5 screens. I have a small FvwmButtons
> Panel in the top left corner with my most often used applications, how do i
> make that FvwmButtons to appear in the top left corner on all screens?

You have to start a separate instance for each screen.

> Q2: And i am also wondering how i specify that for example a program named
> nssv only starts at screen5 +0+0?

To force the application on a particular screen you can use

  Style nssv StartsOnScreen 4

(replace "nssv" by the window name, class or resource).

> Q3: If i want to maximize a window on screen1 to all screens. Maximize, only
> maximizes to the current screen.

You can tell fvwm on which screen to maximize the window (or in
this case the global screen):

  Maximize screen g
