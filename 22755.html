On Wed, 2 Sep 1998, Dieter Meinert wrote:


> Well, and fvwm1 had the nice feature of a command to toggle window
> scrolling. The man says this is covered by the EdgeScroll
> parameters in the config, but I just want to be able to stop
> scrolling for a little time without restarting the WM. I don't
> see a way to do this with the EdgeScroll config.
>
> Hints welcome.
>
I do this
Key 1 A M EdgeScroll 0 0
Key 2 A M EdgeScroll 100 100

so alt-1 disables scrolling and
   alt-2 re-enables it.
