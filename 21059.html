Have you seen the fOX project? :-) fOX.unixville.com is the home... check
it out... it's still growing :-) The fOX project is based around the
Xclass `95 C++ X11 Toolkit (which lets you create widgets that look
identical to 95 ones). I'd like to see a new window manager come out for
the project, and I don't necessarily want it to look like 95 (actually, I
DON'T want it to look like 95 :) ... I also think it might be a good idea
to start a new window manager from scratch with a new perspective on
things.
