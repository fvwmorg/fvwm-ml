lee <learnman_at_21cn.com> writes:
> This is i do:
> 1. I have already install FVWM into my machine, and i commonly enter
> FVWM from command console (runlevel 3) with the command'learnman>$ startx'
> 2. cp /opt/wm/fvwm/fvwm-2.5.8/modules/FvwmSaveDesk/example/xinitrc
> ~~/.xinitrc
> 3. The .xinitrc file have these sentences:
> ...
> fvwm2 -cmd "FvwmM4 -m4-prefix -m4opt -I$HOME $HOME/.fvwm/.fvwm2rc"

I don't think FvwmSaveDesk is in good shape and I don't recommend
using it. Also using M4 isn't simple.

Try starting fvwm from your .xinitrc with:

exec fvwm

Also remove any .fvwm2rc file that you have and press F1 after you
start fvwm. Choose one of the options from the menu that creates
a config file.
