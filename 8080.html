I recycled some code from the XMMS Titlefader[1] plugin Code. Now I have the thing
I always wanted[1]. It isn't that stable but useable at least. Give it a try but
don't expect to much and tell me about it. I simple used the routine provided by
XMMS Titlefadder and put a little main.cc around it. It doesn't do any error
checking and all and you have to start it with at least one argument (the other
arguments get ignored).

        ./fader "Way to easy"

Let me know what you think about it. If you find a better alternative, please
let me know. You can *download* it from my place[2].

        Thomas

        http://sourceforge.net/projects/xmms-titlefader/
