On Sat, 13 Dec 1997, Craig Hutchison wrote:

> Hi,
>
> Does FVWM or X11R6 have the ability to display a backgroup
> wallpaper. If so, how is it done? If not I would be interested in
> doing it. How difficult do you think it will be? Where in the source
> code would I start? I do ALOT of coding in UNIX using C, but I don't
> have much exposure to the X Windows coding environment.

I don't know what goes on with HP-VUE, but _it_ puts some kind of
wallpaper up which apparently distinct from the X server's background. At
least it appears so to me, the simple user. If I put up an xsetroot or
xpmroot or xview -root <stuff> background and invoke Vue, it covers the
background with whatever it wants (unless it is told to use no backdrop).
When the user exits Vue, the old background is exposed again. How it does
this, I don't know. But I do know that with a Vue-specified background,
invoking xsetroot and cousins will not overwrite it. If you want to do
the xearth thing, for instance, you must select a blank Vue background or
you simply won't ever see xearth. I found these things empirically and
never have understood what Vue does. Is it possible that Mr. Hutchison is
speaking of this kind of feature?

I also believe that FvwmBacker serves this purpose, and has not yet been
mentioned in the several replies I have seen.
