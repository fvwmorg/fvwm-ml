When I switched from slackware to debian a while ago, I had to go in
and add full paths to my .fvwm2rc. It seems that the PATH isn't set to
include the X binaries when fvwm2 initalizes. Try adding the path
(/usr/bin/X11/xpmroot).

Also, did you upgrade to debian from 1.3? On the machine that I
upgraded, I had an old-libc-compiled version of xpmroot in
/usr/local/bin.

- Jim
