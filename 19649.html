>>>>> Grochtmann Raphael M writes:

  RG> I recently changed from t(v)twm to fvwm2 and now I'm wondering
  RG> how it can be that such an extremely usefull feature like twm's
  RG> relative resize has been forgotten :-?

I noticed that it isn't in fvwm, but I don't miss it. Maybe this is
because, in W context, I have bound a mouse button to do resize. In
particular:

,-----
| Mouse 3 WFSIT 4 Resize-or-Maximize
`-----

Where Resize-or-Maximize is the following function:

,-----
| AddToFunc Resize-or-Maximize "M" Resize
| + "D" Maximize 0 100
`-----

Btw, X modifier 4 is bound to my Alt key -- I've got a keyboard with
both Meta and Alt keys. I have reserved Alt for the window manager
and never use it for any application.
