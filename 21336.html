This patch would make possible to configure fvwm2 menus in
with nextish look. Read the man page (command MenuStyle for
a not too much detailed description of the new options).

  If you are not too fond of reading man pages, you can try
the following MenuStyle entry in your .fvwm2rc:

MenuStyle Black Gray41 Blue -*-helvetica-bold-r-normal--12-*-120-120-*-*-iso8859-1 \
           next NWSE Gray31 LightGray 20 WEW MidnightBlue White 20

that make menus resemble that of AfterStep.

I'm currently working in improve the efficiency (now it uses different functions
and command style for Menus and for Title bars) so you can also use Pixmaps for
Menus background (and improve a little the so nice TitleBar style).
