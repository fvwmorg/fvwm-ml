On Wed, Oct 02, 2002 at 06:54:03AM -0800, Bob Crochelt wrote:
> I asked this once before, but couldn't figure it out. I'm running fvwm 2.5.2
> on Red Hat 7.3. Is there support for KDE 3? If not, how to enable it? When
> I use the script file it seems to indicate that KDE is not installed on my
> computer, which it is.
>
I dont use redhat, but for me It didnt work by default because my
sytem uses KDEDIRS and not a KDEDIR envirroment variable, but that was
easily fixed.
      `echo $KDEDIR` if it doesnt output anything then..
try `export KDEDIR=/usr/kde/3`
then: `fvwm-menu-desktop --lang=en --desktop kde-sys --merge-user-menu`
