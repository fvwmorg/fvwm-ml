I would still suggest that your keybindings stop working when one of the
modifiers returned by `xmodmap` on your system are pressed.

> Mouse 1 R N Menu "Utilities" Nop
> Mouse 2 R N Menu "Window-Popup" Nop
> Mouse 3 R N Module FvwmWinList transient

So, if some unknown modifier is pressed, "N" does not work. Use "A" - any.

  Mouse 1 R A Menu "Utilities" Nop
  Mouse 2 R A Menu "Window-Popup" Nop
  Mouse 3 R A Module FvwmWinList transient
