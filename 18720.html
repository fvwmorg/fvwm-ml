Hello,

I hope this isn't a FAQ. I have a FvwmButtons button which invokes the
warp-or-run function on xemacs. Here's the function:

AddToFunc warp-or-run "I" Next [$0 CirculateHit] Iconify -1
+ "I" Next [$0 CirculateHit] Focus
+ "I" None [$0] $1
   
If the xemacs has multiple frames (windows) on different desktops the
original xemacs frame is raised and the desktop is changed to the
desktop holding the original frame.

I would much prefer to raise the xemacs frame on the current desk.
Does anyone know how to achieve this?
