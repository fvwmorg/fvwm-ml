> with little boxes I can check labeled "fvwm2", "twm", "olwm",
> whatever, and then when I check one off, I want to restart using the
> appropriate window manager.

Don't know about your FvwmForm problem, but you could try using the
xsession program instead (available from your local X mirror). You run
it as the last client in .xinitrc or .xsession, or whatever X startup
file you use, and it starts a window manager and manages your session:
e.g.
  xsession -defaultWM fvwm2
is the last line in my .xsession.

It then provides a pulldown menu of window managers you can switch between.

Another big advantage of using xsession is not using the WM itself to
manage the session. This means when you make an erroneous change to
your .fvwm2rc and fvwm2 won't start, you still get X running and can
select another WM while you debug the problem.
