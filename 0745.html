I use control and arrow keys.

# press arrow + control anywhere, and scroll by 1 page
Key Left A C Scroll -100 +0
Key Right A C Scroll +100 +0
Key Up A C Scroll +0 -100
Key Down A C Scroll +0 +100

Here is another..

# vi movement between pages
#
# press shift key + meta (alt) anywhere, and scroll by 1 page
#Key h A SM Scroll -100 0
#Key l A SM Scroll +100 +0
#Key k A SM Scroll +0 -100
#Key j A SM Scroll +0 +100
