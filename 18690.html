>>>>> Jerome Banks writes:

  Jerome> How can an average X program figure out which desktop it has
  Jerome> been brought up on??? And how can you force a window to go
  Jerome> to a particular desktop ( or at least come up on a
  Jerome> particular desktop ) ??? Is this currently possible???

WIBNI all the Style command would be undoable? Then one could send
Fvwm the command

    Style "*" StartsOnDesk foo

Then one could start some application. Then one could send Fvwm the
command

    Style "*" StartsOnCurrentDesk

which would undo the first Style command. (Keeping all the specific
Style "foo" commands, of course!)

Wouldn't you all *love* this? Or isn't it worth the effort?
