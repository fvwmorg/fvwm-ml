>
> Inserting this in fvwm2 likely requires source code hacks, since I

I guess you're right about that.
Fvwm is very configurable, but not completely programmable :)

> not even an option to turn off keyboard control of it

Sure there is !
There are 2 possibilities:

1) Use something like
Key A C Left CursorMove Left (or something alike, I don't remember by heart)

2) You could use xmodmap (if available on your system) to completely map
the mouse to the keyboard.
Then, you won't even need the mouse _buttons_ because they're on the keypad
already.

> Is there a clever way of achieving
> this in fvwm2 as it stands? Has anyone already done this?

As far as I know, nope.

> [*: people who are heads-down typists, like me,

Just try practicing a lot - it really helps !
I usually am a head-down typist too, but I try as much as I can to
force myself looking to the screen instead of the keyboard.
It does take some patience, if you're (just like I) used to looking to
the keyboard when typing.

Hope this helps,
