I actually need *less* functionality from FvwmPager. I prefer a simple array
of plain buttons for a pager-- no outlines or icons denoting windows. The
pager that comes with Gnome is essentially what I want, but I don't want to
run E or AfterStep just to use it. I've fiddled with different values of
colors for Fore/Back/Hifore/Hiback/etc., but it just doesn't quite accomplish
what I'm looking for. I want all the panes to have one default color, and the
current active pane to have a different color. No other adornments. Well,
slight beveling would be nice.
