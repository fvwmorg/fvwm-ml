> Recently I moved my UNIX Xterm environment to the
> Xterm environment using Window NT Reflection. I used

> .fvwm2rc file, but the cursor changed from a simple
> "I" to
> half of "X" about font size 18 in bold. I could not
> find
> anything related to the cursor except the following
> commented
> out lines in my .fvwm2rc file.
>
> # Cursors
> # cursor # cursor type
> #!WARNING: Keyword "Cursor" not handled yet
> #Cursor 6 132
>
> Any help will be much appreciated,

Once the mouse pointer gets inside an application window
fvwm is going to leave it alone.

I'm not familiar with this "Reflection" thing, but I think
you are seeing an interaction between the xterm which should be
asking for an I-Beam cursor, and the X server.

You should always mention the version of fvwm you are using.
Recents betas of fvwm have had some nice work done on cursor
configuration.

Hope this helps.
