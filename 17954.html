On Wed, 25 Sep 1996, Phillip Steege wrote:

|> I have been unable to get cut & paste (mouse operations) to work. Is it
|> because I only have a two button mouse?
|> Any help would be appreciated.

I don't know if you're using XFree86 or another X server ... but in XFree
you have to have `Emulate3Buttons' set. Then, hitting both left&right
mouse buttons simultaneously will be equal to the middle mouse button
(which acts as paste in many apps) on a 3 button mouse.

Hope this helps
