Using Fvwm you can bind an arbitrary key to perform some binding. You
can specify this for the title bar, the frame, the buttons, the root
window, and the window contents itself.

For example, I have bound the "Front" key to do RaiseLower everywhere:

,-----
| # name context modifier action
| Key F15 A A RaiseLower
`-----
