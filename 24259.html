  dl> I've noticed that when I run X with fvwm, my screen goes blank
  dl> after a period of inactivity. Since I'm using fvwm with a touch
  dl> screen and I don't want to accidentally touch a button when I'm
  dl> waking up the video, I'd like to disable the screen saver. I'm
  dl> not sure if it's fvwm that is causing this blanking to happen or
  dl> not. If anybody knows how to disable the screen blanking, please

screen.

>From the sound of it, you've just got the default X server screen
blanking enabled.

See the xset command for how to disable this (man xset).
