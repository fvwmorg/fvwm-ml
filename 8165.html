dermotwhelan_at_e-cog.biz wrote:

> I was wondering if you could help to override a webpage setting the
> window placement
> to 0, 0 (ie., <script> move.window(0,0) ) as it hides the title bar and
> prevents you from controlling the window.

You could create a hotkey to move the window when clicking it somewhere
else but the title. Something like

Mouse 1 WTS 4 move

=> when clicking the left window key you can move the window by clicking
it everywhere.

(I am writing the command as I remember it without checking if it is
correct. But it should give you an idea.)
