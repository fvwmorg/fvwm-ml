You can already do the equivalent by specifying an xpm file that is
completely transparent. This works quite nicely:

/* XPM */
static char *no_cursor_xpm[] = {
/* width height ncolors cpp [x_hot y_hot] */
"1 1 1 1 0 0",
/* colors */
". s none m none c none",
/* pixels */
"."};
