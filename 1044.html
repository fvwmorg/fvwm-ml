So I solved part of my problem with the following:

Module FvwmEvent
*FvwmEventPassID
*FvwmEvent add_window AddWindowFunc

AddToFunc AddWindowFunc
+ "I" WindowID $0 ("Netscape: Error") Close

...

However this does not get me past the License agreement.
If I had included the following:

+ "I" WindowID $0 ("Netscape: License Agreement") Close

The License Agreement comes up each time I delete everything
under ~/.netscape and I do this on a regular basis since this
seems to be the only way I can keep netscape working the right
way each time (I really don't like its caching policies).

So, what I would like to be able to do is have fvwm2 send the
licence agreement an enter keystroke automatically. Is this
possible? This would also be handy for handling other tedious
forms of similar nature.
