A few things aren't going right for me.

1. If I have the following in my .fvwm2rc file it doesn't work:

Style Netscape* TitleIcon mini-nscape.xpm, StartsOnDesk 1, SkipMapping

However:

Style Netscape* StartsOnDesk 1, SkipMapping, TitleIcon mini-nscape.xpm

is fine. Does a TitleIcon command need to be at the end of a line?

2. I have

        Style "*" RandomPlacement

However, when my netscape windows come up they want to 'tile'. Each new
window is a bit lower (vertically) than the next. I assume that this is
to show the different titlebars. I have the geom set to +0+0 for the
netscape windows and really do want it that way. Can I force this?
