Hi,
I've just started using fvwm2 and so far, it behaves pretty much the
same as fvwm 1.x, but one thing bugs me - I can't get the
icons to be placed on the bottom, left to right, as I'm used to with
fvwm 1.24.

In my old fvwm file (.fvwmrc) I have this, which works:

  IconBox 5 -64 -140 -5

So I put this in .fvwm2rc:

   Style "*" DecorateTransient,IconBox 5 -64 -140 -5

But I still get the icons in the top left in one desktop and on the
top right on another. Is fvwm2 trying to avoid placing the icons on
any window? If so, can this behavior be overridden?

Thanks,
     Amir

BTW, my configuration is:
fvwm 2.0.42 with the title-bar-icon patch
xpm 3.4h
SunOS 4.1.4
MIT X11R5
