Hi,

This is certainly a strange problem :-) How far did
you move your computer? Since you say that the mouse
moves around, then I assume that X-window is up and
running ok, and that the problem is with FVWM2 (which
in itself is unusuall!!).

Have you tried another window manager, besides FVWM2
(say for example, twm??), to see if the mouse bindings
work ok there.

In FVWM2, does the root menu appear when you click on
the root window (i.e. the background)? If it does then
this is good, since it means that FVWM2 is recognising
your mouse buttons ok.

You might consider posting your config files to the
list (preferrably in "tar.gz" format), that way we can
test them.....

Also, it might be worth taking a look at
"~/.FVWM2-errors", which is the errors file that is
produced. Maybe that will yield an explination.
(you'll have to check the file which belonged to the
user that logged off prior to the "accident").

I hope I have helped you a little....
Let me know how you get on,

Regards,
