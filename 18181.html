> AIX 3.2 & 4.1

On AIX 4.1 you should be able to go into smit and do the following:
        1) Select "System Management"
        2) Select "Change System User Interface"
        3) Here you have 3 selections -- choose "Command line"
or use:
        /usr/dt/bin/dtconfig -d

You'll need to be logged in as root and then you'll have to reboot the
system.

I'm not aware that the default AIX 3.2 image includes an auto session
manager initialization. You may have the login manager installed and it
could have been setup via the command line on all of 3.2 machines.
You'll probably have to use the dtconfig command line option above to
disable it.
