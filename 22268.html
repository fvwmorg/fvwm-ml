Alan F. Ho writes:

> I'm pretty new to FVWM so please forgive the newbie-lian nature of my
> question. Does anyone have a nice keyboard method of selecting
> windows? Right now I have the following lines in my .fvwmrc:
>
> Key F3 A M Module "WindowList" FvwmWinList
>
> which pops up FvwmWinList, but this still requires mouse selection.

I use:

  Key o A 3 WindowList

The window in the list are numbered, and you can select any window by
typing its number (1-9, and A-Z are used if there are more than 9
windows in the list). Alternatively, you can you the arrow keys to move
up and down the list, and RETURN to select.

If using this, you may want to set

  Style "..." WindowListSkip

for classes of windows you feel confident you never want to select this
way (e.g. xclock).
